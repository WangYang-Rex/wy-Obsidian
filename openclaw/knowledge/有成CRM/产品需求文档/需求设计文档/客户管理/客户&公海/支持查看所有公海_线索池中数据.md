# 支持查看所有公海/线索池中数据

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 2024-1-15 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 1、需求背景

客户公海/线索池较多时，导出数据需要不断切换公海/线索池

添加‘全部公海/线索池’会涉及到不同公海/线索池的场景、操作、字段权限的问题，无法兼容

故单开一个页面，集成展示该企业所有客户/线索信息（未分配、已分配、已作废）
