# 客户增加最近成交时间

| **修订时间** | **版本** | **修订说明** |
|----------------|----------|----------------|
| 20220117 | v1.0 | 新建 |
|  |  |  |

## 1、需求背景

客户需要统计全国所有1级经销商（成交客户）最近2个月没下单的经销商有哪些进行汇总具体的客户名单以及最近一次下订单的时间是什么时候。

## 2、业务流程/架构

无

## 3、需求详情

### 3.1、客户模板增加【最近成交时间】系统字段

| **模块** | **说明** |
|----------|----------|
| 属性 | \+ 类型：日期时间\+ 属性：关联字段\+ 新建编辑页显示：不显示\+ 不支持删除 |
| 开放用户 | 新老用户全部客户模板增加该字段，排序在成交状态字段后面 |
| 取值逻辑 | 1、客户下关联新增合同订单数据，且审批状态为「通过」时，取「合同订单创建时间」的值同步更新为对应客户的「最近成交时间」2、当合同订单删除需重新更新「最近成交时间」，取最新一条合同订单数据的创建时间，如无则时间更新为空 |
