# 支持设置可打印数据范围

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 2025-3-13 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

| **需求** | **描述** |
|----------|----------|
| 设置可打印数据范围 | 1、增加打印设置（不另设功能权限，有打印-列表权限即展示）2、处理\+ 详情页：有对应功能权限\+满足范围内的数据详情才出现【打印】按钮\+ 列表：批量打印时，校验所选数据中，是否存在不满足打印范围的，若存在则弹窗提示如下- 点击【取消】：则返回列表- 点击【继续打印】：则将剩余的N条数据执行打印 |
