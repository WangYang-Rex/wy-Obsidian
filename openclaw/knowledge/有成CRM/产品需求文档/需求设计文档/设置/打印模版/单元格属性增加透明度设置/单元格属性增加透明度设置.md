# 单元格属性增加透明度设置

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 2023-5-5 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 1、需求背景

打印报价单等，需要加盖印章，目前单元格底色不支持透明度设置，导致印章被遮挡。

## 2、需求详情

| **需求** | **描述** |
|----------|----------|
| 单元格属性支持透明度设置 | 1、支持输入正整数，范围为0-1002、\{透明度\}越小，表示底色越透明，为0则表示全透明。 |
