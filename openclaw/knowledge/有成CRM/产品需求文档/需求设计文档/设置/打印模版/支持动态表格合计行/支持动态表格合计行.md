# 支持动态表格合计行

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 2024-9-5 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 1、需求背景

1、北京市电加工研究所有限公司：希望在打印模板的动态表格下面加一行合计行

2、深圳市无新啤酒有限公司：合同订单打印支持统计数量

3、安特激光：用户的动态表格中需要体现合计金额只能在另起一行添加合计金额字段，两个数据之间会有空隙，打印出来不好看

4、北京市电加工研究所有限公司：希望在打印模板的动态表格下面加一行合计行

5、广东互立农业综合开发有限公司

## 2、需求详情

| **需求** | **描述** |
|----------|----------|
| 支持动态表格合计行 | 一、动态表格设置中增加字段合计设置\+ 可设置的字段：对应动态表格数据源中，字段类型为整数、小数、金额、百分比、统计字段、计算字段（返回值类型为整数、小数、金额、百分比）的字段\+ 默认为‘不统计’二、打印效果\+ 表格中字段，不存在需要统计的，合计行不展示\+ 最终统计结果，需要结合动态表格打印数据范围（动态表格总数据10条，但是筛选打印只有2条，那么只统计这两条数据\+ 合计行的边框属性、底色、字段位置/颜色/字体/字体大小等属性，均同表格中其他字段 |
