# 记住上一次核算维度支持设置

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 20220824 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 1、需求背景

存在客户反馈，核算维度自动记住上一次后，用户经常忘记修改，导致提交的信息是错误的。

故需要支持设置，是否要记住上一次填写。

## 2、需求描述

### 2.1 设置-进销存设置

| **需求** | **描述** |
|----------|----------|
| 增加是否要记住上一次填写的辅助核算设置 | 增加字段\{新建时默认填充上一次填写的核算维度\}标记（枚举：是/否）历史/新签客户：统一刷成‘是’ |

![Picture 1](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/WgZOZA8QZj95oqLX/img/a31f031a-85bc-46fb-ab0c-cd7ff169fe50.png?Expires=1774938400&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=YxjJTLJrG6q5QW5kdH%2FL%2BgjZphg%3D "")

### 2.2 涉及模块

若\{新建时默认填充上一次填写的核算维度\}=‘是’，则保持目前逻辑；

若\{新建时默认填充上一次填写的核算维度\}=‘否’，则在新建单据时，不需要自动填充。
