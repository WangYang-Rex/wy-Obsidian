# 审批流程条件判断支持关联对象

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 2023-1-3 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 1、需求背景

青岛众瑞（SKA）客户定制需求

## 2、需求详情

| **需求** | **描述** |
|----------|----------|
| 审批流程 | 一、审批流程条件判断中，支持关联对象（单选）字段配置\+ 比较符：等于、不等于、包含、不包含、为空、不为空\+ 值：支持多选（多选之间的条件为OR的关系，即若配置 关联对象 等于/包含 对象值1,对象值2，则表明只要单据中 关联对象值为对象值1 or 对象值2 就满足条件）二、审批流程条件判断中，支持关联对象（多选）字段配置\+ 比较符：包含、不包含、为空、不为空\+ 值：支持多选（若配置 关联对象 包含 对象值1,对象值2，则表明只要单据中 关联对象值（多个）中只要有对象值1 or 对象值2 ，就满足条件） |
