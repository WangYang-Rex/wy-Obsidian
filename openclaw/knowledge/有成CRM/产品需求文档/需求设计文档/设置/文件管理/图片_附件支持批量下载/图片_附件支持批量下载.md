# 图片/附件支持批量下载

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 2023-10-27 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 1、需求背景

## 客户需求：

当前图片，支持预览时下载，但是只能逐个点开、逐个下载，附件当前不支持下载。

## 2、需求详情

| **需求** | **描述** |
|----------|----------|
| 图片/附件支持批量下载 | 一、支持对象：所有对象二、角色设置：所有对象【导出】功能权限后增加【导出图片/附件】权限三、功能说明1\. 大列表：点击【导出图片/附件】，弹出\[导出图片/附件\]页面1. 页面说明：只在钉钉平台展示2. 页面字段1. 选择字段：必填，多选，可选择主对象/子表单中图片/附件类型字段\+ 子表单名称：取不带模板的子表单名称\+ 图片/附件字段：取不带模板的字段名称2. 分组方式：必填，单选，枚举‘不分组’‘按数据结构层级’3. 操作1. 取消2. 关闭3. 确定\+ 校验- 当前对象用户\+同一对象，导出任务列表有任务状态=‘进行中’的任务时，不支持再次发起导出任务，不关闭页面，toas提示‘请不要频繁导出：当前模块您有进行中的导出任务，请在上一任务结束后再发起。’- 拟下载的文件总容量超出5g，不关闭页面，toast提示‘下载失败：下载的文件超出5G,请缩小数据筛选范围或者减少导出的附件/图片字段’\+ 上述校验通过，则关闭页面，并生成文件下载任务将数据中文件打包下载（只获取oss中的文件，生成的任务归属对应的对象如合同订单），并弹窗提示（同数据导出）- 分组方式=不分组\* 文件夹名称=\{当前对象名称\}\* 文件：对应对象-单据-字段中的所有文件，没有树形结构- 分组方式=按数据结构层级\* 文件夹名称：取当前主对象名称、主对象图片附件字段、子表单名称、子表单图片附件字段\* 文件：对应对象-单据-字段中的文件，展示树形结构2\. oss文件列表：增加【下载文件】按钮1. 未选择任何文件：点击按钮，toast提示‘请选择要删除的文件’2. 已选择文件：点击按钮\* 校验\+ 当前对象用户\+同一对象，导出任务列表有任务状态=‘进行中’的任务时，不支持再次发起导出任务，toas提示‘请不要频繁导出：当前模块您有进行中的导出任务，请在上一任务结束后再发起。’\+ 拟下载的文件总容量超出5g，toast提示‘下载失败：下载的文件超出5G,请缩小数据筛选范围或者减少导出的附件/图片字段’\* 上述校验通过，生成文件下载任务将所选文件打包下载\+ 文件生成任务：归属对象‘服务器文件’\+ 生成的文件：文件名为‘有成crm文件’,文件无其他分组 |
