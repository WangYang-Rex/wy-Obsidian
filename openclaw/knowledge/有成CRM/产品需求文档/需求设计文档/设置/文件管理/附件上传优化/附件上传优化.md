# 附件上传优化

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 2023-9-7 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 1、需求背景
- 附件支持拖拽上传
- 文件上传数量放开

## 2、需求详情

| **需求** | **描述** |
|----------|----------|
| 文件支持拖拽上传 | 一、文件上传数量放开限制\+ 一级菜单名称变更：原先为‘oss文件上传大小限制设置’，改为‘文件管理’\+ 增加‘文件数量限制’：默认均为9（含未勾选情况）二、文件支持拖拽上传\+ 正常：选择文件（可多选）并拖动至对应位置，控件样式变更，且文件上传\+ 异常- 企业文件上传路径为钉盘：文件上传失败并toast提示‘不支持拖拽上传文件至企业钉盘，可在设置-钉钉设置-其他设置中切换存储路径’- 文件格式不匹配：对应文件上传失败并toast提示‘\{文件名\}上传失败，原因：文件格式不支持’- 文件数量超出限制：超出限制数文件上传失败并toast提示‘\{文件名\}上传失败，原因：超出文件上传数量限制（最多\{个数限制\}个）’- 文件大小超出限制：超出大小限制的对应文件上传失败并toast提示‘\{文件名\}上传失败，原因：超出文件大小限制（单个最大\{大小限制\}M）’ |
