# 钉钉企业支持切换文件存储路径

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 2023-5-19 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 1、需求背景

1）钉钉平台企业目前文件存储在企业应用钉盘（总容量100G，企业在钉钉上所有使用的应用都可将文件存储于此），存在企业容量不够的情况；

2）须使用钉钉自带的文件上传组件，限制较大，如不能拖拽上传、windows系统不能批量上传等；

## 2、需求详情

### 2.1 钉钉平台企业支持切换文件存储路径

| **需求** | **描述** |
|----------|----------|
| 增加切换设置及处理 | 一、设置-钉钉配置增加存储路径切换（仅pc端）备注：新开通及历史钉钉企业还是默认为‘企业钉盘’二、文件上传当企业切换为‘有成crm服务器’时，图片/文件上传组件切换为我们自定义的（同企微、飞书），存储路径也变更为oss服务器三、附件预览（新建/编辑/详情）1）文件：文件存储在钉盘时，保持现有逻辑；当文件存储在oss可在应用内及浏览器预览；2）图片：浏览器中对图片上/下一张切换时，将存储在钉盘的图片过滤掉 |
