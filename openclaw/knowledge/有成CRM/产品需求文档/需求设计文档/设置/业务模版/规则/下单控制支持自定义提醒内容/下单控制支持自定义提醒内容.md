# 下单控制支持自定义提醒内容

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 20220822 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 1、需求背景

目前合同订单、采购单在下单时，有下单控制策略，针对弱控制和强控制弹窗提示，目前弹窗提示内容较为笼统，客户希望能自定义。



![Picture 1](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/jP2lRYj1mkda2O8g/img/63dabefb-2106-4410-8d0f-813d1c3c63c8.png?Expires=1774937701&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=MFdhj90mc1f%2B0TWATfnqZuDeWFs%3D "")

## 2、业务流程/架构

无。

## 3、功能清单

| **模块** | **二级模块** | **功能点** |
|----------|----------------|-------------|
| 设置 | 业务模板 | 合同订单/采购单业务模板：业务规则-下单控制中，支持企业自定义提醒内容 |

## 4、需求详情

### 4.1 采购单/采购单产品/合同订单/合同订单产品-业务规则-下单控制



![Picture 2](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/jP2lRYj1mkda2O8g/img/d96d717d-eb07-4bbf-aaef-81355b78c302.png?Expires=1774937701&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=28YwgPPaa7QYgxlLQh4Q%2FGJg30A%3D "")

在合同订单/采购单下单控制模块，增加以下两个字段：

| **字段名称** | **变更类型** | **字段类型** | **是否必填** | **备注** |
|----------------|----------------|----------------|----------------|----------|
| 若控制提醒 | 新增字段 | 多行文本 | 是 |  |
| 强控制提示 | 新增字段 | 多行文本 | 是 |  |

| **需求** | **描述** |
|----------|----------|
| 交互 | \{控制策略\}=‘不控制’时，上述两个新增的字段不展示\{控制策略\}=‘弱控制’时，上述字段‘弱控制提醒’展示\{控制策略\}=‘强控制’时，上述字段‘强控制提示’展示 |
| 功能上线时历史数据处理 | 针对已开通对应模块的企业（含过期的企业）：字段\{弱控制提醒\}赋值‘处于业务规则控制范畴，确认继续提交单据？’字段\{强控制提示\}赋值‘处于业务规则控制范畴，单据无法提交。’ |

### 4.2 采购单/合同订单



![Picture 3](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/jP2lRYj1mkda2O8g/img/12306ff8-68c5-4206-84e9-8732522b7b9c.png?Expires=1774937701&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=hHUQtCO6dw0ZrG5WJKs8%2Fj39WxA%3D "")

| **需求** | **描述** |
