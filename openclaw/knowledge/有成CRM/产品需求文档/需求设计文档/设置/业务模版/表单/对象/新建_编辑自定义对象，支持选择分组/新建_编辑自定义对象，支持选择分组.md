# 新建/编辑自定义对象，支持选择分组

| **修订时间** | **版本** | **修订说明** |
|----------------|----------|----------------|
| 2022-06-14 | v1.0 | 新建 |
|  |  |  |

## 1、需求背景

新建/编辑对象时，希望能快速将对象设置到某个分组之内

## 2、业务流程/架构

无

## 3、功能清单

无

## 4、需求详情

### 4.1、新建/编辑自定义对象，支持选择分组

| **模块** | **说明** |
|----------|----------|
| 新建/编辑自定义对象 | 1、新增“选择分组”2、分组选择范围：不包含首页、设置（可以参考“菜单设置”中的分组）3、选择分组后：   （1）新建自定义对象，选择分组后并保存后，对象自动归类到该分组的最下方。并弹窗提示（参考原型图，文案有修改）（2）编辑自定义对象，选择分组并保存后，对象自动归类到该分组的最下方 |
| 支持快捷新建分组 | 1、点击“新建分组”后，可以创建分组2、创建分组之后，自动选中新创建的分组 |
