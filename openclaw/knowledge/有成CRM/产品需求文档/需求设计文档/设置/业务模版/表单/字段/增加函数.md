## 2、需求详情

| **需求** | **名称** | **描述** |
|----------|----------|----------|
| 增加日期函数 | DATE\_TO\_CHINESE | **用法:**DATE\_TO\_CHINESE(日期/日期时间字段)**描述:**将指定日期转换成中国大写名称**示例:**DATE\_TO\_CHINESEE(2024/01/01) **返回:** 二零二四年一月一日 |
|  | DAY\_OF\_MONTH | **用法:** DAY\_OF\_MONTH(日期/日期时间字段)**描述:**获取该日期在当前月份中的第几天**示例:**DAY\_OF\_MONTH(2024/01/03) **返回:** 3 |
|  | DAY\_OF\_YEAR | **用法:** DAY\_OF\_YEAR(日期/日期时间字段)**描述:**获取该日期在当前年份中的第几天**示例:**DAY\_OF\_YEAR(2024/02/03) **返回:** 34 |
|  | WEEK\_OF\_YEAR | **用法:** WEEK\_OF\_YEAR(日期/日期时间字段)**描述:**获取该日期在当前年份中的第几周**示例:**WEEK\_OF\_YEAR(2024/02/05) **返回:** 6 |
|  | WEEK\_DAY | **用法:** WEEK\_DAY(日期/日期时间字段)**描述:**获取该日期是星期几**示例:**WEEK\_DAY(2024/02/05) **返回:** 1 |
|  | QUARTER | **用法:** QUARTER(日期/日期时间字段)**描述:**获取该日期在当前年份中的第几季**示例:**QUARTER(2024/02/05) **返回:** 1 |
|  | MONTH\_BEGIN | **用法:** MONTH\_BEGIN(日期/日期时间字段)**描述:**获取该日期对应月份的第一天**示例:**MONTH\_BEGIN(2024/02/05) **返回:** 2024/02/01 |
|  | MONTH\_END | **用法:** MONTH\_MONTH\_END(日期/日期时间字段)**描述:**获取该日期对应月份的最后一天**示例:**MONTH\_MONTH\_END(2024/02/05) **返回:** 2024/02/29 |
|  | YEAR\_BEGIN | **用法:** YEAR\_BEGIN(日期/日期时间字段)**描述:**获取该日期对应年份的第一天**示例:**MONTH\_BEGIN(2024/02/05) **返回:** 2024/01/01 |
|  | YEAR\_END | **用法:** YEAR\_END(日期/日期时间字段)**描述:**获取该日期对应年份的最后一天**示例:**YEAR\_END(2024/02/05) **返回:** 2024/12/31 |
| 增加文本函数 | PHONETIC | **用法:**PHONETIC(字段)**描述:**获取汉语拼音首字母**示例:**PHONETIC(杭州光云) **返回:** hzgy |
|  | UPPER | **用法:**UPPER(字段)**描述:**字符串中小写字母转换成大写**示例:**UPPER(abCD123) **返回:** ABCD123 |
|  | LOWER | **用法:**LOWER(字段)**描述:**字符串中大写字母转换成小写**示例:**LOWER(abCD123) **返回:** abcd123 |
| 增加数学函数 | INT\_DOWN | **用法:**INT\_DOWN(数字字段)**描述:** 向下舍入到最接近的整数，如果是负数，返回距离零更远的整数**示例:**INT\_DOWN(-1.9)  **返回:** -2 |
|  | INT\_UPPER | **用法:**INT\_UPPER(数字字段)**描述:** 向上舍入到最接近的整数，如果是负数，返回距离零更近的整数**示例:**INT\_DOWN(-1.9)  **返回:** -1 |
|  | TRNUC | **用法:**TRNUC(数字字段)**描述:**通过删除任何小数部分将数字截断为仅整数部分**示例:**TRNUC(1.9)  **返回:** 1 |
