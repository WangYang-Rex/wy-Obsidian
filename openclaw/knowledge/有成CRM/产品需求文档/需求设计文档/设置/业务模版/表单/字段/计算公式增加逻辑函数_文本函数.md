| **需求** | **名称名** | **描述** |
|----------|-------------|----------|
| 增加运算符 | \>（大于）\<（小于）\>=（大于等于）\<=（小于等于）=（等于）=!（不等于）~~&&（and）~~ | - |
| 增加逻辑函数 | AND | **用法:** AND（判断结果1,判断结果2,判断结果3,...)**描述:** 逻辑与，若多个条件运算结果都为真，则结果为真（结果可用于IF等函数中）**示例:** AND(2\>1,5\>3,7\>5) **返回:** 真 |
|  | OR | **用法:** OR(判断结果1,判断结果2,判断结果3,...)**描述:** 逻辑或，若其中任何一个条件运算结果为真，则结果为真（结果可用于IF等函数中）**示例:** OR(2\>1,5\<3) **返回:** 真 |
|  | NOT | **用法:** NOT(判断结果)**描述:** 若条件为真，则结果为假，反之，结果为真（结果可用于IF等函数中）**示例:** NOT(5\<3) **返回:** 真 |
|  | IS\_NULL | **用法:** IS\_NULL(表达式结果)**描述:** 判断表达式结果是否为空，如果为空，返回真，反之为假（结果可用于IF等函数中）**示例:** ISNULL(6) **返回:** 假 |
|  |  IS\_NUMBER | **用法:** IS\_NUMBER(字段)**描述:** 判断字符串是否可以转为数字，如果可以，返回真，反之为假（结果可用于IF等函数中）**示例:** IS\_NUMBER(6) **返回:** 真 |
|  | ~~CASE~~ | ~~用法:~~~~~~CASE(expression, value1, result1, value2, result2,...,else\_result)**描述:**~~~根据满足不同的条件，返回满足条件的值。其中expression的值与value(n)进行比较，value(x)与expression的值一致，则返回对应的result(n)，如果都不一致，则返回else\_result；其中，result(n)和else\_result必须为同一种类型，返回值类型也为该类型~~~~示例:~~~ CASE(3, 2, 2, 3, 33, 6) ~**返回:**~ 33~~ |
|  | IF | **用法:** IF(逻辑判断表达式, 满足时返回值, 不满足时返回值)**描述:** 判断条件是否为真，如果为真，则返回‘满足时返回值’，否则返回‘不满足时返回值’；其中，‘满足时返回值’和‘不满足时返回值’必须为同一类型，返回值类型也为该类型**示例:** IF(true, 34, 52) **返回:** 34 |
|  |  NULL\_VALU | **用法:** NULL\_VALU(表达式, 替换表达式的值)**描述:** 判断条件是否为空，如果是，则返回‘替换表达式的值’，如果否，返回空；其中，返回值类型也为第二个参数的类型**示例:** NULL\_VALUE(null, 6) **返回:** 6 |
| 文本函数增加 | STR\_STARTWITH | **用法:** STR\_STARTWITH(字段1, 字段2)**描述:** 判断字符串参数1是否以字符串参数2开始，满足时返回真，反之为假（结果可用于IF等函数中）**示例:**STR\_STARTWITH('fxiaoke', 'fx') **返回:** 真 |
|  | STR\_ENDSWITH | **用法:**STR\_ENDSWITH(字段1, 字段2)**描述:** 判断字符串参数1是否以字符串参数2结束，满足时返回真，反之为假（结果可用于IF等函数中）**示例:**STR\_ENDSWITH('fxiaoke','ke') **返回:** 真 |
|  | STR\_EQUALS | **用法:** STR\_EQUALS(字段1, 字段2)**描述:** 判断两个字符串参数是否相等（区分大小写），满足时返回真，反之为假（结果可用于IF等函数中）**示例:**STR\_EQUALS('fxiaoke', 'fxiaoke') **返回:** 真 |
|  | STR\_CONTAINS | **用法:STR\_CONTAINS**(字段1, 字段2)**描述:** 判断字符串参数1中是否包含字符串参数2，满足时返回真，反之为假（结果可用于IF等函数中）**示例:**STR\_CONTAINS('fxiaoke', 'xiao') **返回:** 真 |
