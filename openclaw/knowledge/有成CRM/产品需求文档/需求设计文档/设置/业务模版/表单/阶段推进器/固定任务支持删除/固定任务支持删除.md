# 固定任务支持删除

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 2024-8-5 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 需求详情

| **需求** | **描述** |
|----------|----------|
| 固定任务支持删除 | 一、阶段推进器-固定任务设置\+ 增加‘任务生成后是否允许删除’设置- 历史设置：未设置，=‘否’- 新增固定任务：默认为‘否’\+ 保存时校验- 若该任务\{任务生成时是否允许删除\}='是'，且该任务在本阶段中的‘阶段设置-进入下一阶段必需完成的任务’，保存失败并弹窗提示- 通过上述校验，保存成功\+ 阶段设置- 剔除\{任务生成时是否允许删除\}='是'的任务\+ 任务- 任务详情：是固定任务 且 \{任务生成时是否允许删除\}='是' 且该用户有 任务-删除的权限，则详情展示【删除】按钮，点击可删除任务信息- 任务列表：点击【批量删除】\* 可删除以下任务\+ 临时任务\+ 固定任务 且 \{任务生成时是否允许删除\}='是' \* 若有不满足上述条件的任务，则不支持删除并弹窗提示 |
