# 数据列支持统计（求和/平均值）

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 20221031 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

| **需求** | **描述** |
|----------|----------|
| **业务模板：**子表单字段，增加‘统计方式’设置 | 1、支持设置的字段须满足以下条件：1）新建编辑页可见的字段2）字段类型为整数、小数、金额、百分比2、设置项1）位置：排在默认值设置下方2）设置项说明：内容为‘只在新增编辑页面会展示数据列统计情况，且此功能所有模板共用（修改设置后，其他模板都会生效）’3）选项值：求和、平均值（支持空值）4）交互a.当子表单中，有≥1个字段\{统计方式\}不为空，则在子表单下增加‘汇总’列；b.汇总数据中展示对应字段的统计方式 |
| 新建编辑页 | 1、汇总数据中展示对应字段的统计方式求和：汇总对应列值之和平均值：汇总对应列值之和/有值明细行计数(即剔除空值行）备注：子表单依据字段权限设置为‘可见不可编辑’的，即展示为‘’的，汇总值也同样展示为‘’。2、移动端优化：子表单手动新增/扫码添加操作与数据、单据提交操作之间增加间隔。 |
