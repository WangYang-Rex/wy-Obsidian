# 库存/批次保质期库存业务模板支持设置

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 2023-11-6 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 1、需求背景

1）库存查询中需要引用产品明细相关字段；

2）库存查询中，需要能修改字段名称；

3）库存查询中，能自定义移动端列表字段；

## 2、需求详情

| **需求** | **描述** |
|----------|----------|
| 库存查询/批次保质期库存业务模板支持设置 | 1、业务模板中，只支持添加引用字段；2、历史字段，只支持修改字段名称（新建编辑页可见的字段），其余字段配置如必填等均隐藏；3、模板属性相关配置放开\+ 详情摘要字段默认：无\+ 移动端列表字段默认- 库存查询：产品明细名称、产品类型、规格型号、仓库名称、库存数量、可用库存、单位成本、成本- 批次保质期库存：产品明细名称、产品类型、仓库、库存数量、批次、剩余天数、单位成本、成本4、历史部分字段属性变更\+ 产品编码：引用 产品明细.产品编码（部分历史企业有该字段）\+ 系统编码：引用 产品明细.系统编码\+ 规格型号：引用 产品明细.规格型号\+ 产品单位：引用 产品明细.产品单位\+ 条形码：引用 产品明细.条形码\+ 产品类型：引用 产品明细.产品类型\+ 产品简介：引用 产品明细.产品简介 |
| 库存查询/批次保质期库存 | 增加引用字段值更新按钮及说明（不用加数据权限，mb端不加）功能：点击后，历史数据引用字段重新取值备注：按企业加锁，每隔8小时才能再次点击，小于该时间间隔，点击时toast提示‘近期已执行过更新，每隔8小时才能再次执行’ |
