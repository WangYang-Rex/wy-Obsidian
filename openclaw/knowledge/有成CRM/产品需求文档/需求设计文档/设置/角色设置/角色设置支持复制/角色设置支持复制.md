## 4、需求详情

### 4.1、角色设置支持复制新建角色

#### 4.1.1新建弹窗

点击「新建角色」，展示新建角色弹窗（如下图所示），支持用户以「不复制，直接新建」、「从现有角色复制」两种方式新建角色并定义角色名称和角色说明。



![Picture 1](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/XNkOM5jVZKxYdOY7/img/ed3ffdc7-67b3-4250-93f2-6a86934b342e.png?Expires=1774938210&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=eHqmsMY6TpWSNvSKvmfRGIQH%2Fyo%3D "")

| **名称** | **说明** |  |
|----------|----------|---|
| 新建方式 | 必填默认选中“不复制，直接新建”；（见右图）若用户选择“从现有角色新建”，弹窗展示“现有角色”选择器； |  |
| 现有角色 | 必填展示条件：仅当\{新建方式\}=“从现有角色复制”展示（见右图）;选项范围：当前已存在的角色，过滤"CRM管理员"；提示说明：“现有角色”右侧新增提示说明“角色CRM管理员不支持复制” |  |
| 角色名称 | 提示文案：默认提示文案“请输入角色名称”；当\{新建方式\}=“从现有角色复制”，用户选择现有角色后，现有角色名称复制到「角色名称」输入框中，输入框下方展示提示文案“角色名称重复，请为您的新角色重新命名”（见右图）；用户编辑后提示文案消失最多支持16个字符 |  |
| 角色说明 | 选填；默认提示文案：“请输入角色说明”；最多支持100个字符。 |  |

![Picture 2](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/XNkOM5jVZKxYdOY7/img/fed21b17-827b-435f-9462-2ff64f18ed2c.png?Expires=1774938210&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=wDAhhamdvWag%2FYer2lOVBnruujo%3D "")

点击「确定」，角色列表新增角色

#### 4.1.2保存校验

校验内容：

1）必填项是否为空，若为“是”，则弹出toast提示：“\{为空的必填项名称\}不能为空”，对应输入框标红；



![Picture 3](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/XNkOM5jVZKxYdOY7/img/0a49d10b-405a-4aa7-99bc-875b88898bd4.png?Expires=1774938210&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=FXOQQfHhNTt5XjoPxU6w1i0ewXs%3D "")

2）“角色名称”是否重复，若为“是”，则弹出toast提示：“角色名称已存在，请重新命名”，角色名称输入框标红；



![Picture 4](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/XNkOM5jVZKxYdOY7/img/af30f447-82ec-4175-8983-c22a53fca938.png?Expires=1774938210&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=tI5mHTFQxO1qUbEHfJxdcL%2B1xAE%3D "")

#### 4.1.3保存后跳转页面变更

当前：新建角色完成后,页面不发生跳转

**调整：「新建」保存成功过后跳转****新建角色权限详情页面，**带入数据权限、功能权限



![Picture 5](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/XNkOM5jVZKxYdOY7/img/12359529-af9f-4a74-aec3-b2471f2bd678.png?Expires=1774938210&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=J9X9SdbfiJkb%2BuvOf8PgCMMmJks%3D "")
