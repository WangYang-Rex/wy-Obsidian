# 运营后台上传客服值班表

| **修订时间** | **版本** | **修订说明** |
|----------------|----------|----------------|
| 20220317 | v1.0 | 新建 |
|  |  |  |

## 1、需求背景

每天都有值班客服，希望能通过群消息，通知到所有内部人员

## 2、业务流程/架构

暂无

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 运营后台 | 支持上传客服值班表 |  |
| 钉钉群 | 支持发送值班客服消息 |  |

## 4、需求详情

### 4.1、运营后台支持上传客服值班表
1. 新增“客服值班表”页面。记得配置sso权限，默认后端、前端、产品、客服主管、客服都有权限



![Picture 1](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/4j6OJ5jJ8jv3Jq3p/img/9d68771b-64ed-46d6-aaf5-b77bcb0b1340.png?Expires=1774938854&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=DWvSwBXDyHvvk1SnMZxx6mLkUwU%3D "")
2. 支持下载模版。模版如下图，有一行示例数据



![Picture 2](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/4j6OJ5jJ8jv3Jq3p/img/8bc9520c-7888-4083-9ec7-b80417860085.png?Expires=1774938854&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=F4AL1JuuK02TKqDpHUr9xYSECm4%3D "")
3. 支持上传值班表。上传后覆盖原有值班表
4. 支持下载值班表

### 4.2、钉钉群支持发送值班客服消息
5. 提醒时间：周一至周五每天晚上18:00、周六至周日每天早上9:00
6. 提醒方式：发送到群：有成CRM线上问题反馈群（技术）。并@所有人
7. 提醒内容：

客服值班安排如下：

值班时间：当天日期

值班人员：张三

@所有人
