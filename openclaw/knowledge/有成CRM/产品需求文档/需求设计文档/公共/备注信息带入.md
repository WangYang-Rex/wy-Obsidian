## 1、需求背景

1、针对于产品个性化定制的行业，如家具、机器人等，信息流转存在困难，需要有一个字段标记客户需求并在各个业务中流转；

2、项目信息需要从合同订单带入到采购需求池；

## 2、业务流程/架构

无。

## 3、需求详情

| **模块** | **功能点** |
|----------|-------------|
| 销售机会 | 1、销售机会产品增加【备注】字段；2、赢单生成合同订单时，将明细行【备注】字段默认带入到合同订单产品【备注】中； |
| 报价单 | 1、报价单产品增加【备注】字段；2、操作‘转化为合同订单’时，将明细行【备注】字段默认带入到合同订单产品【备注】中； |
| 合同订单 | 1、生成采购需求池时，将合同订单【关联项目】同步到需求池【关联项目】中2、生成销售出库单时，默认将明细行【备注】字段默认带入到销售出库单产品【备注】中； |
| 销售出库单 | <span style="color: #E8323C;">销售出库单产品增加【备注】字段：新开通企业已存在此字段，功能上线时，历史部分企业要补充刷上这个字段</span> |
| 采购需求池 | 1、增加【关联项目】字段 |
| 采购单 | 1、采购单产品增加【备注】字段；2、将选定的采购需求池【明细行备注】字段默认带入到采购单产品【备注】中； |
| 采购入库单 | 将选定的采购单产品【备注】字段默认带入到采购入库单产品【备注】中； |
