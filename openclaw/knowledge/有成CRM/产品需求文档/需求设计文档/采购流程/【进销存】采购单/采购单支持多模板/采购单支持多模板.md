# 采购单支持多模板

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 20220523 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 1、需求背景

针对不同的采购业务，需要填写的采购单内容不同，例如固定资产采购、普通商品采购、外贸商品采购等。

## 2、业务流程/架构

无。

## 3、功能清单

| **模块** | **二级模块** | **功能点** |
|----------|----------------|-------------|
| 设置 | 业务模板 | 采购单业务对象支持设置多模板 |
| 采购流程 | 采购单 | 1、新建/导入/快捷新建支持选择模板2、列表支持模板筛选； |
| 采购流程 | 采购需求池 | 合并采购/多供应商采购/智能采购支持选择模板 |
