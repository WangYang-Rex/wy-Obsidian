# 支持快捷新建价目表

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 20220523 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 1、需求背景

供应商下持快捷新建价目表（该用户有‘采购价目表-新建’的权限才展示快捷新建入口）。

## 2、业务流程/架构

无。

## 3、功能清单

| **模块** | **二级模块** | **功能点** |
|----------|----------------|-------------|
| 采购流程 | 供应商 | 采购价目表tab增加快捷新建入口，并将供应商信息带入到价目表 |

## 4、需求详情

### 4.1、需求说明（pc端）

入口:供应商-详情-采购价目表tab

结果：跳转到新建采购单的界面，并将对应的供应商带入到采购价目表的‘供应商’中
