# 图表支持调整列宽

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 2023-11-6 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 1、需求背景

目前明细表、统计图/统计图穿透明细列宽是固定的，信息查看不便，需要支持用户自定义调增列宽

## 2、需求详情

| **需求** | **描述** |
|----------|----------|
| 明细表/统计表/交叉表 | 1、新建编辑（不含预览）：支持手动调整列宽，调整后报表对企业用户生效；2、报表查看（不含报表编辑页）：用户可手动调整列宽，不需要记住配置 |
| 统计图穿透明细数据 | ~~默认取：对应用户\+表头字段在列表中的宽度值~~支持用户在此处手动调整，无须同步至列表、无须记住 |
| 首页穿透明细数据 | 默认取：对应用户\+表头字段在列表中的宽度值支持用户在此处手动调整，无须同步至列表、无须记住 |
