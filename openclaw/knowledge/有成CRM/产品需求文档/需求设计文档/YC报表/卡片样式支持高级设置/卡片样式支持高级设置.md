# 卡片样式支持高级设置

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 2024-7-19 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 需求详情

| **需求** | **描述** |
|----------|----------|
| 卡片样式支持高级设置 | 卡片样式=布局3时，增加高级样式设置项\+ 指标值字号：12、14、16、18、20、24、28、36、48、72\+ 指标值颜色：参考柱状图指标色卡，增加纯色\+ 指标字号：12、14、16、18、20、24、28、36、48、72\+ 对齐方式：左对齐、居中、右对齐备注：未设置时则根据系统默认设置 |
| 其他图表样式增加字段对齐方式 |  |
| 所有图表（包括明细表） | 数值型均千分位展示 |
| 报表优化 | 单张图表最小宽度，改为页面1/4（原先是1/2） |
