# PAAS高级功能能做什么

## 便捷应用搭建

痛点
- 公司业务广泛，单一应用难以满足，多应用费用高，且应用间信息不互通，使用门槛高
- 公司存在非标业务，定制开发费用高，难以承受

解决方案
- 自定义应用，快捷新增模块，满足各类业务需求
- 托拉拽添加表单控件，维护0门槛，开发0成本

#### 人员/部门

痛点
- 制单时，需要标记业务的相关人员/部门，如订单的签订人、拜访的随行人等，便于后续的信息追踪和业绩计算等，因人员/部门是在动态变化的，若用文本、单选、关联单选等字段标记，均存在维护门槛高、数据统计困难等问题

解决方案
- 通过人员/部门控件，可直接选择应用内架构（人员/部门）信息
- 可根据所选择的人员（单选/多选）自动获取对应人员所属部门
- 报表中可以人员/部门为分析对象，分析订单、回款、拜访等数据

#### 统计字段

痛点
- 手工统计难，且容易出错
- 下游单据持续在录入，例如合同订单的回款信息陆续录入多笔，统计并展示该订单的回款金额之和，需要根据回款的新建、编辑、删除等操作不断变化，若由人为操作几乎不可能

解决方案
- 通过统计字段，可实现自动统计，且支持多种统计方式：求和、求平均值、计数
- 关联查找统计可下查间接关联关系（如在合同订单中添加‘出库数量合计’，统计链路为合同订单-合同订单产品-出库单产品/出库数量）
- 被统计对象数据范围可设置，如统计回款金额的例子中，只统计审批通过的回款单金额


![Picture 8](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/3M0OzeZ8j1k3Nqze/img/dce4c57f-5e47-4ddb-9832-9e97752d561f.png?Expires=1774164998&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=lMO8c92yZS9au113CQaAjAAJqTc%3D "")

#### 计算字段

痛点
- 针对如统计字段中所示的动态计算剩余待回款金额的场景，同样面临手工计算难的问题
- 针对需要动态计算到期天数的场景，例如合同到期天数、距上次跟进天数等，需要结合当前日期动态更新，逐条数据变动难度极大

解决方案
- 计算字段搭配统计字段使用，动态计算剩余量等信息，如订单中未回款金额=订单金额-已回款金额
- 计算字段搭配逻辑函数使用，动态标记单据状态，如根据客户下单总额，标记客户等级
- 计算字段搭配日期函数，动态计算剩余天数等信息，如根据合同到期时间，动态计算合同到期天数

## 更多流程自动化处理

痛点
- 公司存在非标业务，若定制开发费用高，难以承受

解决方案
- 工作流程可自定义触发动作、触发条件
- 由单据修改触发时，可根据修改前、修改后、发生修改时信息配置触发条件
- 可同时修改目标对象主、子（可多个）信息
- 字段映射时，赋值方式支持获取空值、固定值、联动值、计算值

说明
- 升级旗舰版，企业自定义工作流程可增加至20个

## 更多BI报表搭建

痛点
- 销售经营分析报表所分析的维度、统计的信息、展示的字段均需要能结合企业自身的需求个性化生成，传统软件中预设的报表通常难以完美契合，故需要将系统内的数据逐个模块导出，并在表格中手动加工，费时费力，且无法根据数据变动实时更新、在线查看，经营分析是滞后的

解决方案
- 可获取应用中所有模块及其关联模块所有信息统一分析，分析的维度、图表展示字段、统计的数值统统可以自定义
- 托拉拽方式即可搭建，操作简便
- 支持明细表、柱状图、饼图、漏斗图、kpi卡片、仪表盘、交叉表等多种图表类型

说明
- 升级旗舰版，企业报表总数可增加至20张（最多200张图表）

## 更多消息机器人

痛点
- 客户长时间未跟进、拜访、下单，无法提醒对应业务员跟进促活，容易造成客户流失
- 合同、证件、收付款截止日期等到期前无法主动提醒对应人员，容易超期，造成合同、证件失效，回款超时造成公司资金回流慢甚至坏账

解决方案
- 可自定义发送消息数据筛选范围、动态发送消时间、消息接收人、消息提醒文案，确保重要消息不遗漏

说明
- 升级旗舰版，企业自定义消息模板可增加至20个

## 更多视图形式

痛点
- 项目阶段、任务完成情况展示不直观，不利于项目进度掌控与风险管理

解决方案
- 所有模块均可自定义甘特图展示，通过可视化业务时间线，实现资源合理分配，有助于业务顺利开展
- 视图时间线信息、分组字段、展示字段均可自定义
