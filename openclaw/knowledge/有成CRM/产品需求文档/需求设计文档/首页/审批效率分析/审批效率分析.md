## 1、需求背景

1、针对流程（当前是审批流程），记录流程实例耗时、节点超时时间

2、针对流程，进行流程数量、耗时分析，便于企业分析流程运行情况，优化流程设置及节点效率。

备注：审批表结构

select \* from flow\_instance\_0;---主表， 存储审批流程

select \* from flow\_instance\_summary\_0; 摘要表： 存储摘要信息， 应该用来过滤摘要条件

select \* from flow\_node\_operation\_record\_0; 操作记录表： 用来过滤这个审批流程是否存在操作记录， 一般自动通过的不会存在操作日志， 记录了某个流程人员在某一节点的操作行为以及操作时间

select \* from flow\_user\_task\_0; 任务表，每次到达新节点会为审批人生成待审批的任务，审批人员操作后修改任务状态为完成

select \* from flow\_copy\_to\_0; 简单存储流程中的抄送人
