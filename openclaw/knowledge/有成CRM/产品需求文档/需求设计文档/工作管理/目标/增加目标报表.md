| 需求 | 描述 |
|-----------------------------------------------------------------|-----------------------------------------------------------------|
| 增加目标报表 | 仅pc端增加
- 1. 需要记住当前用户切换的视图类型
- 2. 筛选器
<li style="margin-left:1em">1. 展示指标类型及范围
<li style="margin-left:2em">1. 默认是‘员工指标’，用户‘角色-其他数据权限不为‘本人’，可切换为‘部门指标’
<li style="margin-left:2em">2. 指标可选范围及默认值依据用户‘角色-其他数据权限’
![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/oJGq75k3g19PzlAK/img/1ab6077b-43a5-48bc-b241-b97154ca2cda.png?Expires=1774165162&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=e1ZxtCGs6bY%2F3%2F7HlhyL1ck5cVM%3D "")
<li style="margin-left:1em">2. 展示指标日期维度及范围
<li style="margin-left:2em">3. 默认‘按月’，可切换为‘按季度’‘按半年度’‘按年度’，切换选项日期范围清除并自动填充默认值
<li style="margin-left:2em">4. 日期范围
<li style="margin-left:3em">1. 只能选择年份
<li style="margin-left:3em">2. 默认为当年～当年，如2025～2025
<li style="margin-left:3em">3. 最大年份不能超过当年\+1
<li style="margin-left:1em">3. 展示指标范围
默认‘全部’，可多选，可选企业目标管理中，目标可见=是的指标
- 3. 报表
<li style="margin-left:1em">4. 维度
<li style="margin-left:2em">5. 人员维度
<li style="margin-left:3em">4. 员工：根据所设置的人员筛选范围，展示对应人员
<li style="margin-left:3em">5. 部门：根据所设置的部门筛选范围，展示对应部门（选择‘全公司’，则为所有部门）
<li style="margin-left:2em">6. 日期维度
筛选器中指的是财年，例如设置2025财年设置的是2025.4～2026.3，则按月展示2025～2025年，指的是展示2025.4～2026.3的指标
<li style="margin-left:1em">5. 维度属性
<li style="margin-left:2em">7. 员工：展示员工所属部门
<li style="margin-left:2em">8. 部门：展示部门上级部门（所有）
<li style="margin-left:1em">6. 指标
<li style="margin-left:2em">9. 目标：同目前线上
<li style="margin-left:2em">10. 完成：同目前线上
<li style="margin-left:2em">11. 完成率：完成/目标\*100%
<li style="margin-left:2em">12. 累计完成率：以按月统计为例，2025.2月的累计完成率=（2025.2完成\+2025.1完成）/（2025.2目标\+2025.1目标）\*100%
<li style="margin-left:1em">7. 其他
维度、维度属性、合计行/列均为固定行/列，其余部份是可拖拽查看的
- 4. 其他
<li style="margin-left:1em">8. 数据支持导出 |
