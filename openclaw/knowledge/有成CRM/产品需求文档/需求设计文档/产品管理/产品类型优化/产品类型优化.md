# 产品类型优化

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 2023-7-4 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 1、需求背景
- 产品类型维护编辑框太小：目前宽度520需要支持排序：力辰分级不够：北京君源商厨节能自动化设备有限公司/青岛宏迈机械有限公司覆盖导入/修改：中国人寿云南省分公司（本期不考虑）批量新增：广州三森健康产业科技有限公司（本期不考虑）
- 新建/筛选展示不全：力辰/武汉智易农批科技有限公司搜索：力辰/武汉智易农批科技有限公司展示：目前系统选择产品类型筛选的时候 默认的是最小级的数据 客户希望显示第一级的数据，因为数据太多选择不方便

## 2、需求详情

| **需求** | **描述** |
|----------|----------|
| 产品类型维护优化（仅pc端有） | 一、页面宽度增加到1000px（和选择关联弹窗一样大）二、支持调整~分级及~顺序（同级）三、分级限制改为最多15级\+ 分级名太长，多余部分用...代替四、操作按钮，右对齐 |
| 新建/编辑\+筛选框样式及交互优化 | 筛选框包含：大列表、选择关联弹窗、新建编辑（子表单）、设置-消息/数据共享/模板配置（统计字段）相关搜索：根据搜索字段，模糊匹配所有级别选项值，找到匹配结果后，展示该选项值前置路径信息 |
