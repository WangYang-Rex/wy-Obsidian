# 仓库筛选器支持多选

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 20220614 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 1、需求背景

目前在库存查询、库存预警、批次保质期查询、出入库记录、收发汇总（仅pc端）有仓库筛选器，但是该筛选器只支持单选，但是有较多客户提出，想支持多选。

## 2、业务流程/架构

无

## 3、需求详情

| **模块** | **二级模块** | **功能点** |
|----------|----------------|-------------|
| 库存管理 | 1、库存查询（库存查询、库存预警、批次保质期查询）2、出入库记录3、收发汇总（仅pc端） | 一、仓库筛选器支持多选仓库，并根据选择的仓库筛选展示对应的数据；二、对应多选的样式1、PC端：参见单据新建编辑页样式但不同的是：1）最多展示两个选项值，超过的则按‘...'展示2）选项值限宽2）移动端：参见列表页高级筛选多选样式 |
