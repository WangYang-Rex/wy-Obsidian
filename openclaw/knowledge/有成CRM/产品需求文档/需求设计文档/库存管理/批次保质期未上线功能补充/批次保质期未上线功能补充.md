## 1、需求背景

批次保质期功能已上线，但之前有部分需求因开发周期影响，并未上线，本期补充。

## 2、业务流程/架构

无。

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 出入库单、组装单、调拨单、盘点单 | \+ 新建编辑页：支持多选批次保质期\+ 新建编辑页：生产日期支持选择批次保质期；\+ 新建编辑页：批次保质期相关字段支持批量填充 |  |
| 新建编辑页明细行 | \+ 所有有明细行的新建编辑页：明细行信息录入界面，日期的组件边框较文本等的大些； |  |
| 消息设置 | \+ 批次保质期消息设置 |  |

## 4、需求详情

### 4.1、需求说明

| **模块** | **说明** |
|----------|----------|
| 出入库单、组装单、调拨单、盘点单 | \+ 新建编辑页：支持多选批次保质期（pc端\+移动端）支持多选明细行，选择了N（N\>0）行数据，则复制N-1行操作行数据在操作行下方，  并将选中的数据行\{批次\}、\{生产日期\}、\{保质期\}、\{失效日期\}填充到对应行同名字段；\+ 新建编辑页：生产日期支持选择批次保质期；\+ 新建编辑页：批次保质期相关字段支持批量填充 |
| 新建编辑页明细行 | \+ 所有有明细行的新建编辑页：明细行信息录入界面，日期的组件边框与其他组件一致； |
| 消息设置 | \+ 批次保质期消息设置 |
