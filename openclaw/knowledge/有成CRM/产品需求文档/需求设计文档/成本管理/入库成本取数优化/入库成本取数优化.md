# 入库成本取数优化

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 2024-8-12 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

| **需求** | **描述** |
|----------|----------|
| 增加入库成本取数设置 | 备注：功能上线时，三个逻辑均开启，优先级顺序如下取数逻辑：若企业开启了存货核算（既已完成存货初始化，未完成则默认为空）1\. 取销售出库单成本备注：满足条件的出库单产品有多条，取业务日期、最后修改时间最近的一条2\. 取最近入库成本3\. 取产品成本价 |
| 盘点单产品 | 新开通企业，字段说明统一改为：启用了存货核算时，盘亏的产品明细会自动计算出库成本信息（若填写了会被覆盖），盘盈的产品明细则根据入库成本取数规则默认填充（设置-进销存设置-入库成本设置） |
| 销退入库单产品 | 新开通企业，字段说明统一改为：启用了存货核算时，会根据入库成本取数规则默认填充（设置-进销存设置-入库成本设置） |
