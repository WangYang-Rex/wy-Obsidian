# 工商查询优化

| **修订时间** | **版本** | **修订人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 2023-5-26 | v1.0 | 黄艺平 | 新建 |
|  |  |  |  |

## 1、需求背景

尽量减少不必要的工商查询次数扣减：如回填字段中，未配置联系方式的字段回填，不调用

## 2、需求详情

| **需求** | **描述** |
|----------|----------|
| 工商查询优化 | 一、设置优化二、处理优化1、精准查询\+ 若取消勾选展示 \[联系方式\] ，则在精准查询页面不展示对应tab（不调用相关接口）- 扣减企业工商查询次数2次变更为1次- 回填字段设置中，就算设置了联系方式的映射，确认回填后也不进行回填更新（本质是没调用联系方式的接口，没获取到相关数据）2、模糊查询\+ 回填字段设置中，若未配置联系方式的字段映射，获取数据时不调用联系方式的查询接口，扣减企业工商查询次数2次变更为1次 |
