# 企业下人员列表

> **原文**：[企业下人员列表 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

**请求方式：**post

**请求参数数据类型：**application/json

**请求路径：**/oapi/corp/v1/user/list.json

**请求参数：**

| 参数            | 参数类型 | 必须 | 说明                                        |
|-----------------|----------|------|---------------------------------------------|
| corpId          | String   | 是   | 企业ID                                      |
| corpAccessToken | String   | 是   | 访问企业数据合法性票据                      |
| page            | Integer  | 否   | 默认1                                       |
| pageSize        | Integer  | 是   | 默认20，最大200，如超过200则系统按照200接收 |

**请求示例：**

```json
{
"corpId": "CORP_ID",
"corpAccessToken": "CORP_ACCESS_TOKEN",
"page":1,
"pageSize":20
}
```

**响应结果：**

| 参数           | 参数类型 | 必须 | 说明                       |
|----------------|----------|------|----------------------------|
| department     | String   | 是   | 所在部门Ids，英文逗号拼接  |
| departmentName | String   | 是   | 所在部门名称，英文逗号拼接 |
| userId         | String   | 是   | 用户userId                 |
| userName       | String   | 是   | 用户名称                   |
| userPicUrl     | String   | 是   | 用户头像路径               |

响应结果示例：

```json
{
"apiName": "",
"data": [
{
"department": "crm_1",
"departmentName": "光云科技",
"level": "USER",
"userId": "userId1",
"userName": "userName1",
"userPicUrl": "userPicUrl"
}
],
"message": "操作成功",
"result": 100
}
```
