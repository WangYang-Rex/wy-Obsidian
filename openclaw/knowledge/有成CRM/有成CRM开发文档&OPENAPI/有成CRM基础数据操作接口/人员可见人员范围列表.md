# 人员可见人员范围列表

> **原文**：[人员可见人员范围列表 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

请求方式：post

请求参数数据类型：application/json

请求路径：/oapi/corp/v1/owner/user/list.json

请求参数：

| 参数            | 参数类型 | 必须 | 说明                   |
|-----------------|----------|------|------------------------|
| corpId          | String   | 是   | 企业ID                 |
| corpAccessToken | String   | 是   | 访问企业数据合法性票据 |
| userId          | String   | 是   | 用户userId             |
| deptId          | String   | 否   | 部门ID                 |
| userName        | String   | 否   | 用户名称模糊查询       |

请求示例：

```json
{
"corpId": "CORP_ID",
"userId": "USER_ID",
"corpAccessToken": "CORP_ACCESS_TOKEN",
"deptId": 1,
"userName": ""
}
```

响应结果：

| 参数           | 参数类型 | 必须 | 说明                   |
|----------------|----------|------|------------------------|
| department     | String   | 是   | 部门ids，英文逗号拼接  |
| departmentName | String   | 是   | 部门名称，英文逗号拼接 |
| userId         | String   | 是   | 用户userId             |
| userName       | String   | 是   | 用户名称               |
| userPicUrl     | String   | 是   | 用户头像路径           |

响应结果示例：
