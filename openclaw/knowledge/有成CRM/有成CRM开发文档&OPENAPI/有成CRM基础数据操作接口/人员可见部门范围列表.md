# 人员可见部门范围列表

> **原文**：[人员可见部门范围列表 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

请求方式：post

请求参数数据类型：application/json

请求路径：/oapi/corp/v1/owner/dept/list.json

请求参数：

| 参数            | 参数类型 | 必须 | 说明                   |
|-----------------|----------|------|------------------------|
| corpId          | String   | 是   | 企业ID                 |
| corpAccessToken | String   | 是   | 访问企业数据合法性票据 |
| userId          | String   | 是   | 用户userId             |
| deptParentId    | String   | 否   | 父部门ID               |
| deptName        | String   | 否   | 部门名称模糊查询       |

请求示例：

```json
{
"corpId": "CORP_ID",
"corpAccessToken": "CORP_ACCESS_TOKEN",
"userId":"USER_ID",
"deptParentId":"1",
"deptName":"科技"
}
```

响应结果：

| 参数         | 参数类型 | 必须 | 说明           |
|--------------|----------|------|----------------|
| deptId       | String   | 是   | 部门ID         |
| deptName     | String   | 是   | 部门名称       |
| deptParentId | String   | 是   | 父部门ID       |
| userCount    | Integer  | 是   | 部门下用户数量 |

响应结果示例：

```json
{
"apiName": "",
"data": [
{
"deptId": "crm_111",
"deptName": "光云科技",
"deptParentId": "1",
"userCount": 1,
"users": []
}
],
"message": "操作成功",
"result": 100
}
```
