# 数据字典值列表查询

> **原文**：[数据字典值列表查询 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

请求方式：post

请求参数数据类型：application/json

请求路径：/oapi/corp/v1/dic/list.json

请求参数：

| 参数            | 参数类型 | 必须 | 说明                   |
|-----------------|----------|------|------------------------|
| corpId          | String   | 是   | 企业ID                 |
| corpAccessToken | String   | 是   | 访问企业数据合法性票据 |
| dicValue        | String   | 是   | 字典                   |

**请求示例：**

```json
{
"corpAccessToken": "CORP_ACCESS_TOKEN",
"corpId": "CORP_ID",
"dicValue": "product_units"
}
```

**响应结果：**

| 参数    | 参数类型 | 必须 | 说明               |
|---------|----------|------|--------------------|
| apiName | String   | 否   | api名称            |
| data    | Object   | 否   | 结果数据           |
| message | String   | 是   | 提示信息           |
| result  | int      | 是   | 状态码 100操作成功 |
