# 开放业务对象字段列表

> **原文**：[开放业务对象字段列表 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

接口说明：获取CRM开放业务对象字段列表

**请求说明：**

请求方式：post

请求参数数据类型：application/json

请求路径：/oapi/corp/v1/object/field/list.json

**请求参数说明：**

| 参数            | 参数类型 | 参数必填 | 说明                   |
|-----------------|----------|----------|------------------------|
| corpAccessToken | String   | 是       | 访问企业数据合法性票据 |
| corpId          | String   | 是       | 企业ID                 |
| deployId        | String   | 是       | 业务对象deployId       |
| templateId      | String   | 否       | 模版ID                 |

**请求参数结构：**

```json
{
"corpAccessToken": "CORP_ACCESS_TOKEN",
"corpId": "CORP_ID",
"deployId":"DEPLOY_ID"
}
```

**响应结果说明：**

| 参数 | 参数类型 | 参数必填 | 说明 |
|----|----|----|----|
| codeRole | String | 否 | 自动编码规则 |

| componentType | String | 否 | 组件类型 |

| isVisible | String | 是 | 字段可见*0、否 1、是* |

| referRule | String | 是 | 关联对象规则REFER_SELECT和REFER_OBJECT 使用该字段作为关联条件字段 |
