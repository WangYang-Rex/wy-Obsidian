# 获取销售机会阶段列表

> **原文**：[获取销售机会阶段列表 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

请求方式：post

请求参数数据类型：application/json

请求路径：/oapi/corp/v1/opportunity_stage/list.json

请求参数：

| 参数            | 参数类型 | 必须 | 说明                   |
|-----------------|----------|------|------------------------|
| corpId          | String   | 是   | 企业ID                 |
| corpAccessToken | String   | 是   | 访问企业数据合法性票据 |
| enable          | Boolean  | 否   | 是否启用               |

**请求示例：**

```json
{
"corpAccessToken": "CORP_ACCESS_TOKEN",
"corpId": "CORP_ID",
"enable": true
}
```

**响应结果：**

| 参数    | 参数类型 | 必须 | 说明               |
|---------|----------|------|--------------------|
| apiName | String   | 否   | api名称            |
| data    | Object   | 否   | 结果数据           |
| message | String   | 是   | 提示信息           |
| result  | int      | 是   | 状态码 100操作成功 |
