# 人员可见模版范围列表

> **原文**：[人员可见模版范围列表 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

请求方式：post

请求参数数据类型：application/json

请求路径：/oapi/corp/v1/owner/template/list.json

请求参数：

| 参数            | 参数类型 | 必须 | 说明                   |
|-----------------|----------|------|------------------------|
| corpId          | String   | 是   | 企业ID                 |
| corpAccessToken | String   | 是   | 访问企业数据合法性票据 |
| deployId        | String   | 是   | 业务对象deployId       |
| userId          | String   | 是   | 用户userId             |

请求示例：

```json
{
"corpId": "CORP_ID",
"corpAccessToken": "CORP_ACCESS_TOKEN",
"deployId":"customer",
"userId":"USER_ID"
}
```

响应结果：

| 参数         | 参数类型 | 必须 | 说明             |
|--------------|----------|------|------------------|
| corpId       | String   | 是   | 企业ID           |
| changerId    | String   | 是   | 最近修改人userId |
| modified     | String   | 是   | 最近修改时间     |
| creatorId    | String   | 是   | 最近创建人userId |
| created      | Long     | 是   | 创建时间         |
| deployId     | String   | 是   | 业务对象deployId |
| templateId   | String   | 是   | 模版ID           |
| templateName | String   | 是   | 模版名称         |

响应结果示例：

```json
{
"apiName": "",

{

"templateName": "客户模版1",
}
],
"message": "操作成功",

}
```
