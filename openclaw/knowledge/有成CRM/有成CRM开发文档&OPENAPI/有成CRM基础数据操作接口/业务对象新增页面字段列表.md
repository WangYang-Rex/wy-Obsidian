# 业务对象新增页面字段列表

> **原文**：[业务对象新增页面字段列表 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

对业务对象做新增操作时 需要用到的字段列表

**请求方式：**post

**请求参数数据类型：**application/json

**请求路径：**/oapi/corp/v1/add/field/list.json

**请求参数：**

| 参数            | 参数类型 | 必须 | 说明                   |
|-----------------|----------|------|------------------------|
| corpId          | String   | 是   | 企业ID                 |
| corpAccessToken | String   | 是   | 访问企业数据合法性票据 |
| deployId        | String   | 是   | 业务对象deployId       |
| userId          | String   | 是   | 企业用户userId         |
| tempalteId      | String   | 否   | 模版ID                 |

**请求示例：**

```json
{
"corpAccessToken":"CORP_ACCESS_TOKEN",
"corpId":"CORP_ID",
"deployId":"customer",
"userId":"USER_ID",
"templateId":"TEMPLATE_ID"
}
```
