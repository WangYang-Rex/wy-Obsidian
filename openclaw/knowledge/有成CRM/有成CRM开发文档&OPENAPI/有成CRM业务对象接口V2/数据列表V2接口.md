# 数据列表V2接口

> **原文**：[数据列表V2接口 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

接口说明：获取CRM开放的业务对象数据列表

**请求说明**

请求方式：post

请求参数数据类型：application/json

请求路径：/oapi/corp/v2/data/list.json

**请求参数说明：**

| 参数 | 参数类型 | 参数必填 | 说明 |
|----|----|----|----|
| corpId | String | 是 | 企业ID |
| corpAccessToken | String | 是 | 访问企业数据合法性票据 |
| deployId | String | 是 | 业务对象deployId（如：customer、opportunity、sale_order等） |
| userId | String | 否 | 用户ID，用于数据权限过滤 |
| templateId | String | 否 | 业务对象模板ID |
| formated | Boolean | 否 | 是否格式化数据，默认false。true时返回格式化后的显示值 |
| conditionConfig | List\<Object\> | 否 | 筛选条件组列表，结构同V1列表接口 |
| conditionConfig\[0\].conditions | List\<Object\> | 否 | 字段条件列表 |
| conditionConfig\[0\].conditions\[0\].comparison | String | 是 | 比较符（详见比较符说明） |
| conditionConfig\[0\].conditions\[0\].deployId | String | 是 | 字段所属业务对象deployId |
| conditionConfig\[0\].conditions\[0\].fieldName | String | 是 | 字段英文名称 |
| conditionConfig\[0\].conditions\[0\].fieldTitle | String | 是 | 字段中文名称 |
| conditionConfig\[0\].conditions\[0\].fieldValue | Object | 是 | 筛选条件内容 |
| conditionConfig\[0\].conditions\[0\].domType | String | 是 | 字段类型 |
| orderFields | List\<Object\> | 否 | 排序字段列表 |
| orderFields\[0\].fieldName | String | 是 | 排序字段名称 |
| orderFields\[0\].orderType | String | 是 | 排序方式：asc升序、desc降序 |
| page | Integer | 否 | 页码，默认1 |
| pageSize | Integer | 否 | 每页条数，默认20，最大1000 |

具体可参照 ：[常见comparison查询条件组装格式示例](http://www.ycbg.com/help-center.html?tab=1&id=875)

**响应结果说明：**

| 参数 | 参数类型 | 参数必填 | 说明 |
|----|----|----|----|
| success | Boolean | 是 | 请求成功状态，true成功、false失败 |
| data | Object | 是 | 查询结果对象 |
| data.total | Long | 是 | 查询结果总条数 |
| data.list | List\<Map\<String,Object\>\> | 是 | 结果数据列表 |
| code | String | 是 | 返回码 |
| msg | String | 是 | 返回码描述 |
