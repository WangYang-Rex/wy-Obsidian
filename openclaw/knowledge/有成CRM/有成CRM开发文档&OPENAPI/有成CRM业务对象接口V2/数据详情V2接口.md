# 数据详情V2接口

> **原文**：[数据详情V2接口 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

接口说明：获取CRM开放的业务对象单条数据详情

**请求说明**

请求方式：post

请求参数数据类型：application/json

请求路径：/oapi/corp/v2/data/detail.json

**请求参数说明：**

| 参数 | 参数类型 | 参数必填 | 说明 |
|----|----|----|----|
| corpId | String | 是 | 企业ID |
| corpAccessToken | String | 是 | 访问企业数据合法性票据 |
| deployId | String | 是 | 业务对象deployId |
| dataId | Long | 是 | 数据唯一ID |
| userId | String | 否 | 用户ID，用于数据权限过滤 |
| needSubForm | Boolean | 否 | 是否需要子表单数据，默认false。仅主表单对象生效 |
| formated | Boolean | 否 | 是否格式化数据，默认false。true时返回格式化后的显示值 |

**请求参数结构：**

```json
{
    "corpId": "ding2037cc74cbb0f5e735c2f4657eb6378f",
    "corpAccessToken": "8355993e7777895b9859f4771511530992ad4c",
    "deployId": "customer",
    "dataId": 1154,
    "userId": "manager123",
    "needSubForm": false,
    "formated": false
}
```

**响应结果说明：**

| 参数 | 参数类型 | 参数必填 | 说明 |
|----|----|----|----|
| success | Boolean | 是 | 请求成功状态 |
| data | Map\<String,Object\> | 是 | 数据详情对象，包含该业务对象的所有字段 |
| code | String | 是 | 返回码 |
| msg | String | 是 | 返回码描述 |

**注意事项：**

1\. needSubForm参数仅对主表单对象生效，如客户、订单等对象。子表单对象不支持此参数。

2\. formated为true时，字段值会返回格式化后的显示值（如日期格式化、金额格式化、选项值翻译等）。
