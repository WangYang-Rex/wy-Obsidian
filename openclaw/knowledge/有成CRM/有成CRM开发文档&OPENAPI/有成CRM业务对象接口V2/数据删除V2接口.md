# 数据删除V2接口

> **原文**：[数据删除V2接口 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

接口说明：删除CRM业务对象数据

**请求说明**

请求方式：post

请求参数数据类型：application/json

请求路径：/oapi/corp/v2/data/delete.json

**请求参数说明：**

| 参数            | 参数类型     | 参数必填 | 说明                         |
|-----------------|--------------|----------|------------------------------|
| corpId          | String       | 是       | 企业ID                       |
| corpAccessToken | String       | 是       | 访问企业数据合法性票据       |
| deployId        | String       | 是       | 业务对象deployId             |
| userId          | String       | 是       | 用户ID，执行删除操作的用户   |
| dataIds         | List\<Long\> | 是       | 数据唯一ID列表，支持批量删除 |

**请求参数结构：**

```json
{
    "corpId": "ding2037cc74cbb0f5e735c2f4657eb6378f",
    "corpAccessToken": "8355993e7777895b9859f4771511530992ad4c",
    "deployId": "customer",
    "userId": "manager123",
    "dataIds": [1154, 1155, 1156]
}
```

**响应结果说明：**

| 参数    | 参数类型 | 参数必填 | 说明         |
|---------|----------|----------|--------------|
| success | Boolean  | 是       | 请求成功状态 |
| data    | Object   | 是       | 删除结果对象 |
| code    | String   | 是       | 返回码       |
| msg     | String   | 是       | 返回码描述   |

**注意事项：**

1\. 支持批量删除，dataIds可传入多个数据ID。

2\. 删除操作会同时删除该数据关联的子表单数据。

3\. 部分业务对象有删除权限校验，请确保操作用户有删除权限。

4\. 该接口有请求频率限制，每分钟最多30次。
