# 开放业务对象数据删除

> **原文**：[开放业务对象数据删除 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

请求方式：post （[推荐接入V2版本](https://www.ycbg.com/help-center.html?tab=1&id=1219)）

请求参数数据类型：application/json

请求路径：/oapi/corp/v1/data/delete.json

请求参数：

| 参数            | 参数类型     | 必须 | 说明                   |
|-----------------|--------------|------|------------------------|
| corpId          | String       | 是   | 企业ID                 |
| corpAccessToken | String       | 是   | 访问企业数据合法性票据 |
| userId          | String       | 是   | 用户userId             |
| deployId        | String       | 是   | 业务对象deployId       |
| dataIds         | List\<Long\> | 是   | 业务对象数据主键ID     |

请求示例：

```json
{
"corpAccessToken": "CORP_ACCESS_TOKEN",
"corpId": "CORP_ID",
"deployId": "contract",
"userId": "USER_ID",
"dataIds": [
437
]
}
```

响应结果：

| 参数    | 参数类型 | 必须 | 说明               |
|---------|----------|------|--------------------|
| apiName | String   | 否   | api名称            |
| data    | Object   | 否   | 结果数据           |
| message | String   | 是   | 提示信息           |
| result  | int      | 是   | 状态码 100操作成功 |

响应结果示例：

```json
{
&nbsp;&nbsp;&nbsp;&nbsp;"apiName":&nbsp;"",
&nbsp;&nbsp;&nbsp;&nbsp;"data":&nbsp;null,
&nbsp;&nbsp;&nbsp;&nbsp;"message":&nbsp;"操作成功",
&nbsp;&nbsp;&nbsp;&nbsp;"result":&nbsp;100
}
```
