# 批量删除协作人

> **原文**：[批量删除协作人 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

请求方式：post

请求参数数据类型：application/json

请求路径：/oapi/corp/v1/data/batch/del/collaborator.json

请求参数：

| 参数                | 参数类型 | 必须 | 说明                   |
|---------------------|----------|------|------------------------|
| corpId              | String   | 是   | 企业ID                 |
| corpAccessToken     | String   | 是   | 访问企业数据合法性票据 |
| userId              | String   | 是   | 用户userId             |
| deployId            | String   | 是   | 业务对象deployId       |
| dataIds             | List     | 是   | 数据ID                 |
| collaboratorUserIds | List     | 是   | 协作人userId           |

请求示例：

```json
{
"corpAccessToken": "{CORP_ACCESS_TOKEN}",
"corpId": "{CORP_ID}",
"userId": "{USER_ID}",
"deployId": "sale_order",
"dataIds": [
1
],
"collaboratorUserIds":[
"xxxxxxx"
]
}
```

响应结果：

| 参数    | 参数类型 | 必须 | 说明                       |
|---------|----------|------|----------------------------|
| apiName | String   | 否   | api名称                    |
| data    | Object   | 否   | 结果数据（对应新增数据ID） |
| message | String   | 是   | 提示信息                   |
| result  | int      | 是   | 状态码 100操作成功         |

响应结果示例：

```json
{
"apiName": "",
"data": {
"id": 80082
},
"message": "操作成功",
"result": 100
}
```
