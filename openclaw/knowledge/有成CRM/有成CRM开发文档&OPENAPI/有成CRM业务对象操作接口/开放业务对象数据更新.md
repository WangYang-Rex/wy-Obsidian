# 开放业务对象数据更新

> **原文**：[开放业务对象数据更新 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

请求方式：post （[推荐接入V2版本](https://www.ycbg.com/help-center.html?tab=1&id=1218)）

请求参数数据类型：application/json

请求路径：/oapi/corp/v1/data/update.json

请求参数：

| 参数 | 参数类型 | 必须 | 说明 |
|----|----|----|----|
| corpId | String | 是 | 企业ID |
| corpAccessToken | String | 是 | 访问企业数据合法性票据 |
| userId | String | 是 | 用户userId |
| deployId | String | 是 | 业务对象deployId |
| templateId | String | 否 | 单据模板ID 用于对数据的校验，如果未指定，则以系统预置模板(系统预置模板非默认模板)对数据进行校验 |
| data | Map | 是 | 业务对象数据 |
| dataId | Long | 是 | 业务对象数据主键ID |
| relationDatas | Map | 否 | 关联子对象数据 |
| relationDatas.{子对象deployId} | List | 否 | 子对象数据 |
| relationDatas.{子对象deployId}.id | String | 否 | 关联子对象id |

| message | String   | 是   | 提示信息           |
| result  | int      | 是   | 状态码 100操作成功 |
