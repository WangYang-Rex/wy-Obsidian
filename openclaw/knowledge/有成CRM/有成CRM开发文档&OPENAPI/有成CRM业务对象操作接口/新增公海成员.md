# 新增公海成员

> **原文**：[新增公海成员 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

请求方式：post

请求参数数据类型：application/json

请求路径：/oapi/corp/v1/customer/add/seas/person.json

请求参数：

参数

参数类型

必须

说明

corpId

String

是

企业ID

corpAccessToken

String

是

访问企业数据合法性票据

userId

String

是

用户userId

seasUserId

String

是

公海成员ID

seasId

String

是

公海Id

userType

String

是

成员类型#member=普通成员, admin=管理员

请求示例：

```json
{
"corpAccessToken": "CORP_ACCESS_TOKEN",
"corpId": "CORP_ID",
"userId": "USER_ID",
"seasUserId": 437,
"seasId":"8bba81b335b8449095b6f9d7c97ef9aa",
"userType":"member"
}
```
