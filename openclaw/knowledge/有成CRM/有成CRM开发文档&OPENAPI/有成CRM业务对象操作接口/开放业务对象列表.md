# 开放业务对象列表

> **原文**：[开放业务对象列表 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

接口说明：获取当前企业下允许操作的开放业务对象列表

**请求说明：**

请求方式：post

请求参数数据类型：application/json

请求路径：/oapi/corp/v1/object/list.json

**请求参数说明：**

| 参数            | 参数类型 | 参数必填 | 说明                   |
|-----------------|----------|----------|------------------------|
| corpAccessToken | String   | 是       | 访问企业数据合法性票据 |
| corpId          | String   | 是       | 企业ID                 |

**请求参数结构示例：**

```json
{
"corpAccessToken": "CORP_ACCESS_TOKEN",
"corpId": "CORP_ID"
}
```

**响应结果说明：**

| 参数            | 参数类型 | 参数必填 | 说明             |
|-----------------|----------|----------|------------------|
| apiName         | String   | 是       | 接口apiName      |
| data            | Object   | 否       | 响应数据         |
| data.deployId   | String   | 否       | 业务对象deployId |
| data.deployName | String   | 否       | 业务对象显示名称 |
| data.enable     | Boolean  | 否       | 业务对象可用状态 |
| message         | String   | 是       | 返回码描述       |
| result          | Integer  | 是       | 返回码           |
