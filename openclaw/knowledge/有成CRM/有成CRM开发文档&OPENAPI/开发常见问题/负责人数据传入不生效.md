# 负责人数据传入不生效

> **原文**：[负责人数据传入不生效 - 有成CRM官网帮助中心](https://www.ycbg.com/help-center.html?tab=1&id=641)  
> **导出时间**：2026-06-23

**Q：新增/更新数据时 传入的负责人没有生效**

```json
{
"corpAccessToken": "${corpAccessToken}",
"corpId": "${corpId}",
"userId": "${userId}",
"deployId": "customer",
"templateId": "default_template",
"data": {
"seas_id": "067459210e404c91a6f109753af734fd",
"template_id": "default_template",
"custom_nick": "iloiketomovie",
"custom_name": "jungle",
"charger_id": "${负责人Id}",
"charger_name": "${负责人名称}"
}
}
```

A：负责人推荐直接使用 [更换负责人](http://www.ycbg.com/help-center.html?tab=1&id=659) 接口进行操作；
