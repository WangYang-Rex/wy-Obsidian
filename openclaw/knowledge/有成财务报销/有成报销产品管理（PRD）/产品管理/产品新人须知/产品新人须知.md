## 三、产品工作

**主要工作**：制定季度业务目标和需求规划；答疑/需求池管理；需求开发/测试任务管理；产品方案编制（产品文档）；产品使用手册输出

### 1. 需求月度/季度规划维护

结合客户情况、需求价值，维护每个季度的业务目标和需求规划：
- 📋 https://gykj.yuque.com/izoyhv/tzh23k/gqe1el

### 2. 客户需求池收集/维护

**KA客户（1万-5万）/ SKA（5万以上）交付阶段需求**
- 入口：钉钉报销大客户需求群
- 流程：分模块认领 → 出产品方案 → 评审 → TB建任务交付开发 → 上线验收 → 更新需求进度

**VIP（1万以下）及交付后所有客户需求**
- 入口：有成系列架构 → 有成CRM → 工单 → 财务需求工单（售后）和财务售前需求工单
- 流程：接单 → 出产品方案 → TB建任务交付开发 → 上线验收

### 3. 需求答疑

加入售前售后需求咨询群：
- **售前**：钉钉群-财务报销-产品解决方案交流群
- **售后**：钉钉群-客成需求反馈-财务报销

### 4. 项目开发/测试任务管理

使用 TB 进行管理

### 5. 产品文档（产品方案）

- 📋 产品设计：https://gykj.yuque.com/izoyhv/tzh23k/gxq6vp
- 📋 需求文档格式要求：https://gykj.yuque.com/izoyhv/tzh23k/ie51o0

- **期望时间**：一周一次
- **发布位置**：钉钉工作台 → 光云工作台 → 有成财务管理后台 → 消息推送 → 系统公告
