## 1、需求背景

**客户**：财开

**背景：**1）我们现在子表单中文本类的字段，表格宽度不够，导致输入后展示不全；有些数字组件宽度留得又很宽，展示的时候留空很多；2）部分表单可以拖拽宽度 部分又不支持，在系统中不统一；

## 5、需求详情
1. 需要调整的页面：**单据新增页面、单据详情页面**
2. 涉及的组件：**子表单、费用明细、费用明细-费用分摊、产品明细、关联单据、行程、核销借款**
3. 调整内容：**以上组件支持拖拽调整宽度；并且可以记住当前用户拖拽的宽度，下次进到该类单据页面还是展示此次拖拽的宽度**（现在应该是记住的）**根据不同的字段类型，调整默认宽度**

以下用  **较窄****、****适中****、****较宽**  来描述不同字段类型 预设的默认宽度（具体宽度规范由UI定）

| **字段** | **默认宽度** | **备注** |
|----------|----------------|----------|
| **文本-多行文本** | 较宽 | 一般用于输入长备注单行16个字默认预留三行展示区域，超出部分可拖动上下展示进度条查看 |
| **文本-单行文本** | 适中 | 单行16个字 |
| **数字** | 较窄 | 列表“序号”列宽度可以再调窄点最多6位数 |
| **金额** | 适中 | 最多13位数 |
| **日期（单个日期）** | 适中 |  |
| **日期区间** | 较宽 |  |
| \*\*附件 \*\* | 适中 | 上传附件  按钮标蓝 |
| **部门** | 适中 | 仅支持单选 |
| **员工** | 适中 | 仅支持单选 |
| **同行人-支持多选** | 较宽 | 展示样式需要调整可设置支持是否多选 |
| **同行人-不支持多选** | 适中 | 可设置支持是否多选 |
| **城市** | 适中 | 仅支持单选 |
| **辅助核算** | 适中 | 仅支持单选 |
| **月份范围** | 较宽 |  |
| **角色** | 适中 | 仅支持单选 |
| **发票抬头** | 较宽 | 选择数据后，一行展示一个字段信息 |
| **项目** | 适中 | 选择框中间的分割线去掉 |
| **收款信息** | 较宽 | 加宽，目前信息展示不全 |
| **手机号** | 适中 |  |
