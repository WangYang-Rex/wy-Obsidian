# **1\. 需求背景**

业务员报销时，往往填不清税率税额组件。即便开了根据发票自动填写税率税额组件，业务员核对时也无法理解明细行金额是否填写正确。因此需要：

1、标识出不同发票的“明细行金额”究竟对应了发票上的哪个字段；

2、方便手工填写税率税额组件的业务员，按发票字段正确填写

客户：捷成华视网聚

## **基础设置-自定义费用**

税率税额组件新增“增值税发票、机票行程单明细金额计算填写”开关。

默认关闭。即线上目前效果。

启用时，税率税额组件的明细行金额展示提示字段标签、点击标签展示计算填写入口。

线上存量用户为关闭。

## **填写费用上的税率税额组件时，展示提示字段、计算填写入口**

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/oJGq75kNgxQDLlAK/img/b4ee3226-17be-4162-8c20-7459521282f9.png?Expires=1774157252&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=E3T46Xtwgbe1X3LCPyMp14PB6Yk%3D "")
1. 填写费用入口：

随手记、新建单据、编辑单据、重新提交单据
2. 当税率税额组件中启用了“增值税发票、机票行程单明细金额计算填写”开关时，明细行金额中展示提示字段，根据不同发票类型展示不同值。

提示字段值有：不含税、计税价、票价、发票金额。

具体规则：

1）展示“不含税”的：——编辑状态下点击有价税合计气泡
- 增值税纸质普通发票
- 增值税纸质专用发票
- 增值税电子普通发票
- 增值税电子专用发票
- 通用电子发票
- 电子发票（普通发票）
- 电子发票（专用发票）

2）展示“计税价”的：——编辑状态下点击有票价、燃油附加费气泡
- 机动车发票
- 电子发票（航空运输电子客票行程单）

3）展示票价的：
- 火车票
- 电子发票（铁路电子客票）

4）展示发票金额的：
- 剩余的所有类型，兜底
3. 价税合计气泡

编辑状态下点击不含税标签提示价税合计气泡。

| 元素 | 说明 |
|------|------|
| 价税合计 | 两位小数，可编辑。<br>默认带入明细行金额\+税额，用户可修改。<br>明细行金额或税额为空按0.00处理。<br>点击弹出气泡时选中全部数字部分，定位输入光标。 |
| 税率 | 下拉框，可编辑。<br>默认带入列表上的税率，用户可修改。<br>下拉税率组件同列表组件。 |
| 自动计算 | 点击自动计算时执行以下数据更新并关闭气泡：<br><ul><li>明细行金额=价税合计/（1\+气泡中税率）\*气泡中税率，小数点两位四舍五入，覆盖原列表上的明细行金额。</li><br><li>税率按照气泡中税率覆盖原选择的税率。</li><br><li>税额、可抵扣税额根据原公式自动触发计算。</li></ul><br>**即所有的后续计算逻辑还是基于原来的几个字段，气泡只是为了方便用户录入明细行金额。** |
4. 票价、燃油附加费气泡

| 元素 | 说明 |
|------|------|
| 票价 | 两位小数，可编辑。默认为空。<br>点击弹出气泡时定位输入光标。 |
| 燃油附加费 | 两位小数，可编辑。默认为空。 |
| 自动计算 | 明细行金额=票价\+燃油附加费，小数点两位四舍五入，覆盖原列表上的明细行金额。<br><ul><li>税率不动</li><br><li>税额、可抵扣税额根据原公式自动触发计算。</li></ul> |
