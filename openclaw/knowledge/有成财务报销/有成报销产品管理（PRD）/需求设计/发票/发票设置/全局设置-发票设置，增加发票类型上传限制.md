# **1\. 需求背景**

因为目前我们的发票组件中没有对发票类型做限制，客户随便上传了个行程单图片到发票组件中（发票类型：其他），就走了报销流程。增加了财务的审核工作量。

AI审核虽然能在审核前拦截，但是一方面有商务成本，另一方面用户希望能够前移到提交环节。

客户：北京三吉世纪科技有限公司

## **全局设置-发票设置，增加发票类型上传限制开关**

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/4jKqm0bPMGPWYnw1/img/8ece1663-3d4d-4630-a651-650e84485387.png?Expires=1778767238&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=zhNQXVS6%2FVGI4xJOVtwtLo9OZ9A%3D "")
1. 全局设置-发票设置，增加“发票类型上传限制开关”，默认关闭。提示气泡文案：
> 关闭时，允许上传各种类型的发票；   
> 开启时，根据配置控制特定场景下，仅允许上传指定的发票类型，且用户无法手工修改发票类型：   
> -未配置场景时，则默认所有单据模板、费用类型、组件都会做校验。   
> -配置场景后，则针对配置的场景做校验。   
1. 关闭时，保持线上效果，即允许上传各种类型的发票。
2. 开启时，展示控制规则配置信息表，首次开启初始化默认值：
    - 单据模板：空；
    - 费用类型：空；
    - 组件：空，
    - 允许上传的发票类型：除了其他发票之外的所有发票类型。

控制规则配置信息字段：

| 字段 | 说明 |
|------|------|
| 单据模板 | 多选非必选。<br>树形，关键字模糊查找单据模板名称。<br>为空时，所有单据模板都需要校验。 |
| 费用类型 | 多选非必选。<br>树形，关键字模糊查找费用类型名称。<br>为空时，所有费用类型都需要校验。 |
| 组件 | 多选非必选。<br>可选项：<br>发票组件、合同开票组件、发票核销组件、开票申请组件<br>为空时，即上述组件都需要校验。 |
| 允许上传的发票类型 | 多选必选，可选项：<br><ul><li>增值税发票  vat\_invoice</li><br>包含增值税专用发票/增值税普通发票/增值税电子普通发票/增值税普通发票（卷票）/增值税电子普通发票（通行费）<br><li>定额发票  quota\_invoice</li><br>包含通用定额发票/过路过桥费发票/停车费发票/通用机打发票<br><li>出租车票  taxi</li><br><li>火车票  train\_ticket</li><br><li>电子发票（铁路电子客票）train\_electric\_ticket</li><br><li>机票行程单  air\_ticket</li><br><li>电子发票（航空运输电子客票行程单）air\_electric\_ticket</li><br><li>汽车票  car\_ticket</li><br><li>机动车票  mobile\_ticket</li><br><li>财政发票  financial\_invoice</li><br><li>其它类型  other</li></ul><br>当上传的发票不在配置范围内时，阻断并提示。 |
3. 可对控制规则进行新增，最多新增五条规则。多条规则取交集。到达上限时点击新增提示语：
> 最多新增五条控制规则   



## **功能入口**
1. 如果启用了发票类型上传限制开关，则在：
- 单据编辑页面（新建单据、重新提交、审批中编辑）、
- 费用编辑页面（随手记、我的-消费-添加消费）、发票生成消费

的发票组件、合同发票组件、发票核销组件、开票申请组件上传发票、选择发票入口，在所有校验前增加此校验。
> 已限制不能上传【用户上传的发票类型】   
2. 根据控制规则配置的场景，限制用户无法编辑发票类型，置灰控制。

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/4jKqm0bPMGPWYnw1/img/18b4a6e2-fa2a-4dec-9565-087efecd35db.png?Expires=1778767238&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=AVSRKHK4L8Gd8bskZ8Q2yJROCH4%3D "")
3. 上传/选择多张时，如果有发票触发了此校验，则抛出提示，过滤被限制的发票。剩余发票正常上传。

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/4jKqm0bPMGPWYnw1/img/fd8a1533-5b74-4c7d-9328-f03d46569155.png?Expires=1778767238&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=Xvlr6SC05sbU9lk1AeJfvLCTuUs%3D "")

## **AI报销**

AI生成单据草稿需要兼容这个控制，生成草稿后校验一下是否触发了规则，如果触发则过滤被限制的发票，抛出提示语“已限制不能上传\[用户上传的发票类型”，但剩余发票正常生成单据草稿。（如果全部被限制，也正常生成单据草稿，其中的发票组件都是空着的）
