# **1\. 需求背景**

目前的发票核销组件中只支持逐张选择发票进行上传，不支持一次选择多张发票进行上传。



# **2\. 业务流程/架构**

# **3\. 功能清单**

| 模块 | 功能点 | 备注 |
|------|---------|------|
| 发票核销组件 | 发票核销组件支持一次选择多票上传 |  |
|  |  |  |
|  |  |  |

# **4\. 需求详情**

发票核销组件合并开票、点击新增上传发票时，允许按住ctrl多选shift连选上传多张发票。

上传时依次处理识别、校验、计算核销金额（即轮询单张逻辑）。

如果上传的发票金额超过了未开票金额，也依然遵循目前线上逻辑，这批发票都关联到核销记录上。
