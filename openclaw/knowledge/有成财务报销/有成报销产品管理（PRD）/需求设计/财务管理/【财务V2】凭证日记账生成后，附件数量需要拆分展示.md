# **1\. 需求背景**

现状：

1、财务日记账会存在附件的情况只有两种：1️⃣手动添加的日记账可以自主上传附件 2️⃣通过有成报销同步过来的日记账，会根据报销的附件传过来展示

2、用户期望可以针对于对应日记账下的附件，分消费来统计附件数量，更清晰

# **2\. 业务流程/架构**

# **3\. 功能清单**

|------|---------|------|
| 流水账 | 日记账板块需要支持控制附件明细查看的开启或关闭 |  |
|  |  |  |

# **4\. 需求详情**

### **4.1日记账**

流水账--日记账板块，在更多功能中新增“开启附件明细”功能
1. 上线后初始化展示“开启附件明细”按钮，按钮常亮
2. 点击视为进行附件明细展示，按钮变更为“关闭附件明细”同样常亮

如下两种交互方式：【二选一】

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/oJGq75k7247xAlAK/img/4bf1c3b8-f23e-4bc0-8964-0ada06edd03c.png?Expires=1779974788&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=jrOylLOqGnMRjc5npuTHVvQHfLI%3D "")

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/oJGq75k7247xAlAK/img/38798985-54bb-43ee-b346-5989093aca73.png?Expires=1779974788&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=WKtlsdHtfPVusb5%2FWPPmLttU7%2F0%3D "")
3. 开启附件明细后，页面样式如下：
    1. 附件列表不再仅展示汇总附件数量
    2. 而是根据对应单据下的费用明细来拆分不同明细下的附件数量
    3. 如果存在相同的费用类型名称也无所谓，正常展示即可
    4. 按照费用类型名称：数量（张）的结构展示，有成展示附件查看图标，点击查看附件
4. 点击“关闭附件明细”，视为关闭附件明细展示，按钮重新变为“开启附件明细”



注：若是手动上传的附件不存在费用类型，则无需调整，按照现有的展示
