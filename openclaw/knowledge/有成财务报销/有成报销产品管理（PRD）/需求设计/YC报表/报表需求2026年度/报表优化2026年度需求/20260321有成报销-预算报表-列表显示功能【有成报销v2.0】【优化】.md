## **1\. 需求背景**

### **1.1 需求描述**

有成报销预算报表已增加【列表显示】功能，支持用户自定义控制表格字段列的显示或隐藏，现在剩余报表也需要【列表显示】功能。

### **1.2 需求背景**
- **需求方**：产品部
- **现状问题**：
• 查看有成报表时，无法配置关键字段显示或隐藏

• 报表字段列过多，无法根据使用场景灵活隐藏非必要字段
     • 字体大小和行高不可调整，阅读体验不佳
- **解决思路**：参照【预算报表】的【列表显示】功能，在预算报表中提供一致的列表显示能力

**业务流程**
```plain
用户进入报表页面
    ↓
点击表格右上角【列表显示】按钮
    ↓
弹窗配置字段显示/隐藏、字体大小、行高
    ↓
保存配置，即时生效
    ↓
报表按配置展示数据，导出时同步应用配置
```


## **2\. 功能现状**

**功能位置**：有成报表-企业报表/项目报表/流程效率报表/自定义报表

**现有操作**：
• 用户可查看报表数据表格
• 部分报表支持基础筛选条件
• 支持导出报表数据

**缺失能力**：
• 无法自定义控制表格字段列的显示或隐藏
• 无法调整字体大小和行高
• 导出数据时无法应用字段隐藏配置

### **3.2 详细功能描述**

#### **3.2.1 列表显示功能入口**

**功能概述** 在所有预算报表表格右上角增加【列表显示】按钮，点击弹出配置窗口，参照【单据管理-报销管理-表格显示】的交互设计。

**配置入口** 【有成报表-企业报表/项目报表/流程效率报表/自定义报表-表格右上角】

**配置界面**

![画布 12.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/Mp7ld7b7bWaK8OBQ/img/cfb558c4-a501-4540-89ea-2aee84e9c917.png?Expires=1774156289&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=EtyTZuo0dzpAMrq%2BusEV22M0WGA%3D "")![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/Mp7ld7b7bWaK8OBQ/img/b698808e-9659-46cf-ad05-7818a80c1c6d.png?Expires=1774156289&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=ZmEAdF0t3z9lw%2BELVXzXPw48lO4%3D "")

**默认配置**

| **字段** | **默认状态** | **说明** |
|----------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------|
| 报表内字段 | 勾选显示 | 默认显示 |
| 字体大小 | 中 | 标准字体大小 |
| 行高 | 标准 | 标准行高 |

**配置存储** • 配置按用户\+报表类型存储
• 参照【单据管理-报销管理-表格显示】已有存储方案
• 配置保存后即时生效，刷新页面后保持

#### **3.2.2 字段显示/隐藏控制**

**功能概述** 支持用户通过勾选框控制各字段列在报表表格中的显示或隐藏状态。

**功能逻辑**
```text
┌────────────────────────────────────────────────────────────┐
│ 字段显示控制逻辑                                            │
│                                                             │
│ 1. 弹窗中列出该报表所有可配置字段                            │
│ 2. 用户勾选/取消勾选字段前面的复选框                         │
│ 3. 点击【确定】保存配置                                      │
│ 4. 表格即时刷新，按配置显示/隐藏对应列                       │
│ 5. 隐藏字段的数据在导出时也不包含                            │
└────────────────────────────────────────────────────────────┘
```


**涉及报表范围**


| **报表类型** | **报表名称** | **字段配置** |
|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| 企业报表 | 费用分析报表 | 费用类型、部门、金额、占比等 |
| 项目报表 | 项目预算报表 | 项目、预算总额、占用金额、使用金额、余额等 |
| 流程效率报表 | 审批效率报表 | 单据类型、申请人、审批时长、状态等 |
| 自定义报表 | 用户自定义报表 | 根据用户配置的字段动态生成 |

#### **3.2.3 字体大小与行高设置**

**功能概述** 支持用户自定义表格字体大小和行高，改善报表阅读体验。

**配置选项**

| **设置项** | **选项** | **说明** |
|-------------------------------------------------------------|----------------------------------------------------------|----------------------------------------------------------|
| 字体大小 | 小/中/大 | 对应12px/14px/16px字体 |
| 行高 | 紧凑/标准/宽松 | 对应行高1.5/2.0/2.5倍 |

**效果应用** • 设置后立即生效，表格按配置重新渲染
• 导出Excel/PDF时同步应用字体和行高设置

#### **3.2.4 导出数据同步配置**

**功能概述** 【列表显示】中的字段隐藏设置将同步影响报表导出数据，隐藏字段在导出文件中不包含。

**处理逻辑**
```text
用户点击【导出】
    ↓
系统读取当前用户的【列表显示】配置
    ↓
过滤掉配置为隐藏的字段
    ↓
生成只包含显示字段的导出文件
    ↓
用户下载导出文件
```


**边界处理**

| **场景** | **处理方案** |
|----------------------------------------------------------|----------------------------------------------------------------|
| 用户未配置过列表显示 | 使用默认配置（全部字段显示）导出 |
| 用户隐藏了所有字段 | 提示"至少保留一个显示字段" |
| 导出时配置发生变化 | 以点击导出按钮时的配置为准 |

### **3.3 逻辑互斥与边界处理**

**边界情况处理**

| **边界场景** | **触发条件** | **处理方案** |
|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| 全部字段隐藏 | 用户取消勾选所有字段 | 提示"至少保留一个显示字段"，禁止保存 |
| 单用户多设备 | 用户在不同设备登录 | 配置按用户维度存储，多设备共享 |
| 报表字段升级 | 报表新增字段 | 新增字段默认显示，用户可手动隐藏 |
| 存量用户无配置 | 用户从未设置过列表显示 | 默认显示所有字段，与升级前体验一致 |

**与其他功能的兼容性**

| **相关功能** | **交互关系** | **处理方案** |
|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| 单据管理-表格显示 | 参照实现 | 参照【单据管理-报销管理-表格显示】的存储方案和交互设计 |
| 报表导出 | 数据过滤 | 导出时读取列表显示配置，过滤隐藏字段 |
| 报表筛选 | 独立 | 筛选条件与列表显示配置互不影响 |
| 自定义报表 | 动态字段 | 自定义报表的字段列表根据用户配置动态生成 |

**存量兼容策略**

| **场景** | **处理方案** |
|----------------------------------------------------------|----------------------------------------------------------------|
| 存量用户（无配置记录） | 默认显示所有字段，与升级前体验一致 |
| 升级后首次使用 | 首次点击【列表显示】时，展示默认配置（核心字段显示，辅助字段隐藏） |
| 报表字段变更 | 新增字段默认显示，删除字段自动从配置中移除 |


## **4\. 前后端工作任务**

### **4.1 前端负责内容**

• 【报表页面】表格右上角增加【列表显示】按钮
• 【列表设置弹窗】实现字段显示/隐藏勾选框、字体大小单选、行高单选
• 【表格渲染】根据配置动态显示/隐藏字段列，应用字体大小和行高样式
• 【字段排序】支持拖拽调整字段列显示顺序
• 【导出功能】导出时根据列表显示配置过滤隐藏字段

### **4.2 后端负责内容**

• 【接口】GET /api/report/list-display/config - 获取用户列表显示配置
• 【接口】POST /api/report/list-display/config - 保存用户列表显示配置
• 【接口】GET /api/report/list-display/fields - 获取报表可配置字段列表
• 【数据表】report\_list\_display\_config - 用户列表显示配置表
- user\_id：用户ID
- report\_type：报表类型（企业报表/项目报表/流程效率报表/自定义报表）
- report\_code：报表编码
- config\_json：配置JSON（显示字段、字体大小、行高）
• 【导出逻辑】导出接口根据列表显示配置过滤字段
• 【存量数据】无配置记录时，默认返回全字段显示配置

## **附录**

**A. 典型使用场景**

**场景：北京智源人工智能研究院财务查看收入预算报表**

配置步骤（首次）：
1. 财务管理员进入【有成报表-企业报表-预算执行报表】
2. 点击表格右上角【列表显示】按钮
3. 在弹窗中取消勾选"创建人"、"创建时间"、"备注"等非核心字段
4. 保留勾选"预算总额"、"占用金额"、"使用金额"、"余额"等核心字段
5. 设置字体大小为"中"，行高为"标准"
6. 点击【确定】保存配置

使用步骤（日常）：
1. 财务管理员每次进入预算执行报表，表格只显示核心字段
2. 点击【导出】按钮，下载的Excel只包含显示的核心字段
3. 如需查看完整字段，点击【列表显示】恢复显示或调整配置

结果：报表简洁清晰，聚焦于核心数据，导出文件无需二次处理。
