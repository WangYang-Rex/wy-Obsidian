# 项目列表、项目报表增加时间筛选选项“项目期限起”“项目期限止”

| **修订时间** | **版本** | **修订说明** |
|----------------|----------|----------------|
| 20250617 | v1.0 | 新建 |
|  |  |  |

## 需求背景

历史需求：YCCW-25362 项目报表支持按项目创建时间筛选

客户希望能够筛选出例如某一年度开始的项目，目前是根据项目的创建时间来的。在刚启用系统的时候会录入历史的项目，导致没办法筛选出历史年度开始的项目。

上海良安

## 业务流程/架构

## 功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 项目管理-项目列表 | 增加时间类型筛选条件 |  |
| 报表中心-项目报表 | 增加时间类型筛选条件 |  |
|  |  |  |

## 需求详情

1、项目管理-项目列表在时间筛选的类型中增加两个选项【项目期限起】【项目期限止】。
- 选中【项目期限起】，则筛选项目的“项目期限”组件日期类型为“起止日期”的起时间包含在当前时间范围内的项目（闭区间）。
- 选中【项目期限止】，则筛选项目的“项目期限”组件日期类型为“起止日期”的止时间、**以及日期类型为“日期”的时间，**包含在当前时间范围内的项目（闭区间）。

2、报表中心-项目报表，也增加【项目期限起】【项目期限止】这两个时间筛选选项，逻辑同上。查询出项目后以项目纬度去筛选，数据统计则显示查询项目所关联的所有数据
