## 1、需求背景

目前客户自定义报表的需求众多



![Picture 1](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jz5N8AZlvD/img/5520cc9c-9257-4d77-8fd5-d2700ae6f189.png?Expires=1775016622&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=lCGEw1Jh6wMnZDP3qhE0K7ojDBE%3D "")

需求重点围绕在：

1、想要展示、导出字段缺失

2、不能按照多维度统计（项目\+部门\+费用）

3、自定义计算逻辑不足（统计利润率、占比等情况）

4、表尾汇总统计不足

5、筛选条件、字段排序不足

## 2、业务流程/架构



![Picture 2](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jz5N8AZlvD/img/435b4926-e4f6-4d1b-a848-64b3d4dbbb65.png?Expires=1775016622&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=GStG8FjtD5AxURlw2Ou0gkOksA4%3D "")

| **模块** | **功能点** | **解决问题** |
|----------|-------------|----------------|
| 数据源表 | 数据源表 | 列出众多数据表，后续可不断添加 |
| 获取数据表 | 过滤条件 | 解决不同客户对项目、单据等数据在某些状态下不统计的问题 |
| 组合数据表 | 联表组合 | 解决某些表中字段缺失的问题 |
|  | 数据透视 | 解决多维度统计的问题 |
|  | 自定义字段 | 解决计算占比、利润率等问题 |
| 图表渲染 | 图表类型 | 解决想要展示不同样式的展示 |
