# 辅助核算选项内容展示支持配置

| **修订时间** | **版本** | **修订说明** |
|----------------|----------|----------------|
| 20250626 | v1.0 | 新建 |
|  |  |  |

## 需求背景

用户希望在搜索产品辅助核算时，展示例如规格型号、描述之类的自定义字段，方便选择。

目前系统上基础设置-全局设置中可以控制展示辅助核算的编码还是名称。

重庆聚宝盆。

### 基础档案，辅助核算类别编辑增加“选项内容展示设置”



![Picture 3](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/8K4nyeZXWkeVenLb/img/9fecefe0-641b-4200-926e-a6e14bc1fde7.png?Expires=1778767417&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=Xz1vlCrmsSSj7IDZQTUIt8gbzqs%3D "")
1. 在辅助核算大类上编辑，可见“选项内容展示设置”选项，单选必选，可选：套用全局设置、自定义。历史数据、新用户默认“套用全局设置”。
2. 当选择自定义时，展示公式输入框，用户点击弹出规则编辑器弹框（复用自定义单据-公式计算的规则编辑器”。可选字段为**所有单据字段**，确定后回填至公式输入框。其中需要特别说明的有：日期：开始时间、结束时间，选择单一时间即结束时间。月份范围：开始月份、结束月份。收款信息：收款银行、开户行、收款账号、收款人、备注发票抬头：抬头、税号、开户行、银行账号、固定电话、地址
3. 不支持运算符，前端直接隐藏。
4. 仅支持文本函数，前端直接隐藏其他函数类型。
5. 若用户此处设置了选项内容展示设置自定义，则根据配置的公式展示选择辅助核算控件中的选项内容。生效范围见下。
6. 前端控制下数据最长不可超过 100 字符，超出截断加...

### 新增/编辑页面辅助核算选择控件
7. 主表辅助核算控件、子表辅助核算控件都生效
8. ~在搜索时，展示出来的选项内容都可以被搜索，精确搜索/模糊搜索遵从全局设置。~在搜索时依然还是搜索编码\+名称。
