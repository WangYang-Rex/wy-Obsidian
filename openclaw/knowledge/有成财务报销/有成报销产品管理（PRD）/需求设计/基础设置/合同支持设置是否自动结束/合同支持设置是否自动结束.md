# 合同支持设置是否自动结束

| **修订时间** | **版本** | **修订说明** | **作者** |
|----------------|----------|----------------|----------|
| 20221019 | v1.0 | 新建 | 昭觉 |

## 1、涉及系统

有成报销 PC

## 2、需求背景

目前有成报销侧的合同在支付/付款计划执行完成后就会自动结束，但是会存在用户需要在合同

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 基础设置--全局设置 | 全剧设置增加合同是否自动结束开关 |  |
|  |  |  |
|  |  |  |

## 4、需求详情

### 4.1、需求说明

1、基础设置--全局设置：增加合同是否自动结束的开关



![Picture 1](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/WgZOZA8Qjvd2LqLX/img/b51e4664-c729-4507-ace5-20dd2c627ad8.png?Expires=1778767196&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=dxD3udzc2Bwbc9N8f2klyE7uOOg%3D "")

2、开关默认开启，开启时则保持当前合同结束的逻辑：支付/收款计划完成后自动结束

3、开关选择关闭，则当支付/收款计划完成时，合同不结束，可以手动结束

4、开关选择立即生效
