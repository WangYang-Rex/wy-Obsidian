## 1、需求背景

客户：浙江省财务开发有限责任公司

背景：客户会批量导入员工收款账户，不希望员工添加非工资卡以外的个人收款账户

## 2、业务流程/架构

## 3、功能清单

根据个人需要补充

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 收款账户 | 支持“新增个人账户/对公账户/现金账户/承兑汇票“权限设置 |  |
|  |  |  |
|  |  |  |

## 4、需求详情

### 4.1、需求说明1

**系统：**【有成报销】

1、在收款账户-设置中，增加“个人账户/对公账户/现金账户/承兑汇票”权限设置
- 勾选“允许员工创建账户信息”，显示“个人账户/对公账户/现金账户/承兑汇票”供勾选；默认：已勾选

2、4个账户类型配置项控制的是：1）创建单据；2）个人中心——创建收款账户页面的 账户类型
- 未勾选时，禁止创建此类型的账户（置灰，不可选状态）
