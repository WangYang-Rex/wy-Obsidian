# 委托提单的单据，支持仅展示在提单人（被委托人）单据列表

| **修订时间** | **版本** | **编写人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 20231130 | v1.0 | 小乔 | 新建 |

## 1、需求背景

客户：财开

诉求：被委托人代替委托人提交的单据，希望只展示在被委托人（提单人）单据列表里，不要展示在委托人单据列表里

## 2、业务流程/架构

如复杂需求，放置相关流程/架构图

## 3、涉及系统

**有成报销**

## 4、功能清单

根据个人需要补充

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 系统设置-委托授权 | 增加“委托设置” |  |
| 提单数据同步 | 关闭 委托提单数据同步时，被委托人代委托人提交的单据 不同步到委托人单据列表 |  |
|  |  |  |

## 5、需求详情
1. **系统设置-委托授权页面，增加“委托设置”按钮，可以进行委托相关的规则设置（个人头像-委托授权不加）**设置内容包括：从全局设置移过来的“委托授权权限”设置（详见2）、委托提单数据同步开关委托提单数据同步：默认“开启”（原逻辑）；关闭时，被委托人代替委托人提交的单据  不同步到委托人单据列表“委托提单数据同步”注释文案：开启时，被委托人代替委托人提交的单据，将同步到委托人单据列表中；关闭则取消同步
