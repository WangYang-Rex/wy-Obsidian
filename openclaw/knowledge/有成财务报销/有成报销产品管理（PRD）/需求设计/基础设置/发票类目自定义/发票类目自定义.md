# 发票类目自定义

| **修订时间** | **版本** | **修订说明** |
|----------------|----------|----------------|
| 20230915 | v1.0 | 新建 |

## 1、需求背景

目前系统内预置了一套默认的发票类目，用户自定义了发票类目，没有存储下来；目前税率税额组件和会计科目映射中都会使用到发票类目数据，用户自定义的发票类目无法在这两个功能上使用到；

本次需求：发票类目支持新增/删除发票类目数据，并存储下来可编辑

## 2、业务流程/架构

如复杂需求，放置相关流程/架构图

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 发票类目 | 发票类目支持查看、新增、删除 |  |
|  |  |  |

## 4、需求详情

### 4.1、需求说明

一、全局设置-费用设置中，新增“发票类目”设置入口，点击新开TAB页，打开发票类目查询信息页面；

显示当前所有的发票类目数据，并支持新增修改删除；



![Picture 1](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/oJGq75kamoXgPlAK/img/2741d310-7383-44c9-ac85-a478f088b766.png?Expires=1775017293&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=KFdErhfXBeAP7Z%2FcXX%2BSu2EqB1g%3D "")



![Picture 2](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/oJGq75kamoXgPlAK/img/7d997fac-8d3b-41f7-9d27-3c8809cf04f7.png?Expires=1775017293&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=MuxbG2R1ElcFBtKsw9e95EMDha4%3D "")

一、搜索：支持按照发票类目名称模糊搜索

二、序号：系统自增的序号

三、发票类目：每个发票类目都展示一条记录，需要进行分页处理

四、新建：点击新建，弹出“自定义发票类目”弹框，对应的原来的“自定义关键字”弹框，弹框头部文案更改为新的，输入逻辑保存以前

**本次新增：点击确定后，将用户输入的数据存储下来，后续税率税额组件和会计科目映射中都可以选用到用户新增的值**



![Picture 3](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/oJGq75kamoXgPlAK/img/bf5a2242-5344-474d-9781-3539e5917ee2.png?Expires=1775017293&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=xNjjnofwyeIfUVz1OuuKWvVh3jg%3D "")

原来的“全局设置-费用设置”中编辑功能中的文案也需要进行相应修改：自定义关键字更改为“自定义发票类目”，点开之后的弹框顶部文案也改成“自定义发票类目”；



![Picture 4](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/oJGq75kamoXgPlAK/img/a6f47729-c452-41e5-ad66-85a7d530281e.png?Expires=1775017293&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=ccJ5VbpGErV7vhQDOcoiRDhEI8o%3D "")

五、编辑：点击弹框显示文本输入框编辑，显示当前发票类目的名称，支持输入单条文案做修改；

同条数据修改之后，对其他使用的地方都生效，显示新的名称，已将提交的历史单据不受影响：全局设置-费用设置中的类目名称、税率税额组件中的配置以及选项值、会计科目映射中的选项值；

六、删除
- 单条删除：删除需要二次确认



![Picture 5](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/oJGq75kamoXgPlAK/img/1d79355a-72ce-4544-ae80-4598c21aa6aa.png?Expires=1775017293&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=6QVlEaf5wwqxLRRKN29dnT3k6CQ%3D "")
- 批量删除：点击给二次确认弹框：已选 X条发票类目，删除后不可恢复，确定要删除吗？

点击确定即进行删除操作，取消则不删除；



![Picture 6](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/oJGq75kamoXgPlAK/img/13978fc1-2e0f-4eba-890b-2b65cf761886.png?Expires=1775017293&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=S8KihFDT%2B4QYH4oHdoIWJVxOFVQ%3D "")

七：数据存储涉及到的地方：

1、在基础设置-全局设置-**费用设置**中，编辑发票类目与费用类型对应的关系时，新增保存下来的关键字需要存储下来；

2、在本次新增的**发票类目菜单**页面，新增的类目信息也需要保存下来；

3、**自定义单据和自定义费用中，税率税额组件中，发票类目文案右侧的小ICON，点击直接打开本次新增的发票类目页面；**

**4、上线前的历史数据处理：历史用户自定义的关键字也存储下来，可以在列表和发票类目页面中查询到；**

5、一个企业内的发票类目采用同一套数据，系统内使用到的地方：全局设置-费用设置、发票类目、税率税额组件、会计科目映射中。
