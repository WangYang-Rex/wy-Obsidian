# 字段隐藏处理逻辑

| **修订时间** | **版本** | **修订说明** |
|----------------|----------|----------------|
| 20240606 | v1.0 | 新建 |

## 1、需求背景

目前系统有很多功能可以将单据、费用字段进行隐藏，但是有的隐藏会将字段值清空，有的隐藏会将字段值保留，对用户来讲没有什么感知。

如果不清空，会在用户不知情下，影响审批流、报表、预算占用、会计科目映射等

如果清空，部分场景也会满足不了

函数中的隐藏功能不能清空，否则无法计算。

本次文档，致力于将所有隐藏的场景了解清楚，然后针对性提出问题。

## 2、竞品情况

1、易快报，函数隐藏不清空，审批流可用，清空的配置没有找到

2、汇联易，联动隐藏可以设置是否清空值，在提交单据时清空

## 3、现状

| **功能** | **应用场景** | **现状** | **出现问题频率** | **分析** |
|----------|----------------|----------|----------------------|----------|
| 选项关联字段 | 单据、费用 | 清空 | 不清空前多，现在基本没有 |  |
| 字段勾选隐藏、函数计算隐藏 | 单据、费用 | 不清空 | 还未上线，以前隐藏不参与计算，函数可以解决 | 只能不清空，否则无法计算 |
| 审批流配置字段隐藏 | 单据、费用 | 不清空 | 无 | 审批流主要是要修改，不可清空 |
| 收款场景隐藏开票金额 | 单据 | 不清空 | 中 | 这个要清空，否则影响大 |
| 费用字段「适用单据」根据单据隐藏 | 费用 | 不清空 | 高 | 直觉来讲是清空，配置的目的就是某些单据下不需要这个字段，应该不会出现审批时再需要，否则和审批流隐藏、函数重复。 |

主要影响点：

1、如果配置了辅助核算、项目可以隐藏，但是不清空又有默认值时，报表、预算、审批流、都会取值

## 4、解决思路

1、可以给字段增加是否需要清除隐藏值的配置，触发清空有两种方式

1）字段触发隐藏后立即清空吗，计算时如果有公式引用这个值计算，按清空的值计算

2）另外一个是提单后，统一检查隐藏字段然后清空：计算时，按照原值计算，计算完后再清空，可能不好追溯

2、在审批流、报表、预算等地方设置，隐藏值是否获取。（改动大，不建议）

## 5、需求内容

| **功能** | **应用场景** | **方案** |  |
|----------|----------------|----------|---|
| 选项关联字段 | 单据、费用 | 下架 |  |
| 字段勾选隐藏、函数计算隐藏 | 单据、费用 | 增加是否清空的配置 |  |
| 审批流配置字段隐藏 | 单据、费用 | 不清空 |  |
| 收款场景隐藏开票金额 | 单据 | 清空 |  |
| 费用字段「适用单据」根据单据隐藏 | 费用 | 清空 |  |

1、隐藏增加配置「隐藏后清空数据」并hover注释：

通过勾选隐藏、公式计算隐藏后字段值将会被清空

2、提单时，如果字段隐藏了，则会将值清空。如果其他字段的公式引用了这个字段值计算则会按照空值计算
