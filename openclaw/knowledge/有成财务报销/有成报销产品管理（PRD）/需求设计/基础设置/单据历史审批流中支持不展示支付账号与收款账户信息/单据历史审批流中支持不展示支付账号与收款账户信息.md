## 1、需求背景

客户：裕太微电子股份有限公司

需求：客户希望在单据历史审批流中 出纳节点不展示支付账号与收款账号信息

## 4、功能清单

根据个人需要补充

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 基础设置 | 全局设置-展示设置，增加设置项“审批历史展示支付与收款信息” |  |
| 单据详情 | 审批历史中不展示“收款信息”&“支付信息” |  |
|  |  |  |

## 5、需求详情
1. 基础设置-全局设置-展示设置：增加“**审批历史展示支付与收款信息**”设置项
2. 默认选项：是（原逻辑）


![Picture 3](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/a2QnV4j0bQLxyO4X/img/079289fe-2f54-4857-b94f-c504385d5d85.png?Expires=1774157230&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=xVDvT%2F%2Ftde7r0kmEv%2B8V9KdrCyc%3D "")
3. 选择“**否**”时，单据详情-历史审批流中出纳节点隐藏“**收款信息**”与“**支付信息**”的展示

注意：手机端也要兼容
