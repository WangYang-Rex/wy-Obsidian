# 支持用户发布报销制度通知

| **修订时间** | **版本** | **编写人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 20230216 | v1.0 | 小乔 | 新建 |

## 1、需求背景

财务定期会制定/更改报销相关的制度，希望能在系统内部发布全员的通知，并在首页展示

通知内容要求：文本、链接、附件

## 2、业务流程/架构

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 基础设置 | 增加「内部通知」二级菜单；支持增删改内部通知 |  |
| 首页 | PC-首页增加“通知”展示块，首页内容设置增加“通知”选项；MB首页增加【内部通知】展示点击-支持跳转消息中心 |  |
| 角色权限 | 功能授权：增加“内部通知”选项 |  |

## 4、需求详情

### 4.1、需求说明1

**系统：**【有成报销】

**一、基础设置-增加「内部通知」二级菜单**

**1、二级菜单「内部通知」列表页面**
- 列表支持按发布时间、标题/关键字搜索
- 内容较多时，分页展示
- 列表展示字段【标题】【发布状态】【发布时间】【操作】发布状态：已发布、未发布；发布状态支持点击筛选，筛选项：全部状态/未发布/已发布发布时间：定时发布时，取定时发布时间；即时发布时，取发布完成时间已发布状态的通知，支持“复制编辑”，点击“复制编辑”创建一条新的通知，并带入已发布的通知内容

**二、首页**

**1、PC：增加“通知”展示块，并支持内容是否展示设置**
- 报表块上方增加“通知”展示
- 未读通知，标记红点
- 首页每条通知展示【通知标题】【发布时间】，标题过长时，不换行展示，用“...”展示
- 无消息时，展示缺省图，提示：暂无通知
- 展示顺序：按时间倒序
- 默认展示最近5条消息，支持进度条展示



![Picture 5](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/WgZOZA8QjvLLLqLX/img/2042625b-5e6c-4d8b-8736-03b1962b435b.png?Expires=1775017344&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=R7ceE%2FszwPfmWCygzIMtqDvSzuw%3D "")
- 点击“全部”跳转消息中心消息中心增加「内部通知」tab通知列表中，未读通知标记红点支持发布时段、标题/关键字搜索状态选项：全部状态、已读、未读支持时间倒序/时间正序，默认时间倒序



![Picture 6](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/WgZOZA8QjvLLLqLX/img/38463dfb-0a4c-406b-bcbf-63a5d1f96404.png?Expires=1775017344&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=PoUqgFaH1Xsb3vCN%2Bm%2BbYhkmduY%3D "")
- 点击通知，弹窗查看消息详情附件支持预览/下载



![Picture 7](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/WgZOZA8QjvLLLqLX/img/132d200c-040d-4df4-9ada-4c60ff30aaaf.png?Expires=1775017344&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=q7jfhO9CplhWcpM0NLw8YCxLvAs%3D "")
- 首页内容设置中增加“通知”选项，支持设置是否在首页展示。默认：勾选



![Picture 8](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/WgZOZA8QjvLLLqLX/img/4407be40-3e4b-40e2-9944-771015c94866.png?Expires=1775017344&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=BFuNgMk0GLmVEb%2F19j6aBJPNMVE%3D "")

**2、MB-【内部通知】展示**
- 根据发布时间，在首页展示一条【内部通知】，未读用红点标记
- 点击通知，跳转消息中心-内部通知
- 点击具体通知，可查看通知详情
