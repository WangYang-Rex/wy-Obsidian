# **1\. 需求背景**

三方商旅平台通过月结方式结算费用时，员工不需要上传发票。

剩下员工自行垫付的消费，如果员工需要报销的话，则需要上传发票。

目前我们通过基础设置-自定义费用-发票控件上规则编辑器上可以控制必填条件，但却少了费用是否月结的条件。

客户：浙江尚驰（泛嘉）、赛旺（携程）

# **3\. 功能清单**

|  |  |  |

# **4\. 需求详情**
1. 基础设置-自定义费用，规则编辑器增加费用主体字段-是否月结（布尔）。

如果当前费用为三方生成的月结费用，则为真，否则为假。

自定义单据暂不需要加。
