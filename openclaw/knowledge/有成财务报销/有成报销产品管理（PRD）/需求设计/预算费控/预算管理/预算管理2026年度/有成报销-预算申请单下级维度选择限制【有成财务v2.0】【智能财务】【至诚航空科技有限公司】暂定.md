## **1\. 需求背景**

### **1.1 需求描述**

在预算申请单选择预算时，添加下级维度会展示所有可选维度（按提交人、按部门、按辅助核算、按项目、按成本中心）。需要支持在预算管理的基础设置中配置可选择的维度范围，实现只展示指定维度的效果。

### **1.2 需求背景**
- **需求方**：财务管理人员
- **现状问题**：
• 预算申请单添加下级维度时默认展示全部维度选项
• 部分企业只需使用特定维度（如仅使用"辅助核算-项目"）
• 展示多余维度会增加用户选择困扰，降低操作效率
- **解决思路**：在预算管理基础设置中增加"预算申请单下级维度配置"，支持勾选允许选择的维度类型

**业务流程**
```plain
财务管理员进入预算管理基础设置
↓
配置预算申请单可选的下级维度（如仅勾选"辅助核算"和"项目"）
↓
员工提交预算申请单选择预算时
↓
点击"+下级维度"仅展示配置的维度选项
```

## **2\. 功能现状**

**功能位置**：预算费控 → 预算管理 → 基础设置 → 预算权限

**现有配置项**：
- 预算数据权限
- 允许员工看到预算金额
- 允许审批人看到单据详情中的预算信息
- 预算信息展示设置

**说明**：允许修改历史预算、预算仅可编辑最末级维度金额与本次需求无关

**缺失能力**：
- 无预算申请单下级维度选择范围配置
- 添加下级维度时固定展示5个选项：按提交人、按部门、按辅助核算、按项目、按成本中心

**下级维度选项现状**：

| **选项** | **说明** |
|----------------------------------------------------------|----------------------------------------------------------|
| 按提交人 | 按申请人维度细分预算 |
| 按部门 | 按申请部门维度细分预算 |
| 按辅助核算 | 按辅助核算维度细分预算 |
| 按项目 | 按项目维度细分预算 |

### **3.2 详细功能描述**

#### **3.2.1 预算申请单下级维度配置**

**位置**：预算费控 → 预算管理 → 基础设置

**界面原型**
```plain
预算申请单维度配置
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
□ 启用维度限制
    
    启用后，预算申请单添加下级维度时仅展示勾选的维度选项
    
    ☑ 按提交人
    ☑ 按部门
    ☐ 按辅助核算
    ☑ 按项目
    ☐ 按成本中心
    
    [保存配置]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```



**配置项说明**

| **配置项** | **类型** | **默认值** | **说明** |
|-------------------------------------------------------------|----------------------------------------------------------|-------------------------------------------------------------|----------------------------------------------------------|
| 启用维度限制 | 开关 | 关闭 | 控制是否启用维度限制功能 |
| 按提交人 | 复选框 | 勾选 | 启用限制时可选，控制是否展示该维度 |
| 按部门 | 复选框 | 勾选 | 启用限制时可选，控制是否展示该维度 |
| 按辅助核算 | 复选框 | 勾选 | 启用限制时可选，控制是否展示该维度 |
| 按项目 | 复选框 | 勾选 | 启用限制时可选，控制是否展示该维度 |
| 按成本中心 | 复选框 | 勾选 | 启用限制时可选，控制是否展示该维度 |

**交互规则** • 开关关闭时：下方维度复选框禁用，预算申请单展示全部维度选项（保持现有逻辑）
• 开关开启时：下方维度复选框启用，仅勾选至少一个维度才能保存
• 首次开启时：默认全部勾选，用户按需取消
• 至少保留一个维度被勾选，全部取消时给出提示："请至少选择一个维度"

**配置存储** • 配置按企业维度存储（全局配置）
• 配置保存后即时生效
• 配置权限：财务管理员可操作

#### **3.2.2 预算申请单维度选择弹窗**

**位置**：预算申请单 → 预算申请组件 → 选择预算 → \+下级维度

**功能说明**

根据预算管理基础设置中的配置，过滤"\+下级维度"下拉菜单中展示的选项。

**展示规则**

| **配置状态** | **维度选项展示** |
|----------------------------------------------------------------|----------------------------------------------------------------------|
| 维度限制开关关闭 | 展示全部5个选项：按提交人、按部门、按辅助核算、按项目、按成本中心 |
| 维度限制开关开启 | 仅展示配置中勾选的维度选项 |

**示例场景**

**场景A**：配置仅启用"按项目"
```plain
+下级维度
    └─ 按项目
```



**场景B**：配置启用"按辅助核算"和"按项目"
```plain
+下级维度
    ├─ 按辅助核算
    │       └─ 弹出辅助核算选择框
    └─ 按项目
            └─ 弹出项目选择框
```



**场景C**：配置启用全部维度（与现状一致）
```plain
+下级维度
    ├─ 按提交人
    ├─ 按部门
    ├─ 按辅助核算
    ├─ 按项目
    └─ 按成本中心
```



**二级维度选择**

当选择"按辅助核算"时，需进一步选择具体的辅助核算类别（如"项目"、"客户"等）：
• 弹出辅助核算类别选择框
• 列出企业启用的所有辅助核算类别
• 选择后生成"辅助核算-\{类别名称\}"的维度条件

### **3.3 逻辑互斥与边界处理**

**边界情况处理**

| **边界场景** | **触发条件** | **处理方案** |
|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| 配置全部取消 | 用户取消所有维度勾选后尝试保存 | 保存时校验，提示"请至少选择一个维度"，阻止保存 |
| 配置开关关闭 | 维度限制开关处于关闭状态 | 预算申请单展示全部维度，忽略下方复选框状态 |
| 存量申请单 | 配置修改时已存在的预算申请单 | 不影响存量单据，仅影响新创建/编辑的申请单 |
| 无可用维度 | 配置开启但企业未启用任何辅助核算 | 按辅助核算选项隐藏或置灰，提示"未启用辅助核算" |
| 辅助核算类别为空 | 选择按辅助核算但企业无辅助核算类别 | 弹出提示"未配置辅助核算类别"，无法选择该维度 |

**维度层级互斥规则（现有业务规则）**

预算申请单新增下级维度时，存在以下层级互斥限制（需保持兼容）：

| **维度类型** | **层级添加限制** | **说明** |
|----------------------------------------------------------------|----------------------------------------------------------------------|----------------------------------------------------------|
| 辅助核算 | 不限制 | 可在同一链路的多个层级添加不同辅助核算类别 |
| 按提交人 | 仅一次 | 在某一链路上仅能在某个层级添加一次，其他层级不允许添加 |
| 按部门 | 仅一次 | 在某一链路上仅能在某个层级添加一次，其他层级不允许添加 |
| 按项目 | 仅一次 | 在某一链路上仅能在某个层级添加一次，其他层级不允许添加 |
| 按成本中心 | 仅一次 | 在某一链路上仅能在某个层级添加一次，其他层级不允许添加 |

**示例说明**：
• 若预算链路已包含"按部门"维度，则后续层级不能再添加"按部门"（包括不同部门值）
• 若预算链路已包含"辅助核算-项目"，仍可继续添加"辅助核算-客户"或其他维度

**与其他功能的兼容性**

| **相关功能** | **交互关系** | **处理方案** |
|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| 预算报表维度筛选 | 独立功能 | 预算报表的维度筛选不受此配置影响，保持原有逻辑 |
| 预算编制维度选择 | 独立功能 | 预算编制页面的维度选择不受此配置影响 |
| 动态预算维度 | 关联功能 | 动态预算的维度选择同样受此配置控制（可选扩展） |
| 预算数据权限 | 关联功能 | 本配置控制可选维度范围，数据权限控制可操作的预算维度，两者叠加生效 |

## **附录**

**A. 典型使用场景**

场景：某企业仅使用"项目"维度管理预算

配置步骤：
1. 财务管理员进入预算费控 → 预算管理 → 基础设置
2. 找到"预算申请单维度配置"模块
3. 开启"启用维度限制"开关
4. 取消勾选"按提交人"、"按部门"、"按辅助核算"、"按成本中心"
5. 仅保留勾选"按项目"
6. 点击保存

使用步骤：
1. 员工进入申请单 → 新建预算申请单
2. 填写基本信息后点击"选择预算"
3. 选中预算后点击"\+下级维度"
4. 下拉菜单仅显示"按项目"一个选项
5. 选择后弹出项目选择框，选择具体项目

结果：员工只能按项目维度申请预算，避免了其他维度的干扰
