# **1\. 需求背景**

## **1.1 目标用户**

财务管理员、预算管理员

## **1.2 需求来源**

东艾科尖端薄膜（南通）有限公司客户

## **1.3 背景**

### **1.3.1 现有功能说明**

有成报销目前支持自定义费用模版中加入税率税额组件，有税率税额组件的单据业务逻辑如下：
```plain
有成单据
    └── 费用明细组件
            └── 添加费用类型（支持自定义），含税率税额组件
                    └── 上传发票
                            ├── 识别发票信息
                            ├── 计算发票金额
                            └── 计算税额
```


```plain
有成单据
    └── 含税率税额组件     
       └── 上传发票
       ├── 识别发票信息
       ├── 计算发票金额
       └── 计算税额
```



### **1.3.2 预算管理现状**

预算管理中已支持"税额不占用预算"功能，即某些类型的发票税额可以不占用预算额度。当前配置方式为：
- 在预算管理-基础设置中配置"不占预算发票"类型
- 支持按发票属性（如增值税专用发票、普通发票等），发票类别进行配置
- 配置后，符合条件的报销单/付款单中的发票税额不计入预算占用

### **1.3.3 客户业务场景**

东艾科公司的业务特点：
1. **多费用类型管理**：公司内部定义了多种费用类型（如差旅费、招待费、办公费、材料费等）
2. **差异化预算管控需求**：不同类型的费用，其发票税额是否占用预算有不同的管理要求
    - 差旅费、招待费：税额需要占用预算
    - 材料费、设备采购费：税额不占用预算
3. **自定义费用类型**：有成报销支持自定义费用类型
4. **特殊业务场景：**设置了增值税发票税额不占用预算配置，但涉及进项税转出的费用类型，即使上传了增值税发票，税额需要占用预算

## **1.4 核心痛点**

### **痛点：无法按发票类别与费用类型维度组合配置税额不占用预算**

**现状**：现有功能仅支持按发票类型配置，不支持按"费用类型"维度组合

**影响**：无法满足"涉及进项税转出的费用类型下的增值税发票税额占用预算，其余费用类型喜爱的增值税发票税额不占用预算"的精细化管理需求

## **2.2 单据内费用类型预算占用计算流程**

**现有逻辑：**
- **单据内有消费明细组件且消费明细内配置了税率税额组件，优先从此处取发票税额；**
- **单据内仅有税率税额组件，从此处取发票税额；-当前仅付款单类型单据可在单据内配置税率税额组件**
- **单据内无消费明细也无税率税额组件，取不到税额**
```plain
单据提交时
    ↓
遍历费用明细组件中的每一行
    ↓
对于每个费用明细：
    1. 获取费用类型
    2. 获取发票金额、税额
    3. 查询税额不占用预算配置
       IF 费用类型在配置列表中 THEN
           预算占用金额 = 消费金额 - 税额（不含税金额）
       ELSE
           预算占用金额 = 消费金额（含税金额）
       END IF
    4. 累计该明细的预算占用金额
    ↓
汇总所有明细的预算占用金额
```



前置条件：
- 付款单据和借款单据内允许直接配置税率税额组件，且同时费用类型和金额组件
```plain
单据提交时
    ↓
找到单据内税率税额组件，费用类型组件和金额组件
    ↓
    1. 获取费用类型
    2. 获取单据金额、税额
    3. 查询税额不占用预算配置
       IF 费用类型在配置列表中 THEN
           预算占用金额 = 单据金额 - 税额（不含税金额）
       ELSE
           预算占用金额 = 单据金额（含税金额）
       END IF
```

## **4.2 核心功能**

### **4.2.1 配置维度选择**

**配置方式说明**：



| **配置方式** | **说明** | **适用场景** |
|----------------------------------------------------------------|----------------------------------------------------------|----------------------------------------------------------------|
| 按单据\+发票 | 同时满足不占预算单据和不占预算费用类型 | 灵活组合 |
| 按单据\+费用类型 | 同时满足不占预算单据和不占预算费用类型 |  |
| 按发票\+费用类型 | 同时满足不占预算发票和不占预算费用类型 |  |
| 按单据\+费用类型\+发票 | 同时满足不占预算发票，不占预算费用类型和不占预算单据 |  |

### **4.2.2 费用类型多选组件**

**组件功能说明**：



| **功能** | **说明** |
|----------------------------------------------------------|----------------------------------------------------------|
| 树形展示 | 费用类型按分类层级展示 |
| 父子联动 | 勾选父节点自动全选子节点；子节点部分选中时父节点显示半选状态 |
| 搜索筛选 | 支持按费用类型名称 |
| 已选展示 | 选择框内展示已选费用类型，支持单个移除或一键清空 |

### **4.2.3 预算占用计算逻辑**

**前置条件：单据中消费明细组件包含税率税额组件**
```plain
计算公式：
IF 费用类型在税额不占用配置列表中 THEN
    预算占用金额 = 消费金额 - 税额（不含税金额）
ELSE
    预算占用金额 = 消费金额（含税额）
END IF
```

**前置条件：单据中仅有税率税额组件，费类型组件和金额组件**
```plain
计算公式：
IF 费用类型在税额不占用配置列表中 THEN
    预算占用金额 = 单据金额 - 税额（不含税金额）
ELSE
    预算占用金额 = 单据金额（含税额）
END IF
```





## **4.3 默认策略**



| **场景** | **默认行为** |
|----------------------------------------------------------|----------------------------------------------------------------|
| 新开通企业 | 默认为不开启“税额不占用预算”功能 |
| 配置页面初始状态 | 默认“不占用预算费用类型”为空 |
| 费用类型多选组件 | 交互显示按照已有选择费用类型组件交互 |

## **4.4 存量兼容**



| **场景** | **处理方案** |
|----------------------------------------------------------|----------------------------------------------------------------|
| 存量客户1-税额不占用预算配置已开启，且“不占预算发票”为空，“不占预算单据”必有配置项 | 保持原有配置不变基础上，“不占预算费用类型”置空 |
| 存量客户3-税额不占用预算配置未开启， | “不占预算费用类型”规则不生效 |
| 升级后新增配置 |  开启“税额不占用预算”，显示“不占预算费用类型”配置 |
| 审批中的单据 | 按提交时的配置计算预算占用，不受配置变更影响 |

## **4.5 涉及范围**

# **5\. 功能互斥与依赖**

## **5.1 功能依赖**



| **功能A** | **功能B** | **说明** |
|-----------------------------------------------------------|-----------------------------------------------------------|----------------------------------------------------------|
| 税额不占用预算-费用类型配配置 | 配置了消费明细组件的单据 | 强依赖，规则生效需配置税率税额组件和消费明细组件 |
| 税额不占用预算-费用类型配置 | 费用类型管理 | 强依赖，需要获取企业的费用类型数据 |
| 税额不占用预算-费用类型配置 | 预算占用计算 | 强依赖，计算时需读取配置 |
| 税额不占用预算-费用类型配配置 | 付款，借款单 | 强依赖，规则生效需配置税率税额组件和消费明细组件或无消费组件时配置了税率税额组件，费用类型组件和金额组件 |

## **5.2 功能影响**



| **功能A** | **功能B** | **影响说明** |
|-----------------------------------------------------------|-----------------------------------------------------------|----------------------------------------------------------------|
| 税额不占用预算-费用类型配置 | 单据预算占用 | 影响税额是否计入预算占用 |
| 费用类型停用 | 税额不占用配置-停用前已选择 | 存量数据不变，下次修改配置时，弹出tips“已更新可用费用类型”，选择项内被停用的费用类型不显示 |
| 费用类型停用 | 税额不占用配置-停用后已选择 | 存量数据不变，下次修改配置时，弹出tips“已更新可用费用类型”，选择项内被停用的费用类型不显示 |
