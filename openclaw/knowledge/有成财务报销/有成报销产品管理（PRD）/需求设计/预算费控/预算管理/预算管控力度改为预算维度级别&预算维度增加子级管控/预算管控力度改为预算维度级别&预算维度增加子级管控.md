## 4、需求详情

### 4.1、预算基础设置&预算管理列表

1、预算管理列表中，去除“超标限制”字段



![Picture 3](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/yBRq1ZPKppPkpOdv/img/4b03f785-6cd7-45cd-a84c-cad3144caea4.png?Expires=1775016963&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=9SuKFtVjF0y1D126Yrd7B2njth0%3D "")

2、基础设置中，取消“预算超额限制”选项，增加“超额管控模式”，可选“单维度管控”，“父级维度管控”，每个选项的备注如下：

单维度管控：每个预算维度均可设置管控力度父级维度管控：每个预算维度可设置是否管控下级维度，当下级申请金额超过父级额度时，才会管控

![Picture 4](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/yBRq1ZPKppPkpOdv/img/22414fac-dacf-4346-b26c-414a0b0cebc8.png?Expires=1775016963&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=S18uPt9RAf0RbCh8EfcSrnSfRYQ%3D "")

3、默认勾选，单维度管控

### 4.2、预算维度编辑

#### 4.2.1、管控力度&总控下级

1、编辑预算维度时，增加年度、月底管控力度，可选“提示警告”、“禁止提交”、“不提示”，默认选择“提示警告”

| **基础设置** | **展示内容** |
|----------------|----------------|
| 周期管控-按月 | 年度管控，月度管控 |
| 周期管控-季度 | 年度管控，季度管控 |
| 周期管控-年度 | 年度管控 |
| 自定义周期-按月 | 总额管控，月度管控 |
| 自定义周期-按季 | 总额管控，季度管控 |
| 自定义周期-按年 | 总额管控，年度管控 |
| 自定义周期-无周期 | 总额管控 |

2、基础设置中，超额管控模式选择“单维度管控”，明细中没有“总控下级”；

 选择“父级维度管控”，明细中有“总控下级”

3、总控下级可选择：开启、关闭，开启增加备注：

开启后，下级及其子级预算维度超过当前维度的可用额度时，将会按照当前维度的管控力度控制。例：销售部中，差旅费剩余0、管理费剩余1w，开启后差旅费可报销1w，超过1w后，会根据销售部管控力度报错

#### 4.2.1、批量设置

1、新增预算维度按钮右侧增加“批量设置”按钮

2、点击后会有弹框展示，设置内容：

可将所有已经保存的维度进行修改年度管控：月度管控：

3、其中，展示的管控力度需要和维度中保持一致，即：

| **基础设置** | **展示内容** |
|----------------|----------------|
| 周期管控-按月 | 年度管控，月度管控 |
| 周期管控-季度 | 年度管控，季度管控 |
| 周期管控-半年 | 年度管控，半年管控 |
| 周期管控-年度 | 年度管控 |
| 自定义周期-按月 | 总额管控，月度管控 |
| 自定义周期-按季 | 总额管控，季度管控 |
| 自定义周期-按年 | 总额管控，年度管控 |
| 自定义周期-无周期 | 预算管控 |

4、确定好后，会将所有维度的管控力度设置为弹框中的设置规则，确定好后将会把所有已经保存过维度的管控力度覆盖更新。

### 4.3、命中逻辑&提单报错效果

#### 4.3.1、命中逻辑

1、提单时，需要判断“超额管控模式”，如果是“单维度管控”则会按照当前维度中的配置的“管控力度”进行提示

2、如果“超额管控模式"是“父级维度管控”，所有上级没有开启父级管控，则根据当前维度的“管控力度”进行提示

3、，如果“超额管控模式"是“父级维度管控”，则会找这个超标维度上级所有的父级节点中，如果存在一个父级节点的管控力度比本级别高，则会按照父级管控力度执行

4、如果“超额管控模式"是“父级维度管控”，如果不存在父级节点的管控力度比本级别高，则会找距离这个维度最近的开启“总控下级”的预算维度，并根据此父级维度的“管控力度”进行提示

5、如果一个单据中，即有警示提醒的预算维度，也有禁止提交的预算维度，均需要展示，有一项禁止提交的维度即为不能提交。

#### 4.3.2、报错弹框样式

##### 第一个弹框选择维度：

1、第一个选择维度的弹框不在显示超标情况，仅可以选择预算维度

2、“预算总额、剩余预算金额”会受“**允许员工看到预算金额**”这个配置影响，如果允许则可以展示金额，如果不允许则会显示“--”



![Picture 9](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/yBRq1ZPKppPkpOdv/img/9cc9d968-fd79-4b3d-9563-e605b43e9199.png?Expires=1775016963&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=Y1rWTK2qtxPZm4t80tSgMtz5hRY%3D "")

##### 第二个校验弹窗样式：

1、在列表中增加“超额情况”字段，将超额的提示放在此处，如果是父级超标，会提示“父级预算超标”，此时会有详情，点击后会有展示父级的“父级预算单元、预算总额、剩余预算金额”

2、父级的“预算总额、剩余预算金额”会受“**允许员工看到预算金额**”这个配置影响，如果允许则可以展示金额，如果不允许则会显示“--”

3、如果有至少一个禁止提交，下方按钮变为“知道了”，点击后会关闭弹窗

4、如果没有一个禁止提交，下方按钮变为“提交单据”，点击后会提交单据

##### 单据详情（低优先级）

1、每个维度上增加“超额情况”，如果是父级超标，会提示“父级预算超标”，此时会有详情，点击后会有展示父级的“**父级预算单元、预算单元已用、预算单元剩余、预算维度总已用、预算维度总剩余**”

2、父级的“**预算单元剩余、预算维度总已用、预算维度总剩余**”会受“**允许审批人看到单据详情中的预算信息**”这个配置影响，如果允许则可以展示，如果不允许则会显示“--”

3、预算管理员可以不受权限，始终查看
