# **1\. 需求背景**

目前线上我们的预算申请组件仅能支持一个预算内的多个维度，无法选择多个预算内的不同维度。用户有40多个预算，现在分开提申请单非常麻烦。

宁夏安欣、【智能财务】澳斯康

# **3\. 功能清单**

|------|---------|------|
| 新增单据 | 新增单据预算申请组件中，支持选择多个预算 |  |

|  |  |  |

新增单据-预算申请组件

1、选择预算的交互修改，组件中没有维度时，需要用户在弹框中先选择预算。
- 选择预算后，展示对应预算的维度。
- 用户可以切换预算，切换时如果预算中已有选择的维度，保留之前预算维度的选中状态。

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5bP1a0jvlxo/img/e62113db-be5a-4a4e-b3f0-c2a73d2d40a4.png?Expires=1775016943&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=8eFoz1umiSrPA02WLbX4CiFIUdk%3D "")

2、选择预算的列表支持模糊查找预算名称。

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5bP1a0jvlxo/img/754c41fd-6ebc-4875-8b3f-7df671dfe635.png?Expires=1775016943&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=IlYnMwtSkyRePDJB1i8eHxaP9Mc%3D "")

3、跨预算的已选统计挪到右下方。

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5bP1a0jvlxo/img/29209354-c012-4316-8434-feeda7e4e62b.png?Expires=1775016943&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=nELS46VqOa4lsVQpQTV4gKOIe0c%3D "")

4、点击确定将选中的多预算的预算维度，一并更新预算申请组件。
    - 多选预算时，若点击确定，出现部分预算更新失败，提示用户“预算：xxx，预算维度：xxx，周期：xxx，更新失败”
    - 多选预算时，组件展示效果如下

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5bP1a0jvlxo/img/0edc4a00-5386-4dd9-a8fa-461870d93af3.png?Expires=1775016943&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=fztBhm5RSTUcA%2Bm8OcNEzFhideE%3D "")
    - 预算之间按照创建时间从早到晚排序，同一预算的不同预算单元按照单元列表中的顺序排序，同一个预算维度预算期间顺序从早到晚排序。
- 若组件中已经有预算，则点击选择预算时，需要将已经选择的预算、维度和时间，回显到选择预算弹框中（注意线上时间并没有回显，本次一并做上。）

5、单据完成审批调整预算额度时，根据组件中已经选择的预算，同时更新多个预算的额度。

6、导入预算时，模板中已经放出了预算名称，需要根据不同预算的具体维度更新金额。

追加模式下，暂时不开放导入导出入口。
