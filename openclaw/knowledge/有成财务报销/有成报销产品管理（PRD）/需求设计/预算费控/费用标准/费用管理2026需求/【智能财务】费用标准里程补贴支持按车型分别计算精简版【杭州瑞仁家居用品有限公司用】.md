## 1\. 需求背景

### 1.1 需求描述

在智能财务产品-预算费控模块-费用标准中，里程补贴标准新增「按车型分别计算」控制方式。支持按车辆型号（油车/电车等）分别设置补贴单价，参照有成报销已有功能实现。

### 1.2 需求背景
- **需求方**：智能财务客户（企业财务管理员）
- **现状问题**：
• 有成报销产品中，里程补贴标准已支持「按车型分别计算」，可区分油车、电车设置不同单价
• 智能财务的预算费控-费用标准中，里程补贴暂无按车型区分的能力
• 客户实际业务中，油车和电车的里程补贴单价不同（如油车0.80元/公里、电车0.60元/公里），需要分别设置
- **解决思路**：
• 在智能财务费用标准的里程补贴类型中，增加「按车型分别计算」控制方式
• 支持选择「车辆型号」作为车型辅助核算字段
• 标准设置中按不同车型分别配置计价模式和里程单价
• 功能逻辑和交互完全参照有成报销实现

## 2\. 功能现状

### 2.1 有成报销里程补贴按车型计算现状

**位置**：费用标准 → 编辑标准 → 标准类型选择「里程补贴」

**现有能力**：
• 标准类型支持「里程补贴」
• 控制方式支持「按车型分别计算」
• 车型辅助核算支持选择「车辆型号」
• 标准设置表格支持按车型（油车/电车）分别配置适用范围、计价模式、里程单价
• 计价模式支持「单一价格」
• 里程单价支持按币种和数值配置（如 CNY 0.80 元/公里）

**有成报销交互**：
• 控制方式选择「按车型分别计算」后，车型辅助核算必填
• 标准设置表格按行展示不同车型的规则
• 每行包含：适用范围、车型、计价模式、里程单价、操作（上移/下移/删除）
• 支持新增多行，配置多种车型的补贴规则

### 2.2 智能财务费用标准现状

**位置**：预算费控 → 费用标准

**现有能力**：
• 支持创建费用标准，标准类型包含里程补贴
• 控制方式目前仅支持统一计算（不区分车型）
• 标准设置中配置统一的计价模式和里程单价

**现有缺失**：
• 无「按车型分别计算」控制方式
• 无法区分油车、电车等不同车型设置不同补贴单价

### 3.2 详细功能描述

#### 3.2.1 按车型分别计算控制方式

**功能概述**

里程补贴标准的控制方式新增「按车型分别计算」选项，选择后系统按不同车型分别校验和计算补贴金额。

**配置入口**

预算费控 → 费用标准 → 新增/编辑标准 → 标准类型选择「里程补贴」 → 控制方式

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/1GXn45K9gRbexqDQ/img/ee666003-90c9-438b-af6d-183764329e49.png?Expires=1776848273&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=0hA3iYT98Jrq8Yqr3MXZRhK6%2Fnc%3D "")

**控制方式选项**

| 选项 | 说明 |
|------|------|
| 统一计算 | 所有车型使用同一补贴单价（现有逻辑） |
| 按车型分别计算 | 不同车型使用不同补贴单价（新增） |

**变更说明** • 默认选中「统一计算」，保持存量标准不受影响
• 选择「按车型分别计算」后，下方显示「车型辅助核算」选择项
• 切换控制方式时，标准设置区域的内容随之切换

#### 3.2.2 车型辅助核算选择

**功能概述**

选择「按车型分别计算」后，需指定车型辅助核算字段，系统根据该字段值判定单据中的车型。

**字段位置**

控制方式下方，标准设置上方。

**字段配置**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 车型辅助核算 | 下拉选择 | 是 | 选择辅助核算字段，用于识别车型 |

**下拉选项** • 车辆型号（默认推荐）
• 其他自定义辅助核算字段（如车辆类型等）

**提示文案**

控制方式旁显示提示：「规则将对出发地、目的地字段生效，若选择按车型分别计算补贴，则费用中需要有辅助核算字段，请确保需要控制的费用类型中包含对应字段」

#### 3.2.3 按车型标准设置表格

**功能概述**

按车型分别配置补贴规则，每行对应一种车型，支持设置适用范围、计价模式、里程单价。

**表格字段**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 适用范围 | 人员选择器 | 否 | 该规则适用的员工/部门，不选默认全部员工 |
| 车型 | 多选下拉 | 是 | 选择多个车型（如 01油车、02电车），同一车型仅可出现一次 |
| 计价模式 | 下拉选择 | 是 | 目前支持「单一价格」 |
| 里程单价 | 金额输入 | 是 | 币种 \+ 数值 \+ 单位（元/公里） |
| 操作 | 按钮组 | - | 上移 / 下移 / 删除 |

**车型数据来源**

• 车型选项从「车型辅助核算」字段的枚举值中读取
• 如辅助核算「车辆型号」配置有「01油车」「02电车」，则下拉选项显示这两项
• 支持多选，一个规则内可同时配置多个车型共用同一单价

**计价模式**

| 模式 | 说明 |
|------|------|
| 单一价格 | 固定单价 × 里程数 = 补贴金额 |

**里程单价**

• 币种：默认CNY，支持其他币种
• 数值：正数，支持小数（最多2位）
• 单位：元/公里

**示例配置**

| 适用范围 | 车型 | 计价模式 | 里程单价 |
|------------|------|------------|------------|
| 全部员工 | 01油车 | 单一价格 | CNY 0.80 元/公里 |
| 全部员工 | 02电车、03混动 | 单一价格 | CNY 0.60 元/公里 |

**新增规则**

• 点击「新增费用标准」按钮，在表格底部新增一行
• 新增行车型默认空，需手动多选
• 同一车型可在多个规则中重复配置

**排序**

• 支持上移/下移调整规则优先级
• 校验时按从上到下的顺序匹配，匹配到第一条适用的规则即停止

#### 3.2.4 单据校验适配

**功能概述**

员工提交含里程补贴的单据时，系统根据单据中「车型辅助核算」字段的值，匹配对应车型的补贴标准进行校验。

**校验逻辑**
```text
员工提交报销单（含里程补贴费用）
    ↓
读取单据中「车型辅助核算」字段值（如「01油车」）
    ↓
在费用标准中查找里程补贴标准
    ↓
匹配控制方式为「按车型分别计算」的标准
    ↓
按车型匹配对应的里程单价（如油车 0.80元/公里）
    ↓
计算补贴金额 = 里程数 × 里程单价
    ↓
与单据填写金额比对，超出标准则按规则提示/拦截

```

**匹配规则** • 先按适用范围（员工/部门）过滤
• 再按车型精确匹配
• 未匹配到任何规则时，按无标准处理（按现有逻辑执行）

### 3.3 逻辑互斥与边界处理

**边界情况处理**

| **边界场景** | **触发条件** | **处理方案** |
|----------------|----------------|----------------|
| 单据未填写车型 | 单据中车型辅助核算字段为空 | 无法匹配车型规则，按无标准处理 |
| 车型未配置标准 | 单据中的车型在标准中无对应配置 | 按无标准处理，不拦截或按默认提示 |
| 同一车型重复配置 | 标准设置中多行规则包含同一车型 | 允许配置 |
| 切换控制方式 | 从「统一计算」切换为「按车型分别计算」 | 清空原有统一计算的单价配置，需重新按车型配置 |
| 辅助核算字段被删除 | 标准引用的辅助核算字段在后台被删除 | 标准编辑时提示辅助核算字段无效，需重新选择 |
| 车型枚举值变更 | 辅助核算字段的车型选项发生变更 | 存量标准中已配置的车型保留，新增选项可在标准中选用 |

**与其他功能的兼容性**

| **相关功能** | **交互关系** | **处理方案** |
|----------------|----------------|----------------|
| 统一计算标准 | 两种控制方式互斥 | 同一标准只能选择一种控制方式，切换时清空原有配置 |
| 适用范围（员工/部门） | 与车型条件叠加 | 先过滤适用范围，再匹配车型 |
| 费用类型配置 | 车型辅助核算需存在于费用类型中 | 提示用户确保费用类型中已包含车型辅助核算字段 |
| 费用类型配置 | 车型辅助核算需存在于费用类型中 | 提示用户确保费用类型中已包含车型辅助核算字段 |

## 4\. 验收标准

| **序号** | **验收项** | **验收标准** |
|----------|-------------|----------------|
| 1 | 控制方式新增 | 里程补贴标准控制方式下拉中有「按车型分别计算」选项 |
| 2 | 车型辅助核算选择 | 选择「按车型分别计算」后，车型辅助核算字段必填 |
| 3 | 标准设置表格 | 标准设置区域展示按车型配置的表格，包含适用范围/车型/计价模式/里程单价 |
| 4 | 车型下拉选项 | 车型下拉选项与辅助核算字段的枚举值一致 |
| 5 | 里程单价配置 | 可正常配置不同车型的里程单价（如油车0.80、电车0.60） |
| 6 | 重复车型校验 | 同一车型不可重复配置，前端给出提示 |
| 7 | 单据校验-匹配车型 | 提交单据时，系统按单据中的车型匹配对应的补贴单价 |
| 8 | 单据校验-金额计算 | 补贴金额 = 里程数 × 对应车型的里程单价 |
| 9 | 单据未填车型 | 单据未填写车型时，按无标准处理，不报错 |
| 10 | 切换控制方式 | 从统一计算切换到按车型分别计算时，原有配置清空 |
| 11 | 存量兼容 | 存量统一计算的标准不受影响，保持原有逻辑 |
| 12 | 多车型配置 | 一个规则内可选择多个车型，这些车型共用同一单价 |
| 13 | 有成报销一致性 | 功能逻辑、交互、字段与有成报销「按车型分别计算」保持一致 |

## 附录

**A. 典型使用场景**

**场景：财务配置油车和电车的里程补贴标准**
1. 财务管理员进入预算费控 → 费用标准 → 新增标准
2. 标准类型选择「里程补贴」，费用标准名称填写「2026年里程补贴标准」
3. 控制费用类型选择「里程补贴费用」
4. 控制方式选择「按车型分别计算」
5. 车型辅助核算选择「车辆型号」
6. 标准设置中新增两行： 
    - 第一行：车型「01油车」，计价模式「单一价格」，里程单价「CNY 0.80 元/公里」
    - 第二行：车型「02电车」，计价模式「单一价格」，里程单价「CNY 0.60 元/公里」
7. 保存标准
8. 员工提交里程补贴报销单时，系统根据单据中选择的车型自动匹配对应单价进行校验
