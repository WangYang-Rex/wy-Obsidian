## **1\. 需求背景**

### **1.1 需求描述**

费用标准设置新增【继承字段配置】，用户费用标准方案内配置哪些字段继承上一行规则的值。后续点击【新增规则】时，系统自动按继承配置填充字段，仅需修改差异化字段。

### **1.2 需求背景**
- **需求方**：有成报销企业客户（财务管理员）
- **现状问题**：
• 同一人群在多个城市的标准，仅城市级别不同，其他字段完全相同
• 每条规则需重复填写相同字段，配置效率低
- **解决思路**：费用标准方案内配置继承字段，新增规则自动填充，只需修改差异化字段

**业务流程**


```plain
首次配置：
进入费用标准设置 → 点击【继承字段配置】
↓
勾选要继承的字段（适用范围、往返日、日期类型、金额）
↓
取消勾选差异化字段（城市级别）
↓
保存配置

日常使用：
点击【新增规则】
↓
系统自动复制上一行规则的被继承字段值
↓
填写城市级别（差异化字段）
↓
完成
```

### **3.2 详细功能描述**

#### **3.2.1 继承字段配置**

**功能概述** 费用标准方案内配置哪些字段在新增规则时自动继承上一行规则的字段值。

**配置入口** 【预算费控-费用标准-编辑标准-标准设置】页面新增【继承字段配置】按钮。

#### **3.2.2 新增规则（按继承配置）**

**功能概述** 点击【新增规则】时，系统自动按继承字段配置，复制上一行规则的字段值到新规则行。

**业务流程**
```plain
用户点击【新增规则】
↓
在最后一行下方插入新规则行
↓
FOR EACH 字段 IN 字段列表:
  IF 字段在继承配置中勾选 THEN
    新规则[字段] = 上一行规则[字段]
  ELSE
    新规则[字段] = 空值
  END IF
END FOR
↓
光标定位到第一个未继承的字段
```



**效果示例**

继承配置：适用范围☑ 往返日☑ 日期类型☑ 城市级别☐ 金额☑

规则列表：
```plain
┌────┬─────────────┬───────────┬──────────┬──────────┬────────┐
│ #  │ 适用范围    │ 往返日    │日期类型  │ 城市级别 │ 金额   │
├────┼─────────────┼───────────┼──────────┼──────────┼────────┤
│ 1  │ VP及以上    │出发/返回  │ 节假日   │ 一线城市 │ 800   │
├────┼─────────────┼───────────┼──────────┼──────────┼────────┤
│ 2  │ VP及以上    │出发/返回  │ 节假日   │ 【请选择│ 800   │ ← 新增
│    │ （继承）    │ （继承）  │ （继承） │ 】       │（继承）│

```

用户仅需填写城市级别=二线城市

**边界处理**

| **场景** | **处理方案** |
|----------------------------------------------------------|----------------------------------------------------------------|
| 当前无规则 | 新增空规则行（所有字段为空） |
| 上一行规则字段为空 | 继承空值 |
| 切换费用标准方案 | 继承字段配置按费用标准方案独立存储，切换后使用对应配置 |

### **3.3 逻辑互斥与边界处理**

**边界情况处理**

| **边界场景** | **触发条件** | **处理方案** |
|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| 全部勾选 | 用户将所有字段设为继承 | 新增规则完全复制上一行，允许用户自行修改 |
| 全部不勾选 | 用户取消所有字段继承 | 新增规则为空行，与现有【新增规则】行为一致 |
| 删除最后一行后新增 | 删除规则后只剩n-1行，再新增 | 以当前最后一行为基准继承 |

**与其他功能的兼容性**



| **相关功能** | **交互关系** | **处理方案** |
|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| 规则上移/下移 | 独立操作 | 不影响继承配置，新增规则始终继承当前最后一行 |
| 规则删除 | 独立操作 | 删除后新增规则继承新的最后一行 |
| 保存 | 正常保存 | 继承填充的值作为普通值保存 |

## **附录**

**A. 典型使用场景**

场景：VP及以上在三个城市享受不同住宿标准

配置步骤（首次）：
1. 进入费用标准设置
2. 点击【继承字段配置】
3. 取消勾选"城市级别"，其他字段保持勾选
4. 保存配置

使用步骤（日常）：
1. 创建规则1：VP及以上 \+ 一线城市 \+ 1000元（填写全部字段）
2. 点击【新增规则】，系统自动填充：VP及以上 \+ 【请选择】\+ 1000元
3. 填写城市级别=二线城市，保存
4. 点击【新增规则】，系统自动填充：VP及以上 \+ 【请选择】\+ 1000元
5. 填写城市级别=三四线城市，保存

结果：仅需配置一次，后续新增规则自动继承，大幅提升配置效率
