# 首页-增加默认首页设置

| **修订时间** | **版本** | **修订说明** |
|----------------|----------|----------------|
| 20250603 | v1.0 | 首页增加默认首页设置 |

## 1、需求背景

系统管理员自定义了首页无法给员工设为默认首页并生效。目前支持设置首页应用范围但没有生效需要员工自行切换首页，员工一般不会去操作。

## 2、业务流程/架构

首页增加默认首页设置

## 3、功能清单

根据个人需要补充

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 模板编辑页面 | 增加“应用为默认首页”的开关 |  |
| 模板列表 | 增加“设为我的默认”按钮 |  |
| 首页加载 | 加载我的默认模板 |  |

## 4、需求详情

### 4.1、编辑页面增加“应用为默认首页”的开关

增加“应用为默认首页”的开关，打开后当前模板的应用范围人员的默认首页会变成当前首页模板，若关闭且列表没有设置默认首页，则默认首页为上一次进入的首页。

### 4.2、模板列表增加“设为默认”操作

设为默认：针对当前登陆用户生效，每个人都可以设置自己的默认首页。

最新操作的为最新的默认的首页 。
