# 首页支持设置默认模版

| **修订时间** | **版本** | **修订说明** |
|----------------|----------|----------------|
| 20231102 | v1.0 | 新建 |

**1、需求背景**

客户：财开

需求：管理员可以给员工统一设置默认打开的模版

## 2、业务流程/架构

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 首页/自定义首页 | 支持在PC和MB的模版列表设置默认打开的模版，支持PC端在首页模版下拉处设置默认模版。 |  |
|  |  |  |
|  |  |  |

## 4、需求详情

#### 名词定义：

默认模版：即用户进入首页默认打开的模版，一个用户最多只有一个模版是默认的。

#### 4.1  PC端

（1）初始化时，给每个用户分配的那个模版为默认模版，在首页模版下拉列表和PC模版列表页增加了“默认”标签

![Picture 2](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/NpQlK5jN7ERzNqDv/img/42b706c9-89e8-41c3-9aec-aed8fb9448b9.png?Expires=1775016904&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=1jYjsCK9gV%2FREWafIQcDtf%2Bp6bY%3D "")

（2）管理员设置默认模版

在模版列表页将某个模版设为默认，则应用范围的员工的默认模版都设置为该模版



![Picture 3](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/NpQlK5jN7ERzNqDv/img/55640a4d-f470-40c6-8b73-ae15f7e1589c.png?Expires=1775016904&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=RsdLpgcFuimIuOFlsNX31wLpotc%3D "")

（3）员工可以在首页进行模版切换，最后切换的模版更新为该用户的默认模版（管理员在首页切换不影响其他人的默认首页）。

#### 4.2 MB端

（1）初始化时，给每个用户分配的那个模版为默认模版

（2）点击某个模版：将该模版设置为默认模版



![Picture 4](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/NpQlK5jN7ERzNqDv/img/3a454cd4-711e-49f2-9bd8-e99d45f23637.png?Expires=1775016904&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=b1QJ8qld1Y6T%2FdRtSpjg6SH04pk%3D "")

（3）在模版列表页将某个模版设为默认，则应用范围的员工的默认模版都切换为该模



![Picture 5](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/NpQlK5jN7ERzNqDv/img/9130da4f-22e9-4423-9a9c-2ef0b1b0731b.png?Expires=1775016904&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=J9nSE2EQlH%2BD1nPSKENtg3Oy4ZM%3D "")

#### 4.3 极端场景的首页展示逻辑

若用户的默认模版被禁用（例如管理员给员工分配了一个首页且设置为默认模板，又将该默认模版禁用了，且没有再设置其他默认模版），则员工打开首页的展示逻辑为：

（1）若员工还有未被禁用的模版，则展示模版列表第一个未被禁用的模版；

（2）若员工的模版都被禁用了，则展示上次选择的模版（现有逻辑）
