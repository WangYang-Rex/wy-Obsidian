## **1\. 需求背景**

### **1.1 需求描述**

可抵扣发票税额分录设置增加「生效账套」配置项，支持按账套维度独立配置可抵扣发票的价税分离规则，实现同一可抵扣发票在不同账套生成凭证时分别应用各自匹配的分录规则。

### **1.2 需求背景**
- **需求方**：企业财务管理员/系统管理员
- **现状问题**：
• 当前可抵扣发票税额分录设置为全局生效，所有账套共用同一套规则
• 客户实际业务中，不同账套（如总公司账套与分公司账套）对可抵扣发票的税额处理要求可能不同
• 无法实现同一可抵扣发票在多账套凭证生成时按各账套规则分别处理
- **解决思路**：在可抵扣发票税额分录配置中增加「生效账套」字段，支持多选，存量规则默认全部账套生效，实现账套级差异化配置

**业务流程**
```plain
配置阶段：
开启税额分录功能开关
↓
配置生效范围规则（单据范围 + 费用类型 + 发票类别 + 生效账套）
──────────────────────────────────────────────────────────────
差旅报销 交通费 增值税专票 账套A、账套B
日常报销 办公费 增值税普票 全部账套

执行阶段：
可抵扣发票进入系统
↓
单据按账套生成凭证时，开关已开启
↓
匹配规则：单据范围 + 费用类型 + 发票类别 + 生效账套
↓
账套命中规则 → 按价税分离生成税额分录（借：费用 借：进项税 贷：应付/银行）
账套未命中规则 → 按原有逻辑生成（借：费用含税金额 贷：应付/银行）
```

## **2\. 功能现状**

**位置**：会计科目映射 → 基础设置 → 税额分录价税分离生成税额分录设置

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/AJdl65ADzx0vmOke/img/0bee4c65-892d-4d8d-a279-8d03263aade8.png?Expires=1776848515&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=yKPIUJAAdsditP4UVOqckRjUk3U%3D "")

**现有字段**
- 功能开关：总控开关（税额分录是否启用）
- 规则配置表格：
- 单据范围：下拉选择（差旅报销/日常报销/借款单等）
- 费用类型：下拉选择（交通费/办公费/招待费等）
- 发票类别：下拉选择（增值税专票/普票/电子发票等可抵扣发票类型）

**现有操作**
- 可开启/关闭税额分录功能开关
- 开关开启后，可新增、编辑、删除税额分录规则
- 开关关闭时，规则配置区域隐藏
- 规则按配置顺序从上到下匹配，命中即停
- 匹配维度：单据范围 \+ 费用类型 \+ 发票类别
- 匹配到的规则：可抵扣发票按价税分离生成凭证分录
- 未匹配到规则：按原有逻辑生成凭证（无价税分离）

**缺失能力**
- 无法指定规则对哪些账套生效
- 无法实现不同账套用不同规则控制价税分离

### **3.2 详细功能描述**

#### **3.2.1 税额分录功能开关**

**位置**：会计科目映射 → 基础设置 → 税额分录价税分离生成税额分录设置

**配置界面**

![画布 56.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/AJdl65ADzx0vmOke/img/8fbd6805-26ea-49bc-8594-c177bb77b145.png?Expires=1776848515&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=HfkPQX60JaWzmIFKd7ADYyHMecw%3D "")



**交互规则** • 开关默认关闭
• 开关关闭时：规则配置区域隐藏，可抵扣发票按原有逻辑生成凭证（无价税分离）
• 开关开启时：规则配置区域展示，可新增、编辑、删除规则
• 规则表格每行支持删除操作
• 开关关闭时保存：保留已有规则配置但不生效

**默认配置**

| **字段** | **默认状态** | **说明** |
|----------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------|
| 启用税额分录功能 | 关闭 | 存量数据升级后开关默认开启，保持原有行为 |

**配置存储** • 开关状态按全局存储
• 规则按全局存储（非按账套存储）

#### **3.2.2 可抵扣发票税额分录规则-生效账套配置**

**位置**：会计科目映射 → 基础设置 → 税额分录价税分离生成税额分录设置 → 规则配置区域

**新增配置项**

![画布 56.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/AJdl65ADzx0vmOke/img/c581faf5-e110-4bf7-85fe-ee7fab5b6008.png?Expires=1776848515&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=B3%2BZ9vEI9fsjFo%2BC0kEJFlsu4J8%3D "")



**交互规则** • 点击「新增」按钮，表格新增一行空规则，可填写单据范围、费用类型、发票类别、生效账套
• 点击每行「删除」按钮，删除该行规则
• 「生效账套」列支持下拉多选
• 选项包括：全部账套、账套A、账套B、账套C...（动态读取系统已启用账套）
• 未选择任何账套时，默认等价于「全部账套」
• 「全部账套」与其他具体账套互斥，选择「全部账套」后自动清空其他选项
• 编辑存量规则时，若原无账套配置，自动回填「全部账套」

**默认配置**

| **字段** | **默认状态** | **说明** |
|----------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------|
| 生效账套 | 全部账套 | 存量数据升级后默认全部账套生效 |

**配置存储** • 每行规则存储生效账套ID列表，空列表表示全部账套
• 配置保存后即时生效，影响后续可抵扣发票生成的凭证

#### **3.2.3 凭证生成-可抵扣发票账套独立匹配规则**

**触发条件**：税额分录功能开关已开启

**行为分支**

| **场景** | **匹配逻辑** | **分录生成方式** |
|----------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------------|
| 功能开关关闭 | 不进入税额分录逻辑 | 按原有逻辑生成凭证（借：费用含税金额 贷：应付/银行） |
| 账套命中规则 | 单据范围\+费用类型\+发票类别匹配，且该账套在规则生效账套列表中 | 按价税分离生成（借：费用不含税金额 借：进项税 贷：应付/银行） |
| 账套未命中规则 | 单据范围\+费用类型\+发票类别不匹配，或该账套不在规则生效账套列表中 | 按原有逻辑生成凭证（无价税分离） |
| 多规则同时命中 | 按配置顺序从上到下，全部生效 | 按价税分离生成 |

**核心逻辑说明** • 税额分录功能开关关闭时，所有账套均按原有逻辑生成凭证
• 开关开启后，同一可抵扣发票在多个账套生成凭证时，每个账套独立匹配自己的规则
• 匹配维度：单据范围 \+ 费用类型 \+ 发票类别 \+ 生效账套
• 先筛选「生效账套包含当前账套」的规则，再按原有顺序匹配单据范围、费用类型、发票类别
• 命中规则：按价税分离生成凭证分录
• 未命中规则：按原有逻辑生成凭证（无价税分离）

**业务流程**
```plain
可抵扣发票关联到单据费用明细（含费用类型）
↓
单据触发凭证生成
↓
判断税额分录功能开关
↓
关闭 → 按原有逻辑生成凭证（无价税分离）
↓
开启 → 遍历该单据关联的所有目标账套
↓
账套A：
提取单据下所有可抵扣发票及关联的费用类型
↓
筛选生效账套包含「账套A」的规则（或全部账套）
↓
按原顺序匹配单据范围、费用类型、发票类别
↓
命中规则 → 按价税分离生成凭证分录
未命中 → 按原有逻辑生成凭证

账套B：
（同上，独立匹配账套B生效的规则）
```



### **3.3 逻辑互斥与边界处理**

**边界情况处理**

| **边界场景** | **触发条件** | **处理方案** |
|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| 存量兼容-开关 | 已开启税额分录功能 | 升级后开关保持开启状态 |
| 存量兼容-规则 | 已配置的税额分录规则无账套信息 | 升级时自动填充「全部账套」，保持原有行为 |
| 功能开关关闭 | 用户关闭税额分录功能开关 | 规则配置区域隐藏，已配置规则保留但不生效，凭证按原有逻辑生成 |
| 空规则保存 | 新增规则未填写完整字段 | 校验提示，必填字段不能为空 |
| 空值处理 | 新增规则未选择生效账套 | 保存时自动设为「全部账套」 |
| 互斥逻辑 | 同时选择「全部账套」和具体账套 | 选择「全部账套」时清空具体账套；选择具体账套时禁用「全部账套」 |
| 账套删除 | 已配置的账套被删除 | 该规则对该账套不再生效，其他账套不受影响 |
| 无匹配规则 | 某账套未命中任何规则 | 按原有逻辑生成凭证（无价税分离） |
| 多规则冲突 | 同一账套命中多条规则 | 按配置顺序从上到下，全部生效 |
| 非可抵扣发票 | 发票类别不在可抵扣范围内 | 不走税额分录设置规则，按原有逻辑处理 |
| 费用类型匹配 | 费用明细未设置费用类型 | 按原有逻辑处理，全部费用类型 |
| 并发配置 | 多人同时编辑税额分录设置 | 后保存的覆盖先保存的，以最终数据为准 |

**与其他功能的兼容性**

| **相关功能** | **交互关系** | **处理方案** |
|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| 会计科目映射 | 依赖调用 | 税额分录规则匹配后，按会计科目映射生成最终凭证分录（科目、辅助核算等） |
| 多账套凭证生成 | 核心场景 | 开关开启后，同一可抵扣发票在多账套生成时，各账套独立匹配税额分录规则 |
| 发票管理 | 数据关联 | 只处理可抵扣发票，非可抵扣发票不受此配置影响 |
| 费用类型管理 | 联合匹配 | 作为税额分录规则的核心匹配维度之一 |
| 单据范围配置 | 联合匹配 | 与费用类型、发票类别共同构成规则匹配条件 |
| 数据权限 | 配置权限 | 税额分录设置页面按原有权限控制，与账套无关 |

## **附录**

**A. 典型使用场景**

**场景一：总公司与分公司差异化价税分离处理**

配置步骤（首次）：
1. 进入会计科目映射 → 基础设置 → 税额分录价税分离生成税额分录设置
2. 开启「启用税额分录功能」开关
3. 点击「新增」，添加规则：单据范围=差旅报销，费用类型=交通费，发票类别=增值税专票，生效账套=总公司账套
4. 点击「新增」，添加规则：单据范围=差旅报销，费用类型=交通费，发票类别=增值税专票，生效账套=分公司账套A、分公司账套B
5. 点击「保存」

使用步骤（日常）：
1. 员工提交差旅报销单，费用明细包含交通费（关联增值税专票）
2. 单据同时推送至总公司账套和分公司账套生成凭证
3. 总公司账套：命中规则，交通费的可抵扣发票按价税分离生成凭证（借：费用 借：进项税 贷：应付）
4. 分公司账套：命中规则，交通费的可抵扣发票按价税分离生成凭证（借：费用 借：进项税 贷：应付）

结果：同一可抵扣发票在不同账套按各自配置决定是否进行价税分离处理

**场景二：部分账套不需要价税分离**

配置步骤（首次）：
1. 进入会计科目映射 → 基础设置 → 税额分录设置
2. 开启「启用税额分录功能」开关
3. 点击「新增」，添加规则：单据范围=日常报销，费用类型=办公费，发票类别=增值税专票，生效账套=账套A
4. 不配置账套B的规则
5. 点击「保存」

使用步骤（日常）：
1. 员工提交日常报销单，费用类型=办公费，关联增值税专票
2. 单据推送至账套A和账套B生成凭证
3. 账套A：命中规则，按价税分离生成凭证
4. 账套B：未命中规则，按原有逻辑生成凭证（无价税分离）

结果：实现部分账套价税分离、部分账套保持原样的差异化处理

**场景三：存量规则保持兼容**

配置步骤：
1. 原有规则：单据范围=日常报销，费用类型=办公费，发票类别=增值税普票
2. 系统升级后：
- 功能开关自动开启
- 规则生效账套字段自动回填「全部账套」

使用步骤：
1. 员工提交日常报销单，费用类型=办公费，关联可抵扣发票
2. 无论在哪个账套生成凭证，可抵扣发票均按价税分离处理

结果：存量配置无需调整，行为保持一致
