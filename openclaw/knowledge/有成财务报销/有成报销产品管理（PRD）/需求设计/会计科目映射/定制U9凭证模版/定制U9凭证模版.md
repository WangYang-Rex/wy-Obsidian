以下为导出的模版示例，以sheet：凭证（新）这个里面的样式为准，导出模版按照这个sheet里面的样式生成，列与其中保持一致。     

U9凭证导入模板.xlsx (附件: attachments/U9凭证导入模板\_20250701\_144704.xlsx)

| **字段** | **对应报销中的值** |
|----------|-------------------------|
| 行状态 | 默认为“Update” |
| 凭证类别 | 当前凭证字号的“凭证字编码”位置：会计科目映射-凭证字管理 |
| 制单日期 | 当前凭证，生成凭证的时间，精确到日期 |
| 账簿 | 当前凭证对应账套的“账套编码”   位置：会计科目映射-账套编码 |
| 账簿版本 | 不填、为空 |
| 记账期间 | 取凭证日期的期间显示，展示到月份   比如凭证日期为2023 |
| 单号 | 取有成报销单据的单号 |
| 凭证（显示）号 | 凭证字号 |
| 行号 | 行号 |
| 摘要 | 摘要 |
| 币种 | 取当前凭证币种的币种ID位置：基础设置-币种设置 |
| 汇率 | 有外币时，凭证中有汇率，展示对应汇率若是本位币，则默认显示1， |
| 汇率类型 | 不填，为空 |
| 单价 | 填入0 |
| 借方数量 | 填入0 |
| 借方金额(原币) | 1、借方有值，则按照下面规则填入：凭证中有原币填入原币值只有一个本位币值，则原币=本位币，取本位币值填入2、借方没有值，填入0 |
| 借方金额(本币) | 1、借方有值，填入凭证记账金额2、借方没有值，填入0 |
| 贷方数量 | 填入0 |
| 贷方金额(原币) | 1、贷方有值，则按照下面规则填入：凭证中有原币填入原币值只有一个本位币值，则原币=本位币，取本位币值填入2、贷没有值，填入0 |
| 贷方金额(本币) | 1、贷方有值，填入凭证记账金额2、贷方没有值，填入0 |
| 科目.段1 | 取科目编码填入 |
| 科目.段2 | 填入编码：现金流量编码---指的客户辅助核算里面自建的现金流量编码，类别编码为07 |
| 科目.段3 | 填入名称：现金流量名称---指的客户辅助核算里面自建的现金流量名称 |
| 科目.段4 | **填入编码：有外部编码优先取外部 编码，没有则取辅助核算编码**   若关联的辅助核算为”**客户**“，则填入此列，填入编码客户取系统内默认的客户辅助核算类别匹配 |
| 科目.段5 | **填入编码：有外部编码优先取外部 编码，没有则取辅助核算编码**若关联的辅助核算为”**供应商**“，则填入此列供应商取系统内默认的供应商辅助核算类别匹配 |
| 科目.段6 | **填入编码：有外部编码优先取外部 编码，没有则取辅助核算编码**   若关联的辅助核算为”**部门**“，则填入此列部门取系统内默认的部门辅助核算类别匹配 |
| 科目.段7 | **填入编码：有外部编码优先取外部 编码，没有则取辅助核算编码**   若关联的辅助核算为”**职员**“，则填入此列职员取系统内默认的职员辅助核算类别匹配 |
| 科目.段9 | **填入编码：有外部编码优先取外部 编码，没有则取辅助核算编码**   若关联的辅助核算为”**项目**“，则填入此列部门取系统内默认的项目辅助核算类别匹配 |
| 科目.段8、科目.段10 | 不填充值，为空      若出现不属于特定含义的辅助核算列，则暂时先带入任何列 |
