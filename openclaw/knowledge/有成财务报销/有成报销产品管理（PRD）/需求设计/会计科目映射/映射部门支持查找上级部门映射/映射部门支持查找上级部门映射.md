# 映射部门支持查找上级部门映射

| **修订时间** | **版本** | **作者** | **修订说明** |
|----------------|----------|----------|----------------|
| 20220101 | v1.0 | 大武 | 新建 |

## 1、需求背景

用户组织架构大部门不常变动比较稳定，但是大部门下面的小部门，县市和门店相关会经常变动，日常会计科目映射只匹配到大部门即可，这样小部门变动不用再次编辑操作会计科目映射设置内容；

## 3、功能清单

根据个人需要补充

| **模块** | **功能点** | **备注** |

| 辅助核算映射 | 部门映射本级匹配不到就找上级 |  |
|  |  |  |

## 4、需求详情
1. **会计科目映射-基础设置增加配置**：映射部门匹配设置；可选项为：精确匹配和本级未找到时查找上级；单选；默认选择本级未找到时查找上级；当前用户设置精确匹配；
2. **选择精确匹配时**：现有逻辑，在匹配科目或者收支类别或者现金流时仅针对单据上面或者分摊的承担部门所选择的部门查找，查找不到则不匹配；
3. **选择本级未找到时查找上级时**：在匹配科目或者收支类别或者现金流时当单据上或者分摊上的部门进行查找，当找不到对应的部门时，则查找上一级部门，知道找到为止或者找到根节点也没有找到则停止匹配；当匹配规则没有部门时则不需要按照部门查找；
4. **适用范围**：生效范围为凭证科目映射、日记账收支类别映射、现金流科目映射、辅助核算部门映射；
