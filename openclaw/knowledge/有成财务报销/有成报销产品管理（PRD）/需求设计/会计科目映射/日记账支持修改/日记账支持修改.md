### 4.1、需求说明
- 日记账/应收应付日记账都支持编辑，在操作栏增加“编辑”功能，点击编辑，当前行次中的数据可以编辑；
- 点击“确认生成”，所有日记账记录需要保存好；

**日记账/应收应付日记账修改，可编辑的字段：**

| **字段** | **规则说明** |
|----------|----------------|
| 账套 | \+ 取账套列表，下拉筛选\+ 支持模糊搜索 |
| 日期 | 鼠标点击，显示日期组件，点击即选中 |
| 摘要 | \+ 文本输入框、必填 |
| 收支类别 | \+ 选项：取当前记录账套下的收支类别列表，下拉筛选\+ 若切换了账套，则收支类别取当前账套的默认收支类别-收入记录-取收入类别第一个值支出记录-取支出类别第一个值\+ 支持模糊搜索 |
| 银行账号 | \+ 选项：取当前记录账套下的账户管理，下拉筛选\+ 若切换了账套，则默认更改为当前账套的第一个账户\+ 支持模糊搜索 |
| 收入(借方) | \+ 数字输入框\+ 收入/支出只能有一方有值，且必须有一方有值 |
| 支出(贷方) | \+ 数字输入框、必填\+ 收入/支出只能有一方有值，且必须有一方有值 |
| 辅助核算 | \+ 选项，下拉筛选，当前辅助核算值所在类别下的所有辅助核算值\+ 支持模糊搜索 |
| 记账人 | 不支持修改 |

如果有必填项未填，则在点击保存的时候，将对应输入框标红提示。
