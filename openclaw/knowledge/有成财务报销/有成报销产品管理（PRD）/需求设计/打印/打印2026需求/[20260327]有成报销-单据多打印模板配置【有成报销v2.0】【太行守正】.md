## **1\. 需求背景**

### **1.1 需求描述**

支持一个单据类型配置多个打印模板，用户打印时可根据业务场景选择不同模板，满足同一单据在不同场景下的差异化打印需求。

### **1.2 需求背景**

**需求方**：财务人员/系统管理员

**现状问题**：
- 当前一个单据类型仅支持绑定一个打印模板
- 不同部门/场景对同一单据的打印格式要求不同（如财务留存版、业务审批版、外部报送版）
- 需要频繁切换模板或手动调整，效率低且易出错

**解决思路**：支持一个单据类型配置多个打印模板，可设置默认模板，打印时用户自行选择模板（不支持自动匹配）

**业务流程**
```plain
管理员配置多个打印模板
    ↓
用户打印单据时
    ↓
系统展示可用模板列表
    ↓
用户选择模板 → 打印
```

**模板列表操作**

| **操作** | **说明** | **限制** |
|----------------------------------------------------------|----------------------------------------------------------|----------------------------------------------------------|
| 编辑 | 进入模板编辑状态 | 无 |
| 新增 | 新增单据的打印模版 | 高级版打印才可新增打印模版 |
| 删除 | 删除模板 | 默认模板不可删除，需先取消默认 |
| 设为默认 | 设为该单据类型的默认打印模板 | 仅一个默认 |
| 复制 | 复制模板作为新模板基础 | 无 |
| 设计 | 进入模版设计器 | - |

**默认模板规则**
- 一个单据类型仅能设置一个默认模板
- 打印时如未指定模板，自动使用默认模板
- 删除默认模板前需先指定新的默认模板

**新增模板初始化规则**
- 新增模板时，**使用产品原逻辑**自动填充初始模板组件样式
- 初始组件包含：单据标题、申请人信息、费用明细表格、金额合计、审批签名区等标准组件
- 用户可在初始化基础上进行字段增删和样式调整
- **说明**：旧版打印设计器和高级版打印设计器的新增模板设计初始化逻辑保持一致，均使用产品原逻辑

**打印设计器版本权限控制**

有成报销支持两种打印设计器版本，**两个版本互斥使用，同一时间仅能使用一个版本**：

| **版本** | **开通方式** | **功能范围** | **使用状态** |
|----------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| 旧版打印设计器 | 免费，系统默认 | 基础模板编辑功能，单模板 | 默认启用 |
| 高级版打印设计器 | 付费开通 | 高级模板设计功能，支持多模板 | 开通后切换 |

**版本切换规则**
- **互斥原则**：旧版打印设计器和高级版打印设计器**不能同时使用**
- 未开通高级版时：使用旧版打印设计器，仅支持单模板
- 开通高级版后：自动切换至高级版打印设计器，支持多模板功能
- 高级版到期/退订后：自动回退至旧版打印设计器

**未开通高级版时的限制（旧版打印）**
- 进入单据模板管理界面时，仅显示默认打印模板
- **允许编辑默认模板**
- **允许设计默认模板**（进入模板设计器进行字段配置和样式调整）
- **不允许新增模板**（隐藏或禁用新增按钮）
- 页面顶部/空白区域显示开通引导信息：
    - 文案示例：「开通高级版打印设计器，支持多个打印模板配置，满足不同场景打印需求」
    - 操作按钮：「联系客服开通」
    - 点击后跳转客服会话或展示客服联系方式

**开通高级版后的功能（高级版打印）**
- 系统自动从旧版切换至高级版打印设计器
- **允许新增打印模板**：单据模板管理界面显示【新增模版】功能按钮，支持自由新增打印模板
- **允许编辑默认模板**
- **允许设计默认模板**（进入模板设计器进行字段配置和样式调整）
- **允许复制模板**：支持复制已有模板作为新模板基础
- **允许设置默认模板**：支持切换默认模板
- 隐藏开通引导信息
- **历史模板处理**：原有旧版默认模板隐藏，仅显示高级版的打印模版信息

**高级版退回旧版的业务逻辑** 当用户退订高级版或高级版到期时：

| **处理项** | **处理方案** |
|-------------------------------------------------------------|----------------------------------------------------------------|
| 版本切换 | 自动回退至旧版打印设计器 |
| 模板可见性 | 仅显示默认模板，其他模板隐藏但保留数据 |
| 单据打印逻辑 | **走旧版打印业务逻辑**：使用旧版默认模板打印，不弹出模板选择 |
| 模板编辑 | **允许编辑旧版默认模板**；**允许设计旧版默认模板**；禁止新增/复制/切换默认模板 |
| 数据保留 | 高级版模板数据保留，再次开通高级版后恢复可见 |
| 打印行为 | 无论单张还是批量打印，均使用默旧版认模板，不再支持模板选择 |

**存量用户处理**
- 已开通高级版的用户：功能不受影响，正常使用多模板功能
- 未开通高级版的用户：按上述限制规则执行
- 新注册用户：默认使用旧版打印设计器，需付费开通高级版
- 曾开通后退订的用户：回退至旧版，保留多模板数据但隐藏

**配置存储**
- 模板配置按单据类型
- 每个模板独立记录
- 默认模板标记
- ~~~~适用条件存储为JSON~~（本需求不涉及适用条件，由用户打印时自行选择）~~

| 批量打印-同一单据 | 1个模板 | 不弹出选择，直接使用该模板 |
| 批量打印-同一单据 | 多个模板 | 弹出模板选择弹窗，用户自行选择 |

#### **3.2.4 批量打印适配**

**功能概述** 批量打印时，根据选中单据的类型和数量，按以下规则处理：

**批量打印逻辑**

| **选中内容** | **处理方式** |
|----------------------------------------------------------------|----------------------------------------------------------------|
| 不同单据（不同单据ID） | 各单据使用各自的默认打印模板，无需用户选择 |
| 同一单据多张（相同单据ID） | 弹出模板选择弹窗，用户自行选择打印模板 |

**说明**：
- 不同单据打印时，因各单据可能属于不同单据类型，模板不可通用，故直接使用各自默认模板
- 同一单据多张打印时，因是同一单据类型，支持用户选择指定模板统一打印

## **3.3 逻辑互斥与边界处理**

### **边界情况处理**

| **边界场景** | **触发条件** | **处理方案** |
|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| 无默认模板 | 不存在没有默认模版的情况 |  |
| 模板被删除 | 正在使用的模板被删除 | 历史打印记录保留模板快照 |
| 单据无可用模板 | 不存在没有模版的情况 |  |
| 默认模板变更 | 切换默认模板 | 存量单据打印不受影响，使用当时默认模板 |
| 批量打印混合场景 | 同时选中不同单据和同一单据的多张 | 按单据ID分组，不同单据用各自默认，同一单据需选择 |
| 高级版到期/退订 | 用户退订或高级版到期 | 保留已创建的多个模板，**允许编辑默认模板、设计默认模板**，禁止新增/复制/切换模板；**单据打印自动走旧版业务逻辑** |
| 高级版开通前已有模板 | 历史数据中存在多个模板 | 保持现状，按权限规则控制新增/复制操作 |
| **高级版退回旧版打印** | **用户主动退订或到期未续费** | **系统切换至旧版打印设计器，单据打印使用默认模板，不弹出模板选择；允许编辑默认模板、设计默认模板，禁止新增/复制/切换模板，走旧版单模板业务逻辑** |

| **退回旧版逻辑** | **高级版退订后，单据打印自动走旧版业务逻辑（使用默认模板，不弹选择）** |
