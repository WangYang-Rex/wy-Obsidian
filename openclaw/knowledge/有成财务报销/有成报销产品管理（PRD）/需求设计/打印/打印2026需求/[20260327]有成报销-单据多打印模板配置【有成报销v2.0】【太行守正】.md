## **1\. 需求背景**

### **1.1 需求描述**

支持一个单据类型配置多个打印模板，用户打印时可根据业务场景选择不同模板，满足同一单据在不同场景下的差异化打印需求。

### **1.2 需求背景**
- **需求方**：财务人员/系统管理员
- **现状问题**：
• 当前一个单据类型仅支持绑定一个打印模板
• 不同部门/场景对同一单据的打印格式要求不同（如财务留存版、业务审批版、外部报送版）
• 需要频繁切换模板或手动调整，效率低且易出错
- **解决思路**：支持一个单据类型配置多个打印模板，可设置默认模板，打印时支持模板选择

**业务流程**
```plain
管理员配置多个打印模板
↓
为模板设置适用条件
↓
用户打印单据时
↓
系统展示可用模板列表
↓
用户选择模板 → 预览/打印
```

**模板列表操作**

| **操作** | **说明** | **限制** |
|----------------------------------------------------------|----------------------------------------------------------|----------------------------------------------------------|
| 编辑 | 进入模板编辑状态 | 无 |
| 删除 | 删除模板 | 默认模板不可删除，需先取消默认 |
| 设为默认 | 设为该单据类型的默认打印模板 | 仅一个默认 |
| 复制 | 复制模板作为新模板基础 | 无 |
| 设计 | 进入模版设计器 |  |

**默认模板规则**
- 一个单据类型仅能设置一个默认模板
- 打印时如未指定模板，自动使用默认模板
- 删除默认模板前需先指定新的默认模板

**配置存储** • 模板配置按单据类型

• 每个模板独立记录
• 默认模板标记

• 适用条件存储为JSON

#### **3.2.3 打印时模板选择**

**功能概述** 用户在单据详情页或列表页打印时，系统展示可用模板列表，用户可选择使用的模板。用户选择完打印配置项点击确定后，弹出选择打印模版弹窗；若该单据仅有一个打印模版，则不弹出弹窗，使用默认模版打印。

#### **3.2.4 批量打印适配**

**功能概述** 批量打印时，统一使用单据各自默认打印模板。

**批量打印逻辑**

| **选择方式** | **处理逻辑** |
|----------------------------------------------------------------|----------------------------------------------------------------|
| 统一使用匹配的默认模板 | 所有单据使用各自默认打印模版 |

### **3.3 逻辑互斥与边界处理**

**边界情况处理**

| **边界场景** | **触发条件** | **处理方案** |
|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| 无默认模板 | 不存在没有默认模版 | 单据至少有一个打印模版，该打印模版为初始默认打印模版 |
| 模板被删除 | 正在使用中的模板被删除 | 删除后该单据类型打印时回退到默认模板 |
| 存量数据兼容 | 存量单据类型已配置单模板 | 升级时将存量模板自动设为默认模板，功能行为不变 |
| 模板权限 | 用户无某模板的使用权限 | 打印下拉列表中隐藏该模板，如通过URL直接访问则提示"无权限使用此模板" |

**与其他功能的兼容性**

| **相关功能** | **交互关系** | **处理方案** |
|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| 审批流打印节点 | 关联配置 | 审批流打印节点可配置使用的打印模板；如未配置，使用默认模板 |
| 自动打印规则 | 关联配置 | 自动打印规则可配置使用的打印模板；如未配置，使用智能匹配或默认模板 |
| 批量打印 | 功能增强 | 批量打印支持按模板分组，或统一使用选定模板 |
| 移动端打印 | 功能同步 | MB端打印同样支持模板选择，界面适配移动端 |
| 自定义打印字段 | 依赖关系 | 多模板各自独立配置字段，互不影响 |
