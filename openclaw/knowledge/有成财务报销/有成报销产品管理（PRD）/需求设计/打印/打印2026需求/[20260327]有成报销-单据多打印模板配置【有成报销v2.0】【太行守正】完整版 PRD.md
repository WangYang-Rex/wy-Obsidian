## 1\. 需求背景

### 1.1 需求描述

支持一个单据配置多个打印模板，用户打印时可根据业务场景选择不同模板，满足同一单据在不同场景下的差异化打印需求。

### 1.2 需求背景
- **需求方**：财务人员/系统管理员
- **现状问题**：
• 当前一个单据仅支持绑定一个打印模板
• 不同部门/场景对同一单据的打印格式要求不同（如财务留存版、业务审批版、外部报送版）
• 需要频繁切换模板或手动调整，效率低且易出错
- **解决思路**：支持一个单据配置多个打印模板，可设置默认模板，打印时用户自行选择模板

**业务流程**
```
管理员选择单据 → 配置多个打印模板
  ↓
设置默认模板
  ↓
用户打印单据时
  ↓
系统根据场景判断是否弹出模板选择
  ↓
用户选择模板 → 打印

```

## 2\. 功能现状

**功能所在页面路径**：基础设置 → 自定义单据 → 选择单据 → 打印设置 → 打印模板维护

**现有产品层级**：
- 单据类型（如报销单、借款单）→ 单据（如普通报销单、差旅报销单）→ 打印模板
- 打印模板挂在**单据**维度，每个单据独立管理自己的打印模板

**现有字段**：
• 模板名称
• 设计器类型（普通/高级版）
• 模板内容配置

**现有操作能力**：
• 按单据配置单个打印模板（未开通高级版时）
• 开通高级版后支持配置多个打印模板
• 拖拽式字段配置
• 批量打印（50张以下直接预览）

**缺失能力**：
• 未开通高级版时，一个单据仅支持一个模板
• 无启用/禁用控制
• 无默认模板切换能力（未开通高级版时）

**模板列表操作功能**

| 操作 | 说明 | 未开通高级版 | 已开通高级版 |
|------|------|------------------|------------------|
| 编辑 | 进入模板编辑状态，可修改模板名称等基础信息 | ❌ 不可用 | ✅ 可用 |
| 设计 | 进入模板设计器，根据设计器类型进入对应设计器 | ✅ 可用 | ✅ 可用 |
| 删除 | 删除模板 | ❌ 不可用 | ✅ 可用（默认模板不可删除） |
| 设为默认 | 设为该单据的默认打印模板 | ❌ 不可用 | ✅ 可用 |
| 复制 | 复制模板作为新模板基础 | ❌ 不可用 | ✅ 可用 |
| 启用/禁用 | 切换模板的启用状态 | ❌ 不可用 | ✅ 可用（默认模板不可禁用，仅剩一个启用时不可禁用） |

**操作限制说明** • 默认模板不可删除：需先取消默认或设置其他模板为默认后方可删除
• 默认模板不可禁用：需先切换默认给其他启用的模板，才能禁用原默认模板
• 至少一个启用：单据下所有模板至少保留一个启用状态，当仅剩一个启用模板时该模板不可禁用
• 编辑限制：仅能编辑模板名称等基础信息，设计器类型不可修改
• 设计器路由：普通模板点击设计进入普通设计器，高级版模板点击设计进入高级版设计器
• 复制规则：复制时继承原模板的设计器类型，模板名称自动添加「（复制）」后缀，新模板默认启用但不设为默认
• 启用状态变更：禁用的模板在打印选择弹窗中不显示，批量打印时也不使用

**交互规则** • 未开通高级版时，点击【新增模版】或横幅【联系客服开通】→ 弹出「开通高级版打印设计器」弹窗（含权益说明和联系客服按钮）
• 开通高级版后，【新增模版】按钮点击后弹出设计器类型选择弹窗
• 设计按钮根据模板的设计器类型进入对应设计器（普通模板→普通设计器，高级版模板→高级版设计器）
• 普通设计器界面内不显示升级高级版入口
• 模板列表无预览按钮，不支持预览功能

**默认配置**

| 字段 | 默认状态 | 说明 |
|------|------------|------|
| 默认模板 | 每个单据仅一个 | 一个单据下仅能设置一个默认模板（全局唯一，不分普通/高级版） |
| 设计器类型 | 创建时确定 | 普通或高级版，创建后不可修改 |

#### 3.2.3 设为默认

**触发条件**：用户点击某模板的【设为默认】按钮

**规则**：
• 仅启用的模板可设为默认（禁用模板不展示该操作）
• 设为默认后，该模板成为该单据下唯一的默认模板
• 该单据下的原默认模板自动取消默认状态
• 不同单据之间互不影响

### 3.3 逻辑互斥与边界处理

**边界情况处理**

| 边界场景 | 触发条件 | 处理方案 |
|------------|------------|------------|
| 无默认模板 | 默认模板不允许被删除 | - |
| 模板被删除 | 正在使用的模板被删除 | 历史打印记录保留模板快照 |
| 单据无可用模板 | 单据至少有一个默认模版 | - |
| 默认模板变更 | 切换默认模板 | 存量单据打印不受影响，使用当时默认模板 |
| 未开通高级版点击新增 | 用户点击【新增模版】按钮 | 弹出「开通高级版打印设计器」弹窗（含联系客服按钮），不进入创建流程 |
| 未开通高级版点击联系客服 | 用户点击横幅【联系客服开通】 | 弹出「开通高级版打印设计器」弹窗 |
| 开通高级版后新增 | 用户点击【新增模版】按钮 | 弹出设计器类型选择弹窗 |
| 设计器类型选择取消 | 用户在选择弹窗点击取消 | 关闭弹窗，不创建模板 |
| 普通模板设计 | 点击普通模板的设计按钮 | 进入普通设计器 |
| 高级版模板设计 | 点击高级版模板的设计按钮 | 进入高级版设计器 |
| 模板设计器类型修改 | 用户尝试修改已创建模板的设计器类型 | 禁止修改，设计器类型创建时确定且不可变更 |
| 无预览功能 | 用户在模板管理或打印选择时 | 模板列表无预览按钮，选择弹窗无预览功能，需进入设计器查看效果 |
| 高级版到期/退订 | 高级版启用后不存在过期退订 | - |
| 默认模板不可禁用 | 用户点击默认模板的【禁用】 | 操作被禁止，提示"默认模板不可禁用，请先切换默认模板" |
| 仅剩一个启用模板 | 单据下所有模板中仅剩1个启用时尝试禁用 | 禁止禁用，至少保留一个可用模板 |
| 全部模板禁用 | 用户将所有模板都禁用 | 不允许，至少保留一个启用的模板 |
| 禁用模板打印 | 用户打印时 | 禁用的模板不出现在选择弹窗中，不可被选中 |
| 复制模板状态 | 复制模板时 | 新模板默认启用，不设为默认，不受原模板状态影响 |
| 模板启用状态初始化-未开通高级版（新用户） | 新企业未开通高级版 | 普通打印默认模板：启用 |
| 模板启用状态初始化-未开通高级版（存量用户） | 老企业未开通高级版 | 普通打印默认模板：启用（保持） |
| 模板启用状态初始化-已开通高级版（新用户） | 新企业首次开通高级版 | 普通打印默认模板：启用；高级版打印模板：启用，但不默认 |
| 模板启用状态初始化-已开通高级版（存量用户） | 老企业开通高级版 | 当前设计器正在使用的模板设为默认且启用；其他模板保持原有启用状态 |
| 后续开通高级版 | 已存在普通模板的单据开通高级版 | 保持原默认模板不变，不自动切换，由用户手动「设为默认」 |

**与其他功能的兼容性**

| 相关功能 | 交互关系 | 处理方案 |
|------------|------------|------------|
| 打印模板管理 | 核心关联 | 多模板配置入口，按单据维度独立管理 |
| 普通打印设计器 | 核心关联 | 未开通高级版时，默认可以使用普通设计器 |
| 高级版打印设计器 | 核心关联 | 开通高级版后可选择使用 |

| 有成中台 | 权限控制 | 中台启用高级版后，用户端功能解锁 |

## 4\. 验收标准

| 序号 | 验收项 | 验收标准 |
|------|---------|------------|
| 1 | 功能完整性 | 开通高级版后，支持一个单据配置多个打印模板 |
| 2 | 默认模板-唯一性 | 一个单据下仅能设置一个默认模板，不分普通/高级版 |
| 3 | 默认模板-跨单据独立 | 单据A的默认模板不影响单据B的默认模板 |
| 4 | 默认模板-不可禁用 | 默认模板不可禁用，需先切换默认才能禁用 |
| 5 | 模板选择 | 打印时弹出模板选择弹窗，展示所有启用模板，默认选中默认模板 |
| 6 | 模板复制 | 支持复制已有模板，新模板默认启用但不默认 |
| 7 | 批量打印 | 批量打印时各单据使用各自默认模板 |
| 8 | 未开通高级版-新增限制 | 未开通时点击【新增模版】弹出「开通高级版打印设计器」弹窗（含联系客服按钮） |
| 9 | 未开通高级版-联系客服 | 未开通时点击横幅【联系客服开通】弹出开通弹窗 |
| 10 | 未开通高级版-设计功能 | 未开通时允许设计默认模板，进入普通设计器，操作列仅展示「设计」 |
| 11 | 普通设计器界面 | 普通设计器界面内不显示升级高级版入口 |
| 12 | 开通高级版-新增模板 | 开通后点击【新增模版】弹出设计器类型选择弹窗 |
| 13 | 设计器类型选择 | 新增模板时可选择「普通打印设计器」或「高级版打印设计器」 |
| 14 | 模板设计器标记 | 模板列表显示设计器类型标记\[普通\]/\[高级版\] |
| 15 | 设计按钮路由 | 点击设计按钮根据模板类型进入对应设计器 |
| 16 | 设计器类型不可修改 | 已创建模板的设计器类型不可修改 |
| 17 | 多模板混合 | 允许同时存在普通模板和高级版模板 |
| 18 | 无预览功能 | 模板管理列表无预览按钮，打印选择弹窗无预览功能 |
| 19 | 单据类型约束 | 新增/编辑模板时，单据类型字段默认填充且不可修改 |
| 20 | 模板启用/禁用 | 支持启用/禁用模板，默认模板不可禁用，所有模板中仅剩一个启用时不可禁用 |
| 21 | 禁用模板不可打印 | 禁用的模板在打印选择弹窗中不展示，不可被选中 |
| 22 | 新用户-未开通高级版 | 普通打印默认模板初始状态为启用 |
| 23 | 新用户-已开通高级版 | 普通打印默认模板启用且为默认；高级版打印模板启用但不默认 |
| 24 | 存量用户-未开通高级版 | 普通打印默认模板保持启用 |
| 25 | 存量用户-已开通高级版 | 当前设计器正在使用的模板设为默认且启用；其他模板保持原有启用状态 |

## 附录

**A. 典型使用场景**

**场景：财务留存版 vs 业务审批版**

配置步骤（首次）：
1. 管理员进入「基础设置→自定义单据→选择单据→打印设置→打印模板管理」
2. 确认该单据已开通高级版（如未开通，先联系客服开通）
3. 点击【新增模版】，选择「高级版打印设计器」
4. 设计「财务留存版」模板（含详细字段：税额、科目等）
5. 再次点击【新增模版】，选择「高级版打印设计器」
6. 设计「业务审批版」模板（简化版面，关键信息展示）
7. 如需，点击「业务审批版」的【设为默认】将其设为默认模板

使用步骤（日常）：
1. 员工进入单据详情页，点击【打印】
2. 如存在多模板，系统弹出模板选择弹窗（无预览功能），默认选中默认模板
3. 员工根据场景选择「财务留存版」或「业务审批版」
4. 确认后进入打印流程

结果：同一单据支持不同场景的差异化打印

**B. 打印时模板选择场景对照表**

| 打印场景 | 单据 | 模板数量 | 用户操作 | 系统行为 |
|------------|------|------------|------------|------------|
| 单张打印 | 费用报销单 | 1个启用 | 点击打印 → 确定 | 直接使用默认模板，不弹出选择 |
| 单张打印 | 费用报销单 | 4个（2启用） | 点击打印 → 确定 → 选择模板 → 确定 | 弹出模板选择弹窗，仅展示启用的模板 |
| 批量打印 | 不同单据 | - | 选择多张 → 点击打印 | 各单据使用各自默认模板 |
| 批量打印 | 同一单据×3张 | 2个启用 | 选择3张 → 点击打印 → 选择模板 → 确定 | 弹出模板选择弹窗，默认选中默认模板 |
