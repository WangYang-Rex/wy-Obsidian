## 1\. 需求背景

### 1.1 需求描述

在现有打印设计器基础上，将页数显示从"代码写死、底部中央固定"升级为"打印模板级可配置"，支持管理员按模板独立设置页数的显示位置，满足企业对打印格式个性化的需求。

### 1.2 需求背景
- **需求方**：企业财务人员、系统管理员
- **现状问题**：
• 页数作为默认显示项，位置固定在代码中，所有单据打印出来都在底部中央
• 部分企业希望页数显示在右上角或右下角，与现有财务凭证格式对齐
• 无法按模板差异化配置，一刀切影响体验
- **解决思路**：
• 在打印模板设计中增加"页数位置"配置项
• 按模板独立设置，不同单据类型可配置不同位置
• 默认保持现有底部中央，兼容存量用户

**业务流程**
```text
管理员进入打印模板设计器
    ↓
在页眉页脚/打印设置区域配置页数位置
    ↓
    ├─ 底部中央（默认，兼容现有）
    ├─ 右上角
    └─ 右下角
    ↓
员工打印单据时按模板配置渲染页数位置

```

## 2\. 功能现状

### 2.1 打印模板现状

**位置**：基础设置 → 单据模板 → 打印模板管理 → 打印设计器

**现有能力**：
• 打印模板字段拖拽、样式设置
• 页眉页脚设置
• 打印内容选择（发票、审批流程等）

**现有缺失**：
• 页数位置不可配置，固定为底部中央
• 无按模板独立控制页数显示位置的能力

### 2.2 现有页数渲染逻辑
- 页数作为默认显示项，写死在打印渲染逻辑中
- 格式："第X页 / 共X页"
- 位置：页面底部水平居中
> 本次需求需在现有页数渲染逻辑中增加位置变量读取，根据模板配置动态定位。

### 3.2 详细功能描述

#### 3.2.1 打印设计器-页数位置配置

**功能概述**

在打印设计器的页眉页脚设置区域（或打印设置面板），新增"页数位置"配置项，管理员可按模板选择页数显示的位置。

**配置入口**

基础设置 → 单据模板 → 打印模板管理 → 编辑打印模板 → 右侧「打印设置」面板

**配置位置**

在现有「打印设置」面板中，「纸张边距」区域下方新增「页数位置」配置项：

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/yBRq1ZPkmd755Odv/img/c0c7eac6-a555-42a2-89d9-e4a3cb1b5889.png?Expires=1776848167&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=FR4349r17dhFNZsHh141N%2BZO1jI%3D "")
```text
┌─────────────────────────────────────────────┐
│ 打印设置                              [X]   │
├─────────────────────────────────────────────┤
│                                              │
│  标题设置：                                  │
│  [单据类型名称+单号                    ▼]   │
│                                              │
│  页面设置                                    │
│  [纵向                                 ▼]   │
│                                              │

│  ○ 右下角                                    │

│                          [取消] [保存]       │

```

**交互规则** • 默认选中"底部中央"，保持与现有逻辑一致
• 三个选项互斥，单选
• 配置保存后即时生效，仅对当前打印模板生效
• 页数的显示/隐藏逻辑不变（仍作为默认显示项，暂不支持隐藏）

**默认配置**

| 字段 | 默认状态 | 说明 |
|------|------------|------|
| 页数位置 | 底部中央 | 默认保持现有逻辑，兼容存量用户 |

**配置存储** • 配置按打印模板维度存储
• 新增字段 `page_number_position`，枚举类型
• 与现有「打印设置」中的标题设置、页面设置、纸张边距、水印设置同属一个配置组
• 存量打印模板默认值：底部中央

#### 3.2.2 打印渲染-页数位置动态渲染

**渲染规则**

| 模板配置 | 渲染位置 | 说明 |
|------------|------------|------|
| 底部中央 | 页面底部水平居中 | 现有逻辑，默认 |
| 右上角 | 页面右上角，与页面边缘保持标准边距 | 距上边缘、右边缘各保持页边距距离 |
| 右下角 | 页面右下角，与页面边缘保持标准边距 | 距下边缘、右边缘各保持页边距距离 |

**渲染效果示例**

底部中央：
```text
┌─────────────────────────────────────┐
│                                     │
│           [单据内容区域]             │
│                                     │
│           第1页 / 共1页              │
└─────────────────────────────────────┘

```

右上角：
```text
┌─────────────────────────────────────┐
│                     第1页 / 共1页   │
│           [单据内容区域]             │
│                                     │
│                                     │
└─────────────────────────────────────┘

```

右下角：
```text
┌─────────────────────────────────────┐
│                                     │
│           [单据内容区域]             │
│                                     │
│                     第1页 / 共1页   │
└─────────────────────────────────────┘

```

**样式要求** • 字号、字体与现有页数保持一致
• 仅改变位置，不改变页数的显示格式和内容
• 当页数位置为右上角/右下角时，需确保不与页眉/页脚内容发生重叠

### 3.3 逻辑互斥与边界处理

**边界情况处理**

| **边界场景** | **触发条件** | **处理方案** |
|----------------|----------------|----------------|
| 存量打印模板无配置 | 历史模板未设置过页数位置字段 | 默认按"底部中央"渲染，保持现有展示 |
| 页眉内容与页数重叠 | 页数设为右上角，但页眉内容过长 | 页数优先渲染在页眉区域下方，避免重叠；或允许微调垂直偏移（若技术成本允许，首期可简单处理为各自独立区域） |
| 页脚内容与页数重叠 | 页数设为右下角，但页脚内容过长 | 页数优先渲染在页脚区域上方，避免重叠 |
| 多页打印 | 单据内容超过一页 | 每页均按模板配置的页数位置渲染 |
| 单页打印 | 单据内容仅一页 | 正常显示"第1页 / 共1页"，位置按配置 |

**与其他功能的兼容性**

| **相关功能** | **交互关系** | **处理方案** |
|----------------|----------------|----------------|
| 打印模板复制 | 复制模板时页数位置配置一并复制 | 复制后保留原模板的页数位置设置 |
| 打印模板导出/导入 | 模板数据包含页数位置字段 | 导出/导入时同步携带该字段 |
| 批量打印 | 使用各单据的默认模板 | 按各自模板配置的页数位置渲染 |
| 移动端打印 | 打印输出 | 页数位置逻辑与 PC 端保持一致 |

## 4\. 前后端工作任务

### 4.1 前端负责内容
- 【打印设计器】在右侧「打印设置」面板中「纸张边距」下方新增"页数位置"单选配置项
- 【样式适配】确保右上角/右下角页数显示不与页眉页脚内容重叠

### 4.2 后端负责内容
- 【打印模板模型】新增 `page_number_position` 字段，枚举：bottom\_center / top\_right / bottom\_right
- 【模板保存/查询接口】支持页数位置字段的存储和读取
- 【打印渲染接口】根据模板配置的页数位置，动态计算页数在 PDF/打印输出中的坐标位置
- 【存量数据迁移】为所有历史打印模板设置默认值 bottom\_center

## 5\. 验收标准

| **序号** | **验收项** | **验收标准** |
|----------|-------------|----------------|
| 1 | 配置入口 | 管理员可在打印设计器的页眉页脚/打印设置区域看到"页数位置"配置项 |
| 2 | 默认状态 | 新建打印模板和存量打印模板的页数位置默认均为"底部中央" |
| 3 | 底部中央 | 配置为底部中央时，打印输出与现有效果完全一致 |
| 4 | 右上角 | 配置为右上角时，页数显示在页面右上角 |
| 5 | 右下角 | 配置为右下角时，页数显示在页面右下角 |
| 6 | 按模板独立 | 不同打印模板可配置不同的页数位置，互不影响 |
| 7 | 模板复制 | 复制打印模板时，页数位置配置一并复制 |
| 8 | 多页渲染 | 多页单据打印时，每页页数均按配置位置正确显示 |
| 9 | 移动端一致 | 移动端打印输出的页数位置与 PC 端配置一致 |
| 10 | 存量兼容 | 所有历史打印模板在未改动配置的情况下，打印效果保持原有底部中央 |

## 附录

**A. 典型使用场景**

**场景：企业将页数配置到右下角**

配置步骤：
1. 管理员进入「基础设置→单据模板→打印模板管理」
2. 选择"日常报销单-标准打印版"，点击编辑
3. 在打印设置中找到"页数位置"
4. 切换为"右下角"
5. 保存模板

使用结果：
• 员工打印日常报销单时，页数显示在页面右下角
• 其他单据的打印模板不受影响，仍按各自配置展示

**B. 字段说明**

| 字段 | 类型 | 枚举值 | 说明 |
|------|------|---------|------|
| page\_number\_position | VARCHAR(20) | bottom\_center / top\_right / bottom\_right | 页数显示位置 |
