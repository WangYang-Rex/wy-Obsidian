## **一、数据汇总卡片**

```json
SELECT COALESCE(SUM(budget_amount), 0) AS value FROM corp_budget_report_analysis WHERE corp_id = '${corpId}' AND report_id = ${reportId};
```

## **二、分析项**

### **分析项1：月度费用趋势分析**
- **位置**：第2行，第1列
- **分析目的**：识别费用随时间的变化趋势，掌握费用发生的周期性规律   **图表描述**：使用折线图，X轴为月份（从“单据日期”字段提取年月信息，格式化为“YYYY-MM”），Y轴为月度总费用金额（按月份分组对“费用金额”求和，如果存在分摊情况，“费用金额”已为分摊后金额，直接参与计算）。连接各月数据点形成趋势线，按时间顺序排序以确保连续性。
```json
SELECT TO_CHAR(bill_date, 'YYYY-MM') AS month, SUM(amount) AS monthly_amount FROM corp_expense_analysis WHERE corp_id = '${corpId}' AND report_id = ${reportId} AND bill_date BETWEEN '${beginTime}' AND '${endTime}' GROUP BY TO_CHAR(bill_date, 'YYYY-MM') ORDER BY month;
```


### **分析项2：预算使用率分析**
- **位置**：第2行，第2列
- **分析目的**：评估预算的使用效率
- **图表描述**：使用柱状图，X轴为预算管理名称，Y轴为百分比，显示预算已使用金额除以预算总额再乘100%。数据来自预算报表，通过汇总各预算管理名称的预算已使用金额和预算总额。
```json
SELECT budget_name, COALESCE(SUM(budget_amount), 0) AS total_budget, COALESCE(SUM(budget_used_amount), 0) AS total_used, CASE WHEN COALESCE(SUM(budget_amount), 0) = 0 THEN 0 ELSE ROUND(SUM(budget_used_amount) * 100.0 / SUM(budget_amount), 2) END AS usage_rate FROM corp_budget_report_analysis WHERE corp_id = '${corpId}' AND report_id = ${reportId} GROUP BY budget_name ORDER BY usage_rate DESC;
```


### **分析项3：**项目净利润排行榜（TOP 10）

- **分析目的**：识别盈利能力最强项目，聚焦核心利润贡献者项目。
- **图表描述**：使用**柱状图**。左侧Y轴为净利润金额。
    - **图表（TOP 10）**：X轴为“项目名称”（按净利润降序取前10），Y轴为对应项目的“净利润”。

```json
SELECT project_name, profit_amount FROM corp_project_profit_report_analysis WHERE report_id = ${reportId} AND corp_id = '${corpId}' ORDER BY profit_amount DESC LIMIT 10;
```


### **分析项4：**部门费用对比分析
- **位置**：第3行，第2列
- **分析目的**：比较各部门费用支出，识别费用管理重点部门   **图表描述**：使用柱状图，X轴为部门名称（优先使用“分摊部门”字段，如为空则使用“单据部门”作为备用），Y轴为总费用金额（按部门分组对“费用金额”求和）。柱子高度反映各部门费用规模，按金额降序排列以突出重要部门。按照费用金额总额，部门数量不超过100个，剩余合计为其他。
```json
WITH departments AS (SELECT COALESCE(department_path_name, '未知部门') AS department_name, SUM(amount) AS department_amount FROM corp_expense_analysis WHERE corp_id = '${corpId}' AND report_id = ${reportId} AND bill_date BETWEEN '${beginTime}' AND '${endTime}' GROUP BY COALESCE(department_path_name, '未知部门')), ranked_departments AS (SELECT department_name, department_amount, ROW_NUMBER() OVER (ORDER BY department_amount DESC) as rn FROM departments) SELECT CASE WHEN rn <= 100 THEN department_name ELSE '其他' END AS department_name, SUM(department_amount) AS department_amount FROM ranked_departments GROUP BY CASE WHEN rn <= 100 THEN department_name ELSE '其他' END ORDER BY department_amount DESC;
```


### **分析项5：预算申请单追加金额分析**
- **位置**：第4行，第1列
- **分析目的**：分析预算申请单的追加金额情况，识别预算追加较多的领域
- **图表描述**：使用柱状图，X轴为预算管理名称，Y轴为追加金额总和。数据来自预算申请单，通过按预算管理名称分组汇总追加金额。
```json
SELECT budget_name, COALESCE(SUM(budget_adjust_amount), 0) AS total_adjust_amount, COUNT(DISTINCT bill_number) AS bill_count FROM corp_budget_apply_bill_analysis WHERE corp_id = '${corpId}' AND report_id = ${reportId} AND budget_adjust_amount > 0 GROUP BY budget_name ORDER BY total_adjust_amount DESC;
```


### **分析项6：****交易汇率影响分析**
- **位置**：第4行，第2列
- **分析目的**：分析所使用汇率的分布情况，评估汇率波动对账面价值的影响。
- **图表描述**：使用**柱状图**。基于“资金明细”中。X轴为“交易币种”，Y轴为“交易汇率”。可计算每种币种交易汇率的平均值或展示主要汇率区间，反映入账汇率水平。
```json
SELECT trade_currency_code AS currency_code, ROUND(AVG(trade_exchange_rate), 4) AS avg_exchange_rate FROM corp_fund_detail_analysis WHERE corp_id = '${corpId}' AND report_id = ${reportId} AND trade_date >= TO_TIMESTAMP('${beginTime}', 'YYYY-MM-DD HH24:MI:SS') AND trade_date < TO_TIMESTAMP('${endTime}', 'YYYY-MM-DD HH24:MI:SS') AND trade_exchange_rate > 0 GROUP BY trade_currency_code ORDER BY avg_exchange_rate DESC;
```


### **分析项7：**超标单据分析
- **位置**：第5行，第1列
- **分析目的**：评估费用超标情况，识别合规风险点   **图表描述**：使用饼图，两个扇形分别表示“超标”和“未超标”（基于“费用是否超标”字段，将其转换为二进制值如“是/否”），扇形大小对应单据数量占比（按超标状态分组计数“单号”作为唯一标识，计算占比：超标单据数/总单据数×100%）。突出超标部分。
```json
WITH bill_exceed_status AS (SELECT bill_number, MAX(expense_is_exceed) AS bill_exceed_status FROM corp_expense_analysis WHERE corp_id = '${corpId}' AND report_id = ${reportId} AND bill_date BETWEEN '${beginTime}' AND '${endTime}' AND expense_id IS NOT NULL GROUP BY bill_number), exceed_summary AS (SELECT CASE WHEN bill_exceed_status = 1 THEN '超标' ELSE '未超标' END AS exceed_status, COUNT(*) AS bill_count FROM bill_exceed_status GROUP BY CASE WHEN bill_exceed_status = 1 THEN '超标' ELSE '未超标' END), total_bills AS (SELECT COUNT(DISTINCT bill_number) AS total_count FROM corp_expense_analysis WHERE corp_id = '${corpId}' AND report_id = ${reportId} AND bill_date BETWEEN '${beginTime}' AND '${endTime}' AND expense_id IS NOT NULL) SELECT es.exceed_status, es.bill_count, ROUND(es.bill_count * 100.0 / tb.total_count::numeric, 2) AS percentage FROM exceed_summary es CROSS JOIN total_bills tb ORDER BY es.exceed_status DESC;
```


### **分析项8：**项目费用分析
- **位置**：第5行，第2列
- **分析目的**：分析各项目费用支出，优化项目成本管理   **图表描述**：使用柱状图，X轴为项目名称（优先使用“分摊项目”字段，如为空则使用“费用项目”，如归为空则使用单据项目作为备用），Y轴为项目总费用（按项目分组对“费用金额”求和）。按费用金额降序排列，重点关注高费用项目。按照费用总金额，项目不超过100个，剩余合计为其他。
```json
WITH projects AS (SELECT COALESCE(project_name, '无项目') AS project_name, SUM(amount) AS project_amount FROM corp_expense_analysis WHERE corp_id = '${corpId}' AND report_id = ${reportId} AND bill_date BETWEEN '${beginTime}' AND '${endTime}' GROUP BY COALESCE(project_name, '无项目')), ranked_projects AS (SELECT project_name, project_amount, ROW_NUMBER() OVER (ORDER BY project_amount DESC) as rn FROM projects) SELECT CASE WHEN rn <= 100 THEN project_name ELSE '其他' END AS project_name, SUM(project_amount) AS project_amount FROM ranked_projects GROUP BY CASE WHEN rn <= 100 THEN project_name ELSE '其他' END ORDER BY project_amount DESC;
```
