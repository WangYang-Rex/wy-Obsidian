## 1、需求背景

有成报销这边提的加班、请假、外出、出差的申请单，没有同步到飞书考勤，导致飞书用户目前需要（有成报销、飞书）两边提单

## 2、业务流程/架构

接口地址：[<u>https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/reference/attendance-v1/user\_approval/query</u>](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/reference/attendance-v1/user_approval/query)

对接飞书假勤审批接口，将考勤类申请单（请假、外出、加班、出差）同步至飞书，并支持在飞书考勤统计表中查看

### 4.2、应用中心

#### 开通服务
1. 应用中心增加应用模块【飞书假勤】，默认为未开通状态
2. 点击“联系客服开通”，展示客服电话。（后续客服帮助用户创建飞书自建应用，并在后台维护好相关参数进行开通）

3. 开通后，隐藏“开通”按钮，展示“管理配置”、“停用”


![Picture 2](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/eLbnj1bxy2N1rlaN/img/43fc7d9e-ac34-4532-98ce-be862634769c.png?Expires=1774156457&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=yN5dFgs01JSpmqqoK9wr7nEfBZU%3D "")

#### 配置管理

点击“配置管理”跳转页面，支持配置同步到飞书的单据类型/单据同行人假勤数据自动同步/假勤时间设置


![Picture 3](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/eLbnj1bxy2N1rlaN/img/ee97ffca-5ffe-4419-b1b3-577161800c31.png?Expires=1774156457&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=vhOIZGT%2Bz1zVZql%2B5pHfv%2BDWxjo%3D "")

##### 默认样式：开关全部打开状态

##### 配置说明

###### 同步单据选择范围：所有申请单

###### 申请单配置同步单据时，需要对单据模板进行必填字段校验，提示校验结果，并支持跳转到单据表单配置页面进行快捷配置
- **单据模板校验字段：**

| **申请单类型** | **必填字段** |
|-------------------|----------------|
| 加班 | **1、申请人2、起止时间（常用字段“日期”&日期类型：起止日期）3、加班时长（常用字段“常用数字”）** |
| 请假 | **1、申请人2、起止时间（常用字段“日期”&日期类型：起止日期）3、假期时长（常用字段“常用数字”）4、请假理由（常用字段“描述”）** |
| 出差 | **1、申请人2、起止时间（常用字段“日期”&日期类型：起止日期）3、出差理由（常用字段“描述”）4、申请时间** |
| 外出 | **1、申请人2、起止时间（常用字段“日期”&日期类型：起止日期）3、外出时长（常用字段“常用数字”）4、外出理由（常用字段“描述”）5、外出类型（常用字段“常用下拉框”）** |
- **同步单据中存在必填字段未配置时：**


![Picture 4](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/eLbnj1bxy2N1rlaN/img/b2fda2b1-003c-4691-8e67-21f94ad1879d.png?Expires=1774156457&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=1tlMhssveiwt8%2BtfDoGOX9Q12Lg%3D "")

\- 文案：「单据名称」未配置「必填字段」，无法同步至飞书假勤，请先完成单据字段设置\- 取消：取消该单据的添加\- 去设置：跳转至自定义单据页
- **单据模板示例：**

外出申请单-外出类型设置的选项需与飞书中配置的外出类型保持一致

![Picture 5](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/eLbnj1bxy2N1rlaN/img/e99ff090-25c6-4a48-b837-1244f9abdcbb.png?Expires=1774156457&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=2%2Bv8AdZDtB9H5OJjkRwWLPT44ZE%3D "")

###### 加班申请单同步：开启“申请单同步到飞书假勤”，展示以下三个配置项
- 需要同步的单据：默认为空，支持多选（逻辑同其他申请单同步配置）
- 加班日期类型：选项包括：工作日/休息日/节假日；默认选项：工作日（**对应飞书“category”加班类型参数**）
- 加班规则设置：选项包括：无/计为调休/计为加班费/按加班规则计算；默认选项：无（**对应飞书“type”加班规则类型参数**）

| **有成** | **飞书** |
|----------|----------|
| 无 | 0:不关联加班规则 |
| 计为调休 | 1:调休 |
| 计为加班费 | 2:加班费 |
| 按加班规则计算 | 3：关联加班规则，没有调休或加班费 |

###### 同行人员自动同步假勤：开启后，单据中的同行人需要生成对应的假勤数据

###### 保存配置单据数据后，关闭开关然后再次开启，单据配置数据要保留

###### 假勤时间设置：对上午/下午时间段进行设置；必填

#### 停用
4. 点击“停用”，弹出二次确认弹窗


![Picture 6](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/eLbnj1bxy2N1rlaN/img/67415e15-e1d8-4173-8494-ccf66d97f807.png?Expires=1774156457&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=%2FJm%2B11b1vJ5JvuYpKHZ2d5t6vb8%3D "")
5. 确认停用后，隐藏“管理配置”、“停用”按钮，展示“开通”按钮
6. 停用后再次开通，原配置数据保持不变

### 4.4、其他逻辑及参数说明

#### 同步到飞书假勤的“加班、请假、出差、外出”数据可在飞书考勤统计报表中查看
