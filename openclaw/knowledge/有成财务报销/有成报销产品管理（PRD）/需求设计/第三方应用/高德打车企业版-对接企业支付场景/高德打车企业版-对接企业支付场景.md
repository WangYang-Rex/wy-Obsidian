### 4.2、应用中心-高德打车

开通流程、员工管理、后台管理功能都不变，在原基础上做以下调整

#### 管理配置-申请单同步

点击“管理配置”跳转新页面，页面中保留原“默认类型配置”，新增“**有成报销申请单同步到高德**”开关

**有成报销申请单同步到高德**：

##### 默认关闭状态；关闭时，隐藏申请单设置项（如下所示）



![Picture 1](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jzk7dY8lvD/img/3065550b-3dc6-4551-852f-1dd0336f4192.png?Expires=1775016753&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=0K32v%2FXTZHdw3XfkxuI1ZiidSzA%3D "")

##### 开启后，自定义单据自动创建一个“高德打车申请单”模板；并展示申请单设置项

1\. \*\*申请单总限额\*\*：默认为空；非必填（为空时，则不限制）2\. \*\*每笔订单限额\*\*：默认为空；必填；若“申请单总限额”有值，“每笔订单限额”不能超出“申请单总限额”；超出时提示：“每笔订单限额”不允许超出“申请单总限额”3\. \*\*申请单总计可用车次数\*\*：默认为空；非必填（为空时，则不限制）4\. \*\*是否允许跨城用车：\*\*1. 选项包括：允许跨城、不允许跨城、仅临近城市允许跨城2. 默认选项：允许跨城3. \当选择“仅临近城市允许跨城”时，才展示“允许跨城用车名单”；\</span\>未维护跨城数据时，缺省提示：暂未维护城市数据；点击“新增”，列表增加一行，点击“选择城市”展示城市选择组件（单选）；支持删除；

<span style="color: #E4495B;">注：开启开关，维护申请单配置数据保存后，关闭开关-再次开启，原配置数据需保留

##### 高德打车申请单

“有成报销申请单同步到高德”开启开关，在“自定义单据-申请单”类型下，自动创建一个申请单「**高德打车申请单**」

申请人手机号机校验：提交单据时，对申请人手机号进行校验，当手机号为空时，提示：**请联系系统管理员维护好手机号，并完成高德打车信息同步**



![Picture 3](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jzk7dY8lvD/img/61a71fb6-0bf4-430b-851a-f4a649033bde.png?Expires=1775016753&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=ZlodDhrtRAS92fu59GDIvXGQrHM%3D "")

##### 特殊字段对应关系

| **高德打车字段** | **有成报销字段** | **备注** |
|----------------------|----------------------|----------|
| sceneName申请单场景入口展示名称 | 表单字段：申请单名称 |  |
| totalQuotaAmount申请单总限额(分) | 管理配置项：申请单总限额 | 注意进行单位换算元-分 |
| customSceneLabel企业自定义申请单名称 | 固定值：差旅打车 |  |
| startTime权限开始时间 | 审批完成（通过）时间 |  |
| amountLimited单笔限额 | 管理配置项：单笔订单限额 | 注意进行单位换算元-分 |
| allowCrossCity | 管理配置项：是否允许跨城用车 |  |
| whiteCityGroup | 管理配置项：允许跨城用车名单 |  |
| applyTotalCount申请单总计可用车次数 | 管理配置项：申请单总计可用车次数 |  |
