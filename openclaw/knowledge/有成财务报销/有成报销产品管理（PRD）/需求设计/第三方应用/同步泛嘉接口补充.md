# **1\. 需求背景**

现状：

目前系统已经与泛嘉打通，会传输相应的单据信息给到，但是本次泛嘉需要收到发票抬头的信息，需要我们在接口补充提供

# **2\. 业务流程/架构**

# **3\. 功能清单**

| 模块 | 功能点 | 备注 |
|------|---------|------|
| 接口 | 补充发票抬头字段 |  |
|  |  |  |

# **4\. 需求详情**

### **4.1接口补充**

在应用中心--泛嘉国际--管理配置中，申请单补充设置的发票抬头，新增一个“发票抬头”的一级选项

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/Q35O851AGxvgVl9V/img/9566dea6-c78a-48c3-991a-438b0e8f6b1b.png?Expires=1781014998&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=FW7u1tEoyj99KcMBKaEYkOWO7nA%3D "")

选择后，在对应单据同步时，会校验单据中是否存在“发票抬头”组件，如果存在会将该组件信息中的“抬头名称”传给泛嘉

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/Q35O851AGxvgVl9V/img/03e808bc-c838-4e2d-bc5b-85af1e89d859.png?Expires=1781014998&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=bIJ9S1hmkEwO5x9yog35JRTzLcI%3D "")

【对应泛嘉存储字段：invoiceTitle】
