#### 新增



![Picture 2](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/Q35O851bp77kEl9V/img/237fc785-5ef7-4be0-b159-dbd900acfd1e.png?Expires=1779974858&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=qta%2BCVd4ZBI1tnrPphnFkBLLfWE%3D "")

点击【新增】，弹窗展示【新增商品】

| **字段名称** | **说明** |
|----------------|----------|
| 商品名称 | 1、文本输入，max=282、必填3、不可重复 |
| 规格型号 | 1、文本输入，max=182、非必填 |
| 单价 | 1、金额输入，小数点后6位2、非必填 |
| 单位 | 1、文本输入，max=112、非必填 |
| 含税标识 | 1、默认值为 含税，可下拉切换【不含税】2、必填 |
| 税收分类编码 | 1、点击展示【税收分类编码】弹窗，选择税收分类编码2、必填3、系统默认19位 |
| 税编简称 | 1、根据税收分类编码自动带出2、必填 |
| 是否享受优惠政策标识 | 1、默认否，可切换为【是】2、必填 |
| 优惠政策类型 | 1、当【是否享受优惠政策标识】为【否】时，不可编辑当该标识为【是】时，可下拉选择【优惠政策类型】2、枚举值：01-简易征收,02-稀土产品,03-免税,04-不征税,05-先征后退,06-100%先征后退,07-50%先征后退,08-按3%简易征收,09-按5%简易征收,10-按5%简易征收减按1.5%计征,11-即征即退30%,12-即征即退50%,13-即征即退70%,14-即征即退100%,15-超税负3%即征即退,16-超税负8%即征即退,17-超税负12%即征即退,18-超税负6%即征即退 |
| 税率 | 1、默认为0，可下拉切换枚举值：2、必填 |
| 免税类型 | 1、是否享受优惠政策标识=否，税率=0时，免税类型为普通零税率，税率不为0时，免税类型为正常税率2、是否享受优惠政策标识=是，优惠政策类型=【免税】，此时税率只能为0，免税类型=免税3、是否享受优惠政策标识=是，优惠政策类型=【不征税】，此时税率只能为0，免税类型=不征税4、2、是否享受优惠政策标识=是，优惠政策类型 不等于【免税或者不征税】，此时税率为0，免税类型 默认选择【普通零税率】，可下拉切换为【免税、不征税】；税率不为0时，免税类型默认选择【正常税率】5、必填 |
| 备注 | 1、文本输入，max=2002、非必填 |

![Picture 3](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/Q35O851bp77kEl9V/img/55598d6f-abbf-460e-94d8-fc531b599777.png?Expires=1779974858&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=pxOc8KJiionZMHGGAktHurzDEU8%3D "")

点击【确定】：

1、判断必填项是否填写完整，未填写完整，则对应输入框下方提示

2、必填项填写完整后，判断商品编码是否已经存在，如果已经存在则提示：该编码已经存在，请先修改后再操作

3、校验通过后，则保存成功，弹窗关闭，列表页面刷新

#### 编辑

编辑时字段展示和【新增】保持一致，校验逻辑也一致

#### 导入



![Picture 4](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/Q35O851bp77kEl9V/img/25c082c1-6f27-4142-bf4f-8e074ffba7d2.png?Expires=1779974858&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=TXDk0bvyO6fokWnL4lXXwVRVmQU%3D "")

导入时需要先下载模板商品导入模板.xlsx (附件: attachments/商品导入模板\_20250701\_142846.xlsx)

导入时判断：

1、商品编码、商品名称、税收分类编码、是否享受优惠政策、优惠政策类型、税率、免税类型、含税标识 为必填，如果有必填项未填写，则提示：第X行第Y列没有填写完整

3、税收分类编码需要满足税局分类

4、是否享受优惠政策标识和优惠政策类型需要匹配

5、报错要一次性提示

6、数据检查无误后，导入成功，toast提示：导入成功

#### 导出

同系统单据管理的导出逻辑，导出全部/导出选中

#### 删除

点击【删除】，二次弹窗确认，是否删除数据



![Picture 5](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/Q35O851bp77kEl9V/img/9b059588-ac90-4d23-924b-55509ae3e6d5.png?Expires=1779974858&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=ZhToo23mZTc%2BDBdxbqDPnkWMy%2FU%3D "")

点击【确定】，将该商品标记为删除，商品列表刷新

1、对于已经添加到开票申请单中的商品，删除该商品，不影响申请单的数据

2、删除后，开票申请单无法在选择该商品

#### 批量删除

点击【批量删除】：

1、未选中商品，则toast提示：请先选择要删除的商品

2、选中商品，二次确认弹窗：



![Picture 6](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/Q35O851bp77kEl9V/img/8007220d-e9f3-4362-acaa-bfe60aa41170.png?Expires=1779974858&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=1Kj6%2F%2BVkwy8FpczhoQm8%2FAo9L9o%3D "")

点击【确定】，将选中的商品标记为删除。商品列表刷新

1、对于已经添加到开票申请单中的商品，删除该商品，不影响申请单的数据

2、删除后，开票申请单无法在选择该商品
