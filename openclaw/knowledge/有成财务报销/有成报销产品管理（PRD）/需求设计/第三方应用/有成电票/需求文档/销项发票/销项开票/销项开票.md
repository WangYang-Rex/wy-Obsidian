# 1、涉及系统

有成报销 &有成电票 PC & MB

# 2、需求背景

1、提升竞争力：竞品已经做了开票系统，为了与竞争对手保持竞争力并满足市场需求，所以我们也需要发票系统来完善我们的生态圈，并提供高效便捷的发票处理方案

2、业务闭环和集成：将发票系统与现有的业务系统相结合，可以实现业务流程的闭环，提高整体的效率。

3、提高销售客单价：发票系统既可以作为一个独立的系统，也可以作为报销系统和crm的延伸，增加销售机会，提高客单价，从而增加销售人员的收入

# 3、名词解释

| **名词** | **说明** |
|----------|----------|
| 全电票 | 去介质：无需ukey，无需税控盘去版式：xml文件，无行数限制、无销货清单、无长度限制赋码制：无需申领发票号段，直接开票即是系统唯一发票号码授信制：无需事前申请金额、无需票种核定，全电和税盘使用同一个额度标签化：通过标签对电子发票功能、用途、状态的具体分类，要素化：基本要素：纳税人信息&商品信息等特定要素：特别行业&特别品类等附加要素：自己填写的附加要素 |
| 冲红 | 冲红是指对已经开出的发票进行作废或者更正的过程，比如开票有误、销售退回、应税服务终止等情况，就需要开具红字发票普通发票可直接开红票，专票需要填写红字信息表纸质发票可以选择冲红，也可以选择作废电子发票（包括全电发票和税控电票）和机动车增值税发票、成品油发票只能冲红不可作废购方未认证时不需要确认销售方可直接开票 |
| 作废 | 作废时指将已开具的发票或者未填写内容的发票（空白发票）进行无效化仅纸质发票和空白发票可选择作废，纸质发票未跨月且未抄报时才可以作废空白发票作废：为了防止未授权的使用、遗失或者盗窃后的被滥用 |
| 发票类型 | 纸质发票：增值税普通发票、增值税专用发票电子发票：增值税电子普通发票、增值税电子专用发票、电子发票（普通发票）、电子发票（专用发票）税控发票：增值税普通发票、增值税专用发票、增值税电子普通发票、增值税电子专用发票全电发票：电子发票（普通发票）、电子发票（专用发票） |
| 发票小助手 | 供应商提供插件，监控税控设备在线状态 |
| 打印 | 1、发票打印2、开票码打印需要打印插件，打印样式需要自己处理，目前快麦有相关的打印插件 |
| 税率 | 专票不可享受优惠政策且税率不可为0全电专不允许开免税/零税率 |
| 差额征收 | 差额征收只能针对单个商品，不支持折扣和清单开具税控和全电都支持差额征收征收类型和特定业务不可同时存在 |
| 小规模纳税人 | 开具3%的发票时需要选择原因：1、小规模纳税人放弃减按1%征收率原因 ：2： 前期已开具发票，发生销售折让、中止或者退回等情形需要开具红字发票，或者开票有误需要重新开具 3 ：因为实际经营业务需要，放弃享受减按1%征收率征收增值税政策 |
