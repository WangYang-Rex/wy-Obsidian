# 通过部门主管设置审批人，支持多选

| **修订时间** | **版本** | **编写人** | **修订说明** |

## 1、需求背景

客户：河南省哼哼猪企业管理咨询有限公司

背景：实际客户配流程时有这样的要求：一个审批节点可以设置提交人主管和费用承担部门主管或签 

现状：当前按照部门主管设置审批人，只支持设置单个条件

## 2、业务流程/架构

如复杂需求，放置相关流程/架构图

## 3、功能清单

根据个人需要补充

|----------|-------------|----------|

| 出纳节点-会签/或签 |  |  |
|  |  |  |

## 4、需求详情

### 4.1、系统：【有成报销】

### 4.2、通过部门主管匹配：支持多选

##### 需支持的节点类型及审批方式
1. 审批节点-会签/或签/依次审批
2. 出纳节点-会签/或签

##### 交互样式
3. 支持一级/二级多选；
4. 勾选保存后，选择结果较多时，换行展示所有选择结果；
5. 保存勾选结果后，取消“通过部门主管匹配”勾选，再次勾选时，之前选择的数据不变
