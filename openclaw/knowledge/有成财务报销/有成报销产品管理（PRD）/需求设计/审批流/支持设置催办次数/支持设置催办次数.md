# 支持设置催办次数

| **修订时间** | **版本** | **编写人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 20230822 | v1.0 | 小乔 | 新建 |

## 1、需求背景

目前审批流支持对节点设置是否允许提交人催办，但没有次数限制。审批人经常收到多次催办消息，对此比较困扰，希望可以设置允许催办的次数。

## 2、业务流程/架构

如复杂需求，放置相关流程/架构图

## 3、功能清单

根据个人需要补充

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 自定义审批流 | 针对审批节点，支持设置最大催办次数 |  |
| 单据详情 | 催办按钮根据催办次数限制规则 & 已催办次数 展示/隐藏 |  |
|  |  |  |

## 4、需求详情

### 4.1、系统：【有成报销】

### 4.2、自定义审批流
1. 针对**审批节点**进行设置时，当“允许提交人催办”为选中状态时，展示两个单选项（如原型），支持设置限制的最大催办次数；不勾选时样式同当前（不展示单选项）

**不限制催办次数**：默认选项；不对催办次数进行限制

**允许催办次数不超过（）次**：选中时，才可对次数进行编辑；

### 4.3、单据详情—催办

单据详情中催办按钮，根据节点中 **催办次数限制规则** & **实际已催办次数**  进行展示/隐藏
