# 审批节点增加节点备注展示

| **修订时间** | **版本** | **编写人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 20230323 | v1.0 | 小乔 | 新建 |

## 1、需求背景

1、部分企业在定义审批流时，为了给审批人提供审批参考，需要针对审批节点给出该节点备注（一般为当前节点审批人职责或审批规范，内容较长）。



![Picture 1](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/ZWGl05mdKA3BMn34/img/0ae69e8a-1cb8-4b34-8cde-e1986db32c11.png?Expires=1778766782&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=6onYZ6ScSGj4mAz4JxrLQhelouY%3D "")

2、目前审批节点只支持定义最多10个字的名称，在审批节点设置中增加「节点备注」字段，并在审批流中展示节点备注

## 2、业务流程/架构

## 3、功能清单

根据个人需要补充

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 自定义审批流 | 审批节点设置中增加「节点备注」 |  |
| 提交审批 | 审批流程确定弹窗-展示「节点备注」信息 |  |
| 单据打印模板 | 审批流程打印内容-增加「节点备注」 |  |
| 单据详情 | 审批历史-展示「节点备注」信息 |  |
| 单据详情 | 审批流程-展示「节点备注」信息（PC & MB） |  |

## 4、需求详情

### 4.1、系统：【有成报销】PC & MB

### 4.2、自定义审批流
1. **审批节点设置中增加「节点备注」**

自定义审批流-审批节点设置页面，增加「节点备注」字段（历史数据需要刷上该字段）

节点备注：默认为空；非必填项；字数限制最多不超过50字



![Picture 2](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/ZWGl05mdKA3BMn34/img/490b2f78-6614-4835-9328-7a572da8a709.png?Expires=1778766782&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=K1J6t9JSspa%2BPnLTp6fvceb52tQ%3D "")

### 4.3、提交审批-审批流程确定
2. PC端：

提单确定审批流弹窗中，节点名称后展示节点备注；较长时默认展示一行，其他用“...”代替，支持鼠标移入展示全部



![Picture 3](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/ZWGl05mdKA3BMn34/img/93e91798-a8ba-4d13-9237-d8085487ee6b.png?Expires=1778766782&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=8dRk1lYO8fS7B2T%2FBa7hbggkRIk%3D "")
3. MB端：

提单确定审批流弹窗中，节点名称后展示「节点备注」；较长时默认展示一行，支持进行展开/收起操作



![Picture 4](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/ZWGl05mdKA3BMn34/img/a56c1d18-413f-4475-8118-3ca4c690abc1.png?Expires=1778766782&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=U3BcNlAxMY4RFoTlqkylvMUIUho%3D "")

### 4.4、单据打印模板
4. **当前旧版打印**：样式调整：「审批流」「审批人展示」选项平铺展示审批流：仅支持单选审批人展示选项：支持复选两个设置项之间的交互逻辑审批流选择：列表展示审批人展示选项包括：审批人、审批节点、节点备注、审批时间（不可取消勾选）、审批意见（不可取消勾选）审批流选择：平铺展示审批人展示选项包括：审批人、审批节点审批流选择：审批历史隐藏“审批人展示”设置项注：历史数据需要对应进行新样式更新



![Picture 5](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/ZWGl05mdKA3BMn34/img/dac7d6aa-907f-4cdf-9839-e065de80e4ce.png?Expires=1778766782&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=DbHXMHyFpLMBELHKD4w1jvLAhC0%3D "")
5. **高级自定义打印**：系统字段—审批信息中增加「节点备注」字段（这个可以后续再调整）

### 4.5、单据详情
6. **审批历史-展示「节点备注」信息**

节点备注展示在原节点信息下方；默认展示一行，超出部分用...代替，支持鼠标移入展示全部



![Picture 6](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/ZWGl05mdKA3BMn34/img/55c5c412-2119-4e96-929d-6438c84b3395.png?Expires=1778766782&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=%2FtY3sAql2omqrcjHho6sATGYyiQ%3D "")
7. **审批流程-展示「节点备注」信息**

PC端：

1）审批流程：展示在审批人下方，最多展示一行，超出部分用...代替，支持鼠标移入展示全部

2）历史操作记录：同上方审批历史展示样式



![Picture 7](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/ZWGl05mdKA3BMn34/img/f1790f40-4211-4b15-b8bf-c65d5111e39c.png?Expires=1778766782&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=Hhfhymd84UmHBK2f0cP8Uwv56qI%3D "")

MB端：

1）审批流程：展示在节点名称下方，并与节点名称、审批人区分展示；较长时换行展示全部内容

2）历史操作记录：展示在节点名称下方，较长时默认展示一行，支持进行展开/收起操做



![Picture 8](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/ZWGl05mdKA3BMn34/img/67883c6d-778f-4f38-84f9-975d9ce473d0.png?Expires=1778766782&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=3FMdpl5m0vLGw9iZqUfpWCprTiQ%3D "")
