## 1、需求背景

评论目前添加之后就不允许修改和删除，如果用户有错误的情况下没办法修改；增加删除功能方便用户修改评论；

客户--都市花香；

|  |  |  |

## 4、需求详情

### 4.1、评论增加删除

PC端和手机端的评论添加后，增加删除按钮，点击删除按钮，直接将评论删除；

删除按钮控制权限，仅给评论添加人展示，其他人不可见删除按钮；

删除权限无论哪个位置打开，只要是自己添加的评论都可删除；

### 4.2 操作日志

评论删除需要记录操作日志，后期可查；
1. 操作模块增加--单据详情
2. 删除评论日志记录：操作模块：单据详情；操作内容：删除单据单号中评论“评论内容”；例：删除单据BX2012987665567中的评论“评论功能测试”；操作人：删除人名字；操作时间：删除的时间；
