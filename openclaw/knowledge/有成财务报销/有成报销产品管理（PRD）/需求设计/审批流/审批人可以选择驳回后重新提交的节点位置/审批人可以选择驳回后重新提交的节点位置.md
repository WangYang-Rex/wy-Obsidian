# 审批人可以选择驳回后重新提交的节点位置

| **修订时间** | **版本** | **作者** | **修订说明** |
|----------------|----------|----------|----------------|
| 20230614 | v1.0 | 大武 | 新建 |

## 1、需求背景

目前驳回后提交的位置是审批流中直接设置好，客户需要审批人根据情况自行选择驳回后到哪里；

## 2、业务流程/架构

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 自定义审批流-高级设置 | 高级设置增加审批人选择驳回后的重新提交位置 |  |
|  |  |  |
|  |  |  |

## 4、需求详情

### 4.1、审批流高级设置驳回重新提交增加审批人选择
1. 自定义审批流-高级设置-驳回后重新提交到增加选项：**审批人选择节点**；



![Picture 1](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/a2QnV4j05pLxyO4X/img/49b8a376-c318-483f-94b1-33af8a62576c.png?Expires=1775017135&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=e5OOAblgsWYFfmNbr09RiCRVvIo%3D "")

### 4.2 审批人选择驳回后节点
2. 审批人进入单据详情点击驳回按钮的驳回弹窗中增加配置：**驳回后提交人可重新提交到**；
3. 可选选项：开始节点、当前节点、其他节点；必填项，第一次为空，后续**需记录当前登录人上次的选项**；
4. 选择其他节点的话出现已审批过的节点名称进行选择；
5. 选择后，则提交人重新提交时直接跳到选择的节点进行后续的审批；
