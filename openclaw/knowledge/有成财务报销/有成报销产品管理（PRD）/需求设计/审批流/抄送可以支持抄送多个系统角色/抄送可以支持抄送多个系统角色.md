## 1、需求背景

客户：深圳市旭达卓远教育集团有限公司

背景：设置抄送对象时，希望选择多个系统角色进行抄送

## 2、业务流程/架构

## 3、功能清单

| **模块** | **功能点** | **备注** |

| 自定义审批流 | 【抄送】-通过系统角色匹配 |  |
|  | 【审批】-会签/或签-通过系统角色匹配 |  |
|  | 【出纳】-会签/或签-通过系统角色匹配 |  |

## 4、需求详情

### 4.1、需求说明1

**系统：**【有成报销】

**系统路径**：基础设置-自定义审批流

**方案**：节点【抄送】【审批】【出纳】规则设置时，「通过系统角色匹配」可添加多条设置

**1、具体位置**
- 【抄送】节点——抄送设置：通过系统角色匹配
- 【审批】节点——审批方式：会签/或签——设置审批人： 系统分配-通过系统角色匹配    &  抄送设置：通过系统角色匹配
- 【出纳】节点——审批方式：会签/或签——设置审批人： 系统分配-通过系统角色匹配    &  抄送设置：通过系统角色匹配
