# 有成财务管理后台增加更新公告

| **修订时间** | **版本** | **修订说明** |
|----------------|----------|----------------|
| 20220101 | v1.0 | 新建 |
| 20220801 | v2.0 | 大模块升级改造，新老架构合并后 |

## 1、需求背景

有成报销目前没有系统更新公告功能，增加对应功能模块；

## 2、业务流程/架构

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 管理后台 | 管理后台增加系统公告的内容管理 |  |
| 报销消息中心 | 消息中心展示系统公告的内容 |  |
|  |  |  |

## 4、需求详情

### 4.1、需求说明
1. 有成财务后台-系统更新公告-增加有成报销，有成报销的更新公告对有成财务V2版和有成报销以及独立版、飞书和企微版均生效；
2. 系统公告增加配置链接的功能，如果有配置链接则在系统内点击消息首先打开链接的页面；
3. 没有配置链接则打开公告内容生成的页面；
