# **1\. 需求背景**

有些发票票面没有税额，但是可以计算抵扣进项税的，系统提供了“可抵扣税额”的解决方案，但是某些功能上还没有加上或者没有兼容这个字段，有客户反馈需要一并加上。

可抵扣税额的历史需求：

[语雀](https://gykj.yuque.com/izoyhv/tzh23k/zk0ue0kua24c6acy#YD7Kf)

北京汇智有识等

|  |  |  |

# **4\. 需求详情**

## **自定义报表，支持取可抵扣税额**

自定义报表-数据源类型为费用时：
- 增加字段选择“可抵扣税额”，取费用上税率税额组件中的可抵扣税额。
- 字段选择-自定义字段计算对象中也同样新增“可抵扣税额”字段。

## **单据新增、查看详情，消费组件显示支持选择可抵扣税额**

单据新增，字段显示中增加“可抵扣税额”，选中确定后，列表展示和卡片展示的都要展示出费用上的可抵扣税额字段。
