# 单据详情增加关联单据（被关联）展示

| **修订时间** | **版本** | **修订说明** |
|----------------|----------|----------------|
| 20221110 | v1.0 | 新建 |

## 1、需求背景

目前单据关联仅能看到单据主动关联了哪些单据，但是看不出单据被哪些单据关联，没有溯源；

## 2、业务流程/架构

## 3、功能清单

根据个人需要补充

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 单据详情 | 增加关联单据tab |  |
| 关联单据 | 选择单据的列表增加已被关联标记 |  |
|  |  |  |

## 4、需求详情

#### 1\. 单据详情增加关联单据tab

**关联单据**：关联单据tab中对于关联单据直接取关联单据组件中的内容进行展示；如果有收付款是关联了合同的也需要展示；发票核销关联的付款单也需要展示；

**被以下单据关联**：单据作为被动方，被其他单据关联，此处展示关联它的单据；

项目中的关联单据tab需要展示所有关联到此项目的单据，树形结构的项目管理中打开项目详情也需要增加；

列表需要做分页；
