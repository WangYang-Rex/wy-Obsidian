# 辅助核算路径展示优化

| **修订时间** | **版本** | **修订说明** | **作者** |
|----------------|----------|----------------|----------|
| 20221205 | v1.0 | 新建 | 昭觉 |

## 1、涉及系统

有成报销 PC 

## 2、业务背景

目前全局设置的辅助核算路径展示只处理了单据详情的展示，单据列表和导出依然是全路径，没有跟随配置

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 我的-单据&单据管理&待办&项目与合同&凭证日记帐 | 1、列表中辅助核算字段展示2、单据管理列表导出时辅助核算字段展示3、生成凭证--单据列表辅助核算字段展示 |  |
| 导出 | 1、单据列表导出2、报表中单据列表导出 |  |
| 打印 |  |  |
| 预算报表 | 查看预算占用单据 |  |
| 项目报表 | 单据列表 |  |
| 自定义报表 | 数据源为单据或者费用时辅助核算展示 |  |
