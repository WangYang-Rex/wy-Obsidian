### **1\. 需求背景**

#### **1.1 需求描述**

草稿箱中的已驳回、已撤回单据，支持在费用明细组件内直接修改费用类型，与新建草稿单据保持一致的编辑体验。

#### **1.2 需求背景**
- **需求方：** 报销申请人
- **现状问题：**
    - 新建单据存入草稿箱：费用类型**可编辑** ✓
    - 驳回单据回到草稿箱：费用类型**置灰** ✗
    - 撤回单据回到草稿箱：费用类型**置灰** ✗
    - 三者同在草稿箱，编辑权限不一致，用户体验割裂
- **解决思路：** 统一草稿箱内所有单据状态的费用类型编辑权限

**业务流程对比：**
```plain
【现状 - 草稿箱内操作】
新建草稿 → 费用类型可编辑 ✓
驳回单据 → 费用类型置灰 ✗ → 被迫删除重新创建
撤回单据 → 费用类型置灰 ✗ → 被迫删除重新创建

【变更后 - 草稿箱内操作】
新建草稿 → 费用类型可编辑 ✓
驳回单据 → 费用类型可编辑 ✓
撤回单据 → 费用类型可编辑 ✓
```

| **单据来源** | **状态** | **费用类型编辑** | **删除消费释放** |
|----------------------------------------------------------------|----------------------------------------------------------|----------------------------------------------------------------------|----------------------------------------------------------------------|

| 撤回后回到草稿箱 | 已撤回 | **✗ 置灰（需变更）** | **彻底删除，不释放** |

### **3\. 本次变更内容**

#### **3.1 功能变更清单**

| **变更类型** | **功能点** | **功能说明** | **优先级** |
|----------------------------------------------------------------|-------------------------------------------------------------|----------------------------------------------------------------|-------------------------------------------------------------|
| 修改 | 驳回状态编辑权限 | 已驳回单据费用明细费用类型开放编辑 | P0 |
| 修改 | 撤回状态编辑权限 | 已撤回单据费用明细费用类型开放编辑 | P0 |

#### **3.2 详细功能描述**

##### **3.2.1 草稿箱费用类型编辑统一**

**功能概述** 统一草稿箱内三种来源单据（新建、驳回、撤回）的费用类型编辑权限，全部开放可编辑。

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/jP2lRYj8d1X8JO8g/img/be7e2d97-9178-46cf-9d60-404bc03d88af.png?Expires=1778767261&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=S%2BqHi%2FFK3Zh6AXfHYnal4p0Trx8%3D "")

豁免情况：
- 第三方商旅系统订单信息推送到有成报销生成的消费信息，单据被驳回或撤回后回到草稿箱，费用类型 置灰 ✗
- 有成报销内里程补贴打卡定位功能产生的行程信息生成的消费，单据被驳回或撤回后回到草稿箱，费用类型 置灰 ✗



**变更前后对比：**

| **状态** | **变更前** | **变更后** |
|----------------------------------------------------------|-------------------------------------------------------------|-------------------------------------------------------------|
| 草稿（新建） | 可编辑 ✓ | 可编辑 ✓ |
| 已驳回 | **置灰** | **可编辑** |
| 已撤回 | **置灰** | **可编辑** |

**交互设计**
```plain
【变更前】
费用类型：[差旅费]  ← 置灰不可点

【变更后】
费用类型：[差旅费 ▼]  ← 可点击下拉
```



**编辑规则：**
- 点击后弹出费用类型选择器（与新建草稿一致）
- 支持搜索筛选费用类型
- 选择后立即保存至单据
- 消费信息费用类型同步更新

##### **3.2.2 校验与联动（延迟触发）**

**提交前：** 仅修改字段值，不触发任何校验 **提交时：** 统一触发以下校验

| **校验项** | **触发时机** | **处理方案** |
|-------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| 费用标准校验 | 提交时 | 按新费用类型重新校验，超标提示 |
| 预算占用 | 提交时 | 释放原费用类型预算，占用新费用类型预算 |
| 审批流匹配 | 提交时 | **不重新匹配**，保持原审批流 |
| 条件分支 | 提交时 | 重新计算条件分支 |
