# 图片组件支持上传9张以上的图片

| **修订时间** | **版本** | **编写人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 20230511 | v1.0 | 小乔 | 新建 |

## 1、需求背景

背景：用户在进行报销时，一个报销单可能有多笔费用，需要上传多张付款截图，目前我们是限制只能上传9张图片，无法满足这种场景的图片上传需求。

## 2、业务流程/架构

## 3、功能清单

根据个人需要补充

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 图片组件 | 上传图片数量不做限制（不隐藏图片上传按钮） |  |
| 单据/费用详情 | 可展示更多图片 |  |
|  |  |  |

## 4、需求详情

### 4.1、系统【有成报销】PC & MB

### 4.2、【图片】——上传图片数量不做限制

图片数量：不做限制

交互：不管上传图片数量多少，不隐藏“上传图片”按钮；多张图片另起一行展示

### 4.3、单据/费用详情——展示所有上传的图片

PC：点击“更多”，展示所有已上传图片（多张图片换行展示）

MB：图片字段展示全部图片（换行展示）
