# 费用明细-字段显示-应用到所有员工，增加二次确认和操作日志

| **修订时间** | **版本** | **编写人** | **修订说明** |
|----------------|----------|-------------|----------------|
| 20231204 | v1.0 | 小乔 | 新建 |

## 1、需求背景

客户：财开

需求：1）子表单表头字段调整并应用到所有员工，这个操作增加二次确认；2）增加操作日志

## 2、业务流程/架构

如复杂需求，放置相关流程/架构图

## 3、涉及系统

**有成报销**

## 4、功能清单

根据个人需要补充

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 单据 | 单据新增、编辑、查看页面，费用明细子表单 表头字段展示-应用到所有员工 操作增加二次确认 |  |
| 系统设置-操作日志 | 增加“应用到所有员工”操作日志 |  |
|  |  |  |

## 5、需求详情
1. 单据新增/编辑/单据详情页面，修改费用明细表头字段展示时，若当前操作人为系统管理员，修改的字段将被应用到所有员工时：
2. **勾选应用到所有员工**
3. **修改字段显示，点击确定时，应用到所有员工已经被勾选上**

以上两种情况，增加一个二次弹窗确认，文案内容为：勾选后，该表单的字段显示内容将同步到所有员工，是否确认修改？

4. **系统设置-操作日志：增加操作记录**
5. 操作模块：归属于“单据详情”
6. 当字段显示操作针对全员生效时，记录这种操作记录：新增字段显示：【员工姓名】新增了【单据类型】中的字段显示【字段名称1、字段名称2】，并应用到所有员工取消字段显示：【员工姓名】取消了【单据类型】中的字段显示【字段名称1、字段名称2】，并应用到所有员工

示例：张三新增了差旅报销单中的字段显示消费城市、发票金额，并应用到所有员工
