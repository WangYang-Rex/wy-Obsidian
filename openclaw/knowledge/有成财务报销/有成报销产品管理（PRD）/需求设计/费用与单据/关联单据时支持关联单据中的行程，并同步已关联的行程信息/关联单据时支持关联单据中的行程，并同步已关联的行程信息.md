## 1、需求背景

客户：财开

背景：报销单关联出差申请单进行报销，申请单中已经维护了行程信息，希望报销单在关联申请单时，也能同步已关联申请单中的行程，提高提单效率

## 2、业务流程/架构

如复杂需求，放置相关流程/架构图

## 3、功能清单

根据个人需要补充

|----------|-------------|----------|
| 自定义单据 | 「关联单据」组件支持关联行程 |  |
| 行程组件 | 同步已关联的行程信息 |  |
| 单据详情 | 关联单据信息中，可查看关联行程详情 |  |

## 4、需求详情

### 4.1、系统【有成报销】PC & MB

### 4.2、自定义单据-关联单据组件支持关联行程
1. 关联单据组件中，增加“关联行程”列，选中关联单据后，点击“编辑”按钮可关联被关联单据中的行程信息


![Picture 1](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5bz1PExvlxo/img/949f2ae5-7b0a-49dc-8b5d-eafc2f11913f.png?Expires=1774157320&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=fgOvipkx4BmxUcNeRLNNQSL8jCE%3D "")
2. 选择行程信息，保持提交后，关联单据列表行程列展示：已关联行程条数；点击弹出选择行程弹窗


![Picture 2](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5bz1PExvlxo/img/6dc03c68-5aaf-41d6-99f6-94e9b4868ba2.png?Expires=1774157320&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=MhazcYIdUZU3a%2BVyDk9CI%2Bl29%2B0%3D "")

### 4.3、单据行程组件-同步已关联的行程信息

**当单据中存在“行程”组件，并且在关联单据时关联了被关联单据的行程，需要将关联的行程信息同步到单据“行程”组件**

例如：单据A存在行程组件，单据A关联单据B，并关联了单据B的行程a，那么行程a将同步带入到单据A的行程组件中

### 4.4、单据详情-关联单据信息中，可查看关联行程详情

**单据详情——关联单据tab页，关联单据列表可查看已关联的详情信息**

关联单据中关联了行程信息时，展示已关联行程条数，并支持点击查看具体关联行程的详情；若未关联行程，用“--”代替展示
