# 选择收款账户时，支持按不同账户类型分开展示

| **修订时间** | **版本** | **编写人** | **修订说明** |

| 20230324 | v1.0 | 小乔 | 新建 |

## 1、需求背景

有些单据类型，收款账户只会选择个人账户或对公账户等指定类型的账户，为了方便选择，需要在选择收款账户时，能按照不同的账户类型分开展示

|  |  |  |
|  |  |  |

## 4、需求详情

### 4.1、系统：【有成报销】PC & MB

### 4.2、PC端/MB端

1、搜索栏增加账户类型搜索项（PC/MB样式如下）

2、默认：全部账户类型

3、根据选择的账户类型，对收款账户进行类型过滤展示
