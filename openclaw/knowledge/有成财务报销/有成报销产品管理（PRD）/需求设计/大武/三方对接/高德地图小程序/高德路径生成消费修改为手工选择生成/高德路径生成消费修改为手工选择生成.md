# 高德路径生成消费修改为手工选择生成

| **修订时间** | **版本** | **作者** | **修订说明** |
|----------------|----------|----------|----------------|
| 20230412 | v1.0 | 大武 | 新建 |

## 1、需求背景

目前自动生成模式在用户路径中断后又继续生成的会丢失部分路径信息；手工生成可以对路径进行更新，防止数据丢失；

## 2、业务流程/架构

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 里程补贴 | 增加历史路径查看 |  |
|  | 增加手工生成消费 |  |
|  |  |  |

## 4、需求详情

### 4.1、增加查看历史路径
1. 新版里程补贴页面增加“查看历史路径”按钮；
2. 点击进入历史路径页面，参看下图二；
3. 点击“生成消费”可以将对应的路径生成消费；生成成功给出提示“生成成功，可以到我的消费中进行查看”；失败给出失败原因；生成之后刷新当前页面；
4. 路径按照时间倒序排列，最新的在最前面；
5. 已报销的不能再生成消费；（已报销为已经生成消费就算已报销）；
6. 生成的消费删除需要将路径恢复未报销状态展示在历史路径中；
7. 途径点增加文案提示“起终点和途径点可以不填哦”
