# 里程补贴支持员工自己选择是否需要导航

| **修订时间** | **版本** | **作者** | **修订说明** |
|----------------|----------|----------|----------------|
| 20230413 | v1.0 | 大武 | 新建 |

## 1、需求背景

场景：部分用户在跑周边熟悉的路的时候不需要开导航，原本的点位记录更方便，所以希望用到导航的时候才开导航，用不到的时候就还是使用点位记录；

## 2、业务流程/架构

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 里程补贴 | 新版里程补贴增加手工记录点位入口 |  |
| 移动端首页 | 首页增加待生成消费tab展示未生成消费的第三方数据（目前只有里程补贴） |  |
|  |  |  |

## 4、需求详情

### 4.1 新版里程补贴增加手工记录点位的入口
1. 新版里程补贴开启成功之后进入新版页面增加按钮「手工点位记录」；点击回到老版里程补贴页面；
2. 老版里程补贴页面在高德地图导航已经开启的情况下，需要隐藏掉高德开启的入口；
3. 查看历史路径的入口调整到高德提图小程序图标下方；
4. 在老版点位记录一段行程结束并且生成消费之后，再次进入到里程补贴，则打开还是新版页面；
