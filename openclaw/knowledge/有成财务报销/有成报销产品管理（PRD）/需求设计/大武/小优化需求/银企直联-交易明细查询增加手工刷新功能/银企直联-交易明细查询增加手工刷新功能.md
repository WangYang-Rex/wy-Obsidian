# 银企直联-交易明细查询增加手工刷新功能

| **修订时间** | **版本** | **作者** | **修订说明** |
|----------------|----------|----------|----------------|
| 20230209 | v1.0 | 大武 | 新建 |

## 1、需求背景

银行交易明细会存在获取失败的场景，目前失败只能技术去处理，增加手工刷新功能，客户可以自己重新发起查询；

## 2、业务流程/架构

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 有成财务-应用中心-银企直联 | 交易明细查询增加手工刷新 |  |
|  |  |  |
|  |  |  |

## 4、需求详情
1. 应用中心-银企直联-交易明细增加【刷新】按钮；
2. 点击【刷新】按钮则重新去查近三天的交易明细，查询结果需过滤重复数据；
3. 点击【刷新】按钮后后端请求需控制频次，上次结果未返回之前不重新发起新的请求；
