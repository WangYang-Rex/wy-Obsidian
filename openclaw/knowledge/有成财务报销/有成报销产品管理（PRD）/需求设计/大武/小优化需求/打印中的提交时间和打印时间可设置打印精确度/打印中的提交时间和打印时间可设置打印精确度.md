# 打印中的提交时间和打印时间可设置打印精确度

| **修订时间** | **版本** | **作者** | **修订说明** |
|----------------|----------|----------|----------------|
| 20230209 | v1.0 | 大武 | 新建 |

## 1、需求背景

用户打印出的时间只需要精确到日就可以，目前是精确到时分秒；

## 2、业务流程/架构

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 打印设置 | 提交时间和打印时间增加精确度配置 |  |
|  |  |  |
|  |  |  |

## 4、需求详情

打印设置中的提交时间和打印时间增加配置：精确到时间；

此配置在对应的开关打开的时候才展示，默认勾选；

取消勾选之后则打印只需要精确到日即可；
