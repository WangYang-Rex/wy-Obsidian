# 审批节点增加配置审批人选择后续审批人

| **修订时间** | **版本** | **作者** | **修订说明** |
|----------------|----------|----------|----------------|
| 20230420 | v1.0 | 大武 | 新建 |

## 1、需求背景

用户审批流中某些节点强制需要审批人选择，不允许申请人选择；

## 2、业务流程/架构

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 自定义审批流 | 手动选择审批人增加“由审批人选择”的选项 |  |
|  |  |  |
|  |  |  |

## 4、需求详情
1. 审批节点-手动选择增加选项：由审批人选择；
2. 选择后则申请人不允许选择节点的审批人，由前一节点的审批人进行后续审批节点的选择；
3. 加入节点为申请人发起后的第一个节点，则给出提示“审批人为空，且不允许申请人选择，请联系管理员修改配置后再提交”；
