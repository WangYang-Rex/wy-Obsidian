# 凭证日记账生成凭证增加不再生成凭证功能

| **修订时间** | **版本** | **作者** | **修订说明** |
|----------------|----------|----------|----------------|
| 20230210 | v1.0 | 大武 | 新建 |

## 1、需求背景

目前所有需要生成凭证未通过系统生成凭证的单据都会在选择单据的时候出现，但是有些客户有些单据会提前手工在财务系统已经做好凭证了，但是在报销里面会一直存在这个单据未做凭证；容易造成误操作，导致凭证重复；

## 2、业务流程/架构

## 3、功能清单

根据个人需要补充

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 财务管理-凭证日记账 | 生成凭证-选择单据增加不再生成凭证功能 |  |
|  |  |  |
|  |  |  |

## 4、需求详情
1. 凭证日记账-生成凭证-选择单据，勾选对应的单据后出现按钮【不再生成凭证】；
2. 点击不再生成凭证弹出确认弹框提示“标记不再生成凭证之后，选中的单据将以后不能再通过有成报销生成财务凭证，确认请点确定按钮”
3. 已经标记不再生成凭证的单据，后续在生成凭证选择单据的页面不再出现；
4. 日记账不做修改；
