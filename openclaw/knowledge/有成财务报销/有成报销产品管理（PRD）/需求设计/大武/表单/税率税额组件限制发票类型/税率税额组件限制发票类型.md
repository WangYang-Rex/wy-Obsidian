## 1、需求背景

客户不同的费用类型对于发票的种类有要求，目前的税率税额组件是把所有的发票类型全部列出来，员工会有选择错误的情况，为避免员工选择错误，将税率税额组件中增加限制发票类型的设置；

|  |  |  |

## 4、需求详情
1. 自定义单据和自定义费用中的税率税额组件增加配置：限制发票类型；默认为不限制；
2. 点击可下拉框选择限制的发票类型，可多选；不选则为不限制；
3. 设置后员工在选择发票类型的时候只能选择限制的发票类型，其他类型隐藏不可用；
4. 当发票生成消费的时候，如果发票类型不在生成的费用类型中可用，则税率税额组件中的发票类型留空；
