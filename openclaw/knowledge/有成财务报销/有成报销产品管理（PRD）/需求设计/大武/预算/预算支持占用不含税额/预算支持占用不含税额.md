## 1、需求背景

目前预算占用是按照所有的提报金额进行占用的，但是可抵扣的税额部分是后续企业可以收回的一部分成本，并没有真正的支出，所以增加税额不占预算的需求；

## 2、业务流程/架构

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 预算管理 | 增加配置税额不占预算 |  |
| 预算占用 | 扣除可抵扣税额部分 |  |
|  |  |  |

## 4、需求详情

### 4.1、配置税额不占预算
1. 预算管理-基础设置中增加开关：税额不占用预算；默认关闭；
2. 提示文案：开启后，预算占用时税额将不占用预算；仅对报销单、付款单的税额生效；
3. 开关开启后出现发票类型的配置；配置名称：不占预算发票类型；选择系统已有的发票类型；仅针对已选的发票类型不占用预算；

### 4.2 预算税额占用逻辑
4. 税额不占用预算仅针对报销单和付款单生效（后续不确定会不会放开配置单据，目前没有需求场景，暂时写死）；
5. 开启开关之后，针对报销单和付款单中有税额且设置了不占预算的发票类型，占用预算时用预算原本的占用金额-税额部分；
