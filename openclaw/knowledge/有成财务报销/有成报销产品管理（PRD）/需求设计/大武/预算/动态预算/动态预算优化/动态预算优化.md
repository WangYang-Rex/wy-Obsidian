## 1、需求背景

目前的动态预算不同的维度对应的适用范围是一致的，计算出结果是一样的，没办法做到不同的部门等维度不同的公式；

|  |  |  |

## 4、需求详情

### 4.1、动态预算规则增加预算名称选择
1. 错别字修改：预算维度中“纬度”修改为“维度”；
2. 新增动态预算的时候增加预算名称的选择，选择之后后续预算维度仅能此预算下面的维度；
3. 预算名称在预算规则保存之后不允许修改；

### 4.2 不同的预算规则中允许重复选择预算名称
4. 不同的预算规则中允许选择同一个预算名称；
5. 预算名称下面的维度不允许重复选择，每个维度仅支持出现在一个预算规则中；已被选择过的维度置灰；   

### 4.3 适用范围增加申请人
6. 创建动态预算规则时，选择适用范围的条件中增加申请人字段；
7. 弹框标题修改：**从审批条件修改为适用范围；目前线上弹框名称是审批条件；**
