# **1\. 需求背景**
1. 现有打印自定义功能已支持单据字段、系统字段、审批信息等内容，但缺少条形码组件，无法满足客户快速扫条形码识别单据的需求；
2. 用户线下流转单据时，需要通过扫码枪快速获取单据信息，提升审批和核对效率；
3. 客户江苏省人民医院需要单据有条形码；

# **2\. 业务流程/架构**

用户配置打印模板（旧版）

↓

【展示设置】处开启【条形码】

↓

设置【条形码位置调整】或【条形码大小调整】

↓

保存模板并打印单据

↓

扫码枪扫描单据条形码

↓

系统解析条形码内容，定位对应单据

# **3\. 功能清单**

| 模块 | 功能点 | 备注 |
|------|---------|------|
| 展示设置 | 增加【条形码】开启关闭功能 |  |
| 展示设置 | 增加【二维码】开启关闭功能 |  |
| 展示设置 | 增加【审批通过标记/二维码/条形码位置调整】 |  |
| 展示设置 | 增加【水印位置调整】 |  |
| 展示设置 | 增加【审批通过标记/二维码/条形码大小调整】 |  |

# **4\. 需求详情**

### **4.1 旧版打印模版配置**

 4.1.1 【展示设置】新增【二维码】开关，【条形码】开关，将原【水印/审批通过标记/二维码位置调整】开关拆分为【审批通过标记/二维码/条形码位置调整】【水印位置调整】设置，原【审批通过标记/二维码大小调整】变更为【审批通过标记/二维码/条形码大小调整】：

![页面 1 3.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/jP2lRYjjjyNXJO8g/img/d49f43c2-a527-461f-87b5-5645139135cb.png?Expires=1774156781&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=ioZjEjwCSiVrMyt0VJezNpAt9IY%3D "")
1. 开关位置顺序如图所示；
2. 任意单据初始打印模版二维码默认开启，二维码显示在单据上，条形码默认关闭，单据上不显示条形码；
3. 【二维码】和【条形码】不可同时处于开启状态，且【条形码】开关开启时，【二维码】开关关闭，单单据上显示【条形码】不显示【二维码】；


![页面 1 4.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/jP2lRYjjjyNXJO8g/img/70b0f0b3-bdb7-421d-9e96-4ae3f96d7e58.png?Expires=1774156781&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=l1SWm1QNDaq1rjG0KAe4B0Hiirg%3D "")

1.【审批通过标记/二维码/条形码位置调整】和【审批通过标记/二维码/条形码大小调整】开关默认关闭；

2.【二维码】，【条形码】或【审批通过标记】开关开启时，打开【审批通过标记/二维码/条形码位置调整】和【审批通过标记/二维码/条形码大小调整】，才可如图所示对相应的图形进行位置调整或大小调整。

*注：【二维码】开关与【条形码】开关为联动组件，必须有一个为开启状态。*


![页面 1 5.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/jP2lRYjjjyNXJO8g/img/ef276ca6-6a60-48b2-933d-1bcef42b7a1b.png?Expires=1774156781&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=mxxBYe8jEqV3Tmbs%2B0s0GeS6l6w%3D "")

1.【水印位置调整】开关默认关闭；

2.【水印位置调整】开关开启时，若已启用水印，则可对水印位置调整，如图所示；

3.【水印位置调整】开关关闭时，若未启用水印，则提示“未启用水印”，且开关无法开启保持关闭；
