## 1、需求背景

目前支付金额字段为系统字段，不允许修改币种，用户支付为外币支付；

## 2、业务流程/架构

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 自定义单据 | 常用字段增加支付金额字段 |  |
| 新建单据 | 新建单据支持修改支付金额的币种 |  |
| 自定义打印 | 自定义单据的支付金额字段取新建的支付金额组件中的值 |  |

## 4、需求详情

### 4.1、常用字段增加支付金额字段
1. 常用字段中增加支付金额字段；单据初始化模版中默认添加进去；现有用户模板可不用处理；
2. 支付金额字段支持修改显示名称，其他内容不允许设置；直接隐藏金额字段的其他设置内容；
3. 取值逻辑：金额为自动填充，不允许用户自己填写；填充为现有单据的支付金额的值进行填充；允许用户切换币种，币种切换需要根据税率自动计算对应币种的值；

### 4.2 打印中的支付金额取值逻辑
4. 在单据中没有添加支付金额字段时，保持现有逻辑；
5. 在单据中已经添加了支付金额字段时，打印中对应的取值取支付金额组件中的值和币种；
