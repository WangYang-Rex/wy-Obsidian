# 有成财务增加钉盘附件下载

| **修订时间** | **版本** | **作者** | **修订说明** |
|----------------|----------|----------|----------------|
| 20230616 | v1.0 | 大武 | 新建 |

## 1、需求背景

目前客户迁移到有成报销，有成财务原本钉盘的附件无法迁移，用户需要能够下载的入口将数据下载之后保存；

## 2、业务流程/架构

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 单据管理 | 单据管理增加新菜单-钉盘附件下载 |  |
| 权限管理 | 权限管理增加钉盘附件下载 |  |
|  |  |  |

## 4、需求详情

### 4.1、单据管理增加菜单钉盘附件下载
1. 列表展示按照附件维度进行展示，一个附件是一行，一个单据中多个附件则展示多行；单据中附件包含**单据和费用以及子表单**中的附件组件的内容；**附件仅展示保存在钉盘的附件**；
2. 列表内容：附件名称、附件大小、关联单号、单据类型、标题、申请人、单据金额、部门、辅助核算、项目、提交时间、完成时间、操作；其中**操作栏固定在尾部**，不跟滚动条滚动；
3. 筛选：**时间**：支持按照提交时间、完成时间进行筛选；**表头搜索**：关联单号、单据类型、部门；**模糊搜索**：支持输入附件名称、单据标题、申请人模糊搜索；**辅助核算、项目**：使用现有辅助核算和项目搜索组件；

### 4.2 权限管理增加钉盘附件下载的权限
4. 权限管理-报销权限管理中增加「钉盘附件下载」；
5. 系统管理员默认拥有此菜单权限；
6. 无需考虑数据权限；
