# 收付款账户增加联行号的选择

| **修订时间** | **版本** | **修订说明** |
|----------------|----------|----------------|
| 20230109 | v1.0 | 新建 |

## 1、需求背景

新建收款和付款账户时，出现名字相似的支行，用户分不清楚，希望可以有联行号的展示和搜索；

## 2、业务流程/架构

## 3、功能清单

根据个人需要补充

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 收款信息 | 增加联行号的搜索和展示 |  |
| 公司账户 | 增加联行号的搜索和展示 |  |
| 个人中心添加账户 | 增加联行号的搜索和展示 |  |

## 4、需求详情
1. 新建收款账户和公司账户时支行信息下面增加联行号；
2. 联行号在选择支行后能够自动带出；
3. 如果没有选择支行，直接输入联行号，能够自动根据联行号带出对应的银行和支行信息；
4. 手工填写联行号时需要判断联行号是否存在，不存在给出提示“联行号不存在，请检查后再提交”
