## 1. 获取支付账户信息列表（公司账户）

> 维护人：sunhongmin

### 请求地址

| 环境 | URL |
|------|-----|
| 测试环境 | `POST https://yiqbdatatest.superboss.cc/reimburse/corp/payAccount/findPayAccountList.rjson?appKey=xxx&accessToken=xxxx` |
| 线上环境 | `POST https://yiqbdata.superboss.cc/reimburse/corp/payAccount/findPayAccountList.rjson?appKey=xxx&accessToken=xxxx` |

### ContentType

`application/x-www-form-urlencoded; charset=utf-8`

### 请求参数

无（通过 Query 参数传递 appKey 和 accessToken）

### 请求示例

```
POST /reimburse/corp/payAccount/findPayAccountList.rjson?appKey=xxx&accessToken=xxxx
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 否 | API 名称 |
| result | int32 | 否 | 响应码，1xx 表示成功 |
| message | string | 否 | 响应信息 |
| data | object | 否 | 响应实体 |
| └ list | array | 否 | 公司账户列表 |
| &ensp;&ensp;└ id | int32 | 否 | 唯一 ID |
| &ensp;&ensp;└ corpId | string | 否 | 企业 ID |
| &ensp;&ensp;└ bankCode | string | 否 | 银行编码 |
| &ensp;&ensp;└ bankCardNum | string | 否 | 卡号 |
| &ensp;&ensp;└ holdName | string | 否 | 户名 |
| &ensp;&ensp;└ bankName | string | 否 | 银行名 |
| &ensp;&ensp;└ bankFullName | string | 否 | 开户行全称 |
| &ensp;&ensp;└ province | string | 否 | 省 |
| &ensp;&ensp;└ city | string | 否 | 市 |
| &ensp;&ensp;└ type | int32 | 否 | 类型：1=银行账户，2=现金账户，3=支付宝，4=微信 |
| &ensp;&ensp;└ creator | string | 否 | 创建人 |
| &ensp;&ensp;└ creatorName | string | 否 | 创建人姓名 |
| &ensp;&ensp;└ modifier | string | 否 | 修改人 |
| &ensp;&ensp;└ modifierName | string | 否 | 修改人姓名 |
| &ensp;&ensp;└ created | string | 否 | 创建时间 |
| &ensp;&ensp;└ modified | string | 否 | 修改时间 |
| &ensp;&ensp;└ status | int32 | 否 | 是否启用：1=启用，0=禁用 |
| &ensp;&ensp;└ dataType | int32 | 否 | 数据类型：0=现有数据，1=自定义银行+网点，2=自定义网点 |
| &ensp;&ensp;└ visible | int32 | 否 | 可见范围：1=全部可见，2=部分可见 |
| &ensp;&ensp;└ visibleUser | string | 否 | 可见人员 |
| &ensp;&ensp;└ visibleDepartment | string | 否 | 可见部门 |
| &ensp;&ensp;└ authState | string | 否 | 授权状态：0=未授权，1=已授权，2=授权中，3=授权失败 |
| &ensp;&ensp;└ isAuth | boolean | 否 | 是否已授权 |
| &ensp;&ensp;└ signDate | string | 否 | 授权日期 |
| &ensp;&ensp;└ bankDirectStatus | int32 | 否 | 是否银企直连支付：0=否，1=是 |
| &ensp;&ensp;└ userBankDirect | boolean | 否 | 用户银企直连标识 |
| &ensp;&ensp;└ deleteFlag | int32 | 否 | 是否已删除：0=否，1=是 |
| &ensp;&ensp;└ money | string | 否 | 金额（字符串） |
| &ensp;&ensp;└ amount | double | 否 | 金额 |
| &ensp;&ensp;└ typeLable | string | 否 | 企业设置的类型名称 |

### 响应示例

```json
{
    "apiName": "german.rosenbaum",
    "result": 100,
    "message": "success",
    "data": {
        "list": [
            {
                "id": 720,
                "corpId": "19",

                "bankName": "german.rosenbaum",
                "bankFullName": "german.rosenbaum",
                "type": 1,
                "status": 1,
                "dataType": 0,
                "visible": 1,
```

## 2. 新增支付账户信息（公司账户）

> 维护人：sunhongmin

### 请求地址

| 环境 | URL |
|------|-----|
| 测试环境 | `POST https://yiqbdatatest.superboss.cc/reimburse/corp/payAccount/addPayAccount.rjson?appKey=xxx&accessToken=xxxx` |
| 线上环境 | `POST https://yiqbdata.superboss.cc/reimburse/corp/payAccount/addPayAccount.rjson?appKey=xxx&accessToken=xxxx` |

### ContentType

`application/json; charset=utf-8`

### 请求参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| bankCode | string | 否 | 银行编码 |
| bankCardNum | string | 是 | 卡号 |
| holdName | string | 是 | 户名 |
| bankName | string | 否 | 银行名 |
| bankFullName | string | 否 | 开户行全称 |
| province | string | 否 | 省 |
| city | string | 否 | 市 |
| type | int32 | 否 | 类型：1=银行账户，2=现金账户（默认 2） |
| status | int32 | 否 | 是否启用：1=启用，0=禁用（默认 1） |
| dataType | int32 | 否 | 数据类型：0=现有数据，1=自定义银行+网点，2=自定义网点（默认 1） |
| visible | int32 | 否 | 可见范围：1=全部可见，2=部分可见（默认 2） |
| visibleUser | string | 否 | 可见人员（用户 ID，多个逗号分隔） |

### 请求示例

```json
{
    "bankCode": "45207",
    "bankCardNum": "nsul1k",
    "holdName": "german.rosenbaum",
    "bankName": "german.rosenbaum",
    "bankFullName": "german.rosenbaum",
    "province": "gxw7un",
    "city": "yts4vl",
    "type": 1,
    "status": 1,
    "dataType": 0,
    "visible": 1,
    "visibleUser": "2tjctf"
}
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 否 | API 名称 |
| result | int32 | 否 | 响应码，1xx 表示成功 |
| message | string | 否 | 响应信息 |
| data | object | 否 | 响应实体（成功时为空） |

### 响应示例

```json
{
    "apiName": "german.rosenbaum",
    "result": 100,
    "message": "success",
    "data": {}
}
```

| 线上环境 | `POST https://yiqbdata.superboss.cc/reimburse/corp/invoice/getInvoiceTitleList.rjson?appKey=xxx&accessToken=xxxx` |

### ContentType

`application/x-www-form-urlencoded; charset=utf-8`

### 请求参数

无（通过 Query 参数传递 appKey 和 accessToken）

### 请求示例

```
POST /reimburse/corp/invoice/getInvoiceTitleList.rjson?appKey=xxx&accessToken=xxxx
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 否 | API 名称 |
| result | int32 | 否 | 响应码，1xx 表示成功 |
| message | string | 否 | 响应信息 |
