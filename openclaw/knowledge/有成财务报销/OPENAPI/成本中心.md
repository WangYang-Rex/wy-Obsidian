# 成本中心

## 1. 查询成本中心列表

- **请求地址：** `POST /reimburse/data/cost/center/list.rjson`
- **测试环境：** `https://yiqbdatatest.superboss.cc/reimburse/data/cost/center/list.rjson?appKey=xxx&accessToken=xxxx`
- **线上环境：** `https://yiqbdata.superboss.cc/reimburse/data/cost/center/list.rjson?appKey=xxx&accessToken=xxxx`
- **ContentType：** `application/json`

### 请求参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| searchContent | string | 否 | 内容搜索 |

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 否 | api 名称 |
| result | int32 | 否 | 响应码 1xx表示调用成功 |
| message | string | 否 | 响应信息 |
| data | object | 否 | 响应实体信息 |
| └ success | boolean | 否 | 是否成功 |
| └ list | array | 否 | 分页数据 |
|   └ deleted | boolean | 是 | 是否删除（true:删除，false:未删除） |
|   └ id | int32 | 否 | 成本中心id |
|   └ code | string | 否 | 成本中心code |
|   └ name | string | 否 | 成本中心名称 |
|   └ fullName | string | 否 | 成本中心全名称 |
|   └ sort | int32 | 否 | 排序 |
|   └ parentId | int32 | 否 | 父级成本中心id |
|   └ parentName | string | 否 | 父级成本中心名称 |
|   └ route | string | 否 | 全路径 |
|   └ enabled | boolean | 否 | 是否启用 |
|   └ relationData | array | 否 | 关联数据 |
|     └ relationType | enum | 否 | 成本中心关联类型（固定 department） |
|     └ costCenterId | int32 | 否 | 成本中心id |
|     └ relationLabel | string | 否 | 关联部门名称 |
|     └ relationValue | string | 否 | 关联部门id |
|   └ children | array | 否 | 子节点 |
| └ count | int32 | 否 | 记录总数 |
| └ errorMsg | string | 否 | 失败信息 |

### 请求示例

```json
{
    "searchContent": "5t8869"
}
```
