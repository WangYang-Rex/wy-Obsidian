# 辅助核算（表单）

> 基于组件表单的辅助核算操作接口，包含模板查询、新增、详情、修改、标签获取。

---

## 1. 辅助核算模板查询

**请求地址：**
- 测试环境：`POST` https://yiqbdatatest.superboss.cc/reimburse/auxiliary/form/getTemplateList.rjson?appKey=xxx&accessToken=xxxx
- 线上环境：`POST` https://yiqbdata.superboss.cc/reimburse/auxiliary/form/getTemplateList.rjson?appKey=xxx&accessToken=xxxx

**描述：** 辅助核算模板分页查询接口

**ContentType：** `application/json`

### 请求参数（Query）

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| code | string | 否 | 模板编码 |
| page | number | 否 | 当前页数 |
| pageSize | number | 否 | 每页条数 |

### 请求示例

```json
{}
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 是 | 接口名称 |
| result | number | 是 | 响应码 |
| message | string | 是 | 响应信息 |
| data | array | 是 | 模板列表 |
| └ categoryName | string | 是 | 辅助类别名称 |
| └ categoryId | number | 是 | 辅助类别ID |
| └ code | string | 是 | 模板编码 |
| └ id | number | 是 | 模板ID |
| └ name | string | 是 | 模板名称 |
| └ components | array | 是 | 组件列表 |
|   └ code | string | 是 | 组件编码 |
|   └ id | number | 是 | 组件ID |
|   └ label | string | 是 | 组件显示名称 |
|   └ name | string | 是 | 组件名称 |
|   └ type | string | 是 | 组件类型（如 text） |
|   └ configure | object | 是 | 组件配置 |
|     └ optional.require | boolean | 是 | 是否必填（true为必填） |
|     └ maxLength.value | number | 是 | 最大字数限制 |
|     └ option.value | array | 是 | 下拉选项配置 |
|     └ dateType.type | array | 是 | 日期类型选项 |
|     └ dateType.value | string | 是 | 当前选中值 |
|     └ datePeriodType.type | array | 是 | 日期格式选项 |
|     └ datePeriodType.value | string | 是 | 当前选中值 |

### 响应示例

```json
{
    "apiName": "getAuxiliaryItemTemplate",
    "data": [
        {
            "categoryName": "项目",
            "categoryId": 11942,
            "code": "1013113205825728512",
            "components": [
                {
                    "code": "commonName",
                    "configure": {
                        "optional": { "require": true },
                        "maxLength": { "value": 120 },
                        "option": { "value": ["选项一", "选项二", "选项三"] },
                        "dateType": {
                            "type": [{ "lable": "日期", "vale": "single" }],
                            "vale": "multi"
                        },
                        "datePeriodType": {
                            "type": [{ "label": "年-月-日 时-分-秒", "value": "hourAndMinute" }],
                            "value": "hourAndMinute"
                        }
                    },
                    "id": 707,
                    "label": "名称",
                    "name": "名称",
                    "type": "text"
                }
            ],
            "id": 228,
            "name": "客户"
        }
    ],
    "message": "操作成功",
    "result": 100
}
```

## 2. 新增辅助核算

**请求地址：**
- 测试环境：`POST` https://yiqbdatatest.superboss.cc/reimburse/auxiliary/form/save.rjson?appKey=xxx&accessToken=xxxx
- 线上环境：`POST` https://yiqbdata.superboss.cc/reimburse/auxiliary/form/save.rjson?appKey=xxx&accessToken=xxxx

**描述：** 辅助核算新增接口，支持组件表单

**ContentType：** `application/json`

### 请求参数（Body）

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| components | array | 是 | 组件列表 |
| └ bindingVal | object | 是 | 组件绑定值（具体格式详见备注） |
| └ code | string | 是 | 组件编码 |
| templateCode | number | 是 | 模板编码 |
| categoryCode | number | 是 | 辅助类别编码 |

### 请求示例

```json
{
    "components": [
        {
            "bindingVal": "具体示例值详见备注",
            "code": "commonName"
        }
    ],
    "templateCode": 228,
    "categoryCode": 11942
}
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 是 | 接口名称 |
| data | number | 是 | 新增辅助核算ID |
| message | string | 是 | 响应信息 |
| result | number | 是 | 响应码 |

### 响应示例

```json
{
    "apiName": "addAuxiliaryItemOrder",
    "data": 10881,
    "message": "操作成功",
    "result": 100
}
```

## 3. 辅助核算详情

**请求地址：**
- 测试环境：`POST` https://yiqbdatatest.superboss.cc/reimburse/auxiliary/form/get.rjson?appKey=xxx&accessToken=xxxx
- 线上环境：`POST` https://yiqbdata.superboss.cc/reimburse/auxiliary/form/get.rjson?appKey=xxx&accessToken=xxxx

**描述：** 辅助核算详情接口，支持组件表单

**ContentType：** `application/json`

### 请求参数（Body）

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| itemName | string | 是* | 辅助核算名称（编码、名称、ID三传一） |
| categoryCode | string | 是 | 辅助类别ID |
| itemCode | string | 是* | 辅助核算编码（编码、名称、ID三传一） |

> *itemName 和 itemCode 至少传一个

### 请求示例

```json
{
    "itemName": "",
    "categoryCode": "84867",
    "itemCode": ""
}
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 是 | 接口名称 |
| result | number | 是 | 响应码 |
| message | string | 是 | 响应信息 |
| data | object | 是 | 详情数据 |
| └ categoryId | number | 是 | 辅助类别ID |
| └ itemCode | string | 是 | 辅助核算编码 |
| └ itemId | number | 是 | 辅助核算ID |
| └ itemName | string | 是 | 辅助核算名称 |
| └ templateInsCode | string | 是 | 辅助核算模板编码 |
| └ components | array | 是 | 组件列表 |
|   └ bindingVal | string | 是 | 组件绑定值 |
|   └ code | string | 是 | 组件编码 |
|   └ id | number | 是 | 组件ID |
|   └ label | string | 是 | 组件显示名称 |
|   └ name | string | 是 | 组件名称 |
|   └ type | string | 是 | 组件类型 |

### 响应示例

```json
{
    "apiName": "getAuxiliaryItemOrder",
    "data": {
        "categoryId": 11942,
        "itemCode": "9004",
        "itemId": 10881,
        "itemName": "新建辅助核算",
        "templateInsCode": "1013113205825728512",
        "components": [
            {
                "bindingVal": "新建辅助核算",
                "code": "commonName",
                "id": 73865,
                "label": "名称",
                "name": "名称",
                "type": "text"
            }
        ]
    },
    "message": "操作成功",
    "result": 100
}
```

## 4. 修改辅助核算

**请求地址：**
- 测试环境：`POST` https://yiqbdatatest.superboss.cc/reimburse/auxiliary/form/update.rjson?appKey=xxx&accessToken=xxxx
- 线上环境：`POST` https://yiqbdata.superboss.cc/reimburse/auxiliary/form/update.rjson?appKey=xxx&accessToken=xxxx

**描述：** 辅助核算修改接口，支持组件表单

**ContentType：** `application/json`

### 请求参数（Body）

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| oldItemCode | string | 是 | 原辅助选项编码 |
| categoryCode | number | 是 | 辅助类别编码 |
| templateCode | string | 是 | 模板编码 |
| components | array | 是 | 组件列表 |
| └ bindingVal | string | 是 | 组件绑定值 |
| └ code | string | 是 | 组件编码 |

### 请求示例

```json
{
    "oldItemCode": "84867",
    "categoryCode": 11942,
    "templateCode": "9005",
    "components": [
        {
            "bindingVal": "2222",
            "code": "commonName"
        }
    ]
}
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 是 | 接口名称 |
| data | number | 是 | 修改条数 |
| message | string | 是 | 响应信息 |
| result | number | 是 | 响应码 |

### 响应示例

```json
{
    "apiName": "updateAuxiliaryItemOrder",
    "data": 1,
    "message": "操作成功",
    "result": 100
}
```

## 5. 获取辅助核算标签

**请求地址：**
- 测试环境：`POST` https://yiqbdatatest.superboss.cc/reimburse/auxiliary/form/getAuxiliaryTag.rjson?appKey=xxx&accessToken=xxxx
- 线上环境：`POST` https://yiqbdata.superboss.cc/reimburse/auxiliary/form/getAuxiliaryTag.rjson?appKey=xxx&accessToken=xxxx

**描述：** 根据辅助类别获取辅助核算标签列表数据

**ContentType：** `application/json`

### 请求参数（Body）

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| categoryId | number | 是 | 辅助类别ID |

### 请求示例

```json
{
    "categoryId": 11942
}
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 是 | 接口名称 |
| result | number | 是 | 响应码 |
| message | string | 是 | 响应信息 |
| data | array | 是 | 标签列表 |
| └ categoryId | number | 是 | 辅助类别ID |
| └ id | number | 是 | 标签ID |
| └ name | string | 是 | 标签名称 |

### 响应示例

```json
{
    "apiName": "getAuxiTagList",
    "data": [
        {
            "categoryId": 11942,
            "id": 649,
            "name": "水果"
        }
    ],

```
