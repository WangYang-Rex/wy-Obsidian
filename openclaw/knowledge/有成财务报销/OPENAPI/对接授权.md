# 对接授权

## 免登授权接口 — 获取 AccessToken

> 维护人：sunhongmin

任何接口调用前都需要先获取 AccessToken，有效时间 **2 小时**，请自行缓存，不要频繁调用。

### 请求地址

| 环境 | URL |
|------|-----|
| 测试环境 | `POST https://yiqbdatatest.superboss.cc/auth/getAccessToken.rjson` |
| 线上环境 | `POST https://yiqbdata.superboss.cc/auth/getAccessToken.rjson` |

### ContentType

`application/json; charset=utf-8`

### 请求参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| appKey | string | 是 | appKey（有成提供） |
| appSecret | string | 是 | appSecret（有成提供） |

### 请求示例

```json
{
    "appKey": "oskqrq",
    "appSecret": "0t7pog"
}
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 否 | API 名称 |
| result | int32 | 否 | 响应码，1xx 表示成功 |
| message | string | 否 | 响应信息 |
| data | object | 否 | 响应实体 |
| └ accessToken | string | 否 | AccessToken，后续请求携带 |

### 响应示例

```json
{
    "apiName": "youlanda.roob",
    "result": 100,
    "message": "success",
    "data": {
        "accessToken": "jeg880"
    }
}
```
