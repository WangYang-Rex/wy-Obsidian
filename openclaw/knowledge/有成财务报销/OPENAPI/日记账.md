# 日记账

共 2 个接口。基础路径：`https://yiqbdata.superboss.cc`（线上）/ `https://yiqbdatatest.superboss.cc`（测试）。

通用查询参数：`appKey=xxx&accessToken=xxxx`

---

## 1. 日记账查询

**`POST /reimburse/journal/queryListPage.rjson`**

描述：日记账查询

ContentType：`application/json; charset=utf-8`

维护人：jwb

### 请求参数（Body）

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| accountCode | string | 是 | 帐套编码 |
| beginTime | string | 否 | 日记账开始日期（格式 `yyyy-MM-dd HH:mm:ss`） |
| endTime | string | 否 | 日记账结束日期（格式 `yyyy-MM-dd HH:mm:ss`） |
| status | int32 | 否 | 状态筛选：1-待生成 2-生成中 3-生成成功 4-生成失败。传入时仅做查询不修改状态；不传入时将查询出的数据状态改为"生成中"并等待回调 |
| pageSize | int32 | 否 | 返回指定数量，最大 100 条 |

### 请求示例

```json
{
  "accountCode": "55217",
  "beginTime": "2024-01-01 00:00:00",
  "endTime": "2024-12-31 23:59:59",
  "status": 3,
  "pageSize": 10
}
```

## 2. 日记账回调

**`POST /reimburse/journal/callback.rjson`**

描述：日记账生成完成后的回调通知

ContentType：`application/json; charset=utf-8`

维护人：jwb

### 请求参数（Body）

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| accountCode | string | 是 | 帐套编码 |
| cashJournalExternalList | array | 是 | 处理结果列表 |
| └ journalId | int64 | 是 | 日记账 ID |
| └ status | int32 | 是 | 状态：1-成功 2-失败 |
| └ voucherId | int32 | 否 | 外部凭证 ID |

### 请求示例

```json
{
  "accountCode": "55217",
  "cashJournalExternalList": [
    {
      "journalId": 868,
      "status": 1,
      "voucherId": 369
    }
  ]
}
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 否 | api 名称 |
| result | int32 | 否 | 响应码，1xx 表示调用成功 |
| message | string | 否 | 响应信息 |
| data | object | 否 | 响应实体信息（成功时为空对象） |

### 响应示例

```json
{
  "apiName": "string",
  "result": 100,
  "message": "success",
  "data": {}
}
```
