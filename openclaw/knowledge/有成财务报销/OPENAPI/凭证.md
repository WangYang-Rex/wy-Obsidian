## 1. 凭证查询

> 维护人：jwb

### 请求地址

| 环境 | URL |
|------|-----|
| 测试环境 | `POST https://yiqbdatatest.superboss.cc/reimburse/certificate/queryListPage.rjson?appKey=xxx&accessToken=xxxxxa` |
| 线上环境 | `POST https://yiqbdata.superboss.cc/reimburse/certificate/queryListPage.rjson?appKey=xxx&accessToken=xxxxxa` |

### ContentType

`application/json; charset=utf-8`

### 请求参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| status | int32 | 否 | 凭证状态：1=待生成，2=生成中，3=成功，4=失败。传入时仅查询，不传则变为"生成中"等待回调 |
| accountCode | string | 是 | 账套编码 |
| beginTime | string | 否 | 凭证开始日期（如 `2022-01-19 11:11:41`） |
| endTime | string | 否 | 凭证结束日期（如 `2022-01-19 11:11:41`） |
| pageSize | int32 | 否 | 返回数量，最大 100 条 |

### 请求示例

```json
{
    "status": 3,
    "accountCode": "23790",
    "beginTime": "2022-01-19 11:11:41",
    "endTime": "2022-01-19 11:11:41",
    "pageSize": 10
}
```

## 2. 凭证回调

> 维护人：jwb

### 请求地址

| 环境 | URL |
|------|-----|
| 测试环境 | `POST https://yiqbdatatest.superboss.cc/reimburse/certificate/callback.rjson?appKey=xxx&accessToken=xxxx` |
| 线上环境 | `POST https://yiqbdata.superboss.cc/reimburse/certificate/callback.rjson?appKey=xxx&accessToken=xxxx` |

### ContentType

`application/json; charset=utf-8`

### 请求参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| accountCode | string | 否 | 账套编码 |
| certificateIdList | array | 否 | 凭证结果列表 |
| └ certificateId | int64 | 否 | 凭证 ID |
| └ status | int32 | 否 | 状态：1=成功，2=失败 |
| └ voucherCode | string | 否 | 成功后生成到财务系统的凭证编码 |
| └ voucherId | string | 否 | 成功后生成到财务系统的凭证 ID |
| └ reason | string | 否 | 失败原因 |
| └ voucherNumber | string | 否 | 凭证号 |

### 请求示例

```json
{
    "accountCode": "23790",
    "certificateIdList": [
        {
            "certificateId": 120,
            "status": 1,
            "voucherCode": "23790",
            "voucherId": "158",
            "reason": "",
            "voucherNumber": "jw1t6j"
        }
    ]
}
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 否 | API 名称 |
| result | int32 | 否 | 响应码，1xx 表示成功 |
| message | string | 否 | 响应信息 |
| data | object | 否 | 响应实体（成功时为空） |

### 响应示例

```json
{
    "apiName": "nguyet.goodwin",
    "result": 100,
    "message": "success",
    "data": {}
}
```
