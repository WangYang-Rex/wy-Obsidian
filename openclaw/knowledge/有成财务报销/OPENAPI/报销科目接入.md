# 报销科目接入

---

## 1. 新增科目

> 维护人：jwb

### 请求地址

| 环境 | URL |
|------|-----|
| 测试环境 | `POST https://yiqbdatatest.superboss.cc/reimburse/subject/add.rjson?appKey=xxx&accessToken=xxxx` |
| 线上环境 | `POST https://yiqbdata.superboss.cc/reimburse/subject/add.rjson?appKey=xxx&accessToken=xxxx` |

### ContentType

`application/json`

### 请求参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| subjectCode | string | 是 | 科目编码 |
| subjectName | string | 是 | 科目名称（最多 50 个字符） |
| accountCode | string | 是 | 账套编码 |
| categoryCodes | array | 否 | 辅助类别编码，如 `["001","002"]` |
| categoryNames | array | 否 | 辅助类别名称 |
| currencyCodes | string | 否 | 外币核算 code，逗号分隔 |
| isFund | boolean | 否 | 是否为资金科目 |

### 请求示例

```json
{
    "subjectCode": "79132",
    "subjectName": "neal.bahringer",
    "accountCode": "79132",
    "categoryCodes": ["zp16g7", "wzg04z"],
    "categoryNames": ["i1cgn7", "6dn9vm"],
    "currencyCodes": "q3u0y0",
    "isFund": true
}
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 否 | API 名称 |
| result | int32 | 否 | 响应码，1xx 表示成功 |
| message | string | 否 | 响应信息 |
| data | object | 否 | 响应实体（成功时为空） |

### 响应示例

```json
{
    "apiName": "neal.bahringer",
    "result": 100,
    "message": "success",
    "data": {}
}
```

## 2. 修改科目

> 维护人：jwb

### 请求地址

| 环境 | URL |
|------|-----|
| 测试环境 | `POST https://yiqbdatatest.superboss.cc/reimburse/subject/update.rjson?appKey=xxx&accessToken=xxxx` |
| 线上环境 | `POST https://yiqbdata.superboss.cc/reimburse/subject/update.rjson?appKey=xxx&accessToken=xxxx` |

### ContentType

`application/json; charset=utf-8`

### 请求参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| subjectCode | string | 是 | 修改后的科目编码 |
| subjectName | string | 是 | 修改后的科目名称 |
| code | string | 是 | 要修改的科目编码（原编码） |
| accountCode | string | 是 | 账套编码 |
| categoryCodes | array | 否 | 修改后的类别编码，如 `["001","002"]`；传 `[]` 清空类别 |

### 请求示例

```json
{
    "subjectCode": "23790",
    "subjectName": "nguyet.goodwin",
    "code": "23790",
    "accountCode": "23790",
    "categoryCodes": ["ow0ve8", "ua3crm"]
}
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 否 | API 名称 |
| result | int32 | 否 | 响应码，1xx 表示成功 |
| message | string | 否 | 响应信息 |
| data | object | 否 | 响应实体（成功时为空） |

### 响应示例

```json
{
    "apiName": "nguyet.goodwin",
    "result": 100,
    "message": "success",
    "data": {}
}
```
