# 发票

> 发票查询相关接口。

---

## 1. 获取发票列表

**请求地址：**
- 测试环境：`POST` https://yiqbdatatest.superboss.cc/reimburse/invoice/invoiceList.rjson?appKey=xxx&accessToken=xxxx
- 线上环境：`POST` https://yiqbdata.superboss.cc/reimburse/invoice/invoiceList.rjson?appKey=xxx&accessToken=xxxx

**描述：** 分页查询发票列表

**ContentType：** `application/json`

### 请求参数（Body）

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| beginTime | string | 是 | 查询起始时间（yyyy-MM-dd HH:mm:ss） |
| endTime | string | 是 | 查询结束时间（yyyy-MM-dd HH:mm:ss） |
| timeType | number | 是 | 时间类型：1=创建时间，0=开票时间 |
| orderCode | string | 否 | 关联单据编号 |
| pageSize | number | 是 | 每页条数 |
| pageNo | number | 是 | 页码 |

### 请求示例

```json
{
    "beginTime": "2024-08-01 00:00:00",
    "endTime": "2024-10-31 23:59:59",
    "timeType": 0,
    "orderCode": "",
    "pageSize": 20,
    "pageNo": 1
}
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| count | number | 是 | 总记录数 |
| success | boolean | 是 | 是否成功 |
| totalAmount | number | 是 | 合计金额 |
| list | array | 是 | 发票列表 |
| └ id | number | 是 | 发票ID |
| └ invoiceNo | string | 是 | 发票号码 |
| └ code | string | 是 | 发票代码 |
| └ invoiceType | string | 是 | 发票类型（见下方枚举） |
| └ invoiceAmount | string | 是 | 发票总额 |
| └ taxAmount | string | 是 | 税额合计 |
| └ taxRate | string | 是 | 税率 |
| └ content | string | 是 | 发票内容 |
| └ invoiceDate | string | 是 | 开票日期 |
| └ buyerName | string | 是 | 购买方名称 |
| └ taxesNo | string | 是 | 购买方纳税识别号 |
| └ bank | string | 是 | 购买方银行 |
| └ buyerAdress | string | 是 | 购买方地址 |
| └ sellUserName | string | 是 | 销售方 |
| └ sellTaxesNo | string | 是 | 销售方税号 |
| └ sellerBank | string | 是 | 销售方银行 |
| └ sellerAdress | string | 是 | 销售方地址 |
| └ checkCode | string | 是 | 校验码 |
| └ isVerified | string | 是 | 验真状态（true=已验真，false=未验真，非增值税发票为空） |
| └ sourceFrom | number | 是 | 来源（1=手工录入，2=OCR，3=扫码，4=支付宝） |
| └ note | string | 是 | 开票备注 |
| └ deptName | string | 是 | 部门 |
| └ extDeptId | string | 是 | 外部部门ID |
| └ costCenter | string | 是 | 成本中心 |
| └ orderCode | string | 是 | 关联单据编号 |
| └ feeId | string | 是 | 消费ID |
| └ cityForm | string | 是 | 开票地区 |
| └ resourceUrl | string | 是 | 发票URL |
| └ createUserName | string | 是 | 创建人名称 |
| └ crateUserId | string | 是 | 创建人ID |
| └ createTime | string | 是 | 创建时间 |
| └ goodsInfos | array | 是 | 商品明细列表 |
|   └ id | number | 是 | 商品ID |
|   └ name | string | 是 | 商品名称 |
|   └ specification | string | 是 | 规格型号 |
|   └ unit | string | 是 | 单位 |
|   └ unitPrice | number | 是 | 单价 |
|   └ quantity | number | 是 | 数量 |
|   └ preTaxMoney | number | 是 | 不含税金额 |
|   └ taxMoney | number | 是 | 税额 |
|   └ taxRate | number | 是 | 税率 |
|   └ goodTotalAmount | number | 是 | 商品总额 |
|   └ sourceFrom | string | 是 | 来源 |
|   └ corpId | string | 是 | 企业ID |
|   └ createUser | string | 是 | 创建人ID |
|   └ createTime | number | 是 | 创建时间（毫秒时间戳） |
|   └ modifyTime | number | 是 | 更新时间（毫秒时间戳） |

**发票类型枚举（invoiceType）：**

| 值 | 说明 |
|----|------|
| vat_invoice_special | 增值税专票 |
| vat_invoice_general | 增值税普票 |
| vat_invoice_electronic | 电子增值税发票 |
| vat_invoice_electronic_dedicated | 增值税电子专用发票 |
| general_invoice | 普通发票 |
| air_ticket | 机票行程单 |
| taxi | 出租车票 |
| train_ticket | 火车票 |
| quota_invoice | 定额发票 |
| car_ticket | 汽车票 |
| mobile_ticket | 机动车票 |
| etc_ticket | ETC发票 |
| general_electronic_invoice | 通用电子发票 |
| financial_invoice | 财政发票 |
| other | 其它类型 |
