# 收款信息（收款账户）

## 1. 获取收款信息

- **请求地址：** `POST /reimburse/receipt/getBankCardInfoList.rjson`
- **测试环境：** `https://yiqbdatatest.superboss.cc/reimburse/receipt/getBankCardInfoList.rjson?appKey=xxx&accessToken=xxxx`
- **线上环境：** `https://yiqbdata.superboss.cc/reimburse/receipt/getBankCardInfoList.rjson?appKey=xxx&accessToken=xxxx`
- **ContentType：** `application/x-www-form-urlencoded;charset=utf-8`

### 请求参数

无

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 否 | api 名称 |
| result | int32 | 否 | 响应码 1xx表示调用成功 |
| message | string | 否 | 响应信息 |
| data | array | 否 | 响应实体信息 |
| └ cardId | int32 | 否 | 收款信息唯一ID |
| └ bankCardNum | string | 否 | 卡号 |
| └ holdName | string | 否 | 户名 |
| └ bankName | string | 否 | 银行名 |
| └ bankFullName | string | 否 | 开户行全称 |
| └ province | string | 否 | 省 |
| └ city | string | 否 | 市 |
| └ type | int32 | 否 | 类型，1：个人账户，2：对公账户 |
| └ creator | string | 否 | 创建人 |
| └ creatorName | string | 否 | 创建人姓名 |
| └ sourceType | int32 | 否 | 来源，1：用户创建，2：管理者创建，3：导入 |

### 响应示例

```json
{
    "apiName": "german.rosenbaum",
    "result": 100,
    "message": "success",
    "data": [
        {
            "cardId": 181,
            "bankCardNum": "iwt75c",
            "holdName": "german.rosenbaum",
            "bankName": "german.rosenbaum",
            "bankFullName": "german.rosenbaum",
            "province": "co0p52",
            "city": "n6uboy",
            "type": 1,
            "creator": "w8jkcp",
            "creatorName": "german.rosenbaum",
            "sourceType": 1
        }
    ]
}
```

## 2. 新增收款信息

- **请求地址：** `POST /reimburse/receipt/payeeAccounts/addUserBankCardInfo.rjson`
- **测试环境：** `https://yiqbdatatest.superboss.cc/reimburse/receipt/payeeAccounts/addUserBankCardInfo.rjson?appKey=xxx&accessToken=xxxx`
- **线上环境：** `https://yiqbdata.superboss.cc/reimburse/receipt/payeeAccounts/addUserBankCardInfo.rjson?appKey=xxx&accessToken=xxxx`
- **ContentType：** `application/json; charset=utf-8`

### 请求参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| holdName | string | 是 | 户名 |
| bankCardNum | string | 是 | 卡号 |
| bankName | string | 是 | 银行名 |
| province | string | 否 | 省 |
| city | string | 否 | 市 |
| bankFullName | string | 否 | 开户行全称 |
| type | int32 | 否 | 账户类型，1：个人账户，2：对公账户 |
| visible | int32 | 否 | 可见类型，1：全员可见，2：部分可见 |
| visibleUser | string | 否 | 可见员工id列表，逗号分隔 |
| shared | boolean | 否 | 是否开启共享 |
| sourceType | int32 | 否 | 来源，1：用户创建，2：管理者创建，3：导入 |

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 否 | api 名称 |
| result | int32 | 否 | 响应码 1xx表示调用成功 |
| message | string | 否 | 响应信息 |
| data | object | 否 | 响应实体信息 |

### 请求示例

```json
{
    "holdName": "german.rosenbaum",
    "bankCardNum": "nbywe8",
    "bankName": "german.rosenbaum",
    "province": "zub7hx",
    "city": "xjdjye",
    "bankFullName": "german.rosenbaum",
    "type": 1,
    "visible": 1,
    "visibleUser": "rxyo80",
    "shared": true,
    "sourceType": 1
}
```

### 响应示例

```json
{
    "apiName": "german.rosenbaum",
    "result": 100,
    "message": "success",
    "data": {}
}
```
