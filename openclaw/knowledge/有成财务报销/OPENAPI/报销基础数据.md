# 报销基础数据

> 拉取单据/费用模板相关接口，用于获取模板结构和字段配置。

---

## 1. 拉取单据/费用模板

**请求地址：**
- 测试环境：`POST` https://yiqbdatatest.superboss.cc/reimburse/base/getTemplateList.rjson?appKey=xxx&accessToken=xxxx
- 线上环境：`POST` https://yiqbdata.superboss.cc/reimburse/base/getTemplateList.rjson?appKey=xxx&accessToken=xxxx

**描述：** 拉取单据/费用模板（扁平结构）

**ContentType：** `application/json`

### 请求参数（Body）

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| needParentTemplate | boolean | 否 | 是否需要返回父级模板 |

### 请求示例

```json
{
    "needParentTemplate": true
}
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 否 | 接口名称 |
| result | number | 否 | 响应码（1xx表示成功） |
| message | string | 否 | 响应信息 |
| data | array | 否 | 模板列表 |
| └ templateCode | string | 否 | 关联模板编码 |
| └ parentTemplateCode | string | 否 | 父级模板编码 |
| └ templateType | number | 否 | 关联模板类型：1-单据，2-费用 |
| └ orderTemplateType | number | 否 | 单据组件类型：1=报销单，2=借款单，3=付款单，4=费用类型，5=申请单，6=合同，7=项目，8=收款单 |
| └ templateName | string | 否 | 关联模板名称 |
| └ code | string | 否 | 组件实例编码 |
| └ type | string | 否 | 组件实例类型 |
| └ name | string | 否 | 字段名称 |
| └ label | string | 否 | 字段标签（显示名称） |
| └ require | boolean | 否 | 字段是否必填 |
| └ categoryId | number | 否 | 辅助核算类别ID（仅辅助核算字段有值） |
| └ configureObject | object | 否 | 组件配置 |
| └ subformComponentList | array | 否 | 子表单组件列表 |
|   └ code | string | 否 | 组件实例编码 |
|   └ type | string | 否 | 组件实例类型 |
|   └ name | string | 否 | 字段名称 |
|   └ label | string | 否 | 字段标签（显示名称） |
|   └ require | boolean | 否 | 字段是否必填 |
|   └ configureObject | object | 否 | 组件配置 |

### 响应示例

```json
{
    "apiName": "getTemplateList",
    "result": 100,
    "message": "success",
    "data": [
        {
            "templateCode": "19228",
            "parentTemplateCode": "19228",
            "templateType": 2,
            "orderTemplateType": 1,
            "templateName": "差旅报销单",
            "code": "19228",
            "type": "text",
            "name": "amount",
            "label": "金额",
            "require": true,
            "categoryId": null,
            "configureObject": {},
            "subformComponentList": [
                {
                    "code": "19228",
                    "type": "text",
                    "name": "detailAmount",
                    "label": "明细金额",
                    "require": true,
                    "configureObject": {}
                }
            ]
        }
    ]
}
```

## 2. 拉取单据/费用模板（树形结构）

**请求地址：**
- 测试环境：`POST` https://yiqbdatatest.superboss.cc/reimburse/base/getTemplateTree.rjson?appKey=xxx&accessToken=xxxx
- 线上环境：`POST` https://yiqbdata.superboss.cc/reimburse/base/getTemplateTree.rjson?appKey=xxx&accessToken=xxxx

**描述：** 拉取单据/费用模板（树形结构）

**ContentType：** `application/json`

### 请求参数

**Query 参数：**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| templateCode | string | 否 | 模板编码 |

**Body 参数：**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| needParentTemplate | boolean | 否 | 是否需要返回父级模板 |
| templateCode | string | 否 | 模板编码 |

### 请求示例

```json
{
    "needParentTemplate": true,
    "templateCode": "19228"
}
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 否 | 接口名称 |
| result | number | 否 | 响应码（1xx表示成功） |
| message | string | 否 | 响应信息 |
| data | array | 否 | 模板列表（树形） |
| └ templateCode | string | 否 | 模板编码 |
| └ parentTemplateCode | string | 否 | 父级模板编码 |
| └ templateName | string | 否 | 模板名称 |
| └ templateType | number | 否 | 模板类型 |
| └ orderTemplateType | number | 否 | 单据组件类型：1=报销单，2=借款单，3=付款单，4=费用类型，5=申请单，6=合同，7=项目，8=收款单 |
| └ configureObject | object | 否 | 组件配置 |
| └ components | array | 否 | 组件列表 |
|   └ code | string | 否 | 组件编码 |
|   └ type | string | 否 | 组件类型 |
|   └ label | string | 否 | 组件标签 |
|   └ name | string | 否 | 字段名称 |
|   └ require | boolean | 否 | 是否必填 |
|   └ configureObject | object | 否 | 组件配置 |
|   └ subformComponentList | array | 否 | 子表单组件列表 |
|     └ code | string | 否 | 组件实例编码 |
|     └ type | string | 否 | 组件实例类型 |
|     └ name | string | 否 | 字段名称 |
|     └ label | string | 否 | 字段标签 |
|     └ require | boolean | 否 | 是否必填 |
|     └ configureObject | object | 否 | 组件配置 |

### 响应示例

```json
{
    "apiName": "getTemplateTree",
    "result": 100,
    "message": "success",
    "data": [
        {
            "templateCode": "19228",
            "parentTemplateCode": "19228",
            "templateName": "差旅报销单",
            "templateType": 1,
            "orderTemplateType": 1,
            "configureObject": {},
            "components": [
                {
                    "code": "19228",
                    "type": "text",
                    "label": "金额",
                    "name": "amount",
                    "require": true,
                    "configureObject": {},
                    "subformComponentList": [
                        {
                            "code": "19228",
                            "type": "text",
                            "name": "detailAmount",
                            "label": "明细金额",
                            "require": true,
                            "configureObject": {}
                        }
                    ]
                }
            ]
        }
    ]
}
```
