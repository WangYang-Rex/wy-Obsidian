# 资金明细

> 资金明细查询接口，用于查询银行账户交易明细。

---

## 1. 资金明细列表

**请求地址：**
- 测试环境：`POST` https://yiqbdatatest.superboss.cc/reimburse/open/fund/detail/list.rjson?appKey=xxx&accessToken=xxxx
- 线上环境：`POST` https://yiqbdata.superboss.cc/reimburse/open/fund/detail/list.rjson?appKey=xxx&accessToken=xxxx

**描述：** 查询资金明细列表

**ContentType：** `application/json`

### 请求参数（Body）

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| acctNo | string | 是 | 支付账户卡号 |
| acctName | string | 是 | 支付账户用户名 |
| startTime | string | 是 | 明细查询开始时间（yyyy-MM-dd） |
| endTime | string | 是 | 明细查询结束时间（yyyy-MM-dd） |
| dcFlag | string | 否 | 借贷标识：d=借（支出），c=贷（收入） |
| toAcctNo | string | 否 | 对方账户卡号 |
| toAcctName | string | 否 | 对方账户名字 |
| page | number | 否 | 页码（默认查询首页10条） |
| pageSize | number | 否 | 分页条目数 |
| sortCode | string | 否 | 排序编码 |
| orderField | string | 否 | 排序字段（如 tranDateTime） |

### 请求示例

```json
{
    "acctNo": "6222021234567890",
    "acctName": "长沙易企云科技有限公司",
    "startTime": "2025-01-01",
    "endTime": "2025-06-30",
    "dcFlag": "",
    "page": 1,
    "pageSize": 10,
    "orderField": "tranDateTime"
}
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 否 | 接口名称 |
| result | number | 否 | 响应码（1xx表示成功） |
| message | string | 否 | 响应信息 |
| data | object | 否 | 响应数据 |
| └ detailList | object | 否 | 交易明细列表 |
|   └ success | boolean | 否 | 是否成功 |
|   └ count | number | 否 | 记录总数 |
|   └ errorMsg | string | 否 | 失败信息 |
|   └ totalAmount | number | 否 | 发票统计总额 |
|   └ totalArriveAmount | number | 否 | 到账总额 |
|   └ totalNoArriveAmount | number | 否 | 未到账总额 |
|   └ list | array | 否 | 明细数据 |
|     └ bankSeqNo | string | 否 | 银行流水号 |
|     └ toAcctNo | string | 否 | 对方账号 |
|     └ toAcctName | string | 否 | 对方户名 |
|     └ tranAmt | number | 否 | 交易金额 |
|     └ currCode | string | 否 | 币种 |
|     └ remark | string | 否 | 摘要 |
|     └ tranDateTimeStr | string | 否 | 交易时间 |
|     └ dcFlag | string | 否 | 借贷标识：d=借（支出），c=贷（收入） |
|     └ acctBal | number | 否 | 余额 |
|     └ usage | string | 否 | 用途 |
|     └ relationOrder | boolean | 否 | 是否关联单据 |
|     └ relationOrderCode | string | 否 | 单据号 |
|     └ receiptUrl | string | 否 | 回单链接 |
| └ corpPayAccount | object | 否 | 账户信息 |
|   └ bankCode | string | 否 | 银行编码 |
|   └ bankCardNum | string | 否 | 卡号 |
|   └ holdName | string | 否 | 户名 |
|   └ bankName | string | 否 | 银行名称 |
|   └ bankFullName | string | 否 | 开户行全称 |
|   └ acctBal | number | 否 | 余额 |
|   └ lastDayReceive | number | 否 | 昨日收入 |
|   └ lastDayExpend | number | 否 | 昨日支出 |
