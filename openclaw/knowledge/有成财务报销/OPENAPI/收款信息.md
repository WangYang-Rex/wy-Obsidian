# 收款信息

## 获取收款信息

> 维护人：swc

### 请求地址

| 环境 | URL |
|------|-----|
| 测试环境 | `POST https://yiqbdatatest.superboss.cc/reimburse/base/getBankCardInfoList.rjson?appKey=xxx&accessToken=xxxx` |
| 线上环境 | `POST https://yiqbdata.superboss.cc/reimburse/base/getBankCardInfoList.rjson?appKey=xxx&accessToken=xxxx` |

### ContentType

`application/x-www-form-urlencoded; charset=utf-8`

### 请求参数

无（通过 Query 参数传递 appKey 和 accessToken）

### 请求示例

```
POST /reimburse/base/getBankCardInfoList.rjson?appKey=xxx&accessToken=xxxx
```

### 响应参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| apiName | string | 否 | API 名称 |
| result | int32 | 否 | 响应码，1xx 表示成功 |
| message | string | 否 | 响应信息 |
| data | array | 否 | 收款信息列表 |
| └ cardId | int32 | 否 | 卡 ID |
| └ bankCardNum | string | 否 | 卡号 |
| └ holdName | string | 否 | 户名 |
| └ bankName | string | 否 | 银行名 |
| └ bankFullName | string | 否 | 开户行全称 |
| └ province | string | 否 | 省 |
| └ city | string | 否 | 市 |
| └ type | int32 | 否 | 类型：1=个人账户，2=对公账户 |
| └ creator | string | 否 | 创建人 |
| └ creatorName | string | 否 | 创建人姓名 |
| └ sourceType | int32 | 否 | 来源：1=用户创建，2=管理者创建，3=导入 |
