# 资产卡片变动

| **修订时间** | **版本** | **修订说明** |
|----------------|----------|----------------|
| 20230809 | v1.0 | 新建 |

## 1、需求背景

目前资产卡片信息只允许修改科目关联等信息，若遇到资产价值变动，需要修改资产价值与折旧值时，无法修改，目前无法支持这种场景；

因此，将资产卡片的信息放开更多可修改的地方。

### 4.2、本次可修改的字段：

**本次需求：已折旧的卡片，在当前可修改字段的基础上，支持修改更多卡片信息。**即当前卡片已折旧未折旧都允许修改：

**1、未开始在系统中折旧的资产卡片，允许修改---现有逻辑；**

**2、已清理的卡片不允许修改**

**3、已折旧完成的卡片，可以通过修改预计使用期限继续**

4、根据折旧方式，修改后对应的规则不一样，**本次只支持平均年限法的修改，当前卡片折旧方式为平均年限法时才允许修改。**

| **字段** | **规则** |
|----------|----------|
| 预计使用期限（月） | 一、修改时，预计使用期限不能小于已折旧期间二、修改后，**月折旧额**根据新的使用期限动态变更：月折旧额=期初净值/（预计使用期限-已折旧期间）其他字段不再自动变更三、修改后，根据预计使用期限和已折旧期间判断是否需要计提：1、预计使用期限\>已折旧期间，系统继续折旧2、预计使用期限=已折旧期间,认定为已经折旧完，不再进行计提折旧，卡片变更为折旧完成状态 |
| 原值 | 1、修改后不影响这张卡片之前关联的所有凭证数据2、修改后，这张卡片对应的预计残值、期初净值、月折旧根据计算公式自动变更：预计残值=原值*残值率期初净值=原值-期初累计折旧**月折旧额=（期初净值-预计残值）/（预计使用期限-已折旧期间）**期初累计折旧不变更。**3、新的账期进行计提折旧，每次都取卡片当前最新的月折旧额*\* |
| 税额 | 已经生成购入凭证，不能修改其他状态可修改 |
| 残值率 | 1、修改后不影响这张卡片之前关联的所有凭证数据2、修改后，这张卡片对应的预计残值、月折旧/摊销根据计算公式自动变更预计残值=原值*残值率**月折旧额=（期初净值-预计残值）/（预计使用期限-已折旧期间）*\*期初累计折旧不变更3、新的账期进行计提折旧，每次都取卡片当前最新的月折旧额即可4、残值率支持小数点，最多可输入六位小数 |
| 期初累计折旧/摊销 | 一、默认显示当前卡片已经计算出的累计折旧额二、这个字段修改后，期初净值和月折旧额随着这个字段进行自动变更期初净值=原值-期初累计折旧/摊销月折旧额=（期初净值-预计残值）/（预计使用期限-已折旧期间） |
| 月折旧/摊销 | 一、可修改，修改后每期，都取当前卡片中最新的折旧额进行折旧二、计提折旧时的一个特殊校验：1、每一期进行折旧时，需要增加判断，月折旧额大于月初净值，计提折旧时取净值折旧；2、折旧期间为最后一期，取卡片当前月份的月初净值进行计提。 |

5、当前卡片已进行计提折旧，修改了卡片信息，点击保存提示（修改科目类的不给提示，其他字段修改后给提示），提示文案如下：

  修改卡片可能会影响以后的计提折旧与报表，是否确认保存？

6、保存后，记录本次修改后的值与修改前的内容，并记录下来，后续可以在资产变动记录中查询到；

7、资产卡片列表展示：

原值、残值率、月折旧/摊销、残值：以上字段都显示当前卡片中最新的信息

记录每张卡片每一期的信息，后期可能会需要展示每一期卡片的信息记录

1、支持按变动期间查询，默认当前未结账那一期，可以下拉切换期间；

可筛选的时间范围为账套启用期间到最后有凭证那一个期间--跟目前系统内的期间查询控制一致；

2、一个卡片\+一个字段的修改内容生成一条记录

3、默认显示规则：当前筛选的变动期间内根据变动时间降序排序，变动时间最新的排列在上面；

4、需要进行分页；

5、根据列表的表头部分支持直接搜索，搜索的样式参考下报销中的表头搜索样式。具体列表字段规则如下：

| 变动前 | 记录卡片对应纬度修改前的值 |
| 变动后 | 卡片对应纬度修改后的值 |
| 变动期间 | 卡片修改时对应的当前未结账期间，也是对应的菜单中默认的筛选期间 |

2023年03期     原值   12000

| **字段** | **规则** |
|----------|----------|
| 原值 | **一、展示卡片结束期间的原值信息：**比如筛选期间为2023年01期到2023年03期，原值显示为12000比如筛选期间为2023年01期到2023年01期，原值显示为10000二、变动信息显示：期初值：开始期间的卡片原值调整值：在期间内变动的值，期末值-期初值，若是增加，则正数显示，若是减少，则负数显示，没有调整则为空期末值：结束期间的卡片原值 |
| 残值 | 一、取结束期间的残值显示二、变动信息显示：期初值：开始期间修改的卡片，修改前的残值调整值：在期间内变动的值，期末值-期初值，若是增加，则正数显示，若是减少，则负数显示，没有调整则为空期末值：结束期间卡片的残值 |
| 月折旧率 | 一、只筛选一个期间，月折旧率参照原来公式：月折旧额/原值二、筛选多个期间，且每个期间的折旧额不一致，则月折旧率不自动计算显示 |
| 已使用/总期限 | 总期限显示结束期间的最新期限时间显示 |
| **期初累计折旧** | **1、本次新增的字段，也在勾选显示变动信息后显示****2、显示开始期间的未折旧前的累计折旧值****累计折旧值不再仅仅是每个月折旧额的汇总统计，改为按卡片中的记录，统计卡片当前期间的累计折旧值，会包含调整的值** |
| 每期折旧值 | 展示卡片每个期间进行计提折旧时，卡片中对应期间实际的月折旧/摊销额 |
| **折旧调整值** | **一、本次新增的字段，在勾选显示变动信息后显示****二、若是增加，则正数显示，若是减少，则负数显示，没有调整则为空****期末累计折旧-期初累计折旧-所选期间内的折旧值** |
| 本年累计折旧/摊销 | 原逻辑 |
| 期末累计折旧/摊销 | 期初累计折旧\+期间内每月的折旧值\+期间内折旧调整值始终会等于结束期间的期末累计折旧值 |
| 期末净值 | 原逻辑   原值-期末累计折旧 |
