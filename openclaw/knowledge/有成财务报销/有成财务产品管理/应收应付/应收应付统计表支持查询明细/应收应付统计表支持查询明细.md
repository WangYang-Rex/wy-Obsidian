## 1、需求背景

客户：义乌天屿文化科技有限公司

在查看应收应付统计表时，能跳转打开不同核销类型的明细

## 2、需求详情

**一、应收应付统计表内，点击金额支持跳转到应收核销明细或应付核销明细；**

1、交互：鼠标移入金额位置，出现可点击样式（参照科目余额表），点击之后，新开TAB页，打开核销明细页面，同时将当前的条件作为查询条件带入

2、支持点击的金额列与带入的条件有：

| **金额列** | **跳转的目标页面** | **带入的条件** |
|-------------|-------------------------|-------------------|
| 共用的查询条件有：\+ 期间=应收应付统计表的区间\+ 辅助类别=当前选中的类别\+ 往来单位=列中的名称 |  |  |
| 应收金额 | 应收核销明细 | 辅助类别=当前选中的类别；往来单位=列中的“名称” |
| 已收金额 |  | 辅助类别=当前选中的类别；往来单位=列中的“名称”核销类型=已收款 |
| 坏账金额 |  | 辅助类别=当前选中的类别；往来单位=列中的“名称”核销类型=坏账准备 |
| 对冲金额 |  | 辅助类别=当前选中的类别；往来单位=列中的“名称”核销类型=应付款对冲 |
| 其他金额（应收） |  | 辅助类别=当前选中的类别；往来单位=列中的“名称”核销类型=其他 |
| 应付金额 | 应付核销明细 | 辅助类别=当前选中的类别；往来单位=列中的“名称” |
| 已付金额 |  | 辅助类别=当前选中的类别；往来单位=列中的“名称”核销类型=已付款 |
| 转收入金额 |  | 辅助类别=当前选中的类别；往来单位=列中的“名称”核销类型=计为收入 |
| 对冲金额 |  | 辅助类别=当前选中的类别；往来单位=列中的“名称”核销类型=应收款对冲 |
| 其他金额（应付） |  | 辅助类别=当前选中的类别；往来单位=列中的“名称”核销类型=其他 |
