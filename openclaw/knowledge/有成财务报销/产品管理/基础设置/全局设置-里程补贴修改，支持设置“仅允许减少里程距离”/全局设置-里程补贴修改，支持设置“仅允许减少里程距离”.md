## 1、需求背景

客户公司为了管控里程补贴费用报销，员工修改里程时，仅允许减少里程距离

## 2、业务流程/架构

## 3、功能清单

根据个人需要补充

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 全局设置 | “修改里程补贴”支持设置“仅允许减少里程” |  |
| 里程补贴 | 修改里程时，根据配置，控制修改范围 |  |
|  |  |  |

## 4、需求详情

### 4.1、全局设置
1. 全局设置-其他设置-里程补贴修改，在“允许修改”选项后增加“里程距离修改限制”
2. 只有选中“允许修改”时，才可对“里程距离修改限制”进行选择
3. “里程距离修改限制”选项：无：默认选项，不做限制（同当前逻辑）仅允许减少距离：经允许在<span style="color: #DF2A3F;">初始里程距离</span>基础上减少里程数



![Picture 1](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/ABmOoWbXmgK6WOaw/img/d9eeda05-77e4-4e73-9126-685cf775c9fe.png?Expires=1774156021&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=UNoA95s76QhCLxkiglVQVK7i91k%3D "")

### 4.2、里程补贴

<span style="color: #DF2A3F;">“里程距离修改限制”仅对里程修改范围进行限制，里程金额计算等逻辑不受影响</span>
4. **允许修改；里程距离修改限制：无**修改里程时，不做限制（同现在）
5. **允许修改；里程距离修改限制：仅允许减少距离**修改里程时，仅允许在最初的里程数基础上改小；当距离大于最初里程数时，限制提交并提示：**仅允许在初始里程数上减少距离**示例：里程点击结束时，记录距离为A，A即为<span style="color: #DF2A3F;">初始里程数</span>；修改为B，B的合理范围：0 \<= B \<= A

### 4.3、其他优化
6. 修改里程时，当距离填写为负数时，优化提示语



![Picture 2](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/ABmOoWbXmgK6WOaw/img/9c8d9ea7-f204-4858-9166-99ea96d10dab.png?Expires=1774156021&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=24%2BQttLLYsAZVC4YH631Y1eItd0%3D "")

**现状**：距离修改为负数，提交时提示：**请填写距离**

**调整成**：负数提交时提示：**距离不可为负数**
