## 1、需求背景

部分用户在查看数据时，会通过高级查询进行筛选，查看自己关注的数据；但再次查看数据时，上次设置内容将被清空，用户需要重新设置，操作繁琐

## 2、业务流程/架构

如复杂需求，放置相关流程/架构图  

## 3、涉及系统

**有成财务**

## 4、功能清单

根据个人需要补充

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 设置-系统参数-全局设置 | 增加“高级查询设置”支持设置是否开启“系统默认记住并填充上次设置内容” |  |
| 高级查询 | 开启以上设置后，高级查询框中记住并填充上次设置内容 |  |
|  |  |  |

## 5、需求详情

### 5.1、设置-系统参数-全局设置
1. 全局设置中增加“**高级查询设置**”配置项
2. 默认为“**未勾选**”状态：同当前逻辑（不做上次设置内容填充）
3. 勾选后，高级查询框中默认填充上次设置内容



![Picture 1](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/Mp7ld7bBRjx21OBQ/img/341d40a7-41a8-4e22-a6fa-86d2e74194a2.png?Expires=1774156024&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=8EwM%2Blkv%2FcCaWCRVghvt3WdnnO4%3D "")

### 5.2、高级查询
4. **系统默认记住并填充上次设置内容** 勾选状态：**未勾选**；保持当前逻辑
5. **系统默认记住并填充上次设置内容** 勾选状态：**勾选**；筛选内容默认填充当前用户上次设置内容
