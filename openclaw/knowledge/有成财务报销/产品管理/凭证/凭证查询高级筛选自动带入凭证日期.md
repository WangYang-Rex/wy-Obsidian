# **1\. 需求背景**

用户在查询凭证时，如果先在快速筛选中选择了凭证月份，在高级搜索中还得再选一次，不是很方便。

南羽集团

# **3\. 功能清单**

|------|---------|------|
| 凭证 | 查看凭证-高级查询，自动带入快速筛选框中的凭证日期 |  |
|  |  |  |
|  |  |  |

# **4\. 需求详情**

凭证-查看凭证

1、当高级搜索中的凭证日期没有值的时候，自动带入快速筛选框中的凭证日期。

**开始期间处理成月初、结束期间处理成月末**，自动带出。修改后不再更新外层快速筛选框中的值。

2、当高级搜索中已经进行过查询的时候，带出上次选择的查询日期（线上已有）

启用了调整期时无需带入。
