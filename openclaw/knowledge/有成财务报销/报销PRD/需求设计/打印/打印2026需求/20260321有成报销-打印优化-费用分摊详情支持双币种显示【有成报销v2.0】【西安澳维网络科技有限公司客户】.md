## <span style="color: rgba(0, 0, 0, 0.9);">**1\. 需求背景**</span>

### <span style="color: rgba(0, 0, 0, 0.9);">**1.1 需求描述**</span>

<span style="color: rgba(0, 0, 0, 0.9);">优化单据审批单打印模板，当单据内待支付金额币种非本位币时，费用分摊详情区域同时显示待支付金额币种金额和系统本位币金额，解决当前仅显示本位币（人民币）金额导致的财务对账困难问题。</span>

### <span style="color: rgba(0, 0, 0, 0.9);">**1.2 需求背景**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**需求方**</span><span style="color: rgba(0, 0, 0, 0.9);">：财务管理员</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**现状问题**</span><span style="color: rgba(0, 0, 0, 0.9);">：
• 系统本位币为人民币
• 单据内待支付金额支持多币种选择（USD/EUR/GBP等）
• 外币单据审批通过后，打印预览中费用分摊详情仅显示人民币金额
• 财务对账时需根据待支付金额币种手动反推外币分摊金额，效率低且易出错
• 无法满足跨国企业多币种报销的审计要求</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**解决思路**</span><span style="color: rgba(0, 0, 0, 0.9);">：打印模板读取单据待支付金额币种，非本位币时自动展示双币种金额（待支付币种\+本位币）</span>

<span style="color: rgba(0, 0, 0, 0.9);">**业务流程**</span>
```plain
单据打印模版内开启“打印费用分摊详情”且单据内待支付金额选择外币币种(USD)
    ↓
单据审批完成
    ↓
点击【打印】或【打印预览】
    ↓
系统检测单据待支付金额币种 ≠ 本位币(CNY)
    ↓
费用分摊详情区域展示：
  - 原币金额(USD)：按待支付金额币种显示
  - 本币金额(CNY)：按系统本位币显示
    ↓
财务对账时可同时查看原币和本币金额
```

### <span style="color: rgba(0, 0, 0, 0.9);">**3.2 详细功能描述**</span>

#### <span style="color: rgba(0, 0, 0, 0.9);">**3.2.1 双币种金额展示**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**功能概述**</span> <span style="color: rgba(0, 0, 0, 0.9);">打印模板读取单据内"支付金额"组件的币种，当该币种不等于系统本位币（CNY）时，费用分摊详情表格同时显示"支付金额"组件的币种和本位币金额。</span>

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/54Lq35o54oJ89l7E/img/8283406e-e396-4c4c-8c1d-76b25d730a8d.png?Expires=1774156263&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=lITFBRr6SEWkaFd2WhALii%2BxyEo%3D "")



<span style="color: rgba(0, 0, 0, 0.9);">**币种判断逻辑**</span>
```text
IF 单据.“支付金额”组件选择的币种 == 系统本位币(CNY) THEN
    费用分摊详情显示：分摊金额(CNY)
ELSE
    费用分摊详情显示：原币金额(单据币种) + 本币金额(CNY)
END IF
```



<span style="color: rgba(0, 0, 0, 0.9);">**配置入口**</span> <span style="color: rgba(0, 0, 0, 0.9);">【单据管理-报销单/付款申请单-打印/打印预览-费用分摊详情区域】</span>

<span style="color: rgba(0, 0, 0, 0.9);">**打印效果变更**</span>

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/54Lq35o54oJ89l7E/img/80d9b385-3f66-42ab-ae19-056202d0dec0.png?Expires=1774156263&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=Sd6caKfQGO7%2Fls2IBgwlB2nf7tg%3D "")



<span style="color: rgba(0, 0, 0, 0.9);">**计算逻辑**</span>
```plain
原币分摊金额 = 本币分摊金额 / 单据汇率
本币分摊金额 = 单据分摊比例 × 单据本币支付金额
```

## <span style="color: rgba(0, 0, 0, 0.9);">**附录**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**A. 典型使用场景**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**场景：外企财务打印美元报销单**</span>

<span style="color: rgba(0, 0, 0, 0.9);">背景：
• 系统本位币：人民币（CNY）
• 员工报销美元差旅费，单据待支付金额\$1,400（币种USD）
• 费用需分摊到项目A（60%）和项目B（40%）</span>

<span style="color: rgba(0, 0, 0, 0.9);">使用步骤：</span>
1. <span style="color: rgba(0, 0, 0, 0.9);">员工提交报销单，支付金额组件选择币种USD，金额\$1,400</span>
2. <span style="color: rgba(0, 0, 0, 0.9);">审批流程走完，单据状态变为"审批通过"</span>
3. <span style="color: rgba(0, 0, 0, 0.9);">财务进入单据详情页，点击【打印预览】</span>
4. <span style="color: rgba(0, 0, 0, 0.9);">打印预览页面费用分摊详情显示：</span>


```plain
项目A：$840.00 (USD) = ￥6,000.00 (CNY)
项目B：$560.00 (USD) = ￥4,000.00 (CNY)
```


5. <span style="color: rgba(0, 0, 0, 0.9);">点击【打印】或【导出PDF】，文件同步展示双币种金额</span>
6. <span style="color: rgba(0, 0, 0, 0.9);">财务将打印件作为审计凭证归档</span>

<span style="color: rgba(0, 0, 0, 0.9);">结果：财务无需手动换算，审计时可同时查看原币（与待支付金额一致）和本位币金额，满足跨国企业合规要求。</span>
