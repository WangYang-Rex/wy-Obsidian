## <span style="color: rgba(0, 0, 0, 0.9);">**1\. 需求背景**</span>

### <span style="color: rgba(0, 0, 0, 0.9);">**1.1 需求描述**</span>

<span style="color: rgba(0, 0, 0, 0.9);">支持一个单据类型配置多个打印模板，用户打印时可根据业务场景选择不同模板，满足同一单据在不同场景下的差异化打印需求。</span>

### <span style="color: rgba(0, 0, 0, 0.9);">**1.2 需求背景**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**需求方**</span><span style="color: rgba(0, 0, 0, 0.9);">：财务人员/系统管理员</span>

<span style="color: rgba(0, 0, 0, 0.9);">**现状问题**</span><span style="color: rgba(0, 0, 0, 0.9);">：</span>
- <span style="color: rgba(0, 0, 0, 0.9);">当前一个单据类型仅支持绑定一个打印模板</span>
- <span style="color: rgba(0, 0, 0, 0.9);">不同部门/场景对同一单据的打印格式要求不同（如财务留存版、业务审批版、外部报送版）</span>
- <span style="color: rgba(0, 0, 0, 0.9);">需要频繁切换模板或手动调整，效率低且易出错</span>

<span style="color: rgba(0, 0, 0, 0.9);">**解决思路**</span><span style="color: rgba(0, 0, 0, 0.9);">：支持一个单据类型配置多个打印模板，可设置默认模板，打印时用户自行选择模板（不支持自动匹配）</span>

<span style="color: rgba(0, 0, 0, 0.9);">**业务流程**</span>
```plain
管理员配置多个打印模板
    ↓
用户打印单据时
    ↓
系统展示可用模板列表
    ↓
用户选择模板 → 打印
```

<span style="color: rgba(0, 0, 0, 0.9);">**默认模板规则**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">一个单据类型仅能设置一个默认模板</span>
- <span style="color: rgba(0, 0, 0, 0.9);">打印时如未指定模板，自动使用默认模板</span>
- <span style="color: rgba(0, 0, 0, 0.9);">删除默认模板前需先指定新的默认模板</span>

<span style="color: rgba(0, 0, 0, 0.9);">**新增模板初始化规则**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">新增模板时，</span><span style="color: rgba(0, 0, 0, 0.9);">**使用产品原逻辑**</span><span style="color: rgba(0, 0, 0, 0.9);">自动填充初始模板组件样式</span>
- <span style="color: rgba(0, 0, 0, 0.9);">初始组件包含：单据标题、申请人信息、费用明细表格、金额合计、审批签名区等标准组件</span>
- <span style="color: rgba(0, 0, 0, 0.9);">用户可在初始化基础上进行字段增删和样式调整</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**说明**</span><span style="color: rgba(0, 0, 0, 0.9);">：旧版打印设计器和高级版打印设计器的新增模板设计初始化逻辑保持一致，均使用产品原逻辑</span>

<span style="color: rgba(0, 0, 0, 0.9);">**打印设计器版本权限控制**</span>

<span style="color: rgba(0, 0, 0, 0.9);">有成报销支持两种打印设计器版本，</span><span style="color: rgba(0, 0, 0, 0.9);">**两个版本互斥使用，同一时间仅能使用一个版本**</span><span style="color: rgba(0, 0, 0, 0.9);">：</span>

| <span style="color: rgba(0, 0, 0, 0.9);">**版本**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**开通方式**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**功能范围**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**使用状态**</span> |
|----------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| <span style="color: rgba(0, 0, 0, 0.9);">旧版打印设计器</span> | <span style="color: rgba(0, 0, 0, 0.9);">免费，系统默认</span> | <span style="color: rgba(0, 0, 0, 0.9);">基础模板编辑功能，单模板</span> | <span style="color: rgba(0, 0, 0, 0.9);">默认启用</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">高级版打印设计器</span> | <span style="color: rgba(0, 0, 0, 0.9);">付费开通</span> | <span style="color: rgba(0, 0, 0, 0.9);">高级模板设计功能，支持多模板</span> | <span style="color: rgba(0, 0, 0, 0.9);">开通后切换</span> |

<span style="color: rgba(0, 0, 0, 0.9);">**版本切换规则**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**互斥原则**</span><span style="color: rgba(0, 0, 0, 0.9);">：旧版打印设计器和高级版打印设计器</span><span style="color: rgba(0, 0, 0, 0.9);">**不能同时使用**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">未开通高级版时：使用旧版打印设计器，仅支持单模板</span>
- <span style="color: rgba(0, 0, 0, 0.9);">开通高级版后：自动切换至高级版打印设计器，支持多模板功能</span>
- <span style="color: rgba(0, 0, 0, 0.9);">高级版到期/退订后：自动回退至旧版打印设计器</span>

<span style="color: rgba(0, 0, 0, 0.9);">**未开通高级版时的限制（旧版打印）**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">进入单据模板管理界面时，仅显示默认打印模板</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**允许编辑默认模板**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**允许设计默认模板**</span><span style="color: rgba(0, 0, 0, 0.9);">（进入模板设计器进行字段配置和样式调整）</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**不允许新增模板**</span><span style="color: rgba(0, 0, 0, 0.9);">（隐藏或禁用新增按钮）</span>
- <span style="color: rgba(0, 0, 0, 0.9);">页面顶部/空白区域显示开通引导信息：</span>
    - <span style="color: rgba(0, 0, 0, 0.9);">文案示例：「开通高级版打印设计器，支持多个打印模板配置，满足不同场景打印需求」</span>
    - <span style="color: rgba(0, 0, 0, 0.9);">操作按钮：「联系客服开通」</span>
    - <span style="color: rgba(0, 0, 0, 0.9);">点击后跳转客服会话或展示客服联系方式</span>

<span style="color: rgba(0, 0, 0, 0.9);">**开通高级版后的功能（高级版打印）**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">系统自动从旧版切换至高级版打印设计器</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**允许新增打印模板**</span><span style="color: rgba(0, 0, 0, 0.9);">：单据模板管理界面显示【新增模版】功能按钮，支持自由新增打印模板</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**允许编辑默认模板**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**允许设计默认模板**</span><span style="color: rgba(0, 0, 0, 0.9);">（进入模板设计器进行字段配置和样式调整）</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**允许复制模板**</span><span style="color: rgba(0, 0, 0, 0.9);">：支持复制已有模板作为新模板基础</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**允许设置默认模板**</span><span style="color: rgba(0, 0, 0, 0.9);">：支持切换默认模板</span>

<span style="color: rgba(0, 0, 0, 0.9);">**高级版退回旧版的业务逻辑**</span> <span style="color: rgba(0, 0, 0, 0.9);">当用户退订高级版或高级版到期时：</span>
