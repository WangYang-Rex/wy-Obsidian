# **1\. 需求背景**
1. 目标用户：财务
2. 需求来源：吉林太羽科技客户
3. 背景：

a.多选单据不能直接打印，需要导出下载之后才能操作，老师，你看是什么原因？上个月还可以直

接打印呢，用户仅仅需要批量打印单据面单

b.前序需求为避免批量打印单据内附件过多时，超过1张以上单据批量打印，直接去下载中心导出
4. 核心痛点：单据批量打印时，若只批量打印面单，则支持50张以下面单可直接调用预览窗口
