### <span style="color: rgba(0, 0, 0, 0.9);">**1\. 需求背景**</span>

#### <span style="color: rgba(0, 0, 0, 0.9);">**1.1 需求描述**</span>

<span style="color: rgba(0, 0, 0, 0.9);">审批流评论区@某人时，企业微信工作通知及"待办-我参与的"入口，均根据被@人实时审批权限动态展示审批操作。</span>

#### <span style="color: rgba(0, 0, 0, 0.9);">**1.2 需求背景**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**需求方：**</span> <span style="color: rgba(0, 0, 0, 0.9);">审批人/被@员工</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**现状问题：**</span>
    - <span style="color: rgba(0, 0, 0, 0.9);">被@人收到企业微信工作通知，点击后进入有成报销H5单据详情</span>
    - <span style="color: rgba(0, 0, 0, 0.9);">被@人可在"待办-我参与的"看到@他的单据</span>
    - <span style="color: rgba(0, 0, 0, 0.9);">两个入口均只展示单据详情，即使被@人有审批权限也无法操作</span>
    - <span style="color: rgba(0, 0, 0, 0.9);">需返回待办列表找到单据才能审批，路径冗余</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**解决思路：**</span> <span style="color: rgba(0, 0, 0, 0.9);">企业微信员工点击被@单据工作通知打开页面时，实时读取该账号在审批流中的权限，根据权限展示带审批操作的单据详情；"我参与的"页面同步按此逻辑改造</span>

<span style="color: rgba(0, 0, 0, 0.9);">**业务流程：**</span>
```plain
被@人收到通知/进入我参与的列表
  ↓
点击单据进入详情
  ↓
实时读取该员工在审批流的权限
  ↓
权限判定
  ├─ 当前节点有权限 → 展示单据详情 + 审批操作
  └─ 无权限 → 仅展示单据详情
```

### <span style="color: rgba(0, 0, 0, 0.9);">**2\. 功能现状**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**功能所在页面路径：**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">企业微信：工作通知 → 有成报销@提醒 → 单据详情（H5）</span>
- <span style="color: rgba(0, 0, 0, 0.9);">有成APP：待办 → 我参与的 → 单据详情</span>

<span style="color: rgba(0, 0, 0, 0.9);">**当前能力：**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">评论区@某人，被@人收到工作通知</span>
- <span style="color: rgba(0, 0, 0, 0.9);">被@人可在"我参与的"列表查看单据</span>
- <span style="color: rgba(0, 0, 0, 0.9);">单据详情展示完整单据信息</span>

<span style="color: rgba(0, 0, 0, 0.9);">**当前缺失的能力：**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">未根据被@人实时审批权限动态展示审批操作入口</span>

#### <span style="color: rgba(0, 0, 0, 0.9);">**3.2 详细功能描述**</span>

##### <span style="color: rgba(0, 0, 0, 0.9);">**3.2.1 实时权限读取与渲染**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**核心逻辑**</span> <span style="color: rgba(0, 0, 0, 0.9);">进入单据详情页时，实时调用权限查询接口，根据返回结果渲染页面。</span>

<span style="color: rgba(0, 0, 0, 0.9);">**权限判定规则**</span>

| <span style="color: rgba(0, 0, 0, 0.9);">**场景**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**实时权限状态**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**页面展示**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**说明**</span> |
|----------------------------------------------------------|----------------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------|
| <span style="color: rgba(0, 0, 0, 0.9);">场景1</span> | <span style="color: rgba(0, 0, 0, 0.9);">当前审批节点处理人</span> | <span style="color: rgba(0, 0, 0, 0.9);">单据详情 \+ 审批操作区</span> | <span style="color: rgba(0, 0, 0, 0.9);">有审批权限</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">场景2</span> | <span style="color: rgba(0, 0, 0, 0.9);">非当前节点（提前@）</span> | <span style="color: rgba(0, 0, 0, 0.9);">仅单据详情</span> | <span style="color: rgba(0, 0, 0, 0.9);">无审批权限</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">场景3</span> | <span style="color: rgba(0, 0, 0, 0.9);">已审批完成/流程结束</span> | <span style="color: rgba(0, 0, 0, 0.9);">仅单据详情</span> | <span style="color: rgba(0, 0, 0, 0.9);">无审批权限</span> |

<span style="color: rgba(0, 0, 0, 0.9);">**权限读取时机：**</span> <span style="color: rgba(0, 0, 0, 0.9);">进入单据详情页时实时查询，非缓存</span>

##### <span style="color: rgba(0, 0, 0, 0.9);">**3.2.2 企业微信H5单据详情**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**入口：**</span> <span style="color: rgba(0, 0, 0, 0.9);">工作通知 → @提醒 → 点击打开H5</span>

<span style="color: rgba(0, 0, 0, 0.9);">**有权限时：**</span>
```plain
┌─────────────────────────┐
│ 单据详情               │
│                         │
│ [审批操作区]            │
│ [同意] [驳回] [转交]    │
│                         │
│ 单据信息...             │
└─────────────────────────┘
```



<span style="color: rgba(0, 0, 0, 0.9);">**无权限时：**</span>
```plain
┌─────────────────────────┐
│ 单据详情               │
│                         │
│ （无审批操作区）        │
│                         │
│ 单据信息...             │
└─────────────────────────┘
```



##### <span style="color: rgba(0, 0, 0, 0.9);">**3.2.3 我参与的页面迭代**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**页面路径：**</span> <span style="color: rgba(0, 0, 0, 0.9);">待办 → 我参与的</span>

<span style="color: rgba(0, 0, 0, 0.9);">**列表页变更：**</span>

| <span style="color: rgba(0, 0, 0, 0.9);">**字段**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**展示规则**</span> |
|----------------------------------------------------------|----------------------------------------------------------------|
| <span style="color: rgba(0, 0, 0, 0.9);">单据标题</span> | <span style="color: rgba(0, 0, 0, 0.9);">展示</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">单据状态</span> | <span style="color: rgba(0, 0, 0, 0.9);">展示</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">@来源标识</span> | <span style="color: rgba(0, 0, 0, 0.9);">展示</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">审批快捷入口</span> | <span style="color: rgba(0, 0, 0, 0.9);">**新增**</span><span style="color: rgba(0, 0, 0, 0.9);">，有权限时展示\[审批\]按钮</span> |

<span style="color: rgba(0, 0, 0, 0.9);">**详情页变更：**</span>

### <span style="color: rgba(0, 0, 0, 0.9);">**3.3 逻辑互斥与边界处理**</span>

| <span style="color: rgba(0, 0, 0, 0.9);">**边界场景**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**触发条件**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**处理方案**</span> |
|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| <span style="color: rgba(0, 0, 0, 0.9);">权限实时变更</span> | <span style="color: rgba(0, 0, 0, 0.9);">进入页面时有权限，操作前被转交</span> | <span style="color: rgba(0, 0, 0, 0.9);">提交时二次校验，提示"当前节点已处理"</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">并发操作</span> | <span style="color: rgba(0, 0, 0, 0.9);">多人同时点击审批</span> | <span style="color: rgba(0, 0, 0, 0.9);">提交时按现有并发控制，后提交者提示"已被处理"</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">网络异常</span> | <span style="color: rgba(0, 0, 0, 0.9);">权限查询接口失败</span> | <span style="color: rgba(0, 0, 0, 0.9);">降级为仅展示详情，提示"权限获取失败，请重试"</span> |

| <span style="color: rgba(0, 0, 0, 0.9);">单据已作废</span> | <span style="color: rgba(0, 0, 0, 0.9);">查看时单据已作废</span> | <span style="color: rgba(0, 0, 0, 0.9);">展示作废状态，无审批操作</span> |
