### <span style="color: rgba(0, 0, 0, 0.9);">**1\. 需求背景**</span>

#### <span style="color: rgba(0, 0, 0, 0.9);">**1.1 需求描述**</span>

<span style="color: rgba(0, 0, 0, 0.9);">草稿箱中的已驳回、已撤回单据，支持在费用明细组件内直接修改费用类型，与新建草稿单据保持一致的编辑体验。</span>

#### <span style="color: rgba(0, 0, 0, 0.9);">**1.2 需求背景**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**需求方：**</span> <span style="color: rgba(0, 0, 0, 0.9);">报销申请人</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**现状问题：**</span>
    - <span style="color: rgba(0, 0, 0, 0.9);">新建单据存入草稿箱：费用类型</span><span style="color: rgba(0, 0, 0, 0.9);">**可编辑**</span> <span style="color: rgba(0, 0, 0, 0.9);">✓</span>
    - <span style="color: rgba(0, 0, 0, 0.9);">驳回单据回到草稿箱：费用类型</span><span style="color: rgba(0, 0, 0, 0.9);">**置灰**</span> <span style="color: rgba(0, 0, 0, 0.9);">✗</span>
    - <span style="color: rgba(0, 0, 0, 0.9);">撤回单据回到草稿箱：费用类型</span><span style="color: rgba(0, 0, 0, 0.9);">**置灰**</span> <span style="color: rgba(0, 0, 0, 0.9);">✗</span>
    - <span style="color: rgba(0, 0, 0, 0.9);">三者同在草稿箱，编辑权限不一致，用户体验割裂</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**解决思路：**</span> <span style="color: rgba(0, 0, 0, 0.9);">统一草稿箱内所有单据状态的费用类型编辑权限</span>

<span style="color: rgba(0, 0, 0, 0.9);">**业务流程对比：**</span>
```plain
【现状 - 草稿箱内操作】
新建草稿 → 费用类型可编辑 ✓
驳回单据 → 费用类型置灰 ✗ → 被迫删除重新创建
撤回单据 → 费用类型置灰 ✗ → 被迫删除重新创建

【变更后 - 草稿箱内操作】
新建草稿 → 费用类型可编辑 ✓
驳回单据 → 费用类型可编辑 ✓
撤回单据 → 费用类型可编辑 ✓
```

#### <span style="color: rgba(0, 0, 0, 0.9);">**3.2 详细功能描述**</span>

##### <span style="color: rgba(0, 0, 0, 0.9);">**3.2.1 草稿箱费用类型编辑统一**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**功能概述**</span> <span style="color: rgba(0, 0, 0, 0.9);">统一草稿箱内三种来源单据（新建、驳回、撤回）的费用类型编辑权限，全部开放可编辑。</span>

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/jP2lRYj8d1X8JO8g/img/be7e2d97-9178-46cf-9d60-404bc03d88af.png?Expires=1775709713&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=RuVoQH0kDlO%2F5DVVwmW9KRj5OXk%3D "")



<span style="color: rgba(0, 0, 0, 0.9);">**变更前后对比：**</span>

| <span style="color: rgba(0, 0, 0, 0.9);">**状态**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**变更前**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**变更后**</span> |
|----------------------------------------------------------|-------------------------------------------------------------|-------------------------------------------------------------|
| <span style="color: rgba(0, 0, 0, 0.9);">草稿（新建）</span> | <span style="color: rgba(0, 0, 0, 0.9);">可编辑 ✓</span> | <span style="color: rgba(0, 0, 0, 0.9);">可编辑 ✓</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">已驳回</span> | <span style="color: rgba(0, 0, 0, 0.9);">**置灰**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**可编辑**</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">已撤回</span> | <span style="color: rgba(0, 0, 0, 0.9);">**置灰**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**可编辑**</span> |

<span style="color: rgba(0, 0, 0, 0.9);">**交互设计**</span>
```plain
【变更前】
费用类型：[差旅费]  ← 置灰不可点

【变更后】
费用类型：[差旅费 ▼]  ← 可点击下拉
```



<span style="color: rgba(0, 0, 0, 0.9);">**编辑规则：**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">点击后弹出费用类型选择器（与新建草稿一致）</span>
- <span style="color: rgba(0, 0, 0, 0.9);">支持搜索筛选费用类型</span>
- <span style="color: rgba(0, 0, 0, 0.9);">选择后立即保存至单据</span>
- <span style="color: rgba(0, 0, 0, 0.9);">消费信息费用类型同步更新</span>
