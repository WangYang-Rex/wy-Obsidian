## <span style="color: rgba(0, 0, 0, 0.9);">**1\. 需求背景**</span>

### <span style="color: rgba(0, 0, 0, 0.9);">**1.1 需求描述**</span>

<span style="color: rgba(0, 0, 0, 0.9);">在委托授权功能中，增加"已审批单据可见性"配置，控制委托人是否可以在【已审批】列表中查看被委托人代为审批完成的单据。</span>

### <span style="color: rgba(0, 0, 0, 0.9);">**1.2 需求背景**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**需求方**</span><span style="color: rgba(0, 0, 0, 0.9);">：财务</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**现状问题**</span><span style="color: rgba(0, 0, 0, 0.9);">：
• 企业启用委托授权后，委托人将审批权限委托给被委托人
• 被委托人完成单据审批后，委托人无法在自己的【已委托】列表中追踪这些单据
• 委托人需要事后追溯被委托人代为审批的单据时，缺乏统一查看入口
• 不同企业对数据可见性有不同要求：有的需要委托人可见以便监督，有的出于隐私考虑不需要展示</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**解决思路**</span><span style="color: rgba(0, 0, 0, 0.9);">：
• 在【基础设置-全局设置-单据设置-委托授权权限】中增加"委托单据可见性"开关
• 在待办-审批中增加“已委托”，展示委托人委托出去的任务</span>

<span style="color: rgba(0, 0, 0, 0.9);">**业务流程**</span>
```plain
委托人发起委托授权
↓
被委托人代为审批单据
↓
单据审批完成
↓
【已审批单据可见性=开启】→ 同步显示在委托人"已审批"列表
【已审批单据可见性=关闭】→ 不显示在委托人"已审批"列表（仅被委托人可见）
```

#### <span style="color: rgba(0, 0, 0, 0.9);">**3.2.2 委托人已审批列表-委托审批单据展示**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**功能概述**</span> <span style="color: rgba(0, 0, 0, 0.9);">当"授权人已委托单据可见"开启时，被委托人代为审批完成的单据将显示在委托人的【已委托】列表中”。委托人仅能查看委托出去的单据，无法对单据进行操作。委托人委托出去的审批任务无论在审批节点哪个环节，委托人都可在“已委托”查看。</span>

<span style="color: rgba(0, 0, 0, 0.9);">**业务流程**</span>
```plain
用户访问【待办-审批-已委托】
↓
系统查询当前用户作为委托人的委托关系
↓
查询委托人委托出的单据
↓
【已委托单据可见性=开启】→ 合并显示到委托人【待办-审批-已委托】列表
【已委托单据可见性=关闭】→ 不显示
↓
【已委托】列表展示委托的单据
```

<span style="color: rgba(0, 0, 0, 0.9);">**A. 典型使用场景**</span>

<span style="color: rgba(0, 0, 0, 0.9);">场景：财务主管委托会计代为审批报销单，需要监督审批情况</span>

<span style="color: rgba(0, 0, 0, 0.9);">配置步骤（首次）：</span>
1. <span style="color: rgba(0, 0, 0, 0.9);">系统管理员进入【系统设置-委托授权】</span>
2. <span style="color: rgba(0, 0, 0, 0.9);">点击【新增委托】，选择委托授权人（财务主管）、委托被授权人（会计）并保存</span>
3. <span style="color: rgba(0, 0, 0, 0.9);">在【全局设置-单据设置-委托授权】区域勾选"授权审批可用人员"，配置可审批人员范围</span>
4. <span style="color: rgba(0, 0, 0, 0.9);">确认"已委托单据可见性"开关为开启状态</span>
5. <span style="color: rgba(0, 0, 0, 0.9);">保存委托配置</span>

<span style="color: rgba(0, 0, 0, 0.9);">使用步骤（日常）：</span>
1. <span style="color: rgba(0, 0, 0, 0.9);">会计登录系统，在【待办】中查看并审批报销单</span>
2. <span style="color: rgba(0, 0, 0, 0.9);">会计审批完成后，单据状态变更</span>
3. <span style="color: rgba(0, 0, 0, 0.9);">财务主管登录系统，进入【待办-审批-已委托】</span>
4. <span style="color: rgba(0, 0, 0, 0.9);">在已委托列表中可查看会计代为审批的单据</span>

<span style="color: rgba(0, 0, 0, 0.9);">结果：财务主管可在自己的已委托列表中监督会计的审批情况，满足内控审计要求</span>
