### <span style="color: rgba(0, 0, 0, 0.9);">**1\. 需求背景**</span>

#### <span style="color: rgba(0, 0, 0, 0.9);">**1.1 需求描述**</span>

<span style="color: rgba(0, 0, 0, 0.9);">自定义审批节点的超时设置，支持选择"天"或"小时"作为时间单位，包括超时触发时间和提醒频率，满足客户对短时效审批的精细化控制需求。</span>

#### <span style="color: rgba(0, 0, 0, 0.9);">**1.2 需求背景**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**需求方：**</span> <span style="color: rgba(0, 0, 0, 0.9);">系统管理员/审批流配置人员</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**现状问题：**</span>
    - <span style="color: rgba(0, 0, 0, 0.9);">超时设置仅支持"天"为单位（如超过3天提醒、每1天提醒一次）</span>
    - <span style="color: rgba(0, 0, 0, 0.9);">客户有短时效审批场景（如2小时内必须审批），天级粒度太粗无法满足</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**解决思路：**</span> <span style="color: rgba(0, 0, 0, 0.9);">超时设置增加"小时"单位选项，超时触发时间和提醒频率均支持天/小时二选一</span>

<span style="color: rgba(0, 0, 0, 0.9);">**业务流程：**</span>
```plain
进入审批流配置 → 选择审批节点 → 超时设置
  ↓
选择时间单位：[天] / [小时]
  ↓
设置超时时间：超过 [X] 天/小时 时处理
  ↓
选择处理动作：提醒/自动通过
  ↓
如选提醒：设置提醒频率 每 [Y] 天/小时 提醒一次
```

### <span style="color: rgba(0, 0, 0, 0.9);">**2\. 功能现状**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**功能所在页面路径：**</span> <span style="color: rgba(0, 0, 0, 0.9);">单据模板-审批流设置-审批节点-超时设置</span>

<span style="color: rgba(0, 0, 0, 0.9);">**当前能力：**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">超时设置开关：开启/关闭</span>
- <span style="color: rgba(0, 0, 0, 0.9);">时间单位：天（唯一选项）</span>
- <span style="color: rgba(0, 0, 0, 0.9);">超时时间：输入天数（整数）</span>
- <span style="color: rgba(0, 0, 0, 0.9);">处理动作：</span>
    - <span style="color: rgba(0, 0, 0, 0.9);">发送消息提醒，每 \[X\] 天提醒一次，直到审批结束</span>
    - <span style="color: rgba(0, 0, 0, 0.9);">自动审批通过</span>

<span style="color: rgba(0, 0, 0, 0.9);">**当前缺失的能力：**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">不支持小时级超时设置</span>
- <span style="color: rgba(0, 0, 0, 0.9);">提醒频率不支持小时级</span>

#### <span style="color: rgba(0, 0, 0, 0.9);">**3.2 详细功能描述**</span>

##### <span style="color: rgba(0, 0, 0, 0.9);">**3.2.1 超时设置界面**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**位置：**</span> <span style="color: rgba(0, 0, 0, 0.9);">节点设置-审批超时设置</span>

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/8K4nyeZ6r24VPnLb/img/9e59755c-9077-4291-b3da-b213088f34ae.png?Expires=1775709698&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=pqwzanELsouDRhbMj81G%2BFYWeAY%3D "")

<span style="color: rgba(0, 0, 0, 0.9);">**界面变更：**</span>
```plain
【变更前】
☑ 当前节点停留超过 [ 3 ] 天时，处理如下：
   ○ 发送消息提醒，每 [ 1 ] 天提醒一次，直到当前节点审批结束
   ○ 自动审批通过

【变更后】
☑ 当前节点停留超过 [ 3 ] (○ 天  ● 小时) 时，处理如下：
   ○ 发送消息提醒，每 [ 1 ] (○ 天  ● 小时) 提醒一次，直到当前节点审批结束
   ○ 自动审批通过
```



<span style="color: rgba(0, 0, 0, 0.9);">**单位选择规则：**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">两个单位选择器独立，互不影响</span>
- <span style="color: rgba(0, 0, 0, 0.9);">超时时间和提醒频率单位二选一（不能同时选天和小时）</span>
- <span style="color: rgba(0, 0, 0, 0.9);">选择"小时"后，输入框清空，需手动输入小时值</span>

<span style="color: rgba(0, 0, 0, 0.9);">**输入范围：**</span>

| <span style="color: rgba(0, 0, 0, 0.9);">**单位**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**最小值**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**最大值**</span> |
|----------------------------------------------------------|-------------------------------------------------------------|-------------------------------------------------------------|
| <span style="color: rgba(0, 0, 0, 0.9);">天</span> | <span style="color: rgba(0, 0, 0, 0.9);">1</span> | <span style="color: rgba(0, 0, 0, 0.9);">按原逻辑不变</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">小时</span> | <span style="color: rgba(0, 0, 0, 0.9);">1</span> | <span style="color: rgba(0, 0, 0, 0.9);">23</span> |

##### <span style="color: rgba(0, 0, 0, 0.9);">**3.2.2 存量配置处理**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**保留策略：**</span> <span style="color: rgba(0, 0, 0, 0.9);">存量配置按原"天"单位保留，显示为：</span>
```plain
☑ 当前节点停留超过 [ 3 ] (● 天  ○ 小时) 时，处理如下：
   ○ 发送消息提醒，每 [ 1 ] (● 天  ○ 小时) 提醒一次...
```



<span style="color: rgba(0, 0, 0, 0.9);">**修改规则：**</span> <span style="color: rgba(0, 0, 0, 0.9);">用户可手动切换单位为"小时"，切换后输入框清空，需重新输入。</span>

### <span style="color: rgba(0, 0, 0, 0.9);">**3.3 逻辑互斥与边界处理**</span>

| <span style="color: rgba(0, 0, 0, 0.9);">**边界场景**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**触发条件**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**处理方案**</span> |
|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| <span style="color: rgba(0, 0, 0, 0.9);">存量数据</span> | <span style="color: rgba(0, 0, 0, 0.9);">已配置天单位的审批流</span> | <span style="color: rgba(0, 0, 0, 0.9);">保留原配置，显示"天"单位，可手动改小时</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">单位切换</span> | <span style="color: rgba(0, 0, 0, 0.9);">天→小时或小时→天</span> | <span style="color: rgba(0, 0, 0, 0.9);">清空对应输入框，需重新输入，不自动换算</span> |
