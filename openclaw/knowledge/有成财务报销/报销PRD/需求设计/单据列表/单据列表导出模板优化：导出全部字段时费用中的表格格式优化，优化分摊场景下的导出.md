# **1\. 需求背景**

在费用上维护了表格，目前导出单据时展示内容可读性不好，需要优化。

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/5VLqXLbWRZPQwqX1/img/20c86cea-41a2-4f6f-9416-2cba5823afc7.png?Expires=1774157421&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=wduP03M4XWN4RrNsKBf09qduxv0%3D "")

当单据存在分摊情况需要导出时，若主表辅助核算字段为空则会导出分摊辅助核算，但当主表辅助核算有值时，只会导出主表的辅助核算，看不到分摊的辅助核算信息，需要优化。

# **4\. 需求详情**

1、报销管理、借款管理、申请管理、付款管理、收款管理，选择“导出全部字段，包含费用明细（含分摊），若单据中无明细模块，则导出文件中不包含明细数据”、“<span style="color: rgba(0, 0, 0, 0.85);">导出全部字段，包含费用明细，若单据中无明细模块，则导出文件中不包含明细数据</span>”导出时，费用中表格字段格式按照以下规则优化：
- 费用表格中每行数据格式：行【行号】：【表头名称1】：【具体值】，【表头名称2】：【具体值】，.......；
- 各行数据需要做单元格内换行。

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/5VLqXLbWRZPQwqX1/img/9468618f-a0b0-4315-97c5-a4eb57f4066c.png?Expires=1774157421&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=E3ytngrrpgFemgAJTyy6xYFbg4Q%3D "")

2、报销管理、借款管理、申请管理、付款管理、收款管理，选择“导出全部字段，包含费用明细（含分摊），若单据中无明细模块，则导出文件中不包含明细数据”导出时：

1）“承担人”列挪到“费用承担部门”列前面

2）根据所选单据费用分摊维度，动态生成费用分摊列，若涉及辅助核算则列名为“费用承担【辅助核算类别名称】”

3）新增“费用承担备注”列，导出费用分摊中的备注

4）单据主表辅助核算列去掉费用分摊时的特殊逻辑，直接根据主表辅助核算实际值导出。
