## 1、需求背景

目前常用字段一旦点击保存之后，修改对应的字段配置都仅在当前的单子内生效，后续每次增加字段都要重新修改配置，操作不便捷；

## 2、业务流程/架构

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 自定义单据 | 自定义保存为常用字段配置支持更新 |  |
| 自定义费用 | 自定义保存为常用字段配置支持更新 |  |
|  |  |  |

## 4、需求详情
1. 自定义保存为常用的字段，保存后增加“更新常用字段配置”功能，点击更新后，则将常用字段的配置更新为最新的配置；
2. 历史已经使用过的单据中的配置还使用旧的配置；
3. 单据和费用的编辑表单均需要；
