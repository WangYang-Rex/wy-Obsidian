## 1、需求背景

目前预算预警所有维度仅支持同一个配置，不能区分周期和年度，用户原始需求月度提示警告，年度禁止提交；

## 2、业务流程/架构

## 3、功能清单

| **模块** | **功能点** | **备注** |
|----------|-------------|----------|
| 预算管理 | 预算超额限制将周期和年度分开设置 |  |
|  |  |  |
|  |  |  |

## 4、需求详情

### 4.1、需求说明1
1. 预算管理-新建/编辑预算-基础设置中的预算超额限制修改为两个维度的限制；
2. 年度和周期，其中周期要根据周期选择的是月度或者季度或者半年度展示；
3. 如果为自定义周期则年度变更为总额；
4. 设置之后预算的超标提醒需要根据设置的内容进行提醒是否允许提交；
