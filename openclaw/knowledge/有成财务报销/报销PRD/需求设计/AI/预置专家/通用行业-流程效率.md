------员工审批效率统计底表corp\_user\_approval\_efficiency\_analysis\----

按照审批人(approval\_user\_id)维度统计每个月的审批数据，以及审批效率分析

id  int8   not null , 主键id 

corp\_id   varchar(100)  not null , 企业id 

report\_id  int8   not null, 报告id

approval\_user\_id  varchar(100)  not null  ,//审批人id

approval\_user\_name  varchar(100)  not null  ,//审批人名称

approval\_user\_role\_names  varchar(500) ,//审批人所属角色名称，多个英文逗号分隔

approval\_user\_department\_id int8  not null   ,//审批人所属部门id

approval\_user\_department\_name varchar(100)   not null ,//审批人所属部门名称

approved\_bill\_count  int8  not null  ,//审批单据量

pending\_approval\_bill\_count int8   not null ,//待审单据量

avg\_day\_approval\_bill\_count   int8   not null ,//日均审单量

avg\_approval\_duration int8   not null ，//平均审批用时，单位分钟

longest\_approval\_duration  int8   not null ，//最长审批用时，单位分钟

shortest\_approval\_duration int8   not null ，//最短审批用时，单位分钟

avg\_day\_approval\_rate numeric(18,2)  not null ，//日均完成率 百分比数值，例如83.25

statistic\_period  varchar(7)   NOT NULL,//统计月份期间，格式yyyy-MM
