## <span style="color: rgba(0, 0, 0, 0.9);">**1\. 需求背景**</span>

### <span style="color: rgba(0, 0, 0, 0.9);">**1.1 需求描述**</span>

<span style="color: rgba(0, 0, 0, 0.9);">费用标准设置新增【规则复制】功能，允许用户对已存在的费用规则进行复制，快速生成一条字段值相同的新规则，仅需修改差异化字段即可完成配置。</span>

### <span style="color: rgba(0, 0, 0, 0.9);">**1.2 需求背景**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**需求方**</span><span style="color: rgba(0, 0, 0, 0.9);">：有成报销企业客户（财务管理员）</span>

<span style="color: rgba(0, 0, 0, 0.9);">**现状问题**</span><span style="color: rgba(0, 0, 0, 0.9);">：</span>
- <span style="color: rgba(0, 0, 0, 0.9);">同一人群在多个城市的标准，仅城市级别不同，其他字段（适用范围、往返日、日期类型、金额）完全相同</span>
- <span style="color: rgba(0, 0, 0, 0.9);">每条规则需重复填写相同字段，配置效率低</span>
- <span style="color: rgba(0, 0, 0, 0.9);">~~**原方案（继承字段）痛点**~~</span><span style="color: rgba(0, 0, 0, 0.9);">~~：需要前置配置继承字段，且只能继承最后一行，不够灵活~~</span>

<span style="color: rgba(0, 0, 0, 0.9);">**解决思路变更**</span><span style="color: rgba(0, 0, 0, 0.9);">：</span>
- <span style="color: rgba(0, 0, 0, 0.9);">~~~~原方案：全局配置继承字段，新增时自动继承上一行~~~~</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**新方案**</span><span style="color: rgba(0, 0, 0, 0.9);">：提供【复制】操作，用户可主动选择复制任意已有规则，复制后插入到指定位置，所有字段完全复制，用户自行修改差异化字段</span>

<span style="color: rgba(0, 0, 0, 0.9);">**业务流程**</span>
```plain
创建规则1：VP及以上 + 一线城市 + 1000元
↓
点击规则1的【复制】按钮
↓
在规则1下方插入新规则，字段值完全相同
↓
修改城市级别为二线城市
↓
保存完成
```

## <span style="color: rgba(0, 0, 0, 0.9);">**2\. 功能现状**</span>

<span style="color: rgba(0, 0, 0, 0.9);">费用标准管理位于【预算费控-费用标准-编辑标准-标准设置】。</span>

<span style="color: rgba(0, 0, 0, 0.9);">**规则表格字段**</span><span style="color: rgba(0, 0, 0, 0.9);">：适用范围（多选）、往返日（多选）、日期类型（单选）、城市级别（单选）、坐席（单选，交通类）、金额。</span>

<span style="color: rgba(0, 0, 0, 0.9);">**规则操作**</span><span style="color: rgba(0, 0, 0, 0.9);">：新增规则（空行）、删除、上移、下移。</span><span style="color: rgba(0, 0, 0, 0.9);">**无复制功能**</span><span style="color: rgba(0, 0, 0, 0.9);">。</span>

### <span style="color: rgba(0, 0, 0, 0.9);">**3.2 详细功能描述**</span>

#### <span style="color: rgba(0, 0, 0, 0.9);">**3.2.1 规则复制功能**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**功能概述**</span> <span style="color: rgba(0, 0, 0, 0.9);">用户可对任意已存在的费用规则进行复制，系统生成一条字段值完全相同的新规则，插入到被复制规则的下一行，用户仅需修改差异化字段。</span>

<span style="color: rgba(0, 0, 0, 0.9);">**复制入口**</span> <span style="color: rgba(0, 0, 0, 0.9);">规则列表每行新增【复制】按钮，位于【操作】顺序最后。</span>
```plain
规则列表操作栏：
┌────┬─────────────┬───────────┬──────────┬──────────┬────────┬─────────────────┐
│ #  │ 适用范围    │ 往返日    │日期类型  │ 城市级别 │ 金额   │ 操作            │
├────┼─────────────┼───────────┼──────────┼──────────┼────────┼─────────────────┤
│ 1  │ VP及以上    │出发/返回  │ 节假日   │ 一线城市 │ 800   │ [删除][上移][下移][复制]│
├────┼─────────────┼───────────┼──────────┼──────────┼────────┼─────────────────┤
│ 2  │ VP及以上    │出发/返回  │ 节假日   │ 二线城市 │ 800   │ [删除][上移][下移][复制]│ ← 复制生成
└────┴─────────────┴───────────┴──────────┴──────────┴────────┴─────────────────┘
```

<span style="color: rgba(0, 0, 0, 0.9);">**复制逻辑**</span>
```plain
用户点击某行的【复制】按钮
↓
系统读取被复制行的所有字段值
↓
在被复制行下方插入新规则行
↓
新规则所有字段值 = 被复制行对应字段值
↓
光标定位到新规则的第一个可编辑字段（适用范围）
↓
用户修改差异化字段后保存
```

<span style="color: rgba(0, 0, 0, 0.9);">**复制效果示例**</span>

<span style="color: rgba(0, 0, 0, 0.9);">复制前：</span>
```plain
┌────┬─────────────┬───────────┬──────────┬──────────┬────────┐
│ #  │ 适用范围    │ 往返日    │日期类型  │ 城市级别 │ 金额   │
├────┼─────────────┼───────────┼──────────┼──────────┼────────┤
│ 1  │ VP及以上    │出发/返回  │ 节假日   │ 一线城市 │ 1000  │
└────┴─────────────┴───────────┴──────────┴──────────┴────────┘
```

<span style="color: rgba(0, 0, 0, 0.9);">用户点击第1行【复制】后：</span>
```plain
┌────┬─────────────┬───────────┬──────────┬──────────┬────────┐
│ #  │ 适用范围    │ 往返日    │日期类型  │ 城市级别 │ 金额   │
├────┼─────────────┼───────────┼──────────┼──────────┼────────┤
│ 1  │ VP及以上    │出发/返回  │ 节假日   │ 一线城市 │ 1000  │
├────┼─────────────┼───────────┼──────────┼──────────┼────────┤
│ 2  │ VP及以上    │出发/返回  │ 节假日   │ 一线城市 │ 1000  │ ← 复制生成（字段完全相同）
│    │ （可复制）  │ （可复制）│（可复制）│ 【修改为│（可复制）│
│    │             │           │          │ 二线城市】│        │
└────┴─────────────┴───────────┴──────────┴──────────┴────────┘
```

<span style="color: rgba(0, 0, 0, 0.9);">用户仅需修改城市级别为"二线城市"，保存即可。</span>

#### <span style="color: rgba(0, 0, 0, 0.9);">**3.2.2 复制后编辑**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**光标定位**</span> <span style="color: rgba(0, 0, 0, 0.9);">复制生成新规则后，光标自动定位到新规则的【适用范围】字段，用户可按Tab键或点击切换字段进行编辑。</span>

<span style="color: rgba(0, 0, 0, 0.9);">**字段编辑**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">所有字段均可编辑，与正常新增规则一致</span>
- <span style="color: rgba(0, 0, 0, 0.9);">用户可保留部分字段不变，修改差异化字段</span>
- <span style="color: rgba(0, 0, 0, 0.9);">支持一次复制后不修改直接保存（完全复制）</span>

<span style="color: rgba(0, 0, 0, 0.9);">**连续复制**</span> <span style="color: rgba(0, 0, 0, 0.9);">用户可复制刚生成的新规则，实现连续快速复制：</span>
```plain
规则1：一线城市标准
↓ 复制
规则2：二线城市标准（修改城市级别）
↓ 复制规则2
规则3：三四线城市标准（修改城市级别）
```



### <span style="color: rgba(0, 0, 0, 0.9);">**3.3 逻辑互斥与边界处理**</span>
