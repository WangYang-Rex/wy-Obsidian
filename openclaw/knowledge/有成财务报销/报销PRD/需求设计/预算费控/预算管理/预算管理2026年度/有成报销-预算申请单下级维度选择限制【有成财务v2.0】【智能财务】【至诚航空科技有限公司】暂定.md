## <span style="color: rgba(0, 0, 0, 0.9);">**1\. 需求背景**</span>

### <span style="color: rgba(0, 0, 0, 0.9);">**1.1 需求描述**</span>

<span style="color: rgba(0, 0, 0, 0.9);">在预算申请单选择预算时，添加下级维度会展示所有可选维度（按提交人、按部门、按辅助核算、按项目、按成本中心）。需要支持在预算管理的基础设置中配置可选择的维度范围，实现只展示指定维度的效果。</span>

### <span style="color: rgba(0, 0, 0, 0.9);">**1.2 需求背景**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**需求方**</span><span style="color: rgba(0, 0, 0, 0.9);">：财务管理人员</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**现状问题**</span><span style="color: rgba(0, 0, 0, 0.9);">：
• 预算申请单添加下级维度时默认展示全部维度选项
• 部分企业只需使用特定维度（如仅使用"辅助核算-项目"）
• 展示多余维度会增加用户选择困扰，降低操作效率</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**解决思路**</span><span style="color: rgba(0, 0, 0, 0.9);">：在预算管理基础设置中增加"预算申请单下级维度配置"，支持勾选允许选择的维度类型</span>

<span style="color: rgba(0, 0, 0, 0.9);">**业务流程**</span>
```plain
财务管理员进入预算管理基础设置
↓
配置预算申请单可选的下级维度（如仅勾选"辅助核算"和"项目"）
↓
员工提交预算申请单选择预算时
↓
点击"+下级维度"仅展示配置的维度选项
```

### <span style="color: rgba(0, 0, 0, 0.9);">**3.2 详细功能描述**</span>

#### <span style="color: rgba(0, 0, 0, 0.9);">**3.2.1 预算申请单下级维度配置**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**位置**</span><span style="color: rgba(0, 0, 0, 0.9);">：预算费控 → 预算管理 → 基础设置</span>

<span style="color: rgba(0, 0, 0, 0.9);">**界面原型**</span>
```plain
预算申请单维度配置
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
□ 启用维度限制
    
    启用后，预算申请单添加下级维度时仅展示勾选的维度选项
    
    ☑ 按提交人
    ☑ 按部门
    ☐ 按辅助核算
    ☑ 按项目
    ☐ 按成本中心
    
    [保存配置]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```



<span style="color: rgba(0, 0, 0, 0.9);">**配置项说明**</span>

| <span style="color: rgba(0, 0, 0, 0.9);">**配置项**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**类型**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**默认值**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**说明**</span> |
|-------------------------------------------------------------|----------------------------------------------------------|-------------------------------------------------------------|----------------------------------------------------------|
| <span style="color: rgba(0, 0, 0, 0.9);">启用维度限制</span> | <span style="color: rgba(0, 0, 0, 0.9);">开关</span> | <span style="color: rgba(0, 0, 0, 0.9);">关闭</span> | <span style="color: rgba(0, 0, 0, 0.9);">控制是否启用维度限制功能</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">按提交人</span> | <span style="color: rgba(0, 0, 0, 0.9);">复选框</span> | <span style="color: rgba(0, 0, 0, 0.9);">勾选</span> | <span style="color: rgba(0, 0, 0, 0.9);">启用限制时可选，控制是否展示该维度</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">按部门</span> | <span style="color: rgba(0, 0, 0, 0.9);">复选框</span> | <span style="color: rgba(0, 0, 0, 0.9);">勾选</span> | <span style="color: rgba(0, 0, 0, 0.9);">启用限制时可选，控制是否展示该维度</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">按辅助核算</span> | <span style="color: rgba(0, 0, 0, 0.9);">复选框</span> | <span style="color: rgba(0, 0, 0, 0.9);">勾选</span> | <span style="color: rgba(0, 0, 0, 0.9);">启用限制时可选，控制是否展示该维度</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">按项目</span> | <span style="color: rgba(0, 0, 0, 0.9);">复选框</span> | <span style="color: rgba(0, 0, 0, 0.9);">勾选</span> | <span style="color: rgba(0, 0, 0, 0.9);">启用限制时可选，控制是否展示该维度</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">按成本中心</span> | <span style="color: rgba(0, 0, 0, 0.9);">复选框</span> | <span style="color: rgba(0, 0, 0, 0.9);">勾选</span> | <span style="color: rgba(0, 0, 0, 0.9);">启用限制时可选，控制是否展示该维度</span> |

<span style="color: rgba(0, 0, 0, 0.9);">**交互规则**</span> <span style="color: rgba(0, 0, 0, 0.9);">• 开关关闭时：下方维度复选框禁用，预算申请单展示全部维度选项（保持现有逻辑）
• 开关开启时：下方维度复选框启用，仅勾选至少一个维度才能保存
• 首次开启时：默认全部勾选，用户按需取消
• 至少保留一个维度被勾选，全部取消时给出提示："请至少选择一个维度"</span>

<span style="color: rgba(0, 0, 0, 0.9);">**配置存储**</span> <span style="color: rgba(0, 0, 0, 0.9);">• 配置按企业维度存储（全局配置）
• 配置保存后即时生效
• 配置权限：财务管理员可操作</span>

#### <span style="color: rgba(0, 0, 0, 0.9);">**3.2.2 预算申请单维度选择弹窗**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**位置**</span><span style="color: rgba(0, 0, 0, 0.9);">：预算申请单 → 预算申请组件 → 选择预算 → \+下级维度</span>

<span style="color: rgba(0, 0, 0, 0.9);">**功能说明**</span>

<span style="color: rgba(0, 0, 0, 0.9);">根据预算管理基础设置中的配置，过滤"\+下级维度"下拉菜单中展示的选项。</span>

<span style="color: rgba(0, 0, 0, 0.9);">**展示规则**</span>

| <span style="color: rgba(0, 0, 0, 0.9);">**配置状态**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**维度选项展示**</span> |
|----------------------------------------------------------------|----------------------------------------------------------------------|
| <span style="color: rgba(0, 0, 0, 0.9);">维度限制开关关闭</span> | <span style="color: rgba(0, 0, 0, 0.9);">展示全部5个选项：按提交人、按部门、按辅助核算、按项目、按成本中心</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">维度限制开关开启</span> | <span style="color: rgba(0, 0, 0, 0.9);">仅展示配置中勾选的维度选项</span> |

<span style="color: rgba(0, 0, 0, 0.9);">**示例场景**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**场景A**</span><span style="color: rgba(0, 0, 0, 0.9);">：配置仅启用"按项目"</span>
```plain
+下级维度
    └─ 按项目
```



<span style="color: rgba(0, 0, 0, 0.9);">**场景B**</span><span style="color: rgba(0, 0, 0, 0.9);">：配置启用"按辅助核算"和"按项目"</span>
```plain
+下级维度
    ├─ 按辅助核算
    │       └─ 弹出辅助核算选择框
    └─ 按项目
            └─ 弹出项目选择框
```



<span style="color: rgba(0, 0, 0, 0.9);">**场景C**</span><span style="color: rgba(0, 0, 0, 0.9);">：配置启用全部维度（与现状一致）</span>
```plain
+下级维度
    ├─ 按提交人
    ├─ 按部门
    ├─ 按辅助核算
    ├─ 按项目
    └─ 按成本中心
```



<span style="color: rgba(0, 0, 0, 0.9);">**二级维度选择**</span>

<span style="color: rgba(0, 0, 0, 0.9);">当选择"按辅助核算"时，需进一步选择具体的辅助核算类别（如"项目"、"客户"等）：
• 弹出辅助核算类别选择框
• 列出企业启用的所有辅助核算类别
• 选择后生成"辅助核算-\{类别名称\}"的维度条件</span>

### <span style="color: rgba(0, 0, 0, 0.9);">**3.3 逻辑互斥与边界处理**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**边界情况处理**</span>

| <span style="color: rgba(0, 0, 0, 0.9);">**边界场景**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**触发条件**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**处理方案**</span> |
|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| <span style="color: rgba(0, 0, 0, 0.9);">配置全部取消</span> | <span style="color: rgba(0, 0, 0, 0.9);">用户取消所有维度勾选后尝试保存</span> | <span style="color: rgba(0, 0, 0, 0.9);">保存时校验，提示"请至少选择一个维度"，阻止保存</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">配置开关关闭</span> | <span style="color: rgba(0, 0, 0, 0.9);">维度限制开关处于关闭状态</span> | <span style="color: rgba(0, 0, 0, 0.9);">预算申请单展示全部维度，忽略下方复选框状态</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">存量申请单</span> | <span style="color: rgba(0, 0, 0, 0.9);">配置修改时已存在的预算申请单</span> | <span style="color: rgba(0, 0, 0, 0.9);">不影响存量单据，仅影响新创建/编辑的申请单</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">无可用维度</span> | <span style="color: rgba(0, 0, 0, 0.9);">配置开启但企业未启用任何辅助核算</span> | <span style="color: rgba(0, 0, 0, 0.9);">按辅助核算选项隐藏或置灰，提示"未启用辅助核算"</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">辅助核算类别为空</span> | <span style="color: rgba(0, 0, 0, 0.9);">选择按辅助核算但企业无辅助核算类别</span> | <span style="color: rgba(0, 0, 0, 0.9);">弹出提示"未配置辅助核算类别"，无法选择该维度</span> |

<span style="color: rgba(0, 0, 0, 0.9);">**维度层级互斥规则（现有业务规则）**</span>

<span style="color: rgba(0, 0, 0, 0.9);">预算申请单新增下级维度时，存在以下层级互斥限制（需保持兼容）：</span>

| <span style="color: rgba(0, 0, 0, 0.9);">**维度类型**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**层级添加限制**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**说明**</span> |
|----------------------------------------------------------------|----------------------------------------------------------------------|----------------------------------------------------------|
| <span style="color: rgba(0, 0, 0, 0.9);">辅助核算</span> | <span style="color: rgba(0, 0, 0, 0.9);">不限制</span> | <span style="color: rgba(0, 0, 0, 0.9);">可在同一链路的多个层级添加不同辅助核算类别</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">按提交人</span> | <span style="color: rgba(0, 0, 0, 0.9);">仅一次</span> | <span style="color: rgba(0, 0, 0, 0.9);">在某一链路上仅能在某个层级添加一次，其他层级不允许添加</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">按部门</span> | <span style="color: rgba(0, 0, 0, 0.9);">仅一次</span> | <span style="color: rgba(0, 0, 0, 0.9);">在某一链路上仅能在某个层级添加一次，其他层级不允许添加</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">按项目</span> | <span style="color: rgba(0, 0, 0, 0.9);">仅一次</span> | <span style="color: rgba(0, 0, 0, 0.9);">在某一链路上仅能在某个层级添加一次，其他层级不允许添加</span> |
| <span style="color: rgba(0, 0, 0, 0.9);">按成本中心</span> | <span style="color: rgba(0, 0, 0, 0.9);">仅一次</span> | <span style="color: rgba(0, 0, 0, 0.9);">在某一链路上仅能在某个层级添加一次，其他层级不允许添加</span> |

<span style="color: rgba(0, 0, 0, 0.9);">**示例说明**</span><span style="color: rgba(0, 0, 0, 0.9);">：
• 若预算链路已包含"按部门"维度，则后续层级不能再添加"按部门"（包括不同部门值）
• 若预算链路已包含"辅助核算-项目"，仍可继续添加"辅助核算-客户"或其他维度</span>

<span style="color: rgba(0, 0, 0, 0.9);">**与其他功能的兼容性**</span>

| <span style="color: rgba(0, 0, 0, 0.9);">**相关功能**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**交互关系**</span> | <span style="color: rgba(0, 0, 0, 0.9);">**处理方案**</span> |
|----------------------------------------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| <span style="color: rgba(0, 0, 0, 0.9);">预算报表维度筛选</span> | <span style="color: rgba(0, 0, 0, 0.9);">独立功能</span> | <span style="color: rgba(0, 0, 0, 0.9);">预算报表的维度筛选不受此配置影响，保持原有逻辑</span> |

<span style="color: rgba(0, 0, 0, 0.9);">**A. 典型使用场景**</span>

<span style="color: rgba(0, 0, 0, 0.9);">场景：某企业仅使用"项目"维度管理预算</span>

<span style="color: rgba(0, 0, 0, 0.9);">配置步骤：</span>
1. <span style="color: rgba(0, 0, 0, 0.9);">财务管理员进入预算费控 → 预算管理 → 基础设置</span>
2. <span style="color: rgba(0, 0, 0, 0.9);">找到"预算申请单维度配置"模块</span>
3. <span style="color: rgba(0, 0, 0, 0.9);">开启"启用维度限制"开关</span>
4. <span style="color: rgba(0, 0, 0, 0.9);">取消勾选"按提交人"、"按部门"、"按辅助核算"、"按成本中心"</span>
5. <span style="color: rgba(0, 0, 0, 0.9);">仅保留勾选"按项目"</span>
6. <span style="color: rgba(0, 0, 0, 0.9);">点击保存</span>

<span style="color: rgba(0, 0, 0, 0.9);">使用步骤：</span>
1. <span style="color: rgba(0, 0, 0, 0.9);">员工进入申请单 → 新建预算申请单</span>
2. <span style="color: rgba(0, 0, 0, 0.9);">填写基本信息后点击"选择预算"</span>
3. <span style="color: rgba(0, 0, 0, 0.9);">选中预算后点击"\+下级维度"</span>
4. <span style="color: rgba(0, 0, 0, 0.9);">下拉菜单仅显示"按项目"一个选项</span>
5. <span style="color: rgba(0, 0, 0, 0.9);">选择后弹出项目选择框，选择具体项目</span>

<span style="color: rgba(0, 0, 0, 0.9);">结果：员工只能按项目维度申请预算，避免了其他维度的干扰</span>
