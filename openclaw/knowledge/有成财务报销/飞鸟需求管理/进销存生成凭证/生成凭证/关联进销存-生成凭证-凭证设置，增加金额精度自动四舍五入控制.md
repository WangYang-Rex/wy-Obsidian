## 1\. 需求背景

线上用户crm中金额精度超过2位小数的，目前无法生成凭证，会报错。

# **2\. 业务流程/架构**

# **3\. 功能清单**

| 模块 | 功能点 | 备注 |
|------|---------|------|
| 关联进销存 | 生成凭证-凭证设置 |  |
|  |  |  |
|  |  |  |

# **4\. 需求详情**

关联进销存-生成凭证-凭证设置，不论生成的是凭证还是凭证\+日记账，均增加金额精度自动四舍五入选项：

1、勾选中后，金额自动按照两位小数四舍五入后生成凭证/日记账；

2、未勾选时，若精度金额大于2位小数，则报错。

线上用户默认不勾选。
