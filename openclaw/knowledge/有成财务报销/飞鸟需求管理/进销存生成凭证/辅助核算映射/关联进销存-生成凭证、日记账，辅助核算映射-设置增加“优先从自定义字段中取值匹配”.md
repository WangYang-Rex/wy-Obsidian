# **1\. 需求背景**

用户在crm表单上有两个客户字段，一个系统的一个是自定义的，用户希望根据系统字段生成凭证。目前系统会报错。

|  |  |  |

# **4\. 需求详情**

关联进销存-生成凭证、日记账，辅助核算映射-设置增加“优先从自定义字段中取值匹配”选项。

（1）勾选选项时，若在crm业务对象中存在多个同一类辅助核算字段，优先取自定义的字段，取不到再取系统预置的。若自定义字段也有多个，那还是和线上一样报错。

（2）不勾选时，优先取系统的字段，取不到再取自定义的。

历史用户不勾选。
