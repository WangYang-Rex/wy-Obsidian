## <span style="color: rgba(0, 0, 0, 0.9);">**1\. 需求背景**</span>

### <span style="color: rgba(0, 0, 0, 0.9);">**1.1 需求描述**</span>

<span style="color: rgba(0, 0, 0, 0.9);">支持用户按照《民间非营利组织会计制度（2024年版）》（财会〔2024〕25号）新建账套，初始化标准科目体系及三大报表（含公式）。</span>

### <span style="color: rgba(0, 0, 0, 0.9);">**1.2 需求背景**</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**需求方**</span><span style="color: rgba(0, 0, 0, 0.9);">：使用有成财务的民间非营利组织客户（基金会、社会团体、民办非企业单位等）</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**现状问题**</span><span style="color: rgba(0, 0, 0, 0.9);">：
• 2024年12月财政部发布《民间非营利组织会计制度（2024年版）》（财会〔2024〕25号），2026年1月1日起施行，科目体系和报表格式均有调整
• 现有账套仅支持2004年版制度，无法直接新建2024年版账套
• 客户从旧账套迁移到新账套时，需手工配置标准科目和三大报表公式，工作量巨大且易出错
• 现有报表公式编辑需逐行配置，缺乏批量操作能力</span>
- <span style="color: rgba(0, 0, 0, 0.9);">**解决思路**</span><span style="color: rgba(0, 0, 0, 0.9);">：
• 新增2024年版制度账套模板，初始化时自动带入56个标准科目
• 三大报表（资产负债表/业务活动表/现金流量表）预置2024年版标准公式
• 兼容现有科目批量导入功能，支持新旧科目映射</span>

<span style="color: rgba(0, 0, 0, 0.9);">**业务流程**</span>
```plain
客户创建新账套
  ↓
选择会计制度：民间非营利组织会计制度（2024年版）
  ↓
系统自动初始化：
  ├── 56个标准会计科目（含新增科目）
  └── 三大报表模板（含标准公式）
  ↓
客户按需调整：
  ├── 科目设置：导出→修改→导入（已有功能）
  └── 报表公式：导出Excel→批量修改→导入
  ↓
新账套启用，支持旧账套余额导入
```

### <span style="color: rgb(6, 10, 38);">核心变化解读</span>

<span style="color: rgb(6, 10, 38);">根据最新的财会〔2024〕25号文，你需要特别注意以下几个科目的调整：</span>
1. <span style="color: rgb(6, 10, 38);">新增科目：</span>
    - <span style="color: rgb(6, 10, 38);">1403 其他长期投资：用于核算持有时间超过1年的除股权和债权以外的其他长期投资。</span>
    - <span style="color: rgb(6, 10, 38);">3201 以前年度净资产调整：专门用于核算本年度发现的前期差错更正。</span>
    - <span style="color: rgb(6, 10, 38);">5201 税金及附加：核算业务活动发生的各项税金及附加。</span>
    - <span style="color: rgb(6, 10, 38);">5501 资产减值损失：从“管理费用”中拆分出来，单独核算。</span>
2. <span style="color: rgb(6, 10, 38);">名称与核算变更：</span>
    - <span style="color: rgb(6, 10, 38);">文物资源：原“文物文化资产”重新分类调整。</span>
    - <span style="color: rgb(6, 10, 38);">应付职工薪酬：替代原有的“应付工资”，核算内容更全面。</span>
    - <span style="color: rgb(6, 10, 38);">应交税费：替代原有的“应交税金”。</span>
    - <span style="color: rgb(6, 10, 38);">长期股权投资：取消了权益法，统一采用成本法核算，这大大简化了后续计量工作。</span>
3. <span style="color: rgb(6, 10, 38);">净资产处理简化：</span>
    - <span style="color: rgb(6, 10, 38);">取消了限定性净资产在使用时重分类为非限定性净资产的要求，期末直接将限定性收入和费用结转至限定性净资产。</span>

## <span style="color: rgba(0, 0, 0, 0.9);">**附录**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**A. 典型使用场景**</span>

<span style="color: rgba(0, 0, 0, 0.9);">**场景：客户从旧账套迁移到新账套（2024年版制度）**</span>

<span style="color: rgba(0, 0, 0, 0.9);">配置步骤（首次）：</span>
1. <span style="color: rgba(0, 0, 0, 0.9);">进入【系统管理-账套管理-新建账套】</span>
2. <span style="color: rgba(0, 0, 0, 0.9);">填写账套名称，选择会计制度"民间非营利组织会计制度（2024年版）"</span>
3. <span style="color: rgba(0, 0, 0, 0.9);">确认提示信息后，点击确定创建账套</span>
4. <span style="color: rgba(0, 0, 0, 0.9);">系统自动初始化56个标准科目和三大报表标准公式（含详细的取数规则说明）</span>
5. <span style="color: rgba(0, 0, 0, 0.9);">进入【基础设置-科目设置】，导出科目Excel，根据旧账套调整科目（如新增明细科目）</span>
6. <span style="color: rgba(0, 0, 0, 0.9);">导入调整后的科目Excel</span>
7. <span style="color: rgba(0, 0, 0, 0.9);">依次完成业务活动表、现金流量表的公式调整</span>

<span style="color: rgba(0, 0, 0, 0.9);">使用步骤（日常）：</span>
1. <span style="color: rgba(0, 0, 0, 0.9);">月末进入报表管理查看三大报表，系统自动按新公式计算</span>
2. <span style="color: rgba(0, 0, 0, 0.9);">如需调整公式，重复步骤7-9</span>

<span style="color: rgba(0, 0, 0, 0.9);">结果：客户成功创建2024年版制度账套，科目和报表公式按新规配置完成，支持后续记账和报表生成。</span>
