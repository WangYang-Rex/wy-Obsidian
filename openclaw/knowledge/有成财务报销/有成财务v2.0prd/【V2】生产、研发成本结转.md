# **4\. 需求详情**

### **4.1新增结转**

在结账板块右上角新增按钮“新增结转”，按钮常亮

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/1GXn45KoxeoddqDQ/img/636b8628-787a-4535-9b43-0e78c39ef872.png?Expires=1781780484&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=BdhP0fjzM4qkjkoO2FIxd9iLkJI%3D "")

点击后弹窗如下：

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/1GXn45KoxeoddqDQ/img/e233a694-a267-4b34-94fe-4e70d7d00964.png?Expires=1781780484&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=Xb%2BtXgCo24B0XjoGgPplmUgmYjg%3D "")
1. 凭证日期：默认取当前月最后一天。可手动修改（作为生成凭证的日期）
2. 摘要：默认为空，手动维护。（作为生产的结转卡片的标题名称）
3. 结转转出科目：交互逻辑参考结转损益的选项，枚举值就是当前账套维护的科目
4. 结转转入科目：交互逻辑参考结转损益的选项，枚举值就是当前账套维护的科目
5. 结转规则：
    1. 无：默认为无。相当于没有特殊要求
    2. 辅助核算：右侧提示语“根据转出科目的辅助核算值对结转分录进行过滤”选择辅助核算后，交互变更如下

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/1GXn45KoxeoddqDQ/img/64f63483-1662-4ddb-a04b-ae0469233505.png?Expires=1781780484&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=x2utPwguN7iWdp4bgEDfuRWZ%2FF4%3D "")
- 操作：添加/删除【最少保留一行】
- 辅助核算：下拉单选框，枚举值为转出科目维护的辅助核算类型
- 匹配值：文本框。若有多个值，用“；”隔空

注：每行的辅助核算不能重复

规则维护完成后点击确定进行校验，必填项以及选择辅助核算后是否所有值均填写完成，否则提示“必填项未填写”

### **4.2结转逻辑**

确认保存后生成结转卡片

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/1GXn45KoxeoddqDQ/img/57422c3e-ec40-4ea1-9cd1-855d69d73cdc.png?Expires=1781780484&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=1Mobk29oAbCsCYW8awysG0J0qaE%3D "")
1. 卡片标题名取，规则的摘要信息
2. 设置按钮：点击打开刚维护的规则页面，相关信息需要回显示
3. 凭证：未生成凭证时为空
4. 生成凭证：即根据规则生成凭证。
    1. 若结转规则为“无”，则视为将转出科目的所有余额都结转。
- <span style="color: rgb(15, 17, 21);">**转出科目**</span><span style="color: rgb(15, 17, 21);">余额在</span><span style="color: rgb(15, 17, 21);">**借方**</span><span style="color: rgb(15, 17, 21);">时：</span>
- <span style="color: rgb(15, 17, 21);">借：转入科目</span>
- <span style="color: rgb(15, 17, 21);">贷：转出科目</span>
- <span style="color: rgb(15, 17, 21);">**转出科目**</span><span style="color: rgb(15, 17, 21);">余额在</span><span style="color: rgb(15, 17, 21);">**贷方**</span><span style="color: rgb(15, 17, 21);">时：提示“科目余额为负数，请检查异常情况”，阻止凭证生成</span>


    2. 若结转规则为“辅助核算”，则视为将转出科目中所有匹配到辅助核算值的余额进行结转。
- <span style="color: rgb(15, 17, 21);">**转出科目**</span><span style="color: rgb(15, 17, 21);">余额在</span><span style="color: rgb(15, 17, 21);">**借方**</span><span style="color: rgb(15, 17, 21);">时：</span>
- <span style="color: rgb(15, 17, 21);">借：转入科目</span>
- <span style="color: rgb(15, 17, 21);">贷：转出科目</span>
- <span style="color: rgb(15, 17, 21);">**转出科目**</span><span style="color: rgb(15, 17, 21);">余额在</span><span style="color: rgb(15, 17, 21);">**贷方**</span><span style="color: rgb(15, 17, 21);">时：提示“科目余额为负数，请检查异常情况”，阻止凭证生成</span>
5. 生成凭证后，按钮变为重新生成
6. 生成的记账凭证，凭证号规则为：当期最后一张凭证号\+1
