# **4\. 需求详情**

### **4.1记账凭证打印**

凭证--查看凭证--打印功能，新增勾选项“打印附件”

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/Mp7ld7b9y1yEpOBQ/img/f294152d-aec9-4356-a83f-ec45ca04a371.png?Expires=1781014974&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=odH9%2B8zlL%2BRv6%2BvjGO6kxiLCzv8%3D "")

默认未勾选，右侧存在悬浮提示图标“选择打印附件后，会将所选类型附件合并为一个文件，所以无法在线预览并打印，需要前往下载中心下载后打印”

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/Mp7ld7b9y1yEpOBQ/img/4b203809-fbb0-4a09-850a-12731801488f.png?Expires=1781014974&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=e2BMBUi7sDt5M706%2Bg6AulGdj6I%3D "")

用户勾选该选项后，右侧出现9个多选项：
- 全部：默认不选中，选中时会同步勾选下面的8个选项，取消勾选也会取消如下的8个选项。当8个选项中存在任意一个未被选中时，该选项也同样取消选中
- PNG、JPG、JPEG、PDF、DOC、DOCX、XLS、XLSX：默认选中“png、pdf、jgp、jpeg”
- 注：仅会打印对应类型的附件；当一个都没选中时，视为本次不打印挂接的附件

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/Mp7ld7b9y1yEpOBQ/img/11a21cc6-fbdc-4ae1-83c4-66455ce899d8.png?Expires=1781014974&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=raYvwf%2F7cMVB2VbgVN9S4mFL5WQ%3D "")

选中后，点击“预览并打印”按钮逻辑如下：
    1. 会按照上述规则生成打印文件样式
    2. 在原有打印的文件基础上，再将挂接附件也进行合并，最终汇总为一个附件放在下载中心
    3. 异步处理，发起成功后提示“请前往下载中心等待生成打印附件”![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/Mp7ld7b9y1yEpOBQ/img/bf172c8a-f521-4ba5-b55e-c4f118442d61.png?Expires=1781014974&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=c1mAf5LYx2U8Cj9nqF9xNNV%2BXXs%3D "")
    4. 若不存在上传的附件或没有所选类型附件，则打印预览时不做展示；若存在，则每个附件固定以新的一页开始进行展示

选中后，点击“下载”按钮逻辑同原来一致
