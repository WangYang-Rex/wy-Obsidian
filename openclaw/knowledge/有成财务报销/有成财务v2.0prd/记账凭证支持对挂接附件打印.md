# **4\. 需求详情**

### **4.1外币核算**

在基础设置--币别设置， 补充“币别展示规则”

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/4EZlweZJmKVXXqxA/img/928dddbb-301f-4f69-add5-7a70ca570c13.png?Expires=1778766904&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=X6JuryLyHitTq0Gb%2FOX8NlKGnO0%3D "")

存在两个规则：
1. 记账凭证是否展示币别：【功能上线时默认选择“是”】
    1. 选择是：所有的记账凭证都展示币别字段列
    2. 选择否：所有的记账凭证都隐藏币别字段列

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/4EZlweZJmKVXXqxA/img/f07aed2f-a5fa-43ff-9349-559f4d651330.png?Expires=1778766904&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=qf7r7rxdlcS2tr5ejEClteBqTRI%3D "")
2. 记账凭证打印、导出是否携带币别值：
    3. 选择是：打印、导出记账凭证时，币别字段存在数据
    4. 选择否：打印、导出记账凭证时，币别字段不存在数据





注：移除掉原本科目配置中的“外币核算”的开关控制的记账凭证上的币别字段列规则。
