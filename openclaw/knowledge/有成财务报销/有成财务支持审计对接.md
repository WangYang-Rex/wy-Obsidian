# **1\. 需求背景**

使用有成财务的企业需要接受审计，目前需要导出excel表格，让审计公司手工建立审计账套，比较麻烦。

市面上主流的审计软件均可接受国标数据接口的文件导入生成审计账套，以审计大师为例：

http://www.cpa8.com/news.asp?dcat=16&cat=30

[柠檬云导出标准接口操作说明.doc]

虽然国标更新了2024版本（感兴趣的可以看下[https://std.samr.gov.cn/gb/search/gbDetailed?id=2AD0270638D2091BE06397BE0A0A2D62](https://std.samr.gov.cn/gb/search/gbDetailed?id=2AD0270638D2091BE06397BE0A0A2D62)），但因审计工具更新频率极低，故优先比照竞品支持主流版本。

## **界面入口**

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/1GXn45KQDzRdkqDQ/img/78667ed7-149e-463e-9fa3-52baafaf63cc.png?Expires=1775709378&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=I5AaZo8GmU7v8ZWiK9UDu4%2FPfo0%3D "")



限制用户单选，建账期间（含）～开账期间（含）的单一年度。
> 导出压缩包后解压，运行审计软件取数工具-标准数据接口，选择解压后的文件GSSM.TXT导入即可。   
> 注意：   
> 1、暂不支持单科目上绑定多辅助的情况。   
> 2、数据包内不含固定资产数据，如有需要可独导出excel表格。   

点击后提示用户前往下载中心下载压缩包，账套id\_账套名称-导出年度\_时间戳.zip，仅支持下载，不支持预览。

## **文件整体结构**

账套id\_账套名称-导出年度\_时间戳.zip内的结构

| 文件名 | 含义 |
|---------|------|
| BMXX.TXT | 部门信息 |
| CHXX.TXT | 存货信息 |
| DZZB.TXT | 电子账簿 |
| GSSM.TXT | 公司说明 |
| GYSXX.TXT | 供应商信息 |
| JZPZ.TXT | 记账凭证 |
| KHXX.TXT | 客户信息 |
| KJKM.TXT | 会计科目，禁用的科目也要导出 |
| KMYE.TXT | 科目余额 |
| XJLXX.TXT | 现金流信息 |
| XMXX.TXT | 项目信息 |
| ZYXX.TXT | 职员信息 |
| 自定义辅助核算类别拼音缩写XX.TXT | 例如账套内有辅助核算类别为自定义，那么压缩包内就会多一个ZDYXX.TXT的文件，重名了则加个(1)后缀，如ZDYXX(1).TXT，ZDYXX(2).TXT |
