需求现状：
审批流设计器中，同类型节点配置高度相似时，每次添加新节点都需从头勾选10\+个配置项，操作繁琐易遗漏。当前仅支持鼠标悬停显示【删除】按钮，点击【\+】按钮弹出九宫格/网格布局的悬浮面板选择节点类型生成空白节点。

客户预期：
在节点卡片右上角增加【复制】图标按钮，点击后缓存该节点配置进行快速复制；

## 三、需求详情

### 3.1 增加【节点复制按钮】功能

**流程：** 鼠标悬停节点卡片 → 右上角显示【复制】按钮 → 点击复制 → 缓存节点配置 → 显示成功提示

**位置：** 审批流设计器中任意节点卡片的右上角（与【删除】按钮并列）

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/3BMqYyb4rvK3bqwZ/img/ac7fbeda-ae85-4fb7-ac54-6f8b3ee2bf9c.png?Expires=1779975190&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=z6N0I4zb1F8KmgwioUnIlIf1kMc%3D "")

**功能描述：**

• 鼠标悬停节点时，右上角显示【删除】和【复制】两个操作按钮
• 点击【复制】后，系统将该节点全部配置缓存到当前页面会话中
• 弹出"已复制'XX'的配置"轻量提示
• 刷新页面或切换审批流：复制缓存自动清空
• 复制源节点被删除：缓存配置保留，仍可继续粘贴；粘贴后节点独立存在

### 3.2 增加【粘贴节点】功能

**流程：** 点击蓝色圆形【\+】新增节点按钮 → 弹出悬浮面板 → 点击"粘贴节点" → 生成新节点并加载复制配置

**位置：** 审批流设计器 → 点击蓝色圆形【\+】新增节点按钮 → 弹出九宫格/网格布局悬浮面板

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/3BMqYyb4rvK3bqwZ/img/092ad2bb-3aa6-4442-afbc-1ee1c5dbe6f5.png?Expires=1779975190&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=KsJO3JsLuLeWVLPllescZAjke0M%3D "")

**功能描述：**

• 存在复制缓存时，悬浮面板中增加"粘贴节点（复制自：XX）"格子，显示复制源节点名称
• 无复制缓存时，"粘贴节点"格子完全不显示
• 点击"粘贴节点"后，在当前位置插入新节点并加载复制的配置
• 新节点生成与复制源同类型，完整加载复制源的配置（除节点名称外）
• 节点名称自动添加"副本"后缀，同个节点复制多个时按副本1、副本2、副本N递增
• 跨分支粘贴：在条件分支内点击【\+】时，若缓存来自主分支，仍可正常粘贴
• 重名保存：允许保存，不强制校验节点名称唯一性

### 3.3 【审批流设计器全屏】

**流程：** 点击工具栏【全屏】按钮 → 隐藏导航栏/Tab栏/侧边栏 → 画布扩展至全屏 → 点击【退出全屏】恢复

**位置：** 审批流设计器顶部工具栏右侧

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/3BMqYyb4rvK3bqwZ/img/8f03455b-6109-4f4a-8ded-7c8c95a205e8.png?Expires=1779975190&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=TW9SA3qoBbzAAc8mX56OL33MJkI%3D "")

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/3BMqYyb4rvK3bqwZ/img/8ed2fd38-fa95-4d44-b4bb-03cf5e1bc212.png?Expires=1779975190&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=xTPwNR4iV8LQo9kzclNGzqtBoQQ%3D "")

**功能描述：**

• 工具栏右侧增加【全屏】按钮
• 点击后隐藏顶部导航栏、二级Tab栏、左侧窄边栏，审批流画布扩展至占据整个浏览器可视区域
• 全屏状态下按钮变为【退出全屏】，点击后恢复原有布局
• 全屏/退出全屏过程中，节点配置和当前操作状态保持不变
• 刷新页面后恢复默认非全屏状态

### 3.4 【支持复制的节点】

**复制范围：**审批节点，审批节点，出纳节点，发起人节点
