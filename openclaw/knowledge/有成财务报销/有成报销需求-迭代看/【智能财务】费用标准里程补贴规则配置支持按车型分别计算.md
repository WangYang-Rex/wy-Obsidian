# **需求背景**

## **需求现状：**

  智能财务的企业客户在费用标准-标准设置-控制方式配置时无法按车型分别配置补贴规则，企业客户需要按照油车和电车分别配置

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5byVkKMLlxo/img/626b96cf-19f0-4130-8b37-fc1d5e6c7115.png?Expires=1779975183&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=eCHlQ2GafEXMMOL7mH8aV5wFmWc%3D "")

##   **客户预期**

  企业可以通过【按车型分别计算】去配置油车和电车的不同补贴标准



# **功能清单**

| 模块 | 功能点 | 备注 |
|------|---------|------|
| 控制方式-按车型分别计算 | 增加【按车型分别计算】配置功能，满足用户按照车型分别设置补贴标准 |  |
| 里程补贴定位打卡-车型选择 | 增加车型选择：油车/电车 |  |
| 费用类型私车公用扩展字段 | 支持车型选择，开始结束里程填写和日期 |  |

# **需求详情**

## 增加【按车型分别计算】配置功能

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5byVkKMLlxo/img/75b2feb4-d9d7-4b07-b49a-36d39bd2ab9a.png?Expires=1779975183&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=hIshkbTJfhiTpbOZZ2N4QuUARkU%3D "")

**位置**：预算费控 → 费用标准 → 标准设置 → 控制方式

**配置项**

| 档位 | 说明 |
|------|------|
| 按车型分别计算 | 不同车型不同标准：油车/电车（新增） |

**切换规则**：
•  默认「按车型统一计算」，存量保持不变
• 切换为「按车型分别计算」后，显示「车型辅助核算」必填项
• 切换控制方式时，原有配置清空

## 车型-辅助核算选择

**字段**：车型-辅助核算（下拉选择，无选项时视为全选）

**选项来源**：设置-辅助设置-辅助核算字段列表（如车辆类型）

## 费用类型【私车公用】-增加选择车型（油车/电车），开始里程和结束里程

**位置：**智能财务-填写单据-选择费用【私车公用】

![画布 32.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5byVkKMLlxo/img/29fd5965-0ad2-4da6-90cc-151938f09e2e.png?Expires=1779975183&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=%2FuLtgGmlTFIUV1evMy3KsZcGr%2FI%3D "")
