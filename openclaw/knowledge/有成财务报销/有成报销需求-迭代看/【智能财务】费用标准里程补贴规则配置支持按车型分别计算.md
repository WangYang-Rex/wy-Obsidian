# **需求背景**

## **需求现状：**

  智能财务的企业客户在费用标准-标准设置-控制方式配置时无法按车型分别配置补贴规则，企业客户需要按照油车和电车分别配置

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5byVkKMLlxo/img/626b96cf-19f0-4130-8b37-fc1d5e6c7115.png?Expires=1781015316&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=QEypcv1mbiuwwmuzzknoPUOraYw%3D "")

##   **客户预期**

  企业可以通过【按车型分别计算】去配置油车和电车的不同补贴标准



# **功能清单**

| 模块 | 功能点 | 备注 |
|------|---------|------|
| 控制方式-按车型分别计算 | 增加【按车型分别计算】配置功能，满足用户按照车型分别设置补贴标准 |  |

# **需求详情**

## 3.1 费用标准-增加【按车型分别计算】配置功能

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5byVkKMLlxo/img/75b2feb4-d9d7-4b07-b49a-36d39bd2ab9a.png?Expires=1781015316&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=vQdWsrrbKp%2BCeGSDWG4cHN0QHy0%3D "")

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5byVkKMLlxo/img/09bccca1-394d-4261-9a36-864b5fd19e44.png?Expires=1781015316&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=MrdHzgNm5sFGMGcVcoW3gRIT03U%3D "")

**位置**：预算费控 → 费用标准 → 标准设置 → 控制方式

**配置项**

| 档位 | 说明 |
|------|------|
| 按车型分别计算 | 不同车型不同标准：油车/电车（新增） |

**功能说明**：
- 默认「按车型统一计算」，存量保持不变
- 切换为「按车型分别计算」后，显示「车型」配置项，默认为空，为空视为全选
- 点击「车型」弹出“选择辅助核算”弹窗，选择自定义辅助核算里的辅助核算项，支持多选
- 切换控制方式时，原有配置清空
- 车型中选中的辅助核算项名称与“里程补贴”费用中车型名称做匹配，匹配成功则规则生效

## 3.2  支出类别增加固定支出“里程补贴”

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5byVkKMLlxo/img/acadde06-618a-4b5c-9b7e-c8d65af66075.png?Expires=1781015316&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=L%2FwhqgWtWu4FhzVi6FjwUEPTGZU%3D "")

**功能说明**：
- 支出类别增加固定“里程补贴”类别，不允许删除和添加子类别，允许编辑；隐藏删除和添加子类别按钮；创建日期显示“-”，创建人显示“系统”
- 使用随手记-里程补贴打卡定位生成的费用自动使用该费用类型“里程补贴”，并将打卡定位记录带入该费用内
- 该费用内默认固定字段【我的车型，日期，里程数，出发地和目的地】
- 给存量用户和新用户的支出刷新“里程补贴”类别，后续搭配费用标准配置的规则去控制提单时校验计算“私车公用”的里程补贴的报销标准

## 3.3 里程补贴定位打卡-增加选择车型（油车/电车）和里程信息生成的费用中带入车型信息

**位置：**智能财务-随手记-里程补贴-选择车型-打卡定位

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5byVkKMLlxo/img/5e0e4b16-420e-4d47-9cec-95fb52b51d7e.png?Expires=1781015316&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=Qv4IExrL3CyMGHghL%2BICyZfxeDw%3D "")![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5byVkKMLlxo/img/b3a4b641-0f6f-4028-b7c1-0305f2f72bc6.png?Expires=1781015316&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=E5U8MuTabqsZ2LH9gTqE0oBTA24%3D "")![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5byVkKMLlxo/img/6b7abb67-d59d-4c87-ac10-2ff66bfd6227.png?Expires=1781015316&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=ztiMfx03bHR5jmhtP4K57jnfyOk%3D "")

![画布 75 Copy.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5byVkKMLlxo/img/60c641c6-df72-4e59-86dc-1b1ed4a46b81.png?Expires=1781015316&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=ZvFmoUKHQkuJSRU%2FBpYCU1AI%2Foo%3D "")


- 里程补贴打卡定位界面支持选择车型，车型固定可选项“油车和电车”，仅单选
- 选中某车型后，才可开始打卡定位，若未选择车型点击“记录”，提示“选择任一车型后才可开始打卡定位”
- 当前打卡行程未结束时，不允许切换车型；开始新的打卡行程前，可以切换车型
- 打卡行程结束后生成随手记费用，打卡行程信息生成“里程补贴”费用，本次生成的费用金额由配置的费用标准规则自动计算，后续使用该费用去报销
