## 〇、现有逻辑说明（供参考）

### 0.1 附件打印数据源

勾选「附件」打印时，getImageAndAttachmentUrlList() 方法从三个数据源收集附件：

| 序号 | 数据源 | 说明 |
|------|---------|------|
| ① | 单据表单中的附件组件 | type=attachment |
| ② | 费用明细中的附件组件 | — |
| ③ | 审批记录中的附件 | 调用 getAttachmentUrlImgAndPdfByOrderOperation() |

### 0.2 审批记录附件取值逻辑

getAttachmentUrlByOrderOperation() 遍历 order\_operation\_info 表中所有操作记录（审批、评论、抄送），从每条记录的 attachmentUrl 字段提取 downloadUrl。

### 0.3 评论附件写入

评论接口 /comment → saveOrderOperation()：
- 评论保存时 type 默认设为 2（评论类型）
- attachmentUrl 字段保存评论区上传的附件信息（JSON 数组，含 downloadUrl）
- 同时写入 attachment\_relation 关联表，来源标记为 comment

### 0.4 过滤规则

| 规则 | 说明 |
|------|------|
| 仅最新审批流 | 驳回重审的旧审批流附件不打印 |
| 仅最新审批节点 | 驳回/退回后中间段节点的附件不打印 |
| 评论记录不被去重 | type=2 且 nodeCode 为空，分配随机 UUID，不会被去重过滤 |
| 仅图片和 PDF 格式 | imgAndPdf=true 时，只打印图片和 PDF 格式的附件 |
> **结论**：勾选「附件」打印时，已包括审批流评论中上传的附件。本次需求不改动附件收集逻辑，仅优化提示文案和扩展打印格式。

## 一、需求背景

**需求现状**：
现有「附件」打印选项已支持打印审批流评论区的附件，但界面上无任何说明，用户无法感知。且附件打印仅支持图片和 PDF 格式，不支持 Word 和 Excel。

**客户预期**：
1. 「附件」选项的提示文案明确说明支持打印单据中附件和审批流评论区附件
2. 附件打印格式扩展，支持 Word（.doc/.docx）和 Excel（.xls/.xlsx）
3. 打印格式支持文件勾选

## 三、功能需求

### 3.1 打印内容选择-附件选项提示文案优化

**流程**：点击「打印」→ 打印选项弹窗 → 「附件」选项旁展示提示文案

**位置1**：有成报销各模块 → 单据列表页/详情页 → 点击「打印」按钮 → 打印选项弹窗

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/8oLl952xA3oVglap/img/62d70104-a903-44c8-97d2-7dd8f974dbdf.png?Expires=1781780825&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=ru2S7f4ky2Quoj95LtMjf5BKuyM%3D "")

**功能描述**：
- 在「附件」选项以灰色辅助文字展示提示文案
- 勾选支持打印的文件格式时，文案内容：「打印单据中的附件及审批流评论区的附件   支持的格式有：JPG、JPEG、PNG、PDF、Doc、Docx、Xls、Xlsx」
- 适用所有支持打印功能的单据类型（报销单、借款单、付款单、合同、收款单等）

### 3.2 附件打印格式扩展-全设置配置增加附件打印格式配置功能

**流程**：全局设置→ 附件设置→ 附件打印→ 勾选「附件格式」→ 打印设置勾选「附件」 →系统收集附件 → 格式校验（图片/PDF/**Word/Excel**）→ 合并打印

**位置**：全局设置→ 附件设置→ 附件打印

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/8oLl952xA3oVglap/img/196d1ba8-4192-4fa5-8d82-2b35163238c9.png?Expires=1781780825&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=MjRuDYRicfFgnCOCVbcmnIAfolI%3D "")



**功能描述**：
- 在现有图片和 PDF 格式基础上，新增支持 Word（.doc/.docx）和 Excel（.xls/.xlsx）格式
- 存量用户和新增用户【附件打印】中【JPG、JPEG、PNG、PDF】默认已勾选，【（Doc、Docx）、（Xls、Xlsx）】默认未勾选
- Word/Excel 文件需先转换为 PDF 或图片后合并到打印 PDF 中，与现有图片/PDF 统一输出
- 转换失败时跳过该附件并记录日志，不影响其他附件正常打印
- 适用于所有通过「附件」选项打印的附件：单据表单附件、费用明细附件、审批流评论区附件等
