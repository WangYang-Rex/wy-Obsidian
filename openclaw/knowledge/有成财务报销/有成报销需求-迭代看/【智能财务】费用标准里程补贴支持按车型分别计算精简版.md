# 【智能财务】费用标准里程补贴支持按车型分别计算精简版

> **原文链接**：[【智能财务】费用标准里程补贴支持按车型分别计算精简版 - 钉钉文档](https://alidocs.dingtalk.com/i/nodes/QOG9lyrgJP3OYPnGTzxpA0OyVzN67Mw4?utm_scene=team_space)
> **导出时间**：2026-04-23

## 二、功能需求

### 2.1 按车型分别计算控制方式

**位置**：预算费控 → 费用标准 → 新增/编辑标准 → 标准类型「里程补贴」

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/1GXn45K9gRbexqDQ/img/ee666003-90c9-438b-af6d-183764329e49.png?Expires=1776917548&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=BY0HWlrxr1aRFcJDlFd9X2HazUk%3D)

**控制方式选项**：

| 选项 | 说明 |
|------|------|
| 统一计算 | 所有车型同一单价（现有逻辑，默认） |
| 按车型分别计算 | 不同车型不同单价（新增） |

**切换规则**：
• 默认「统一计算」，存量标准不受影响
• 切换为「按车型分别计算」后，显示「车型辅助核算」必填项
• 切换控制方式时，原有配置清空

### 2.2 车型辅助核算选择

**字段**：车型辅助核算（下拉选择，必填）

**选项来源**：辅助核算字段列表（如车辆型号）

**提示**：「规则将对出发地、目的地字段生效，若选择按车型分别计算补贴，则费用中需要有辅助核算字段」

### 2.3 按车型标准设置表格

**表格列**：适用范围 \| 车型 \| 计价模式 \| 里程单价 \| 操作

**字段说明**：

| 字段 | 说明 |
|------|------|
| 适用范围 | 员工/部门选择器，不选默认全部 |
| 车型 | 下拉选择，选项来自辅助核算枚举值 |
| 计价模式 | 单一价格 |
| 里程单价 | 币种 \+ 数值 \+ 元/公里 |

**示例配置**：

| 适用范围 | 车型 | 计价模式 | 里程单价 |
|------------|------|------------|------------|
| 全部员工 | 01油车 | 单一价格 | CNY 0.80 元/公里 |
| 全部员工 | 02电车 | 单一价格 | CNY 0.60 元/公里 |

**约束**：
• 支持上移/下移调整优先级
• 支持新增/删除行

### 2.4 单据校验逻辑
```
提交含里程补贴的单据
    ↓
读取单据中「车型辅助核算」字段值
    ↓
匹配对应车型的里程单价
    ↓
补贴金额 = 里程数 × 对应车型单价
    ↓
与填写金额比对，超标准则提示/拦截

```

**匹配规则**：先按适用范围过滤 → 再按车型精确匹配 → 匹配到第一条即停止

### 2.5 端适配

| 端 | 适配说明 |
|---|------------|
| PC端 | 费用标准新增/编辑页面完整支持 |
| 移动端 | 单据提交时按车型匹配标准，校验结果在MB端正常展示 |
