# 20260302预算管理-允许按比例手工结转支持自动结转【锅圈（上海）】【有成报销v2.0】精简版

### 2.1 结转规则选项调整

**位置**：预算费控 → 预算管理 → 基础设置 → 预算结转 → 结转规则设置

![画布 23.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/meonarb59Rj58qXx/img/ee7b57b8-711b-4c01-9fc0-abc74b806fec.png?Expires=1776917564&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=NzHCR0ch%2BydMZcfOUU4wxroqogI%3D)

**变更内容**：
- 新增 **"余额全额自动结转"**：按周期自动将剩余余额全部结转至下一期间
- 新增 **"余额比例自动结转"**：按周期自动按配置比例结转剩余余额
- 原 **"余额按比例结转"** 文案修改为 **"余额比例手工结转"**

**配置规则**：
- 四种结转方式互斥，单选
- 配置保存后即时生效
- 自动结转执行时间由预算周期决定，无需单独配置执行时间

### 2.2 自动结转执行逻辑

**执行时机**：

| 预算周期 | 自动结转执行时间 |
|------------|------------------------|
| 年度 | 财务开始年度月份的次年首月首日凌晨 |
| 半年度 | 前/后6个月月末后次日凌晨 |
| 季度 | 每季度末次日凌晨 |
| 月度 | 每月末次日凌晨 |

**执行流程**：
1. 系统定时扫描所有启用自动结转的规则
2. 遍历规则下的预算维度，判断当前是否为期末次日
3. 检查当前期间余额\>0、目标期间已创建、无进行中的手工结转
4. 计算结转金额 = 当前期间剩余预算 × 结转比例
5. 执行结转、更新预算额度、生成操作日志和变更记录
6. 汇总执行结果并发送通知给预算管理员

**边界处理**：
- 非执行日：跳过，不记录
- 余额为零：跳过，记录"无需结转"
- 存在手工结转进行中：跳过，记录冲突并发送失败通知
- 目标期间未创建：跳过，记录并发送失败通知
- 系统异常：记录失败日志，10分钟后重试，最多3次

### 2.3 操作日志、变更记录与通知

**操作日志与变更记录**：
- 自动结转操作记录到操作日志
- 变更记录新增【结转规则】字段，显示"余额全额结转/余额比例手工结转/余额比例自动结转"

**消息通知（固定发给预算管理员）**：
- **成功通知**：告知本次执行规则数、成功/失败条数，引导查看操作日志
- **失败通知**：列出失败明细及原因（如冲突、目标期间未创建等）
