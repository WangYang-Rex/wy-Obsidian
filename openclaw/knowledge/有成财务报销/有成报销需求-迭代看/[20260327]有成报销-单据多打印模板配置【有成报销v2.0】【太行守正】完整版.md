# 单据多打印模板配置 - 需求文档
> **版本**：v1.14 \| **产品**：有成报销v2.0 \| **客户**：太行守正   **修订记录**：v1.14(20260421)修正产品架构：打印模板挂在单据维度，不是单据类型维度；每个单据仅允许一个默认模板

## 〇、现有产品架构（供参考）

### 0.1 单据与单据类型的关系

有成报销的产品层级：
```
单据类型（如：报销单、借款单、申请单）
  ↓
单据（如：普通报销单、差旅报销单——自定义表单定义）
  ↓
打印模板（每个单据下可配置多个打印模板）

```
- **单据类型**：系统预设或自定义的分类，用于区分业务类型
- **单据**：在单据类型下创建的自定义表单，包含字段、流程、打印等配置
- **打印模板**：挂在**单据**维度，每个单据可配置自己的打印模板
> 本次需求的操作对象是**单据**下的打印模板，不是单据类型。   

---

## 一、快速概览

### 1.1 这个需求要做什么？

**一句话**：单据下支持配置多个打印模板，每个单据仅允许一个默认模板，打印时按需选择，并支持启用/禁用控制。

**具体效果**：
- 有成报销中每个**单据**（如普通报销单、差旅报销单）可独立配置多个打印模板
- **每个单据下只能有一个默认模板**（不同单据之间互不影响）
- 模板列表支持启用/禁用操作
- 打印时仅展示启用的模板

### 1.2 谁需要这个功能？

| 角色 | 痛点 | 解决方案 |
|------|------|------------|
| 财务人员 | 不同场景需要不同打印格式 | 一个单据配置多个模板，按需选择或批量打印用默认 |
| 系统管理员 | 临时模板不想删但又不想被选中 | 禁用模板，保留但不可用 |
| 企业员工 | 打印时模板太多干扰选择 | 管理员禁用不常用模板，简化选择 |

### 1.3 核心价值
- ✅ 单据维度独立管理，普通报销单、差旅报销单等各自有自己的默认模板
- ✅ 每个单据下默认模板唯一，避免打印时默认选择混乱
- ✅ 模板启用/禁用灵活控制，不用删除即可临时下线

说明：先选择单据（如普通报销单），再在该单据下维护打印模板。不同单据的模板列表完全独立。每个单据的高级版开通状态独立，互不影响。

#### 2.1.1 未开通高级版时的状态
- 单据下仅显示一个普通版默认模板
- 操作列仅展示「设计」，无启用/禁用/设为默认/复制等操作
- 页面顶部展示升级指引横幅：「开通高级版打印设计器，支持多个打印模板配置」
- 点击横幅「联系客服开通」或操作栏「新增模版」→ 弹出「开通高级版打印设计器」弹窗

#### 2.1.2 已开通高级版时的状态

**列表字段**：

| 字段 | 说明 |
|------|------|
| 模板名称 | 用户自定义 |
| 设计器类型 | \[普通\] / \[高级版\] |
| 启用状态 | ● 启用 / ○ 禁用 |
| 默认模板 | ● 默认 / - |
| 创建人 | 创建者姓名 |
| 创建时间 | YYYY-MM-DD |
| 操作 | 编辑 / 设计 / 删除 / 复制 / 设为默认 / 启用 / 禁用 |

**操作规则**：
- **每个单据下只能有一个默认模板**（不同单据之间互不影响）
- 默认模板**不可删除**
- 默认模板**不可禁用**（需先切换默认给其他启用的模板，才能禁用）
- 禁用的模板**不可设为默认**
- 仅剩一个启用模板时**不可禁用**
- 设为默认时，该单据下的原默认模板**自动取消默认**
- 复制产生的新模板**默认启用，但不设为默认**（与原模板状态无关）
- 每个单据至少保留一个启用的模板

### 2.2 启用/禁用功能

**触发条件**：用户点击操作区的「启用」或「禁用」按钮

**启用**：
- 切换模板状态为「启用」
- 启用后该模板可在打印选择弹窗中被选中

**禁用**：
- 切换模板状态为「禁用」
- 默认模板不可禁用（操作入口置灰或点击提示）
- 仅剩一个启用模板时不可禁用
- 禁用后打印选择弹窗中不展示
- 批量打印时不使用禁用的模板

### 2.3 设为默认

**触发条件**：用户点击某模板的「设为默认」按钮

**规则**：
- 仅**启用的模板**可设为默认（禁用模板不展示该操作）
- 设为默认后，该模板成为该单据下**唯一的默认模板**
- 该单据下的原默认模板自动取消默认状态
- 不同单据之间互不影响（普通报销单的默认模板不影响差旅报销单的默认模板）
- 打印时如无需选择（仅1个启用模板），直接使用默认模板

**行为分支**：

| 场景 | 模板数量（启用状态） | 处理方式 |
|------|------------------------------|------------|
| 单张打印 | 1个启用 | 直接使用，不弹出选择 |
| 单张打印 | 多个启用 | 弹出选择弹窗，仅展示启用的模板，默认选中该单据的默认模板 |
| 批量打印-不同单据 | - | 各单据使用各自单据下的默认模板 |
| 批量打印-同一单据 | 1个启用 | 直接使用默认模板 |
| 批量打印-同一单据 | 多个启用 | 弹出选择弹窗，仅展示启用的模板，默认选中默认模板 |

### 2.5 默认模板初始化规则

| 场景 | 默认模板 | 启用状态 | 说明 |
|------|------------|------------|------|
| **新单据，未开通高级版** | 普通版模板 | 启用 | 该单据下仅1个普通版模板 |
| **新单据，已开通高级版** | 普通版模板 | 启用 | 该单据下普通版为默认；高级版模板也启用，但不默认 |
| **存量单据** | 当前设计器正在使用的模板 | 启用 | 该单据下当前使用的模板设为默认且启用；其他模板保持原有启用状态 |
| **后续开通高级版** | 保持原默认不变 | - | 该单据下不自动切换默认，由用户手动「设为默认」 |
