需求现状：
当前有成报销中，每个单据（如普通报销单、差旅报销单）仅支持配置一个打印模板。不同场景需要不同打印格式时，只能不断修改同一个模板或删除重建。系统管理员临时模板不想删除但又不想被选中，没有灵活的控制手段。本次支持每个单据配置多个打印模板，并支持启用/禁用和默认模板设置。

客户预期：
单据下支持配置多个打印模板

## 二、功能清单

| 模块 | 功能点 | 备注 |
|------|---------|------|
| 基础设置-打印模板维护 | 模板列表（已开通高级版） | 支持编辑/设计/删除/复制/设为默认/启用/禁用 |
| 基础设置-打印模板维护 | 启用/禁用功能 | 仅已开通高级版时展示 |
| 基础设置-打印模板维护 | 设为默认 | 每个单据下只能有一个默认模板 |
| 单据详情-打印 | 打印时模板选择 | 单张/批量打印时按需选择 |
| 基础设置-打印模板维护 | 默认模板初始化规则 | 新单据/存量单据/开通高级版等不同场景 |

**功能描述：**
- 先选择单据（如普通报销单），再在该单据下维护打印模板。不同单据的模板列表完全独立
- 模版名称：不作唯一性校验，允许重名，字数限制30字

**未开通高级版时的状态：**

• 单据下仅显示一个普通版默认模板
• 操作列仅展示「设计」，无启用/禁用/设为默认/复制等操作
• 页面顶部展示升级指引横幅："开通高级版打印设计器，支持多个打印模板配置"
• 点击横幅「联系客服开通」或操作栏「新增模版」→ 弹出「开通高级版打印设计器」弹窗

**已开通高级版时的状态：**

列表字段：

| 字段 | 说明 |
|------|------|
| 模板名称 | 用户自定义 |
| 设计器类型 | \[普通\] / \[高级版\] |
| 启用状态 | ● 启用 / ○ 禁用 |
| 默认模板 | ● 默认 / - |
| 创建人 | 创建者姓名 |
| 创建时间 | YYYY-MM-DD |
| 操作 | 编辑 / 设计 / 删除 / 复制 / 设为默认 / 启用 / 禁用 |

操作规则：
• 每个单据下只能有一个默认模板（不同单据之间互不影响）
• 默认模板不可删除
• 默认模板不可禁用（需先切换默认给其他启用的模板，才能禁用）
• 禁用的模板不可设为默认
• 仅剩一个启用模板时不可禁用
• 设为默认时，该单据下的原默认模板自动取消默认
• 复制产生的新模板默认启用，但不设为默认（与原模板状态无关）
• 每个单据至少保留一个启用的模板

### 3.5 【默认模板初始化规则】

**流程：** 新单据创建/存量单据升级/开通高级版时 → 按规则初始化默认模板

**位置：** 单据创建或高级版开通时自动触发

**功能描述：**

| 场景 | 默认模板 | 启用状态 | 说明 |
|------|------------|------------|------|
| 存量客户打印单据，未开通高级版 | 普通版模板 | 启用 | 该单据下仅1个普通版模板 |
| 存量客户打印单据，已开通高级版 | 当前设计器正在使用的模板 | 启用 | 该单据下当前使用的模板设为默认且启用，其他模版统一禁用 |
| 存量单据 | 当前设计器正在使用的模板 | 启用 | 该单据下当前使用的模板设为默认且启用；其他模板统一禁用 |
| 新客户后续开通高级版 | 普通版模版 | 启用 | 该单据下当前使用的模板设为默认且启用，高级版模版根据用户剩余可增模版数量添加模版 |
