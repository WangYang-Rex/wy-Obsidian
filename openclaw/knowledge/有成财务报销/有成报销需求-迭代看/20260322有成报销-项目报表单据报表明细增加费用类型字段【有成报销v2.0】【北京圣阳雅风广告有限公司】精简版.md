### **1\. 需求背景**

#### **需求说明**

在项目报表-单据纬度报表的分摊明细中，增加"费用类型"字段列。

#### **现状痛点**
- 报表-单据纬度明细只能看到分摊金额、分摊部门/项目，缺少费用类型
- 财务需要点击进入单据详情才能查看费用类型，操作繁琐
- 费用类型是财务分析和成本核算的关键维度，缺失影响报表效率

#### **预期结果**
- 单据报表明细直接展示费用类型
- 取值来源于费用明细组件中的费用类型字段
- 报表导出同步包含费用类型字段

### **3\. 功能描述**

#### **核心逻辑**
```plain
用户进入项目报表-单据报表
↓
查看单据列表
↓
展开单据分摊明细
↓
明细展示新增「费用类型」字段列
```

**功能概述** 在项目报表-不同单据纬度报表的明细展示区域，增加"费用类型"字段列，显示该笔分摊对应费用明细的费用类型。

**展示位置-示例** 报表中心-项目报表-报销单纬度-单据明细区域

#### **数据取值与展示规则**

| **规则** | **说明** |
|----------------------------------------------------------|----------------------------------------------------------|
| 取值来源 | 费用明细组件或费用类型组件 |
| 多级费用类型 | 展示完整路径，用"-"连接，如：差旅费-交通费-火车票 |
| 费用类型为空 | 展示为"-" |
| 费用类型已停用 | 名称后标注"(已停用)" |
| 一费用多分摊 | 同一费用类型分摊到多个维度时，每行都展示该费用类型 |
| 多费用合并分摊 | 按实际费用类型分别展示，分摊金额按费用拆分 |
| 无分摊数据 | 保持原逻辑 |
| 历史数据 | 存量单据按新逻辑补充费用类型字段 |
