# 合同结束审批流记录展示精简版

> **原文链接**：[合同结束审批流记录展示精简版 - 钉钉文档](https://alidocs.dingtalk.com/i/nodes/lyQod3RxJK3mne2NT45kgNjlJkb4Mw9r?utm_scene=team_space)
> **导出时间**：2026-04-23


### 2.1 PC端合同结束审批流记录

**位置**：单据管理 → 应收合同/应付合同 → 单据详情 → 审批流程tab

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/4j6OJ5jv7QN1Jq3p/img/ce562153-1190-4df3-9f6d-76479d9dd9af.png?Expires=1776917550&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=npuucBQ0XHYZ94711ak%2BaSKa6GI%3D "")![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/4j6OJ5jv7QN1Jq3p/img/bd64f6b8-f1fc-4854-8b24-ee6e8170ae46.png?Expires=1776917550&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=WBL0ukubY1rpVft4hWbKmg%2BHcl0%3D)

**展示内容**：

| 字段 | 内容 |
|------|------|
| 节点类型 | 开始 |
| 操作人 | 公司-姓名（如「京竹-李亮亮」） |
| 动作描述 | 手动结束了合同 |
| 操作时间 | YYYY-MM-DD HH:mm:ss |

**展示位置**：审批流时间轴最底部，所有正常审批节点之后追加。

**展示样式**：完全参照申请单「停用」记录样式（节点图标、字体、颜色、间距一致）。

### 2.2 MB端合同结束审批流记录

**位置**：MB端 → 单据详情 → 审批流程tab → 历史操作记录

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/4j6OJ5jv7QN1Jq3p/img/de6eb794-2cd5-4230-8de9-b0461517b4e1.png?Expires=1776917550&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=BSp9CwzobNh2oTWs0tob719PKMo%3D "")![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/4j6OJ5jv7QN1Jq3p/img/d3021560-6589-4122-8f51-d6567d97ddb3.png?Expires=1776917550&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=3q%2FUFIInGCoGjjEbJwVCek7CRGg%3D)

**展示内容**：与PC端一致，时间格式简化为 MM-DD HH:mm。

**展示位置**：历史操作记录列表最底部。

**展示样式**：完全参照申请单「停用」MB端样式。

### 2.3 端适配

| 端 | 适配说明 |
|---|------------|
| PC端 | 审批流程tab时间轴底部追加结束记录 |
| 移动端 | 审批流程tab历史操作记录列表底部追加 |
