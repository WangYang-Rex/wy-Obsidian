## 一、快速概览

### 1.1 这个需求要做什么？

**一句话**：把费用明细「动态可选范围」规则里「字段值」从单选改成多选，让一条规则能配多个字段值，减少重复规则。

**具体效果**：
• 规则中字段值支持勾选多个（如部门=\[财务部、技术部、行政部\]）
• 7种联动组件全部支持字段值多选
• 多个字段值对应同一组费用类型时，只需1条规则（原需N条）
• 存量单选规则正常兼容，编辑后可升级为多选

### 1.2 谁需要这个功能？

| 角色 | 痛点 | 解决方案 |
|------|------|------------|
| 系统管理员 | N个部门用相同费用类型，要配N条重复规则，维护麻烦 | 一条规则字段值多选，替代N条 |

### 1.3 核心价值

✅ 规则配置效率提升，N条合并为1条
✅ 后续维护成本降低，改费用类型范围只需改1处
✅ 存量数据兼容，无感知升级

## 二、功能需求

### 2.1 字段值多选

**位置**：自定义单据 → 表单编辑器 → 费用明细组件 → 可选范围设置 → 动态可选范围 → 设置规则

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/4EZlweZ280YNYqxA/img/759a59f9-0075-47cc-a028-172e4a8e7e6b.png?Expires=1778767562&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=WcJq7RKxs8cuvuqis0Gh%2FsRep5c%3D "")

**变更前 vs 变更后**：

|  | 变更前 | 变更后 |
|---|---------|---------|
| 字段值 | 单选下拉（1个值） | 多选标签（多个值，复选框勾选） |
| 规则示例 | 部门=财务部→办公费、差旅费 | 部门=\[财务部、技术部、行政部\]→办公费、差旅费 |
| 规则数量 | 3个部门需3条规则 | 3个部门只需1条规则 |

**交互规则**：
• 点击字段值区域弹出下拉选择器，复选框勾选多个值
• 已选值以标签形式展示，可逐个删除
• 多选字段值之间为 **OR** 关系，任一匹配即生效 

### 2.2 联动字段类型兼容

**完整支持列表**（7种）：

| 组件类型 | 说明 |
|------------|------|
| 部门 | 单据中的部门组件 |
| 辅助核算 | 单据中的辅助核算组件 |
| 成本中心 | 单据中的成本中心组件 |
| 项目 | 单据中的辅助核算（项目）组件 |
| 常用下拉框 | 单据中的常用下拉框组件 |
| 下拉选择 | 单据中的下拉选择组件 |
