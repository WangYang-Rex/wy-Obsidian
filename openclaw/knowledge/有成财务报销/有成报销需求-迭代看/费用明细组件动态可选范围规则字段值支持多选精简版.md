# 费用明细组件动态可选范围规则字段值支持多选精简版

> **原文链接**：[费用明细组件动态可选范围规则字段值支持多选精简版 - 钉钉文档](https://alidocs.dingtalk.com/i/nodes/jb9Y4gmKWr7lmxrahj6gb9llVGXn6lpz?utm_scene=team_space)
> **导出时间**：2026-04-23


### 2.1 字段值多选

**位置**：自定义单据 → 表单编辑器 → 费用明细组件 → 可选范围设置 → 动态可选范围 → 设置规则

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/4EZlweZ280YNYqxA/img/759a59f9-0075-47cc-a028-172e4a8e7e6b.png?Expires=1776917551&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=QDEQh0wDKUOizAX61hKNbGfcH20%3D)

**变更前 vs 变更后**：

|  | 变更前 | 变更后 |
|---|---------|---------|
| 字段值 | 单选下拉（1个值） | 多选标签（多个值，复选框勾选） |
| 规则示例 | 部门=财务部→办公费、差旅费 | 部门=\[财务部、技术部、行政部\]→办公费、差旅费 |
| 规则数量 | 3个部门需3条规则 | 3个部门只需1条规则 |

**交互规则**：
• 点击字段值区域弹出下拉选择器，复选框勾选多个值
• 已选值以标签形式展示，可逐个删除
• 多选字段值之间为 **OR** 关系，任一匹配即生效

### 2.2 联动组件类型扩展

| 组件类型 | 说明 |

| 项目 | 单据中的辅助核算（项目）组件 |

| 下拉选择 | 单据中的下拉选择组件 |
